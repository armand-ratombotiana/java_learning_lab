# Pedagogic Guide: Helidon

## Learning Path
1. Understand Helidon SE vs MP approaches
2. Create simple REST endpoints
3. Add configuration and health checks
4. Integrate with databases
5. Build reactive microservices

## Key Concepts
- **Helidon SE**: Lightweight reactive framework
- **Helidon MP**: MicroProfile implementation
- **WebServer**: Reactive HTTP server
- **Config**: Configuration management
- **Health**: Health check APIs
