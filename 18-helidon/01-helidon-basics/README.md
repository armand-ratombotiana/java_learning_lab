# Helidon Basics

Learn Helidon SE and MP for lightweight microservices.

## Overview
Helidon is a lightweight Java microservices framework developed by Oracle. It provides two programming models: Helidon SE (reactive) and Helidon MP (MicroProfile).

## Key Features
- Lightweight and fast startup
- Reactive HTTP client/server
- MicroProfile full implementation
- Health check support
- GraalVM native image support

## Version
- Helidon: 4.0.0
- Java: 21
