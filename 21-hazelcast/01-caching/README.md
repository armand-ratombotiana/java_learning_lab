# Hazelcast Caching

Learn Hazelcast distributed cache and data grid.

## Overview
Hazelcast provides distributed in-memory caching and data grid capabilities. It enables fast access to data across clustered applications.

## Key Features
- Distributed cache (IMap)
- Near cache for performance
- Distributed collections
- Topic/Queue messaging
- Entry processors
- Spring Boot integration

## Version
- Hazelcast: 5.3.0
- Java: 21
