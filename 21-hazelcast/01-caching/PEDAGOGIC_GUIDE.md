# Pedagogic Guide: Hazelcast

## Learning Path
1. Understand distributed caching
2. Create IMap for caching
3. Configure near cache
4. Use distributed collections
5. Implement entry processors

## Key Concepts
- **IMap**: Distributed map
- **Near Cache**: Local cache for speed
- **HazelcastInstance**: Cluster node
- **Entry Processor**: Distributed compute
- **Partitions**: Data distribution
