# Exercises - Selenide

## Exercise 1: Basic Test
Write a test that opens a page, fills a form, and submits it.

## Exercise 2: Assertions
Use shouldHave() to verify page title, text presence, and element visibility.

## Exercise 3: Page Objects
Create a LoginPage class with methods for username, password, and submit.

## Exercise 4: Dynamic Elements
Test elements that appear/disappear based on user actions.

## Exercise 5: Screenshot Configuration
Configure Selenide to capture screenshots on failure and save to custom directory.
