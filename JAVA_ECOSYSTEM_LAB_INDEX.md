## 🚀 Getting Started

### Prerequisites
- Java 11+ installed
- Maven 3.6+ installed
- Git installed
- IDE (IntelliJ IDEA or VS Code)

### Quick Start
```bash
# Clone the repository
git clone https://github.com/your-repo/java-learning-lab.git

# Navigate to project
cd java-learning-lab

# Build the project
mvn clean install

# Run tests
mvn test
```

### First Steps
1. Choose your learning path
2. Start with the first module
3. Read PEDAGOGIC_GUIDE.md
4. Complete exercises
5. Take quizzes
6. Prepare for interviews
