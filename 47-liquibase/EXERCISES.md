# Exercises - Liquibase

## Exercise 1: Create XML Changelog
Write an XML changelog that creates a `customers` table with id, name, email columns.

## Exercise 2: Add Rollback
Create a changeset with a proper rollback attribute.

## Exercise 3: Use Contexts
Modify a changeset to run only in 'dev' and 'test' contexts.

## Exercise 4: Generate Changelog
Use the diff command to generate a changelog between two databases.

## Exercise 5: Tag and Rollback
Create a tag, make changes, then rollback to the tagged version.
