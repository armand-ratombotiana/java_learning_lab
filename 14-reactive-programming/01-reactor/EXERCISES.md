# Exercises: Project Reactor

## Basic Exercises

1. **Flux Basics**: Create Flux stream
   - Emit items with Flux.just()
   - Subscribe to Flux

2. **Mono Basics**: Use Mono for single value
   - Create Mono.just()
   - Handle single result

3. **Operators**: Transform data with operators
   - Use map() transformation
   - Use filter() for condition

4. **Error Handling**: Handle errors
   - Use onErrorReturn()
   - Use onErrorResume()

## Intermediate Exercises

5. **Backpressure**: Handle backpressure
   - Use limitRequest()
   - Use subscribe()

6. **Windowing**: Window elements
   - Use window() operator
   - Batch elements

7. **Combining**: Combine streams
   - Use merge()
   - Use zip()

## Advanced Exercises

8. **Parallel Processing**: Use parallel processing
   - Use flatMapParallel() for parallel execution
