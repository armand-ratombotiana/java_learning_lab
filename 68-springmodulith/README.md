# Spring Modulith

## Overview
Spring Modulith enables modular monolith architecture with clear module boundaries and optional microservices migration path.

## Key Features
- @ApplicationModule annotation
- Module dependency management
- Event publishing between modules
- Module verification
- Internal package visibility

## Project Structure
```
68-springmodulith/
  spring-modulith/
    src/main/java/com/learning/springmodulith/SpringModulithLab.java
```

## Running
```bash
cd 68-springmodulith/spring-modulith
mvn compile exec:java
```
