# Pedagogic Guide - Grafana

## Learning Path

### Phase 1: Dashboard Basics
1. Dashboard and panel creation
2. Data source configuration
3. Query editor usage
4. Time range and refresh

### Phase 2: Visualization Types
1. Graph (line, bar, stacked)
2. Stat and gauge panels
3. Tables and heatmaps
4. Text and annotations

### Phase 3: Queries and Transformations
1. PromQL integration
2. Math and transformations
3. Field calculations
4. Organize and sort data

### Phase 4: Variables and Templating
1. Query variables
2. Custom values
3. Chained variables
4. Dynamic filtering

### Phase 5: Alerting
1. Alert rules and conditions
2. Notification channels
3. Alert evaluation and state
4. Routing and grouping

## Panel Configuration

| Setting | Purpose |
|---------|---------|
| Legend | Show/hide series names |
| Axis | Scale and unit selection |
| Thresholds | Color zones |
| Data links | Clickable links |
| Alerts | Alert rules |

## Best Practices
- Use variables for dynamic values
- Set appropriate refresh intervals
- Create alert thresholds carefully
- Document dashboard purpose
- Use annotations for events
