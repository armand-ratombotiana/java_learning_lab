# Exercises: Creational Patterns

## Basic Exercises

1. **Singleton**: Implement thread-safe singleton
   - Double-checked locking
   - Enum singleton

2. **Factory Method**: Create factory implementation
   - Define product interface
   - Create concrete factories

3. **Abstract Factory**: Implement abstract factory
   - Create related product family
   - Implement concrete factory

## Intermediate Exercises

4. **Builder**: Implement builder pattern
   - Create fluent builder
   - Build immutable objects

5. **Prototype**: Implement prototype pattern
   - Clone objects with copy constructor
   - Use clone method

## Advanced Exercises

6. **Complex Builder**: Build complex object tree
   - Use method chaining
   - Implement director
