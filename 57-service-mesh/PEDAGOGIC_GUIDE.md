# Pedagogic Guide - Istio Service Mesh

## Learning Path

### Phase 1: Fundamentals
1. Understand service mesh concept
2. Install Istio in cluster
3. Inject sidecars

### Phase 2: Intermediate
1. VirtualService configuration
2. DestinationRule policies
3. Traffic splitting

### Phase 3: Advanced
1. Security (mTLS, authorization)
2. Observability setup
3. Multi-cluster Istio

## Key Resources

| Resource | Purpose |
|----------|---------|
| VirtualService | Traffic routing rules |
| DestinationRule | Load balancing, subsets |
| Gateway | Ingress/Egress |
| AuthorizationPolicy | Access control |

## Common Patterns
- Canary deployments
- A/B testing
- Circuit breaking
- Rate limiting
