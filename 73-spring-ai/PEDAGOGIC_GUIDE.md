# Pedagogic Guide: Spring AI

## Learning Path

### Phase 1: Spring AI Basics (Week 1)
- Understand Spring AI architecture
- Configure first AI provider
- Build simple chat endpoint

### Phase 2: ChatClient Deep Dive (Week 1-2)
- Master prompt construction
- Learn structured output
- Handle streaming responses

### Phase 3: Embeddings & Vectors (Week 2-3)
- Understand vector embeddings
- Configure embedding models
- Implement vector stores

### Phase 4: RAG Systems (Week 3-4)
- Build document processing pipeline
- Implement semantic search
- Create Q&A applications

### Phase 5: Production Patterns (Week 4)
- Error handling and retries
- Observability and monitoring
- Security best practices

## Key Concepts

| Concept | Description |
|---------|-------------|
| ChatClient | Fluent API for LLM interactions |
| PromptSpec | Prompt specification builder |
| Advisor | Middleware for chat pipeline |
| VectorStore | Embedding storage and retrieval |
| Document | Text with metadata |
