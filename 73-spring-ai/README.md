# 73 - Spring AI

A module for integrating AI capabilities into Spring Boot applications using Spring AI framework.

## Overview

Spring AI provides a unified API for working with various AI providers, making it easy to integrate LLMs, embeddings, and vector databases into Spring applications.

## Key Topics

- **Chat Client**: Unified chat interface across providers
- **Embedding Models**: Text-to-vector conversion
- **Vector Stores**: Embedding storage and retrieval
- **Document Processing**: PDF, text document handling
- **Structured Output**: Type-safe AI responses
- **Multi-Provider Support**: OpenAI, Anthropic, Google, Ollama

## Technology Stack

- Spring AI
- Spring Boot 3.x
- OpenAI Client
- Anthropic Client
- Vector Store implementations

## Getting Started

```bash
cd 73-spring-ai
mvn compile exec:java
```

## Requirements

- Java 17+
- Spring Boot 3.2+
- Maven 3.8+
- API keys for AI providers
