## Key Concepts

### Core Components
- **ChatClient** - Text generation with LLMs
- **ImageClient** - Image generation
- **EmbeddingClient** - Vector embeddings
- **Vector Store** - Semantic search storage

### AI Models
- **LLMs** - Large Language Models (GPT, Claude, etc.)
- **Embeddings** - Semantic vector representations
- **Multimodal** - Text, image, audio processing

### Patterns
- **RAG** - Retrieval Augmented Generation
- **Prompt engineering** - Optimal input design
- **Function calling** - Tool integration
