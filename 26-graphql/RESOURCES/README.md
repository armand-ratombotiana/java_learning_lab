## Key Concepts

### Query Language
- **Query** - Read data
- **Mutation** - Write data
- **Subscription** - Real-time updates

### Schema Elements
- **Types** - Object, Scalar, Enum, Interface, Union
- **Fields** - Properties with resolvers
- **Arguments** - Parameterized queries
- **Directives** - Conditional logic

### Design Principles
- Hierarchical, client-driven
- Strongly typed schema
- No over-fetching or under-fetching
- Introspection support
