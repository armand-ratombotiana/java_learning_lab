# 76 - Oracle Tribuo

A machine learning library from Oracle for building ML models in Java.

## Overview

Tribuo provides a unified interface for common ML tasks with built-in support for classification, regression, clustering, and anomaly detection.

## Key Topics

- **Classification**: Linear models, SVM, Trees, Ensemble methods
- **Regression**: Linear, Tree-based, SVR
- **Clustering**: K-Means, DBSCAN
- **Feature Processing**: Normalization, PCA
- **Model Evaluation**: Confusion matrix, metrics
- **Data Loading**: CSV, JSON data sources

## Technology Stack

- Oracle Tribuo
- LibSVM, XGBoost (via Tribuo)
- Maven dependencies

## Getting Started

```bash
cd 76-tribuo
mvn compile exec:java
```

## Requirements

- Java 11+
- Maven 3.6+
