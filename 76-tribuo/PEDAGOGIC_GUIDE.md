# Pedagogic Guide: Oracle Tribuo

## Learning Path

### Phase 1: ML Review (Week 1)
- Review classification and regression
- Understand train/test splits
- Learn evaluation metrics

### Phase 2: Tribuo Basics (Week 1-2)
- Set up Tribuo project
- Load datasets (CSV)
- Create basic classifiers

### Phase 3: Advanced Models (Week 2-3)
- Linear models and SVM
- Tree-based methods
- Ensemble approaches

### Phase 4: Feature Engineering (Week 3-4)
- Data normalization
- Dimensionality reduction with PCA
- Feature selection

### Phase 5: Production ML (Week 4)
- Model serialization
- Performance optimization
- Integration patterns

## Key Concepts

| Concept | Description |
|---------|-------------|
| Dataset | Collection of examples with features |
| Trainer | Algorithm for fitting models |
| MutableDataset | Dataset that can be modified |
| Output | Target variable (Label, Regression) |
| Feature | Input variable for prediction |
