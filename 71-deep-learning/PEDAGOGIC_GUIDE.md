# Pedagogic Guide: Deep Learning

## Learning Path

### Phase 1: Neural Network Foundations (Week 1-2)
- Understand perceptrons and activation functions
- Learn backpropagation and gradient descent
- Study loss functions (MSE, Cross-Entropy)

### Phase 2: DJL Framework (Week 2-3)
- Set up DJL with Maven dependencies
- Build SequentialBlock models
- Configure trainers and optimizers

### Phase 3: CNN & Computer Vision (Week 3-4)
- Convolutional layers and pooling
- Common architectures (LeNet, AlexNet, ResNet)
- Image classification with pre-trained models

### Phase 4: Advanced Topics (Week 4-5)
- Transfer learning and fine-tuning
- Model optimization and deployment
- ONNX model export

## Key Concepts

| Concept | Description |
|---------|-------------|
| Forward Propagation | Computing output from input through layers |
| Backpropagation | Computing gradients for weight updates |
| Activation Function | Non-linear transformation (ReLU, Sigmoid, Tanh) |
| Dropout | Regularization technique to prevent overfitting |
| Batch Normalization | Stabilizing layer inputs |
