# Neural Network Architecture Diagram

Visual representations of deep learning neural network concepts.

## Simple Neuron

```
┌─────────────────────────────────────────────────────────────┐
│                     SINGLE NEURON                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│      x₁ ── w₁ ─┐                                             │
│                │                                             │
│      x₂ ── w₂ ─┼──→ Σ(wᵢxᵢ) + b ──→ activation ──→ output  │
│                │                                             │
│      x₃ ── w₃ ─┘                                             │
│                                                              │
│  Formula: output = activation(Σ(wᵢxᵢ) + b)                  │
│                                                              │
│  where:                                                      │
│    xᵢ = input values                                        │
│    wᵢ = weights                                             │
│    b  = bias                                                │
│    activation = ReLU, Sigmoid, Tanh, etc.                   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## Feedforward Neural Network

```
┌─────────────────────────────────────────────────────────────┐
│                  FEEDFORWARD NETWORK                         │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   INPUT       HIDDEN          HIDDEN         OUTPUT         │
│   LAYER       LAYER 1         LAYER 2        LAYER          │
│                                                              │
│   ┌───┐      ┌───┐           ┌───┐          ┌───┐          │
│   │ ○ │      │ ○ │           │ ○ │          │ ○ │ y₁       │
│   └───┘      └───┘           └───┘          └───┘          │
│     │          ╲│╱             ╲│╱             │             │
│   ┌───┐        │               │            ┌───┐          │
│   │ ○ │───────►│──────────────►│───────────►│ ○ │ y₂       │
│   └───┘        │               │            └───┘          │
│     │          ╲│╱             ╲│╱             │             │
│   ┌───┐        │               │            ┌───┐          │
│   │ ○ │───────►│──────────────►│───────────►│ ○ │ y₃       │
│   └───┘        │               │            └───┘          │
│     │          ╲│╱             ╲│╱                           │
│   ┌───┐        │               │                           │
│   │ ○ │───────►│──────────────►│                           │
│   └───┘                                          INPUT:     │
│     │                                           [x₁,x₂,x₃] │
│   ┌───┐                                                       │
│   │ ○ │──────────────────────────────────────────────────►   │
│   └───┘                                                       │
│                                                              │
│   x₁,x₂,x₃...  ──→  h₁,h₂,h₃...  ──→  h₁,h₂...  ──→  y₁,y₂ │
│                                                              │
│   Layer 0          Layer 1           Layer 2         Layer 3 │
│   (input)          (hidden)          (hidden)       (output) │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## Convolutional Neural Network (CNN)

```
┌─────────────────────────────────────────────────────────────┐
│                      CNN ARCHITECTURE                        │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  INPUT (Image)                                        │   │
│  │  e.g., 224×224×3 (RGB)                               │   │
│  └──────────────────────────────────────────────────────┘   │
│                          │                                    │
│                          ▼                                    │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  CONV1: 64 filters, 7×7, stride 2                    │   │
│  │  → BatchNorm → ReLU → MaxPool                        │   │
│  └──────────────────────────────────────────────────────┘   │
│                          │                                    │
│                          ▼                                    │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  CONV BLOCK × N                                       │   │
│  │  3×3 conv × 64 → BatchNorm → ReLU → MaxPool         │   │
│  └──────────────────────────────────────────────────────┘   │
│                          │                                    │
│                          ▼                                    │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  RESIDUAL BLOCKS (skip connections)                    │   │
│  │                                                      │   │
│  │    ┌──────────┐                    ┌──────────┐     │   │
│  │    │ Conv2D   │                    │ Conv2D   │     │   │
│  │    │ BatchNorm│──┐    ┌───────────►│ BatchNorm│     │   │
│  │    │ ReLU     │  │    │            │ ReLU     │     │   │
│  │    └──────────┘  │    │            └──────────┘     │   │
│  │                  │    │                   │           │   │
│  │                  └───►│───────────────────┘           │   │
│  │                       │  + (add and ReLU)            │   │
│  └──────────────────────────────────────────────────────┘   │
│                          │                                    │
│                          ▼                                    │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  Global Average Pooling                               │   │
│  │  (each feature map → single value)                    │   │
│  └──────────────────────────────────────────────────────┘   │
│                          │                                    │
│                          ▼                                    │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  Dense: 1000 units (Softmax)                          │   │
│  │  OUTPUT: Class probabilities                           │   │
│  └──────────────────────────────────────────────────────┘   │
│                                                              │
│  Key Operations:                                             │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  Conv2D    │  Spatial feature extraction              │   │
│  │  Pooling   │  Dimensionality reduction                │   │
│  │  BatchNorm │  Internal covariate shift reduction      │   │
│  │  Dropout   │  Regularization                          │   │
│  └──────────────────────────────────────────────────────┘   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## Recurrent Neural Network (RNN)

```
┌─────────────────────────────────────────────────────────────┐
│                    RNN UNFOLDED                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   ┌─────┐      ┌─────┐      ┌─────┐      ┌─────┐          │
│   │  ○  │─────►│  ○  │─────►│  ○  │─────►│  ○  │   ...   │
│   │ xₜ  │      │ xₜ₊₁│      │ xₜ₊₂│      │ xₜ₊₃│          │
│   └──┬──┘      └──┬──┘      └──┬──┘      └──┬──┘          │
│      │            │            │            │              │
│      │            │            │            │              │
│      ▼            ▼            ▼            ▼              │
│   ┌─────┐      ┌─────┐      ┌─────┐      ┌─────┐          │
│   │  ●  │─────►│  ●  │─────►│  ●  │─────►│  ●  │   ...   │
│   │hₜ₋₁ │      │  hₜ │      │hₜ₊₁│      │hₜ₊₂│          │
│   └─────┘      └─────┘      └─────┘      └─────┘          │
│                                                              │
│   ┌──────────────────────────────────────────────────────┐ │
│   │  hₜ = tanh(W_xh·xₜ + W_hh·hₜ₋₁ + b_h)               │ │
│   │  yₜ = softmax(W_hy·hₜ + b_y)                         │ │
│   │                                                      │ │
│   │  Same weights shared across all time steps!         │ │
│   └──────────────────────────────────────────────────────┘ │
│                                                              │
│   RNN Variants:                                               │
│   ┌──────────────────────────────────────────────────────┐ │
│   │  LSTM  │  Long Short-Term Memory (gates: input,       │ │
│   │        │  forget, output + cell state)                │ │
│   │  GRU   │  Gated Recurrent Unit (update, reset gates)   │ │
│   └──────────────────────────────────────────────────────┘ │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## Transformer Architecture

```
┌─────────────────────────────────────────────────────────────┐
│                    TRANSFORMER                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   ENCODER                       DECODER                     │
│   ┌───────────┐                 ┌───────────┐               │
│   │ Input     │                 │ Output    │               │
│   │ Embeddings│                 │ Embeddings│               │
│   │ + Pos Enc │                 │ + Pos Enc │               │
│   └─────┬─────┘                 └─────┬─────┘               │
│         │                           │                       │
│         ▼                           │                       │
│   ┌───────────┐                     │                       │
│   │  Multi-   │                     │                       │
│   │  Head     │◄───► Add & LayerNorm│                       │
│   │  Self     │                     │                       │
│   │  Attn     │                     │                       │
│   └─────┬─────┘                     │                       │
│         │                           │                       │
│         ▼                           ▼                       │
│   ┌───────────┐                 ┌───────────┐             │
│   │  Feed     │                 │  Multi-   │             │
│   │  Forward  │◄───► Add & Norm │  Head     │             │
│   └─────┬─────┘                 │  Self     │             │
│         │                       │  Attn     │             │
│         ▼                       └─────┬─────┘             │
│   ┌───────────┐                     │                       │
│   │  N ×      │                 ┌─────┴─────┐             │
│   │  Encoder  │───────────────►│  Encoder- │             │
│   │  Blocks   │                 │  Decoder  │             │
│   └───────────┘                 │  Attn     │             │
│                                └─────┬─────┘             │
│                                      ▼                     │
│                                ┌───────────┐             │
│                                │  Feed     │             │
│                                │  Forward  │             │
│                                └─────┬─────┘             │
│                                      ▼                     │
│                                ┌───────────┐             │
│                                │  Linear   │             │
│                                │  Softmax  │             │
│                                └─────┬─────┘             │
│                                      ▼                     │
│                                ┌───────────┐             │
│                                │   Output  │             │
│                                └───────────┘             │
│                                                              │
│   Self-Attention:                                            │
│   ┌──────────────────────────────────────────────────────┐ │
│   │  Attention(Q,K,V) = softmax(QKᵀ/√d) × V               │ │
│   │                                                      │ │
│   │  Q = Query, K = Key, V = Value (all learned)         │ │
│   │  Multi-head = parallel attention with different Q,K,V│ │
│   └──────────────────────────────────────────────────────┘ │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## Training Loop

```
┌─────────────────────────────────────────────────────────────┐
│                    TRAINING LOOP                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   for epoch in range(num_epochs):                            │
│       ┌─────────────────────────────────────────────────┐   │
│       │  for batch in train_loader:                      │   │
│       │      ┌────────────────────────────────────────┐ │   │
│       │      │  1. Forward Pass                        │ │   │
│       │      │     predictions = model(batch.inputs)  │ │   │
│       │      │                                         │ │   │
│       │      │  2. Compute Loss                        │ │   │
│       │      │     loss = criterion(predictions,      │ │   │
│       │      │                  batch.targets)        │ │   │
│       │      │                                         │ │   │
│       │      │  3. Backward Pass                       │ │   │
│       │      │     loss.backward()                     │ │   │
│       │      │     gradients computed automatically    │ │   │
│       │      │                                         │ │   │
│       │      │  4. Update Weights                      │ │   │
│       │      │     optimizer.step()                    │ │   │
│       │      │     optimizer.zero_grad()               │ │   │
│       │      └────────────────────────────────────────┘ │   │
│       │  ┌────────────────────────────────────────────┐ │   │
│       │  │  Validation / Metrics                       │ │   │
│       │  │  if epoch % eval_interval == 0:             │ │   │
│       │  │      metrics = evaluate(model, val_data)    │ │   │
│       │  └────────────────────────────────────────────┘ │   │
│       └─────────────────────────────────────────────────┘   │
│                                                              │
│   Key Components:                                             │
│   ┌──────────────────────────────────────────────────────┐ │
│   │  Loss Function  │  Measures error (MSE, CrossEntropy)  │ │
│   │  Optimizer      │  Updates weights (SGD, Adam, etc.)    │ │
│   │  Backprop       │  Computes gradients via chain rule   │ │
│   │  Regularization │  Prevent overfitting (Dropout, L2)  │ │
│   └──────────────────────────────────────────────────────┘ │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
