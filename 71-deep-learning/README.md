# 71 - Deep Learning

A comprehensive module for learning deep learning concepts and implementation using DJL (Deep Java Library).

## Overview

This module covers neural network fundamentals, CNN architectures, and transfer learning using DJL - a powerful deep learning framework for Java.

## Key Topics

- **Neural Network Basics**: Layers, activations, loss functions, optimizers
- **DJL Framework**: Building and training models in pure Java
- **CNN Architectures**: Convolutional networks for image processing
- **Transfer Learning**: Pre-trained models and fine-tuning
- **Multi-Framework Support**: PyTorch, TensorFlow, MXNet, ONNX

## Technology Stack

- DJL (Deep Java Library)
- PyTorch Engine
- TensorFlow Engine
- MXNet Engine
- ONNX Runtime

## Getting Started

```bash
cd 71-deep-learning
mvn compile exec:java
```

## Requirements

- Java 17+
- Maven 3.8+
- For GPU support: CUDA Toolkit
