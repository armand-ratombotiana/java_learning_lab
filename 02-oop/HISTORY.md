# History of OOP

## Key Milestones

### 1967 - Simula
- First OOP language
- Introduced classes, inheritance, virtual methods

### 1972 - Smalltalk
- Alan Kay's vision of "messaging"
- Pure OOP - everything is an object
- Influenced Java's design

### 1983 - C++
- Bjarne Stroustrup added OOP to C
- Multiple inheritance introduced

### 1995 - Java
- Simplified C++ syntax
- Single inheritance with interfaces
- Garbage collection

### 2004 - Java 5
- Generics added
- Annotations
- Enhanced for loop

### 2014 - Java 8
- Default methods in interfaces
- Lambda expressions
- Functional interfaces
