# Exercises - Kafka Connect

## Exercise 1: JDBC Source Connector
Configure a JDBC source connector to stream database changes to Kafka.

## Exercise 2: Elasticsearch Sink
Set up an Elasticsearch sink connector to index Kafka topics.

## Exercise 3: Single Message Transform
Add SMT to convert message key format.

## Exercise 4: Connect REST API
Use REST API to create, pause, and delete connectors.

## Exercise 5: File Source
Implement a simple file source connector to read log files.
