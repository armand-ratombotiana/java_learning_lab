# Apache Spark - Data Pipeline

## Overview
Apache Spark is a unified analytics engine for large-scale data processing with batch and streaming capabilities.

## Key Features
- RDD and DataFrame APIs
- Spark SQL for queries
- MLlib for machine learning
- Structured Streaming
- Cluster deployment

## Project Structure
```
62-data-pipeline/
  apache-spark/
    src/main/java/com/learning/data/spark/SparkLab.java
```

## Running
```bash
cd 62-data-pipeline/apache-spark
mvn compile exec:java
```

## Concepts Covered
- Transformations (map, filter, reduce)
- Actions and lazy evaluation
- DataFrame operations
- Spark SQL queries
