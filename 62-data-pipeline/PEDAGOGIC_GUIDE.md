# Pedagogic Guide - Apache Spark

## Learning Path

### Phase 1: Fundamentals
1. Spark architecture and execution
2. RDD transformations and actions
3. SparkSession setup

### Phase 2: Intermediate
1. DataFrame and Dataset APIs
2. Spark SQL queries
3. UDFs and window functions

### Phase 3: Advanced
1. Structured Streaming
2. Performance tuning
3. Cluster deployment

## Key Concepts

| Concept | Description |
|---------|-------------|
| RDD | Resilient Distributed Dataset |
| Transformation | Lazy operation |
| Action | Triggers execution |
| DAG | Direct Acyclic Graph |

## Transformations
- map, flatMap, filter
- reduceByKey, groupByKey
- join, union

## Actions
- collect, count, first
- save, foreach
