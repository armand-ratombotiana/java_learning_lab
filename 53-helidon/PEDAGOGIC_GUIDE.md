# Pedagogic Guide - Helidon

## Learning Path

### Phase 1: Fundamentals
1. Create Helidon MP project
2. Understand routing and handlers
3. Configure application

### Phase 2: Intermediate
1. MicroProfile annotations (@Path, @GET)
2. Health and metrics endpoints
3. Reactive WebClient usage

### Phase 3: Advanced
1. SE programming model
2. Helidon DB client
3. GraalVM native build

## Programming Models

| Model | Description |
|-------|-------------|
| Helidon MP | MicroProfile, Jakarta EE compatible |
| Helidon SE | Lightweight, functional style |

## Key Features
- Fast startup
- Small footprint
- Native image support
- Built-in observability
