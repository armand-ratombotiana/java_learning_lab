# 28 - GraphQL (Spring Boot)

GraphQL API development with Spring Boot and Spring GraphQL. Covers `@QueryMapping` annotation for query resolvers, schema-first approach, type definitions in GraphQL schema files, controller-based data fetching, and Product API example.

## Prerequisites

- Java 11+
- Maven 3.x
- Spring Boot 3.x

## Key Concepts

- Spring GraphQL starter and auto-configuration
- `@QueryMapping` annotation for query resolver methods
- Schema-first GraphQL development
- GraphQL schema types and field definitions
- Controller-based data fetching with `@Controller`
- Integration with Spring Boot ecosystem

## Module Structure

- `graphql-learning/` - Spring Boot GraphQL application

## Learning Objectives

- Build GraphQL APIs with Spring Boot and Spring GraphQL
- Implement query resolvers with annotations
- Define GraphQL schemas for type-safe APIs

## Estimated Time

- 2-3 hours

## How to Build

```bash
cd 28-graphql
mvn clean package
```

Run the Spring Boot application:

```bash
cd graphql-learning
mvn spring-boot:run
```
