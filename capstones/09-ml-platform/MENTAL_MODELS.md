# Mental Models: ML Platform

- **Pipeline = Assembly Line**: Raw data goes in, trained model comes out, deployed to serving.
- **Feature Store = Ingredient Warehouse**: Pre-computed features ready for any recipe (model).
- **Model Registry = Library Catalog**: Every model version cataloged with metadata for traceability.
- **Serving = Restaurant Kitchen**: Online is à la carte (single order), batch is catering (mass production).
- **A/B Testing = Taste Test**: Compare two recipes side by side; customers vote with their behavior.
- **Monitoring = Thermometer**: Check if the model is still "cooking" correctly or if something is burning (drift).
