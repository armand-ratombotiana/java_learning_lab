# History: ML Platform

- **2010-2015**: ML models trained in notebooks, manually deployed via ad-hoc scripts (no platform)
- **2015**: Google TFX (TensorFlow Extended) — first comprehensive ML platform
- **2016**: MLflow (Databricks) — open-source ML lifecycle management
- **2017**: Kubeflow — ML on Kubernetes
- **2018**: SageMaker (AWS) — managed ML platform
- **2020s**: Feature stores (Feast, Tecton), model registries, automated retraining
- **2024+**: LLM-specific platforms (LangSmith, Weights & Biases), MLOps for foundation models
