# History: Recommendation Engine

- **1990s**: First collaborative filtering systems (GroupLens for Usenet, Tapestry for email)
- **2000**: Amazon item-to-item collaborative filtering (patented)
- **2006**: Netflix Prize competition — BellKor's ensemble approach wins with 10% improvement
- **2008**: Matrix factorization becomes dominant approach
- **2010s**: Deep learning enters recommendations (YouTube DNN, Neural Collaborative Filtering)
- **2020s**: Transformer-based models (BERT4Rec, SASRec); graph neural networks (PinSage); real-time personalization with feature stores
