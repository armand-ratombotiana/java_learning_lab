# History: Search Engine

- **1960s**: First information retrieval systems (SMART, Salton's magic)
- **1980s**: Boolean retrieval models; LexisNexis, Dialog
- **1990s**: Web search emerges (Archie, Gopher, AltaVista, Excite)
- **1998**: Google founded — PageRank revolutionizes web search
- **1999**: Apache Lucene project started by Doug Cutting
- **2000s**: TF-IDF widely adopted; Lucene becomes standard Java search library
- **2010s**: Elasticsearch (2010), Solr (distributed search) built on Lucene
- **2020s**: Semantic search (vector + keyword hybrid), dense retrieval, LLM-powered search
