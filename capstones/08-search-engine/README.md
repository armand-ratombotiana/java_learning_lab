# Search Engine

## Overview
Elasticsearch-like full-text search engine with inverted indexing, relevance ranking, and faceted search.

## Features
- Full-text search with TF-IDF
- Boolean queries (AND, OR, NOT)
- Phrase and fuzzy matching
- Faceted search
- Result highlighting

## Quick Start
```bash
cd 08-search-engine
docker-compose up -d
```
