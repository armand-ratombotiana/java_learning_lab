# History: Fraud Detection

- **Pre-2000**: Manual review of suspicious transactions; rule-based systems with static thresholds
- **2000-2010**: Automated rule engines; velocity checks; blacklists
- **2010-2015**: Machine learning adoption; logistic regression models; random forests
- **2015-2020**: Deep learning; real-time scoring; feature stores; graph-based fraud networks
- **2020s**: Federated learning for privacy; transformer models; real-time stream processing with Flink; explainable AI (SHAP, LIME)
