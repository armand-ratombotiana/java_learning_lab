# Exercises: Vector Database

## Beginner
1. Implement cosine distance and L2 distance functions
2. Add a simple brute-force kNN search (no index, O(n*d))
3. Implement a basic WAL with append and replay
4. Create a REST endpoint for collection CRUD

## Intermediate
5. Implement the HNSW graph construction and search algorithms
6. Add metadata filtering with inverted index
7. Implement product quantization for vector compression
8. Add batch vector insert with atomic commit

## Advanced
9. Implement IVF index (k-means clustering + HNSW per cluster)
10. Add concurrent read-write support with fine-grained locking
11. Implement a segment-based compaction strategy (like LSM-tree)
12. Build a distributed version with consistent hashing and replication
