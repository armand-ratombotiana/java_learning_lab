# History: Vector Database

- **1990s-2000s**: Traditional nearest neighbor search (kd-trees, R-trees) — fine for low dimensions, fail at high dimensions (curse of dimensionality)
- **2000s-2010s**: Approximate methods emerge — LSH (Locality-Sensitive Hashing), IVF, Product Quantization
- **2011**: HNSW paper by Malkov & Yashunin (became the most popular ANN algorithm)
- **2017**: Faiss library by Facebook (GPU-accelerated vector search)
- **2019**: First vector database startups: Pinecone, Weaviate, Qdrant, Milvus
- **2020s**: Vector databases become essential infrastructure for LLM-based applications (RAG, memory, agents)
