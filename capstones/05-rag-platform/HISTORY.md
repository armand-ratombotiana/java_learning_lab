# History: RAG Platform

- **2017**: "Attention Is All You Need" — Transformer architecture enables modern LLMs
- **2020**: Lewis et al. "Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks" (RAG paper)
- **2022**: ChatGPT launch; RAG becomes primary pattern for grounding LLMs
- **2023**: LangChain and LlamaIndex emerge as RAG frameworks; advanced RAG techniques (HyDE, Self-RAG, CRAG)
- **2024-2025**: Agentic RAG (tool use + retrieval), multi-modal RAG, GraphRAG, evaluation frameworks (RAGAS, TruLens)
