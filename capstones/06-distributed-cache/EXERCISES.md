# Exercises: Distributed Cache

## Beginner
1. Implement a simple LRU eviction policy using LinkedHashMap
2. Add a new command (INCR: increment integer value)
3. Implement TTL expiration with a background cleanup thread

## Intermediate
4. Implement the consistent hashing ring with virtual nodes
5. Add async replication to N-1 replica nodes
6. Implement the gossip protocol for cluster membership
7. Build a client library with automatic node routing

## Advanced
8. Add RDB snapshot persistence (periodic dump to disk)
9. Implement read-replica support (subscribe to replication stream)
10. Add Lua scripting support (like Redis EVAL)
11. Implement cross-datacenter replication (async, with conflict resolution)
12. Build a rebalancing system for adding/removing nodes without downtime
