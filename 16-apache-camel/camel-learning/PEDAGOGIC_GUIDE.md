# Pedagogic Guide: Apache Camel

## Learning Path
1. Understand what Camel is - integration framework with Enterprise Integration Patterns
2. Learn Routes and Components basics
3. Practice data transformation with Processors
4. Build REST integrations
5. Add error handling and retry logic

## Key Concepts
- **Routes**: Define message flow from source to destination
- **Components**: Adapters for different protocols (file, HTTP, JMS, etc.)
- **EIPs**: Enterprise Integration Patterns (Aggregator, Splitter, Router)
- **Data Format**: XML, JSON, CSV transformations
- **Context**: Camel application container
