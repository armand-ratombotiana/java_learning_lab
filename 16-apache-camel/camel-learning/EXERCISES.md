# Exercises: Apache Camel

## Exercise 1: Simple Route
Create a route that reads from a file endpoint and logs the content.

## Exercise 2: Transform Data
Use a processor to transform JSON to XML format.

## Exercise 3: REST Integration
Create a REST endpoint that accepts POST requests and routes to a queue.

## Exercise 4: Error Handling
Add error handling with dead letter queue for failed messages.

## Exercise 5: Content-Based Routing
Route messages to different endpoints based on a header value.
