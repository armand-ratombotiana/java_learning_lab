# Apache Camel Learning

Learn Apache Camel for enterprise integration patterns and routing.

## Overview
Apache Camel is a powerful integration framework that implements Enterprise Integration Patterns (EIPs). It provides a fluent API for routing and transforming data betweendifferent systems.

## Key Features
- Enterprise Integration Patterns implementation
- 300+ components for various protocols
- Data transformation and format conversion
- REST support with camel-rest
- Spring Boot integration

## Build & Run
```bash
mvn clean install
mvn spring-boot:run
```

## Version
- Camel: 4.4.0
