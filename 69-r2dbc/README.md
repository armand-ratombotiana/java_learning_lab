# R2DBC - Reactive Database

## Overview
R2DBC (Reactive Relational Database Connectivity) provides non-blocking database access using reactive streams for modern applications.

## Key Features
- Reactive database access
- Spring Data R2DBC
- Reactive repositories
- Transaction support
- Multi-database support

## Project Structure

```bash
cd 69-r2dbc/r2dbc-learning

```

## Concepts Covered
- R2DBC repositories

- PostgreSQL, MySQL, SQL Server
- H2, MariaDB
