# 80 - Computer Vision

A module for computer vision tasks using JavaCV, OpenCV, and DJL.

## Overview

This module covers image processing, object detection, classification, and OCR using Java-based computer vision libraries.

## Key Topics

- **JavaCV/OpenCV**: Image processing and manipulation
- **Object Detection**: YOLO, Haar Cascades
- **Image Classification**: Pre-trained models with DJL
- **OCR**: Tesseract for text recognition
- **Video Processing**: Real-time video analysis
- **Image Transformations**: Filters, contours, features

## Technology Stack

- JavaCV (OpenCV bindings)
- DJL Computer Vision
- Tess4J (Tesseract wrapper)
- YOLO models

## Getting Started

```bash
cd 80-computer-vision
mvn compile exec:java
```

## Requirements

- Java 11+
- Maven 3.6+
- OpenCV native libraries
