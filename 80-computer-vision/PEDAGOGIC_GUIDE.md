# Pedagogic Guide: Computer Vision

## Learning Path

### Phase 1: Image Basics (Week 1)
- Image representation
- Color spaces and channels
- Basic operations

### Phase 2: Image Processing (Week 1-2)
- Filtering and convolution
- Edge detection
- Morphological operations

### Phase 3: Object Detection (Week 2-3)
- Haar Cascade basics
- YOLO architecture
- Real-time detection

### Phase 4: Deep Learning CV (Week 3-4)
- Pre-trained models
- Transfer learning
- Model serving

### Phase 5: Applications (Week 4)
- OCR implementation
- Video analysis
- Custom applications

## Key Concepts

| Concept | Description |
|---------|-------------|
| Convolution | Kernel-based image transformation |
| CNN | Convolutional Neural Network |
| YOLO | You Only Look Once detector |
| IoU | Intersection over Union |
| NMS | Non-Maximum Suppression |
