# Exercises: Terraform

## Basic Exercises

1. **Provider Setup**: Configure AWS provider
   - Write provider block
   - Configure credentials

2. **Basic Resources**: Create S3 bucket
   - Write resource definition
   - Apply and verify

3. **Variables**: Use input variables
   - Define variable blocks
   - Pass values

4. **Outputs**: Export values
   - Define output blocks
   - Query outputs

5. **State**: Understand state
   - Inspect state file
   - Create backup

## Intermediate Exercises

6. **Modules**: Create reusable modules
   - Write module definition
   - Call module with inputs

7. **Remote State**: Use remote backend
   - Configure S3 backend
   - Enable state locking

8. **Data Sources**: Read existing resources
   - Use data source
   - Reference attributes

## Advanced Exercises

9. **Provisioners**: Execute commands
   - Use local-exec
   - Use remote-exec

10. **Workspaces**: Manage environments
    - Create workspaces
    - Switch environments
