# Pedagogic Guide: Jenkins

## Learning Path

### Phase 1: Foundation (Day 1)
- Understand Jenkins architecture
- Learn master/agent model
- Create first job

### Phase 2: Core Skills (Day 2-3)
- Write Jenkinsfiles
- Configure build triggers
- Manage artifacts

### Phase 3: Advanced (Day 4-5)
- Distributed builds
- Shared libraries
- Quality integration

## Key Concepts

| Concept | Description |
|---------|-------------|
| Master | Controller node |
| Agent | Build executor |
| Job | Build definition |
| Pipeline | CI/CD process as code |
| Stage | Pipeline phase |
| Agent | Executor for jobs |

## Prerequisites
- CI/CD fundamentals
- Basic Groovy knowledge

## Common Pitfalls
- Unstable builds
- Resource exhaustion
- Missing cleanup
