# Exercises: Jenkins

## Basic Exercises

1. **Job Creation**: Create first freestyle job
   - Configure SCM checkout
   - Add build step

2. **Pipeline**: Write Jenkinsfile
   - Create declarative pipeline
   - Configure stages

3. **Build Trigger**: Configure triggers
   - Set up SCM polling
   - Add webhook trigger

4. **Post Build**: Configure notifications
   - Archive artifacts
   - Send email notifications

5. **Agent Configuration**: Set up agents
   - Configure agent nodes
   - Use labels

## Intermediate Exercises

6. **Distributed Builds**: Use agents
   - Configure SSH agents
   - Set up Kubernetes agents

7. **Shared Libraries**: Reuse code
   - Create shared library
   - Call from pipeline

8. **Credentials**: Manage secrets
   - Add credentials
   - Use in pipeline

## Advanced Exercises

9. **Blue Ocean**: Use modern UI
   - Install Blue Ocean
   - Visualize pipelines

10. **Quality Gates**: Add scanning
    - Integrate SonarQube
    - Add test reports
