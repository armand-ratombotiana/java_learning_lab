# Pedagogic Guide: Docker

## Learning Path

### Phase 1: Foundation (Day 1)
- Understand container fundamentals
- Learn Docker CLI basics
- Study images and layers

### Phase 2: Core Skills (Day 2-3)
- Write Dockerfiles for Java apps
- Configure multi-stage builds
- Use Docker Compose

### Phase 3: Advanced (Day 4-5)
- Optimize images for production
- Configure networking
- Implement health checks

## Key Concepts

| Concept | Description |
|---------|-------------|
| Image | Template for containers |
| Container | Running instance of image |
| Dockerfile | Script to build images |
| Docker Compose | Multi-container orchestration |

## Prerequisites
- Java/Maven or Gradle
- Basic command line knowledge

## Common Pitfalls
- Large image sizes
- Port conflicts
- Missing volume mounts
