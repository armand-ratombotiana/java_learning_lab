# Exercises: Docker

## Basic Exercises

1. **Container Basics**: Run your first container
   - Pull and run nginx image
   - Map ports and access app

2. **Dockerfile**: Create a Java application image
   - Write Dockerfile for Spring Boot app
   - Build and tag image

3. **Multi-stage Builds**: Optimize image size
   - Use multi-stage build pattern
   - Separate build and runtime

4. **Docker Compose**: Define multi-container app
   - Configure app + database services
   - Use networking and volumes

5. **Volume Management**: Persist data
   - Use named volumes
   - Mount host directories

## Intermediate Exercises

6. **Optimization**: Reduce image size
   - Use slim JDK images
   - Remove unnecessary layers

7. **Networking**: Configure networks
   - Create custom networks
   - Link containers

8. **Health Checks**: Add container health
   - Configure HEALTHCHECK
   - Test health status

## Advanced Exercises

9. **Security**: Harden containers
   - Run as non-root user
   - Use read-only root filesystem
