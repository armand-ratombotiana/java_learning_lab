# Pedagogic Guide: Helm

## Learning Path

### Phase 1: Foundation (Day 1)
- Understand Helm concepts
- Learn chart structure
- Create first chart

### Phase 2: Core Skills (Day 2-3)
- Write templates with Go templating
- Configure values and overrides
- Manage releases

### Phase 3: Advanced (Day 4-5)
- Use hooks for lifecycle
- Manage dependencies
- Test and validate charts

## Key Concepts

| Concept | Description |
|---------|-------------|
| Chart | Package definition |
| Release | Deployed instance |
| Repository | Chart storage |
| Template | K8s manifest generator |
| Values | Configuration |

## Prerequisites
- Kubernetes fundamentals
- YAML knowledge
- Basic templating

## Common Pitfalls
- Template errors
- Wrong value precedence
- Missing requirements
