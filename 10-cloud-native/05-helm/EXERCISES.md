# Exercises: Helm

## Basic Exercises

1. **Chart Creation**: Create first chart
   - Use `helm create`
   - Explore chart structure

2. **Template**: Write templates
   - Create deployment YAML
   - Add service template

3. **Values**: Configure values
   - Edit values.yaml
   - Override with --set

4. **Install**: Deploy chart
   - Use helm install
   - Verify deployment

5. **Upgrade**: Update release
   - Modify values
   - Use helm upgrade

## Intermediate Exercises

6. **Rollback**: Revert changes
   - Use helm rollback
   - View history

7. **Hooks**: Add lifecycle hooks
   - Configure pre-install hook
   - Test hook behavior

8. **Dependencies**: Manage dependencies
   - Add sub-charts
   - Use requirements.yaml

## Advanced Exercises

9. **Testing**: Test charts
   - Use helm lint
   - Run helm test

10. **Repository**: Share charts
    - Create chart repository
    - Publish charts
