# Exercises: Kubernetes

## Basic Exercises

1. **Pod Basics**: Create first Pod
   - Write Pod YAML manifest
   - Apply and inspect Pod

2. **Deployments**: Manage replica sets
   - Create Deployment with replicas
   - Scale and update

3. **Services**: Expose applications
   - Configure ClusterIP, NodePort
   - Create Ingress

4. **ConfigMaps**: Manage configuration
   - Create ConfigMap
   - Mount as environment/env

5. **Secrets**: Handle sensitive data
   - Create Secrets
   - Mount securely

## Intermediate Exercises

6. **Labels & Selectors**: Organize resources
   - Use labels for grouping
   - Select with matchLabels

7. **Resource Limits**: Manage resources
   - Set requests/limits
   - Configure resource quotas

8. **Health Checks**: Configure probes
   - Add liveness/readiness probes
   - Configure startup probes

## Advanced Exercises

9. **Helm**: Deploy with Helm
   - Use existing charts
   - Create custom charts

10. **Auto-scaling**: Configure HPA
    - Set up Horizontal Pod Autoscaler
