# Pedagogic Guide: Kubernetes

## Learning Path

### Phase 1: Foundation (Day 1-2)
- Understand K8s architecture
- Learn core resources (Pod, Deployment, Service)
- Study namespace and labels

### Phase 2: Core Skills (Day 3-4)
- Create Pods and Deployments
- Configure Services and Ingress
- Use ConfigMaps and Secrets

### Phase 3: Advanced (Day 5)
- Configure resource management
- Implement health checks
- Set up autoscaling

## Key Concepts

| Concept | Description |
|---------|-------------|
| Pod | Smallest deployable unit |
| Deployment | Manages Pod replicas |
| Service | Network abstraction |
| Ingress | HTTP routing |
| ConfigMap | Non-sensitive config |
| Secret | Sensitive data |

## Prerequisites
- Docker fundamentals
- Basic YAML knowledge

## Common Pitfalls
- Missing resource limits
- Incorrect image tags
- Network policy issues
