# Exercises: CI/CD

## Basic Exercises

1. **GitHub Actions**: Create first workflow
   - Create `.github/workflows/ci.yml`
   - Configure build and test jobs

2. **Build Pipeline**: Build on push
   - Set up Maven/Gradle build
   - Configure artifact publishing

3. **Test Automation**: Run tests in CI
   - Execute unit and integration tests
   - Publish test reports

4. **Deployment**: Deploy to environment
   - Configure staging deployment
   - Add approval gates

5. **Artifacts**: Store build artifacts
   - Upload to GitHub Releases
   - Use artifact storage

## Intermediate Exercises

6. **Matrix Builds**: Test multiple versions
   - Configure matrix strategy
   - Test JDK versions

7. **Caching**: Speed up builds
   - Cache dependencies
   - Cache Docker layers

8. **Branch Protection**: Add quality gates
   - Require tests to pass
   - Configure branch rules

## Advanced Exercises

9. **Environment Promotion**: Implement promotion
   - Promote build to staging
   - Promote to production
