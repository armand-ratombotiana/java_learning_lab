# Java AI Integration

## Overview
Java AI integration enables applications to leverage large language models (LLMs) and AI services using Spring AI, LangChain4j, and OpenAI SDKs.

## Key Features
- Spring AI framework
- LangChain4j for LLM applications
- OpenAI and Azure OpenAI
- Prompt engineering
- AI model serving

## Project Structure
```
70-ai/
  java-ai-integration/
    src/main/java/com/learning/ai/AILab.java
```

## Running
```bash
cd 70-ai/java-ai-integration
mvn compile exec:java
```

## Concepts Covered
- Spring AI ChatClient
- LangChain4j basics
- Prompt templates
- AI use cases

## Dependencies
- Spring AI
- LangChain4j
- OpenAI SDK
