
## Exchange Types

```
┌─────────────────────────────────────────────────────────────────────────┐
│  DIRECT EXCHANGE - Exact routing key match                              │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   Producer ──▶ routing_key="order.created"                             │
│                    │                                                    │
│                    ▼                                                    │
│              ┌─────────────┐                                            │
│              │   Direct    │                                            │
│              └─────────────┘                                            │
│                    │                                                    │
│        ┌─────────┴─────────┐                                            │
│        │         │         │                                            │
│        ▼         ▼         ▼                                            │
│    Queue A   Queue B   Queue C                                          │
│    key:      key:      key:                                             │
│   order.    payment.  order.                                            │
│   created   created   created                                           │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  FANOUT EXCHANGE - Broadcast to all bound queues                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   Producer ──▶ (no routing key needed)                                 │
│                    │                                                    │
│                    ▼                                                    │
│              ┌─────────────┐                                            │
│              │   Fanout    │                                            │
│              └─────────────┘                                            │
│                    │                                                    │
│        ┌─────────┴─────────┬─────────┐                                  │
│        ▼                   ▼         ▼                                 │
│    Queue A             Queue B   Queue C   (All receive message)       │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  TOPIC EXCHANGE - Pattern-based routing                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   Producer ──▶ routing_key="order.created.paid"                        │
│                    │                                                    │
│                    ▼                                                    │
│              ┌─────────────┐                                            │
│              │   Topic     │                                            │
│              └─────────────┘                                            │
│                    │                                                    │
│        ┌─────────┼─────────┼─────────┐                                 │
│        ▼         ▼         ▼         ▼                                 │
│    #           *.created  order.#    order.*                           │
│    (all)   (exact)   (prefix)    (any word)                            │
│                                                                          │
│   Patterns: # (any), * (exactly one word)                              │
└─────────────────────────────────────────────────────────────────────────┘
```

## Message Patterns

```
┌─────────────────────────────────────────────────────────────────────────┐
│  POINT-TO-POINT                                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   ┌──────────┐      ┌───────┐      ┌────────────┐                      │
│   │Producer 1│─────▶│ Queue │─────▶│ Consumer 1 │                      │
│   └──────────┘      └───────┘      └────────────┘                      │
│                     (FIFO)                                               │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  WORK QUEUE (Competing Consumers)                                      │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   ┌──────────┐                                 ┌────────────┐          │
│   │Producer  │──────┐                      ┌───│ Consumer 1 │          │
│   └──────────┘      │                      │   └────────────┘          │
│                     │    ┌───────┐         │                          │
│                     ├───▶│ Queue │────┬────┤   ┌────────────┐         │
│                     │    └───────┘    ├───│───│ Consumer 2 │          │
│                     │                   │    └────────────┘         │
│                     │                   ├───   ┌────────────┐         │
│                     │                   └───│──│ Consumer 3 │          │
│                                      │        └────────────┘         │
│   Round-robin distribution                                               │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  PUBLISH/SUBSCRIBE                                                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   ┌──────────┐                                     ┌────────────┐      │
│   │Producer  │─────▶ Fanout ───▶ Queue 1 ──────────▶│ Consumer 1 │      │
│   └──────────┘                    Queue 2 ──────────▶│ Consumer 2 │      │
│                                       Queue 3 ──────────▶│ Consumer 3 │      │
└─────────────────────────────────────────────────────────────────────────┘
```

## Reliable Messaging

```
┌─────────────────────────────────────────────────────────────────────────┐
│  PUBLISHER CONFIRMS                                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   Producer ──▶ Publish ──▶ Broker ──▶ ACK ──▶ Confirmed                 │
│                     │                            │                       │
│                     └──────── NACK ─────────────┘                       │
│                            (republish)                                  │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  CONSUMER ACKNOWLEDGMENT                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│   Broker ──▶ Deliver ──▶ Consumer ──▶ ACK ──▶ Remove from queue        │
│                         │                                               │
│                         └──── NACK ──▶ Requeue ──▶ Redeliver          │
└─────────────────────────────────────────────────────────────────────────┘
```

## Common Patterns Summary

| Pattern | Use Case |
|---------|----------|
| Point-to-Point | Task processing |
| Work Queue | Load distribution |
| Pub/Sub | Notifications |
| Routing | Filtering |
| RPC | Request/Response |
