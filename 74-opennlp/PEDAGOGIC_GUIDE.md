# Pedagogic Guide: Apache OpenNLP

## Learning Path

### Phase 1: NLP Fundamentals (Week 1)
- Understand text processing basics
- Learn about language models
- Explore NLP pipeline concepts

### Phase 2: Core OpenNLP (Week 1-2)
- Master sentence detection
- Implement tokenization
- Apply POS tagging

### Phase 3: Entity Recognition (Week 2-3)
- Understand NER principles
- Extract various entity types
- Handle multi-word entities

### Phase 4: Advanced Tasks (Week 3-4)
- Implement language detection
- Build document classification
- Create custom models

## Key Concepts

| Concept | Description |
|---------|-------------|
| Token | Individual word or punctuation unit |
| Sentence | Grammatically complete unit |
| POS Tag | Grammatical category (noun, verb, etc.) |
| Named Entity | Specific person, place, organization |
| Chunk | Phrase-level grouping |

## Recommended Resources

- OpenNLP Documentation
- "Speech and Language Processing" by Jurafsky & Martin
- Penn Treebank POS Tags Reference

## Assessment

- Build text processing pipeline
- Extract entities from news articles
- Create document classifier
