# 74 - Apache OpenNLP

A module for Natural Language Processing using Apache OpenNLP library.

## Overview

OpenNLP provides Java-based tools for common NLP tasks including sentence detection, tokenization, POS tagging, and named entity recognition.

## Key Topics

- **Sentence Detection**: Split text into sentences
- **Tokenization**: Break text into tokens
- **POS Tagging**: Part-of-speech classification
- **Named Entity Recognition**: Extract entities (person, org, location)
- **Language Detection**: Identify language of text
- **Document Classification**: Text categorization
- **Chunking**: Phrase extraction

## Technology Stack

- Apache OpenNLP
- Pre-trained models (English)
- Maven dependencies

## Getting Started

```bash
cd 74-opennlp
mvn compile exec:java
```

## Requirements

- Java 11+
- Maven 3.6+
- Pre-trained model files
