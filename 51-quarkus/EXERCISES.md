# Exercises - Quarkus

## Exercise 1: Create Entity
Create a Book entity with PanacheEntity that has title, author, and year fields.

## Exercise 2: REST Resource
Create a REST endpoint that returns all books using Book.listAll().

## Exercise 3: Add Query Methods
Add custom query methods to the entity using Panache's find() method.

## Exercise 4: Reactive Route
Implement a reactive route that returns a Uni<Book>.

## Exercise 5: Configure Application
Set up application.properties with database and HTTP server config.
