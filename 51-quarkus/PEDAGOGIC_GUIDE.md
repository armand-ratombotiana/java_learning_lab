# Pedagogic Guide - Quarkus

## Learning Path

### Phase 1: Fundamentals
1. Create a Quarkus project
2. Understand Panache active record pattern
3. Build and run with Maven

### Phase 2: Intermediate
1. REST endpoints with JAX-RS
2. Reactive programming with Mutiny
3. Configuration management

### Phase 3: Advanced
1. Native image compilation
2. Kubernetes deployment
3. Testing with RestAssured

## Panache Patterns

| Pattern | Description |
|---------|-------------|
| PanacheEntity | Active record with built-in methods |
| PanacheRepository | Repository pattern |
| find() | Query by parameters |
| persist() | Save entity |

## Key Advantages
- Fast startup (subatomic)
- Low memory footprint
- Developer joy (live reload)
- Kubernetes native
