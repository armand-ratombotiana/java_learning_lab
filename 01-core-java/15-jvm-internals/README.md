# Module 15: JVM Internals

This module covers the internal workings of the Java Virtual Machine.

## Topics
- Memory regions (Heap, Stack, Metaspace, Code Cache)
- Class loading and ClassLoader hierarchy
- Garbage Collection algorithms
- JIT compilation
- Thread management
- JMX and monitoring

## Running the Lab
```bash
cd 15-jvm-internals
mvn compile exec:java
```
