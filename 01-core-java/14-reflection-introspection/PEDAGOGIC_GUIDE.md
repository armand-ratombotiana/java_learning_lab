# Pedagogic Guide: Reflection & Introspection

## Learning Path
- **Beginner**: Class metadata, getting fields/methods/constructors
- **Intermediate**: Dynamic invocation, private access
- **Advanced**: Proxies, annotations, frameworks

## Key APIs
- `Class.forName()` - Load class by name
- `getDeclaredFields()` - All fields
- `getMethods()` - All public methods
- `setAccessible(true)` - Access private members
- `Method.invoke()` - Call methods dynamically

## Interview Questions
1. How does reflection work?
2. What's the difference between getMethod vs getDeclaredMethod?
3. How do you access private members via reflection?
