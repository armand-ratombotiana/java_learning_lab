# 35 - System Design

Fundamentals of designing scalable, maintainable software systems including architecture patterns, scalability strategies, and distributed systems concepts.

## Overview

- **Topic**: System Design Fundamentals
- **Prerequisites**: Java, data structures, networking basics
- **Duration**: 3-4 hours

## Key Concepts

- Architecture patterns (monolithic, microservices)
- Scalability (horizontal/vertical, caching, partitioning)
- Distributed systems (communication, consistency)
- Database design (SQL vs NoSQL, indexing)

## Getting Started

Run the training code:
```bash
cd 35-system-design
mvn compile exec:java -Dexec.mainClass=com.learning.design.SystemDesignTraining
```

## Module Contents

- Architecture patterns comparison
- Scalability strategies and trade-offs
- Distributed systems patterns
- Database modeling and optimization
