# System Design Exercises

## Exercise 1: Design a URL Shortener
Create a system design for:
- Short URL generation
- Redirect functionality
- Analytics (click tracking)
- Consider: storage, hash function, scalability

## Exercise 2: Design a Twitter Clone
Design core features:
- Post tweets
- Follow/unfollow users
- Home timeline generation
- Consider: write vs read optimization, feed algorithms

## Exercise 3: Scalability Analysis
Given a web app with 1000 users, scale to 1M users:
- Identify bottlenecks
- Propose caching solution
- Design database sharding strategy

## Exercise 4: Database Schema Design
Design normalized schema for an e-commerce system with:
- Users, Products, Orders, Reviews
- Create ER diagram
- Add appropriate indexes

## Exercise 5: Distributed System Pattern
Implement a circuit breaker pattern in Java for resilient service calls.
