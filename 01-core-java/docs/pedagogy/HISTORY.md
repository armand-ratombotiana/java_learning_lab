# History of Core Java

## Origins (1991-1995)

The Java project began in 1991 at Sun Microsystems under the name "Oak." James Gosling and his team aimed to create a language for consumer electronics. The name was changed to "Java" in 1995 after being deemed unsuitable for trademark reasons.

## Java 1.0 (1996)

First public release. Key features included:
- Applet support
- AWT for GUI
- Basic networking
- Security features

## Java 2 Platform (1998)

Major reorganization into three editions:
- J2SE (Standard Edition)
- J2EE (Enterprise Edition)
- J2ME (Micro Edition)

## Modern Era (2006-Present)

- OpenJDK (2006) - Open source implementation
- Java 8 (2014) - Lambda expressions, streams
- Java 11+ (2018+) -模块化系统, records, pattern matching
- Java 21 (2023) - 虚拟线程, pattern matching improvements
