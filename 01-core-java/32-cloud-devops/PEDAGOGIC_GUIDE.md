# Cloud & DevOps - Learning Path

## Beginner
- Understand cloud computing models (IaaS, PaaS, SaaS)
- Learn major AWS/GCP/Azure services
- Understand container basics (Docker)
- Know CI/CD concept and stages

## Intermediate
- Design CI/CD pipelines with GitHub Actions/Jenkins
- Deploy to Kubernetes (minikube/local)
- Write basic Terraform configurations
- Implement monitoring and logging

## Advanced
- Multi-cloud architecture
- GitOps with ArgoCD
- Kubernetes operators
- Cloud security best practices

## Key Concepts
- Infrastructure as Code
- Container orchestration
- Deployment strategies (blue-green, canary)
- Observability (metrics, logs, traces)

## Tools
- Docker, Kubernetes
- GitHub Actions, Jenkins
- Terraform, Pulumi
- Prometheus, Grafana, ELK

## Assessment
- Design deployment architecture for a web app
- Write CI/CD pipeline configuration
- Explain Kubernetes components and resources
