# 32 - Cloud and DevOps

Modern software delivery practices including cloud platforms, CI/CD pipelines, container orchestration, and infrastructure as code.

## Overview

- **Topic**: Cloud Platforms and DevOps Practices
- **Prerequisites**: Java, basic networking
- **Duration**: 3-4 hours

- CI/CD pipeline design

- Infrastructure as Code (Terraform)

```bash
cd 32-cloud-devops

```
