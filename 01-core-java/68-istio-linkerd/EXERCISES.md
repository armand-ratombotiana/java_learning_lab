# Exercises: Istio & Linkerd

## Easy (1-5)
1. Install Istio in Kubernetes
2. Add sidecar injection to namespace
3. Configure destination rules
4. Create virtual service routing
5. Enable mTLS strict mode

## Medium (6-10)
6. Implement canary deployment
7. Add request timeouts
8. Configure circuit breaker
9. Use traffic mirroring
10. Set up egress gateway

## Hard (11-15)
11. Configure JWT authentication
12. Implement workload authorization
13. Create custom Envoy filter
14. Set up multi-cluster mesh
15. Build ingress controller integration
