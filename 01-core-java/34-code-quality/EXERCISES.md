# Code Quality Exercises

## Exercise 1: Identify Code Smells
Given the code sample, identify at least 5 code smells and propose fixes.

## Exercise 2: Extract Method Refactoring
Take a 100-line method and refactor into 5-7 smaller focused methods with clear names.

## Exercise 3: Replace Conditional with Polymorphism
Convert a chain of if-else statements for different types into a proper strategy pattern.

## Exercise 4: Null Safety
Convert code with multiple null checks to use Optional properly.

## Exercise 5: Configure Static Analysis
Add SpotBugs and Checkstyle plugins to a Maven project and fix reported issues.
