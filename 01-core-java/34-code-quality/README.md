# 34 - Code Quality

Writing maintainable, readable, and testable code through clean code principles, refactoring techniques, and static analysis.

## Overview

- **Topic**: Clean Code and Quality Assurance
- **Prerequisites**: Java fundamentals
- **Duration**: 2-3 hours

## Key Concepts

- Clean code principles
- Code smells identification
- Refactoring techniques
- Static analysis tools

## Getting Started

Run the training code:
```bash
cd 34-code-quality
mvn compile exec:java -Dexec.mainClass=com.learning.quality.CodeQualityTraining
```

## Module Contents

- Meaningful names and functions
- Comments vs self-documenting code
- Code smells (bloaters, couplers, dispensables)
- SpotBugs, PMD, Checkstyle, SonarQube
