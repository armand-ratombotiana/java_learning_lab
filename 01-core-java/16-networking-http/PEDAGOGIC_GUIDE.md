# Pedagogic Guide: Networking & HTTP

## Learning Path

- **Beginner**: URL/URI parsing, InetAddress, HttpClient basics
- **Intermediate**: ServerSocket, DatagramSocket, HTTP headers
- **Advanced**: WebSocket, SSL/TLS, connection pooling

## Key Concepts

| Concept | Description |
|---------|-------------|
| HttpClient | Java 11+ HTTP client API |
| TCP Socket | Connection-oriented networking |
| UDP Socket | Connectionless datagrams |
| WebSocket | Full-duplex communication |
