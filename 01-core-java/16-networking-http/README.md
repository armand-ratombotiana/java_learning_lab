# Module 16: Networking & HTTP

Covers Java networking APIs including HTTP client, sockets, and WebSocket.

## Topics
- URL/URI parsing
- InetAddress
- Java 11+ HttpClient
- ServerSocket (TCP)
- DatagramSocket (UDP)
- WebSocket
- Network interfaces

## Running the Lab
```bash
cd 16-networking-http
mvn compile exec:java
```

## Key References
- java.net.http package
- HttpClient, HttpRequest, HttpResponse
