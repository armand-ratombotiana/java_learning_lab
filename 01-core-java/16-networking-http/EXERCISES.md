# Exercises: Networking & HTTP

## Exercise 1: Simple HTTP Client
Create a GET request using Java 11+ HttpClient to fetch data from a public API.
Handle response and parse JSON.

## Exercise 2: REST API Client
Build a client for a REST API with:
- GET, POST, PUT, DELETE methods
- Authentication header
- JSON serialization

## Exercise 3: TCP Server
Implement a simple TCP echo server using ServerSocket.
Handle multiple clients with threads.

## Exercise 4: WebSocket Client
Connect to a WebSocket server and handle messages.
Implement reconnection logic.

## Exercise 5: URL Connection
Use URLConnection to download a file with progress tracking.
