# Exercises: WebAssembly

## Easy (1-5)
1. Compile Java to WASM with TeaVM
2. Create "Hello World" WASM module
3. Load WASM in JavaScript
4. Call Java methods from JS
5. Return values from WASM

## Medium (6-10)
6. Pass complex objects between JS/WASM
7. Handle browser APIs from Java
8. Implement WASI for system calls
9. Debug WASM with source maps
10. Bundle WASM with webpack

## Hard (11-15)
11. Use GraalVM native-image for WASM
12. Create graphics app with WASM
13. Implement multi-threaded WASM
14. Optimize WASM binary size
15. Build server-side WASM service
