# Pedagogic Guide: WebAssembly

## Learning Path
1. **WASM Basics** (1h): Binary format, execution
2. **Java Compilation** (2h): TeaVM, JWebAssembly
3. **Browser Integration** (2h): JS interop
4. **Optimization** (1h): Size, performance

## Assessment
- [ ] Compile Java to WebAssembly
- [ ] Call WASM from JavaScript
- [ ] Handle JS/WASM data exchange
