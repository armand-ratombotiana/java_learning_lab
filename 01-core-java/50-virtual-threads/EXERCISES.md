# Virtual Threads Exercises

## Exercise 1: Create Virtual Threads
Create 10,000 virtual threads using Thread.ofVirtual() and compare memory usage.

## Exercise 2: Migrate Thread Pool
Migrate an ExecutorService implementation to use virtual threads.

## Exercise 3: Structured Concurrency
Implement a structured concurrency pattern using try-with-resources.

## Exercise 4: Scoped Values
Use Thread.ScopedValue for passing context across virtual thread tasks.

## Exercise 5: Handle Pinning
Identify and fix pinning issues when using synchronized with virtual threads.

## Bonus: HTTP Client
Use virtual threads with Java HTTP client for concurrent requests.
