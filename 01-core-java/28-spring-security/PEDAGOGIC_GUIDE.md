# Pedagogic Guide: Spring Security

## Learning Path
- **Beginner**: Basic auth, form login, password encoding
- **Intermediate**: JWT, OAuth2 basics, method security
- **Advanced**: Custom filters, identity providers, CSRF

## Key Concepts
| Concept | Description |
|---------|-------------|
| Authentication | Identity verification |
| Authorization | Access control |
| JWT | JSON Web Token |
| OAuth2 | Authorization framework |
| CSRF | Cross-site request forgery |
