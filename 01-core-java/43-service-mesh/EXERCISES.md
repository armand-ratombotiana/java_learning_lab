# Service Mesh Exercises

## Exercise 1: Traffic Routing
Configure a service mesh to route traffic between two versions of a service.

## Exercise 2: Canary Deployment
Implement canary deployment with 10% traffic to the new version.

## Exercise 3: Circuit Breaker
Add circuit breaker configuration for a failing backend service.

## Exercise 4: Mutual TLS
Configure mutual TLS between services in the mesh.

## Exercise 5: Observability
Set up distributed tracing and metrics collection.

## Bonus: Traffic Shadowing
Implement traffic shadowing to test new versions in production.
