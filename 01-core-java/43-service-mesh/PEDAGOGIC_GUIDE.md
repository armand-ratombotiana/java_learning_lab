# Service Mesh - Learning Path

## Beginner
- Understand service mesh architecture
- Learn sidecar proxy patterns
- Study traffic routing
- Configure mTLS basics

## Intermediate
- Implement canary deployments
- Add circuit breaking
- Configure retry policies
- Set up observability

## Advanced
- traffic shadowing
- Multi-cluster mesh
- Custom extensions
- Security policy design

## Key Concepts
- Sidecar injection
- Zero-trust networking
- Distributed tracing
- Traffic split

## Dependencies
- Istio
- Linkerd
- Consul Connect

## Assessment
- Configure canary deployment with traffic splitting
- Implement mTLS between services
- Add observability with distributed tracing
