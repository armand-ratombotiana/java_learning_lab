## Overview

This capstone project integrates all Java concepts learned throughout the journey. Build a production-ready microservice application demonstrating core competencies.

### Project Requirements

- Spring Boot REST API
- Database integration (JPA)
- Microservices architecture
- Container deployment
- Observability
- CI/CD pipeline
- GitOps deployment
