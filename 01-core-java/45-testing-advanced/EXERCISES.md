# Testing Advanced Exercises

## Exercise 1: Database Integration Tests
Write integration tests using Testcontainers for a repository layer.

## Exercise 2: Mock Service Dependencies
Use Mockito to mock external service calls in unit tests.

## Exercise 3: Property-Based Testing
Use jqwik to implement property-based tests for a data processing function.

## Exercise 4: Mutation Testing
Run Pitest and analyze mutation coverage on critical code paths.

## Exercise 5: Test Fixtures
Create reusable test fixtures with @BeforeEach setup.

## Bonus: Contract Testing
Implement contract tests for REST API compatibility.
