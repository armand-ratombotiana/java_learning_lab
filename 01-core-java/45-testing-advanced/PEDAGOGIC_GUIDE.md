# Testing Advanced - Learning Path

## Beginner
- Understand test pyramid
- Learn integration testing
- Use Testcontainers
- Write effective test cases

## Intermediate
- Mock with Mockito
- Implement test fixtures
- Use property-based testing
- Analyze test coverage

## Advanced
- Mutation testing with Pitest
- Chaos testing
- Contract testing
- Performance testing

## Key Concepts
- Test isolation
- Test data management
- Assertion strategies
- Test maintainability

## Dependencies
- JUnit 5
- Mockito
- Testcontainers
- Pitest

## Assessment
- Write integration tests using Testcontainers
- Implement comprehensive mock coverage
- Run mutation testing and interpret results
