# Data Structures & Algorithms

Core Java module covering fundamental data structures and algorithms essential for software development and technical interviews.

## Topics
- **Foundations**: Arrays, Strings, Two Pointers, Sliding Window
- **Linear Data Structures**: Linked Lists (Singly, Doubly, Circular), Stacks, Queues
- **Hierarchical Data Structures**: Trees, Binary Trees, Binary Search Trees (BST), Heaps, Tries
- **Graphs**: Representation, BFS, DFS, Shortest Path Algorithms, Spanning Trees
- **Algorithmic Paradigms**: Sorting, Searching, Dynamic Programming, Greedy Algorithms, Backtracking
- **Advanced Topics**: Bit Manipulation, Disjoint Set Union (Union-Find)

## Getting Started
```bash
mvn compile
mvn test
```
