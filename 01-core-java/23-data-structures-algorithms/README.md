# Data Structures & Algorithms

Core Java module covering fundamental data structures and algorithms essential for software development and technical interviews.

## Topics
- Arrays and Strings
- Linked Lists (Singly, Doubly, Circular)
- Stacks and Queues
- Trees and Binary Search Trees
- Graphs and Graph Algorithms
- Sorting Algorithms
- Searching Algorithms
- Dynamic Programming basics

## Getting Started
```bash
mvn compile
mvn test
```
