# Exercises: Security & Cryptography

## Exercise 1: Hashing
Implement password hashing with SHA-256.
Compare hashes and verify integrity.

## Exercise 2: AES Encryption
Encrypt and decrypt data using AES-256-GCM.
Handle IV correctly.

## Exercise 3: RSA Key Pairs
Generate RSA key pair (2048-bit).
Encrypt with public key, decrypt with private key.

## Exercise 4: Digital Signature
Sign data with private key.
Verify signature with public key.

## Exercise 5: KeyStore
Store keys in PKCS12 KeyStore.
Load and retrieve keys securely.
