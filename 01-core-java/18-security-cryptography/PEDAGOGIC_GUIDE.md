# Pedagogic Guide: Security & Cryptography

## Learning Path

- **Beginner**: Hashing, Base64 encoding, SecureRandom
- **Intermediate**: AES encryption, RSA key pairs
- **Advanced**: Digital signatures, KeyStore, SSL/TLS

## Key Concepts

| Concept | Description |
|---------|-------------|
| Hashing | One-way function for integrity |
| AES | Symmetric encryption (fast) |
| RSA | Asymmetric encryption (key exchange) |
| Signature | Message authenticity |
