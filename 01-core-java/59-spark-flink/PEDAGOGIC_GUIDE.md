# Pedagogic Guide: Spark & Flink

## Learning Path
1. **Spark Basics** (2h): RDD, DataFrame, transformations
2. **Spark SQL** (2h): Queries, joins, aggregations
3. **Streaming** (2h): Structured Streaming, watermarks
4. **Flink** (2h): DataStream, state, windows

## Assessment
- [ ] Create Spark DataFrame
- [ ] Implement streaming pipeline
- [ ] Use Flink for stateful processing
