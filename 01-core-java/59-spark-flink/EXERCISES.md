# Exercises: Spark & Flink

## Easy (1-5)
1. Create SparkSession and load data
2. Use DataFrame operations (select, filter, groupBy)
3. Implement word count in Spark
4. Write results to Parquet
5. Configure Spark cluster

## Medium (6-10)
6. Implement streaming with Spark Structured
7. Add window functions (tumbling, sliding)
8. Use Flink DataStream API
9. Implement stateful processing
10. Handle late data with watermarks

## Hard (11-15)
11. Optimize Spark joins (broadcast, shuffle)
12. Implement custom aggregate functions
13. Create Flink checkpoint for fault tolerance
14. Build exactly-once processing
15. Tune Spark memory and executors
