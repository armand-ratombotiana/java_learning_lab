# Database Advanced Exercises

## Exercise 1: Connection Pooling
Set up HikariCP with optimal pool size and configure connection timeout handling.

## Exercise 2: Transaction Isolation
Demonstrate differences between isolation levels with explicit locking scenarios.

## Exercise 3: Query Optimization
Analyze query execution plans and optimize a slow query with proper indexing.

## Exercise 4: Optimistic Locking
Implement optimistic locking using version fields for concurrent updates.

## Exercise 5: Read Replica
Configure read replica routing for read-heavy workloads.

## Bonus: Distributed Transaction
Implement two-phase commit for distributed database updates.
