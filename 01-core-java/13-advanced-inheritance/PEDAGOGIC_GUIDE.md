## Learning Path

### Beginner (2-3 hours)
- Understand interface vs abstract class
- Learn default methods
- Practice simple composition

### Intermediate (3-4 hours)
- Master multiple interface implementation
- Solve diamond problems
- Apply mixin patterns

### Advanced (2-3 hours)
- Use sealed classes effectively
- Design flexible hierarchies
- Apply composition patterns

---

## Key Concepts

| Concept | When to Use |
|---------|-------------|
| Interface | Multiple "is-a" relationships |
| Abstract Class | Shared implementation + contract |
| Composition | "has-a" over "is-a" |
| Sealed Class | Control who can extend |

---

## Interview Tips

- Explain diamond problem and solution
- Discuss composition vs inheritance tradeoffs
- Know sealed class syntax and purpose
