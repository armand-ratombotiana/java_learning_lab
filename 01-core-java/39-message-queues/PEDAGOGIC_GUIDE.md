# Message Queues - Learning Path

## Beginner
- Understand async communication benefits
- Learn producer/consumer patterns
- Study queue vs topic differences
- Understand message persistence

## Intermediate
- Implement JMS producers and consumers
- Handle message acknowledgment
- Design message formats
- Add error handling and retries

## Advanced
- Message routing and filtering
- Transaction management
- High availability patterns
- Queue clustering and federation

## Key Concepts
- Asynchronous messaging
- Message durability
- Consumer groups
- Dead letter queues

## Dependencies
- JMS API (jakarta.jms)
- ActiveMQ or RabbitMQ client
- Apache Kafka client

## Assessment
- Design a message schema for an order processing system
- Implement reliable message delivery with retries
- Compare queue vs pub/sub for different scenarios
