# Message Queues Exercises

## Exercise 1: Simple Producer-Consumer
Implement a producer that sends messages to a queue and a consumer that processes them asynchronously.

## Exercise 2: Message Types
Create messages with different priorities and implement a priority queue consumer.

## Exercise 3: Pub/Sub Pattern
Build a pub/sub system where multiple consumers subscribe to the same topic.

## Exercise 4: Dead Letter Queue
Implement a dead letter queue for failed message processing with retry logic.

## Exercise 5: Message Ordering
Handle ordered message processing for dependent operations.

## Bonus: Distributed Queue
Simulate a distributed message queue with multiple producers.
