# 49 - Kotlin for Java Developers

Kotlin is a modern JVM language that interoperates seamlessly with Java. This module covers Kotlin features useful for Java developers transitioning to Kotlin.

## Overview

- **Topic**: Kotlin Language Features
- **Prerequisites**: Java fundamentals
- **Duration**: 2-3 hours

## Key Concepts

- Null safety and smart casts
- Extension functions
- Data classes and sealed classes
- Coroutines basics

## Getting Started

Run the training code:
```bash
cd 49-kotlin-java
mvn compile exec:java -Dexec.mainClass=com.learning.kotlinjava.Lab
```

## Module Contents

- Kotlin syntax for Java devs
- Interop between Java/Kotlin
- Idiomatic Kotlin patterns
