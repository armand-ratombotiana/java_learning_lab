# Kotlin for Java Developers - Exercises

## Exercise 1: Null Safety
Convert a Java method with null checks to Kotlin using null safety features.

## Exercise 2: Data Classes
Replace Java getters/setters with Kotlin data classes.

## Exercise 3: Extension Functions
Add functionality to a Java class using Kotlin extension functions.

## Exercise 4: Simple Coroutine
Implement async processing using Kotlin coroutines.

## Exercise 5: Interop
Use Kotlin code from Java and vice versa, handling nullability.

## Bonus: DSL Builder
Create a type-safe builder for configuring a complex object.
