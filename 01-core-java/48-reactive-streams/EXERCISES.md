# Reactive Streams Exercises

## Exercise 1: Simple Publisher
Implement a custom Publisher using Java 9 Flow API that emits numbers.

## Exercise 2: Backpressure
Create a Subscriber that implements backpressure with request(n) handling.

## Exercise 3: Transform Stream
Build a reactive pipeline that transforms data using operators.

## Exercise 4: Error Handling
Add error handling and recovery to a reactive stream pipeline.

## Exercise 5: Integration
Integrate a reactive stream with a Spring WebFlux controller.

## Bonus: Hot and Cold Streams
Compare behavior of hot vs cold publishers with multiple subscribers.
