# Functional Programming

Core Java module covering functional programming paradigms introduced in Java 8+.

## Topics
- **Core Functional Concepts**: Pure Functions, Immutability, Referential Transparency
- **Lambdas & Method References**: Syntax, scope, and best practices
- **Built-in Functional Interfaces**: Predicate, Function, Consumer, Supplier, Bi-interfaces
- **Stream API Deep Dive**: Intermediate/Terminal operations, lazy evaluation
- **Advanced Collectors**: Grouping, Partitioning, Downstream and Custom Collectors
- **Optional API**: Null-safety, monadic operations (`map`, `flatMap`)
- **Function Composition**: Building complex transformations
- **Functional Design Patterns**: Strategy, Command, Execute Around
- **Parallel Streams**: Thread-safety and performance considerations

## Getting Started
```bash
mvn compile
mvn test
```
