# Exercises: Jakarta EE

## Easy (1-5)
1. Create CDI managed bean
2. Inject dependency with @Inject
3. Add qualifier for implementation
4. Use @ApplicationScoped bean
5. Fire CDI event

## Medium (6-10)
6. Implement JPA entity and repository
7. Add Bean Validation to entity
8. Create JAX-RS REST endpoint
9. Use JSON-B for serialization
10. Implement role-based security

## Hard (11-15)
11. Configure persistence.xml
12. Implement interceptors
13. Create custom validator
14. Add transactional boundaries
15. Implement async CDI events
