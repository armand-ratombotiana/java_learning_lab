# Exercises: ML with Java

## Easy (1-5)
1. Load CSV data with Weka
2. Train decision tree classifier
3. Evaluate model with cross-validation
4. Use Weka Explorer GUI
5. Make predictions on new data

## Medium (6-10)
6. Train random forest classifier
7. Implement linear regression
8. Build clustering model (K-means)
9. Normalize features for training
10. Use DeepLearning4j for MLP

## Hard (11-15)
11. Implement custom feature transformer
12. Tune hyperparameters programmatically
13. Build image classification model
14. Save and load trained models
15. Create ML pipeline with preprocessing
