# Pedagogic Guide: Incident Response

## Learning Path
1. **Response** (1h): Triage, escalation
2. **On-Call** (1h): Best practices
3. **Analysis** (2h): Post-mortems, learning
4. **Prevention** (1h): Reliability engineering

## Assessment
- [ ] Handle production incident
- [ ] Write blameless post-mortem
- [ ] Create runbook
