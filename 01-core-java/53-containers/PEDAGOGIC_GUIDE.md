# Pedagogic Guide: Containers

---

## Learning Path

### Phase 1: Docker Fundamentals (2 hours)
1. Understand container architecture
2. Create first Dockerfile
3. Build and run containers

### Phase 2: Kubernetes (3 hours)
1. Understand K8s architecture
2. Create Pod, Deployment, Service
3. Deploy to local cluster (minikube/kind)

### Phase 3: Production Patterns (3 hours)
1. Multi-stage builds optimization
2. Resource management
3. Health checks and scaling

---

## Assessment Checklist

- [ ] Create Docker image for Java app
- [ ] Understand layer caching
- [ ] Deploy to Kubernetes
- [ ] Configure resources
- [ ] Implement health checks
