# Exercises: Containers

---

## Easy Exercises (1-5)

### Exercise 1: Simple Dockerfile
Create Dockerfile for a simple Java application.

### Exercise 2: Run Container
Build and run the Docker image locally.

### Exercise 3: Volume Mounting
Mount source code for development.

### Exercise 4: Port Mapping
Expose application port to host.

### Exercise 5: Environment Variables
Pass configuration via environment variables.

---

## Medium Exercises (6-10)

### Exercise 6: Multi-Stage Build
Implement multi-stage Dockerfile for Java app.

### Exercise 7: .dockerignore
Create .dockerignore to reduce build context.

### Exercise 8: Kubernetes Pod
Create pod.yaml for Java application.

### Exercise 9: Kubernetes Deployment
Create deployment with replicas.

### Exercise 10: Service Exposure
Expose application via Kubernetes Service.

---

## Hard Exercises (11-15)

### Exercise 11: ConfigMap Integration
Use ConfigMap for application configuration.

### Exercise 12: Resource Limits
Set CPU/memory requests and limits.

### Exercise 13: Liveness Probe
Add health check endpoint and probe config.

### Exercise 14: Horizontal Pod Autoscaler
Configure HPA based on CPU/memory.

### Exercise 15: Ingress Setup
Configure Ingress for external access.

---
