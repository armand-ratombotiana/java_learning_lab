# Pedagogic Guide: Spring REST API

## Learning Path
- **Beginner**: HTTP methods, REST annotations, CRUD
- **Intermediate**: Validation, exception handling, versioning
- **Advanced**: HATEOAS, OpenAPI/Swagger, content negotiation

## Key Concepts
| Concept | Description |
|---------|-------------|
| REST | Representational State Transfer |
| @RestController | REST endpoint marker |
| @RequestMapping | HTTP request mapping |
| @ControllerAdvice | Global exception handling |
| OpenAPI | API documentation (Swagger) |
