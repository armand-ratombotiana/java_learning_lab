# Pedagogic Guide: Observability

## Learning Path
1. **Metrics** (2h): Micrometer, Prometheus
2. **Logging** (1h): Structured logs, ELK
3. **Tracing** (2h): OpenTelemetry, Jaeger
4. **Alerting** (1h): Grafana, Alertmanager

## Key Concepts
- The Three Pillars: Metrics, Logs, Traces
- RED metrics (Rate, Errors, Duration)
- USE metrics (Utilization, Saturation, Errors)

## Assessment
- [ ] Add metrics to Spring Boot app
- [ ] Configure distributed tracing
- [ ] Create dashboard
