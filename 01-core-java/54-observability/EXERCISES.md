# Exercises: Observability

## Easy (1-5)
1. Add Micrometer counter for HTTP requests
2. Create custom metric for business logic
3. Configure JSON logging
4. Add log correlation ID
5. Expose Prometheus endpoint

## Medium (6-10)
6. Add distributed tracing with OpenTelemetry
7. Create Grafana dashboard
8. Configure log aggregation
9. Add health check endpoint
10. Implement alerting rules

## Hard (11-15)
11. Trace database calls with span
12. Build custom exporter
13. Implement sampling strategy
14. Create service map visualization
15. Setup on-call rotation
