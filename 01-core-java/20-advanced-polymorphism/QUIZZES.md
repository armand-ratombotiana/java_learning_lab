# Quizzes: Advanced Polymorphism

Test your knowledge of method hiding, covariant returns, and bridge methods.

## Quiz 1: Static vs Dynamic Binding

**Q1: What happens if a subclass declares a `static` method with the exact same signature as a `static` method in its superclass?**
- A) The compiler throws an error.
- B) The subclass method overrides the superclass method (Dynamic Binding).
- C) The subclass method hides the superclass method (Static Binding).
- D) Both methods execute when called.
*Answer: C*

**Q2: You have `Parent p = new Child();`. Both classes declare a field `int x`. `Parent`'s `x` is 5, `Child`'s `x` is 10. What does `System.out.println(p.x)` print?**
- A) 5
- B) 10
- C) Compile error
- D) RuntimeException
*Answer: A (Fields are resolved based on the reference type, not the object type).*

## Quiz 2: Covariant Returns and Overriding

**Q1: Which of the following is a valid Covariant Return Type?**
- A) Superclass returns `Number`, subclass overriding method returns `Integer`.
- B) Superclass returns `Integer`, subclass overriding method returns `Number`.
- C) Superclass returns `int`, subclass overriding method returns `Integer`.
- D) Superclass returns `void`, subclass overriding method returns `String`.
*Answer: A (Integer is a subtype of Number. C is wrong because primitives and wrappers are not in the same inheritance hierarchy).*

**Q2: Why should you NEVER call an overridable (non-final, non-private) method from a constructor?**
- A) It slows down object creation.
- B) If a subclass overrides the method, the subclass's version will execute before the subclass has finished initializing its fields, likely causing a `NullPointerException`.
- C) It is a syntax error in Java.
- D) It causes a memory leak.
*Answer: B*

## Quiz 3: Interfaces and Generics

**Q1: If a class implements two interfaces that both provide a `default` method with the same signature, what must the class do?**
- A) Nothing, the JVM randomly picks one.
- B) Nothing, the compiler picks the one from the interface listed first in the `implements` clause.
- C) The class must override the method to resolve the ambiguity (it can optionally call `InterfaceName.super.methodName()` inside).
- D) The class cannot implement both interfaces.
*Answer: C*

**Q2: What is a "Bridge Method" in Java?**
- A) A method that connects a Java application to a database.
- B) A synthetic method generated by the compiler to preserve polymorphism when generic types are erased at runtime.
- C) A method used in the Bridge Design Pattern.
- D) A method that allows C++ code to call Java code.
*Answer: B*
