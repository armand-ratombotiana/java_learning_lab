# Exercises: Blockchain

## Easy (1-5)
1. Implement SHA-256 hashing
2. Create Block class with hash
3. Build Genesis block
4. Validate chain integrity
5. Add transactions to block

## Medium (6-10)
6. Implement proof-of-work mining
7. Create wallet with key pair
8. Sign transactions digitally
9. Add Merkle tree for transactions
10. Implement difficulty adjustment

## Hard (11-15)
11. Build P2P node network
12. Implement UTXO model
13. Add smart contract execution
14. Create SPV proof verification
15. Implement sharding concept
