# Pedagogic Guide: Blockchain

## Learning Path
1. **Cryptography** (2h): Hashing, signatures
2. **Blockchain** (2h): Blocks, chain, mining
3. **Transactions** (1h): Signing, verification
4. **Consensus** (2h): PoW, PoS, mechanisms

## Assessment
- [ ] Implement basic blockchain
- [ ] Create block with proof-of-work
- [ ] Sign transactions
