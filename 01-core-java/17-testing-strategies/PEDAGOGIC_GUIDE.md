# Pedagogic Guide: Testing Strategies

## Learning Path

- **Beginner**: JUnit basics, assertions, test structure
- **Intermediate**: Mockito, mocking, parameterized tests
- **Advanced**: Integration tests, test containers, TDD

## Key Concepts

| Concept | Description |
|---------|-------------|
| Unit Test | Test single class/method in isolation |
| Mock | Simulated dependency |
| TDD | Red-Green-Refactor cycle |
| Test Pyramid | Unit > Integration > E2E |
