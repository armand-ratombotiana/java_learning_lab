# Exercises: Spring Native

## Easy (1-5)
1. Create Spring Boot app with web dependency
2. Build native image with native-maven-plugin
3. Compare startup time JVM vs native
4. Add @NativeHint for resource bundling
5. Configure native build pack

## Medium (6-10)
6. Handle reflection with @TypeHint
7. Register serialization with @SerializationHint
8. Add GraalVM Reachability Metadata
9. Configure init vs build time
10. Use Spring Boot performance insights

## Hard (11-15)
11. Migrate existing Spring Boot to native
12. Add native image debugging
13. Implement resource in native image
14. Configure AOT vs JIT tradeoffs
15. Optimize native image size
