# Memory Profiling - Learning Path

## Beginner
- Understand JVM memory model
- Learn heap/stack differences
- Study GC algorithms
- Analyze GC logs

## Intermediate
- Use VisualVM for profiling
- Analyze heap dumps
- Identify memory leaks
- Tune GC parameters

## Advanced
- MAT (Memory Analyzer Tool)
- Allocation profiling
- Soft reference handling
- Off-heap memory tracking

## Key Concepts
- Heap sizing
- GC pause times
- Object retention
- Memory leak patterns

## Dependencies
- VisualVM
- Eclipse MAT
-async-profiler
- GCEasy

## Assessment
- Analyze a heap dump to find memory leaks
- Configure GC for low-latency applications
- Use allocation profiling to find allocation hotspots
