# Memory Profiling - Exercises

## Exercise 1: GC Log Analysis
Configure GC logging and analyze the output using GCEasy to identify GC performance issues.

## Exercise 2: Heap Dump Analysis
Generate a heap dump and use MAT to find memory leaks in a sample application.

## Exercise 3: Object Allocation Tracking
Use VisualVM to track object allocation and identify allocation hotspots.

## Exercise 4: Memory Leak Pattern
Create and fix a common memory leak scenario (e.g., listener Accumulation).

## Exercise 5: GC Tuning
Tune GC parameters to reduce pause times for a latency-sensitive application.

## Bonus: Off-Heap Memory
Implement and profile off-heap memory using DirectByteBuffer.
