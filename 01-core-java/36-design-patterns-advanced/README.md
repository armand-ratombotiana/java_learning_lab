# 36 - Advanced Design Patterns

In-depth coverage of creational, structural, and behavioral design patterns for building flexible and reusable software.

## Overview

- **Topic**: Advanced Design Patterns
- **Prerequisites**: Java OOP, basic design patterns
- **Duration**: 3-4 hours

## Key Concepts

- Creational: Builder, Factory, Abstract Factory, Singleton, Prototype
- Structural: Adapter, Bridge, Composite, Decorator, Facade, Flyweight, Proxy
- Behavioral: Chain of Responsibility, Command, Iterator, Mediator, Memento, Observer, State, Strategy, Template Method, Visitor

## Getting Started

Run the training code:
```bash
cd 36-design-patterns-advanced
mvn compile exec:java -Dexec.mainClass=com.learning.patterns.AdvancedPatternsTraining
```
