# Advanced Design Patterns Exercises

## Exercise 1: Builder Pattern
Implement a builder for a complex object like Order with nested items, discounts, and shipping details.

## Exercise 2: Strategy Pattern
Create a payment system with multiple strategies (CreditCard, PayPal, Crypto) that can be swapped at runtime.

## Exercise 3: Decorator Pattern
Build a coffee shop ordering system with base coffee and various toppings (milk, sugar, caramel) using decorators.

## Exercise 4: Observer Pattern
Implement a stock price monitor that notifies multiple observers (display, logger, alert) when prices change.

## Exercise 5: State Pattern
Create an order processing system with states (New, Paid, Shipped, Delivered, Cancelled) where state transitions are encapsulated.

## Exercise 6: Adapter Pattern
Write an adapter to make a legacy PaymentGateway work with a new modern interface.

## Exercise 7: Facade Pattern
Create a facade for a complex video conversion system hiding multiple internal components.
