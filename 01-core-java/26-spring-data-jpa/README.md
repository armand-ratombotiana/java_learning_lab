# Spring Data JPA

Core Java module covering JPA/Hibernate for database persistence in Spring applications.

## Topics
- Entity Classes
- Spring Data Repositories
- JPA Relationships
- JPQL and Native Queries
- Paging and Sorting
- Transactions

## Getting Started
```bash
mvn spring-boot:run
```
