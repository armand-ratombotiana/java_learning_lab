# Performance Optimization Exercises

## Exercise 1: JMH Microbenchmark
Write a JMH benchmark to compare StringBuilder vs StringBuffer performance.

## Exercise 2: Escape Analysis
Implement object creation that can be stack-allocated through escape analysis.

## Exercise 3: Collection Optimization
Optimize a hot loop by using primitive collections instead of object collections.

## Exercise 4: JIT Inlining
Measure JIT inlining effects by examining JIT logs for method hot spots.

## Exercise 5: Array Performance
Compare performance of array vs ArrayList for primitive data types.

## Bonus: Lock Elision
Implement lock elision usingbiased locking optimization techniques.
