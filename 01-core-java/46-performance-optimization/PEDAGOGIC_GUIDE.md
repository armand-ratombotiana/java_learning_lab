# Performance Optimization - Learning Path

## Beginner
- Understand JIT compilation
- Learn hot spot detection
- Study escape analysis
- Analyze array optimizations

## Intermediate
- Profile with JMH
- Optimize collection access
- Reduce object allocation
- Use primitive types

## Advanced
- SIMD optimizations
- Branch prediction
- Cache-line alignment
- Lock elision techniques

## Key Concepts
- JIT warmup
- Inline caching
- Escape analysis
- Deoptimization

## Dependencies
- JMH
- JProfiler
- VisualVM
-async-profiler

## Assessment
- Profile a method with JMH microbenchmarking
- Identify and optimize allocation hotspots
- Apply code-level optimizations
