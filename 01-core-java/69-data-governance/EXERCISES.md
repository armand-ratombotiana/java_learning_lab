# Exercises: Data Governance

2. Add metadata to datasets
3. Define data ownership
4. Implement data quality checks
5. Add data classification tags

6. Build data lineage tracker
7. Implement GDPR compliance
8. Create data access audit
9. Add PII detection
10. Build data quality dashboard

## Hard (11-15)
11. Implement data versioning
12. Create self-service portal
13. Build policy engine
14. Add data retention policies
15. Integrate governance API
