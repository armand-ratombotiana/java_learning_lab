# Pedagogic Guide: Data Governance

## Learning Path
1. **Catalog** (1h): Metadata, discovery
2. **Quality** (2h): Validation, monitoring
3. **Lineage** (1h): Tracking, impact analysis
4. **Compliance** (2h): GDPR, PII, audit

## Assessment
- [ ] Create data catalog
- [ ] Implement quality checks
- [ ] Track data lineage
