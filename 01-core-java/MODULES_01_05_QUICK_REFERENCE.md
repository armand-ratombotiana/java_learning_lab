## 🔍 Common Patterns

### Module 01: Java Basics
**Pattern 1: Loop and Accumulate**
```java
int sum = 0;
for (int i = 0; i < n; i++) {
    sum += array[i];
}
```

**Pattern 2: Two Pointers**
```java
int left = 0, right = array.length - 1;
while (left < right) {
    // Process
    left++;
    right--;
}
```

---

### Module 02: OOP Concepts
**Pattern 1: Inheritance Hierarchy**
```java
abstract class Animal { }
class Dog extends Animal { }
class Cat extends Animal { }
```

**Pattern 2: Composition**
```java
class Car {
    private Engine engine;
    private Wheels wheels;
}
```

---

### Module 03: Collections Framework
**Pattern 1: Frequency Counter**
```java
Map<String, Integer> freq = new HashMap<>();
for (String word : words) {
    freq.put(word, freq.getOrDefault(word, 0) + 1);
}
```

**Pattern 2: Group By**
```java
Map<String, List<T>> grouped = list.stream()
    .collect(Collectors.groupingBy(T::getCategory));
```

---

### Module 04: Streams API
**Pattern 1: Filter-Map-Collect**
```java
List<Integer> result = numbers.stream()
    .filter(n -> n > 5)
    .map(n -> n * 2)
    .collect(Collectors.toList());
```

**Pattern 2: Reduce**
```java
int sum = numbers.stream()
    .reduce(0, (a, b) -> a + b);
```

---

### Module 05: Concurrency & Multithreading
**Pattern 1: Producer-Consumer**
```java
synchronized void produce(int val) {
    while (available) wait();
    value = val;
    available = true;
    notify();
}
```

**Pattern 2: Thread Pool**
```java
ExecutorService executor = Executors.newFixedThreadPool(3);
executor.submit(() -> { /* task */ });
executor.shutdown();
```
