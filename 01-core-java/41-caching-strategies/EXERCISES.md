# Caching Strategies Exercises

## Exercise 1: In-Memory LRU Cache
Implement an LRU cache with size limit and TTL expiration.

## Exercise 2: Cache-Aside Pattern
Add caching to a data access layer using cache-aside pattern.

## Exercise 3: Write-Through Cache
Build a write-through cache that updates both cache and database.

## Exercise 4: Cache Invalidation
Implement time-based and event-based cache invalidation.

## Exercise 5: Distributed Cache
Use Redis for distributed caching with cache-aside pattern.

## Bonus: Multi-Level Cache
Implement L1 ( Caffeine) and L2 (Redis) multi-level caching.
