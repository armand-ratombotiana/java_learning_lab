# Exercises: Data Engineering

## Easy (1-5)
1. Read CSV file with OpenCSV
2. Write data to JSON file
3. Parse fixed-width format
4. Transform records with Java streams
5. Filter data based on conditions

## Medium (6-10)
6. Process Parquet files
7. Use Avro schema for serialization
8. Build incremental ETL pipeline
9. Handle schema evolution
10. Implement batch processing

## Hard (11-15)
11. Build CDC pipeline from database
12. Implement data validation framework
13. Create data quality checks
14. Build data pipeline with Airflow
15. Implement Slowly Changing Dimensions
