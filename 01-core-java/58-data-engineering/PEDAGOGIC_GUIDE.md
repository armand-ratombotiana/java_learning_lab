# Pedagogic Guide: Data Engineering

## Learning Path
1. **Data Formats** (2h): CSV, JSON, Parquet, Avro
2. **ETL Basics** (2h): Extract, Transform, Load
3. **Processing** (2h): Batch, streaming
4. **Quality** (1h): Validation, monitoring

## Assessment
- [ ] Build simple ETL pipeline
- [ ] Process multiple data formats
- [ ] Implement data validation
