# 40 - gRPC and Protobuf

gRPC is a high-performance RPC framework using Protocol Buffers for efficient binary serialization. It provides fast, type-safe communication between services.

## Overview

- **Topic**: gRPC and Protocol Buffers
- **Prerequisites**: Java networking, distributed systems
- **Duration**: 2-3 hours

## Key Concepts

- Protocol Buffers serialization
- gRPC service definitions
- Streaming (client, server, bidirectional)
- Schema evolution and versioning

## Getting Started

Run the training code:
```bash
cd 40-grpc-protobuf
mvn compile exec:java -Dexec.mainClass=com.learning.grpc.Lab
```
