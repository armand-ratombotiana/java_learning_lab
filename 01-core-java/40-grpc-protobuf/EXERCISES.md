# gRPC and Protobuf Exercises

## Exercise 1: Define a Product Schema
Create a protobuf schema for a product service with product info, pricing, and inventory.

## Exercise 2: Implement a Unary RPC
Build a gRPC service with GetProduct and SearchProducts unary calls.

## Exercise 3: Server Streaming
Implement server-side streaming to push price updates to clients.

## Exercise 4: Client Streaming
Create a batch import feature using client-side streaming.

## Exercise 5: Bidirectional Streaming
Build a real-time chat system using bidirectional streaming.

## Bonus: Add Error Handling
Implement proper error codes and status handling for gRPC calls.
