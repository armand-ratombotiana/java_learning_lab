# Exercises: GitOps

## Easy (1-5)
1. Create Git repository for config
2. Write Kubernetes manifest in YAML
3. Configure ArgoCD application
4. Set up auto-sync policy
5. Create environment branch

## Medium (6-10)
6. Implement multi-environment promotion
7. Add drift detection
8. Configure Git webhook
9. Create rollback mechanism
10. Implement config version tagging

## Hard (11-15)
11. Build GitOps dashboard
12. Create self-service platform
13. Add policy enforcement (OPA)
14. Implement progressive delivery
15. Build multi-cluster orchestration
