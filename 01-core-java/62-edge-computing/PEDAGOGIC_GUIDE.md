# Pedagogic Guide: Edge Computing

## Learning Path
1. **Edge Basics** (1h): Devices, gateways
2. **Deployment** (2h): Java on edge devices
3. **Data** (2h): Local processing, sync
4. **Patterns** (1h): Cloud-edge hybrid

## Assessment
- [ ] Deploy Java to edge device
- [ ] Implement local processing
- [ ] Handle offline scenarios
