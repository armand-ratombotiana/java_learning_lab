# Module 20: Build Tools

Covers Maven and Gradle build tools and automation.

## Topics
- Maven lifecycle and phases
- Essential Maven plugins
- Gradle DSL
- Build automation best practices

## Running the Lab
```bash
cd 20-build-tools
mvn compile exec:java
```

## Key Commands
- `mvn clean compile`
- `mvn test`
- `mvn package`
- `gradle build`
