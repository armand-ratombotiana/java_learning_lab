# API Gateway Exercises

## Exercise 1: Basic Routing
Implement a simple API gateway that routes requests to backend services based on URL paths.

## Exercise 2: Authentication Filter
Add JWT authentication to the gateway with token validation.

## Exercise 3: Rate Limiting
Implement token bucket rate limiting per API key.

## Exercise 4: Request Aggregation
Build a gateway that aggregates product and inventory data from multiple services.

## Exercise 5: Circuit Breaker
Add circuit breaker pattern to handle backend service failures.

## Bonus: Response Caching
Implement caching for frequently accessed Gateway responses.
