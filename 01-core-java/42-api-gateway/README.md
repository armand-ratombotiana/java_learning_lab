# 42 - API Gateway

An API gateway acts as a single entry point for a set of microservices. It handles request routing, composition, authentication, rate limiting, and protocol translation.

## Overview

- **Topic**: API Gateway Implementation
- **Prerequisites**: REST APIs, microservices
- **Duration**: 2-3 hours

## Key Concepts

- Request routing and aggregation
- Authentication and authorization
- Rate limiting and throttling
- Response caching and transformation

## Getting Started

Run the training code:
```bash
cd 42-api-gateway
mvn compile exec:java -Dexec.mainClass=com.learning.apigateway.Lab
```
