# Pedagogic Guide: Spring Boot Basics

## Learning Path
- **Beginner**: Bootstrapping, @SpringBootApplication, dependency injection
- **Intermediate**: External configuration, REST basics, Actuator
- **Advanced**: Auto-configuration, custom starters, profiles

## Key Concepts
| Concept | Description |
|---------|-------------|
| @SpringBootApplication | Main application entry point |
| Spring Boot Starter | Pre-configured dependency sets |
| Auto-configuration | Bean auto-wiring based on classpath |
| Profiles | Environment-specific config |
| Actuator | Production monitoring endpoints |
