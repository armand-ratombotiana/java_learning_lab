# Spring Boot Basics

Core Java module covering Spring Boot fundamentals for building production-ready applications.

## Topics
- **Bootstrapping**: Spring Initializr, `@SpringBootApplication` deep dive
- **Dependency Injection**: Constructor vs Field injection, `@Primary`, `@Qualifier`
- **Component Scanning**: Stereotypes (`@Component`, `@Service`, `@Repository`) and Lifecycles
- **External Configuration**: `application.yml`, `@Value`, `@ConfigurationProperties`
- **Profiles**: Environment-specific beans and configurations
- **REST APIs**: Controllers, Path Variables, Request Bodies, ResponseEntities
- **Actuator**: Health indicators, metrics, and custom production endpoints

## Getting Started
```bash
mvn spring-boot:run
```
