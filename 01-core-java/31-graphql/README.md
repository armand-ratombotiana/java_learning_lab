# 31 - GraphQL

GraphQL is a query language for APIs that provides a complete description of your data and gives clients the power to ask for exactly what they need.

## Overview

- **Topic**: GraphQL API Development
- **Prerequisites**: REST API knowledge, Java fundamentals
- **Duration**: 2-3 hours

## Key Concepts

- Schema Definition Language (SDL)
- Queries, Mutations, Subscriptions
- Resolvers and Data Loaders
- GraphQL Java tools (graphql-java, DGS, SPQR)

## Getting Started

Run the training code:
```bash
cd 31-graphql
mvn compile exec:java -Dexec.mainClass=com.learning.graphql.GraphQLTraining
```

## Module Contents

- GraphQL concepts vs REST

- Query language features
