# GraphQL - Learning Path

## Beginner
- Understand GraphQL vs REST differences
- Learn SDL (Schema Definition Language)
- Write basic queries and mutations
- Understand type system (non-null, lists, scalars)

## Intermediate
- Implement resolvers in Java
- Handle N+1 problems with DataLoader
- Design schemas for real applications
- Add input validation and error handling

## Advanced
- Subscriptions for real-time features
- GraphQL federation for microservices
- Performance optimization and caching
- Security best practices (query complexity, depth limiting)

## Key Concepts
- Single endpoint architecture
- Client-driven data fetching
- Strongly typed schema
- Introspection capabilities

## Dependencies
- graphql-java
- GraphQL SPQR (code generation)
- Netflix DGS (Spring Boot)

## Assessment
- Design a schema for a given domain
- Write queries, mutations for CRUD operations
- Explain N+1 problem and solutions
