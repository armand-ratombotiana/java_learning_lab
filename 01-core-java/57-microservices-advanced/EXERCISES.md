# Exercises: Microservices Advanced

## Easy (1-5)
1. Design order service with orchestration saga
2. Implement compensation for failed step
3. Add saga state tracking
4. Separate read model from write
5. Use Redis for distributed cache

## Medium (6-10)
6. Implement choreograph-based saga
7. Create event store for event sourcing
8. Add snapshotting for event replay
9. Implement read model projection
10. Add eventual consistency handling

## Hard (11-15)
11. Handle saga timeout with monitoring
12. Implement outbox pattern for reliability
13. Add event versioning strategy
14. Create materialized views
15. Implement saga retry with backoff
