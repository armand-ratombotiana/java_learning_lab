# Pedagogic Guide: Serverless

---

## Learning Path

### Phase 1: FaaS Fundamentals (2 hours)
1. Understand serverless vs traditional servers
2. Study Lambda execution model
3. Learn event sources and triggers

### Phase 2: Implementation (3 hours)
1. Create first Lambda with Java
2. Handle API Gateway events
3. Process various event types

### Phase 3: Optimization (3 hours)
1. Measure cold start times
2. Reduce package size
3. Implement best practices

---

## Key Concepts

| Concept | Depth | Priority |
|---------|-------|----------|
| Cold Start | Deep | High |
| Stateless Design | Deep | High |
| Event Sources | Surface | High |
| Cost Model | Surface | Medium |

## Assessment Checklist

- [ ] Understand FaaS vs serverful
