# Pedagogic Guide: GraalVM Native Image

## Learning Path
1. Understand native image concept
2. Build simple native image
3. Configure reflection and resources
4. Optimize for performance
5. Debug native image issues

## Key Concepts
- **Native Image**: Ahead-of-time compilation
- **Substrate VM**: Runtime for native images
- **Reflection**: Configuration for native builds
- **Resources**: Bundle or exclude resources
- **Initialization**: Run vs build time
- **Reachability**: Analysis phase
