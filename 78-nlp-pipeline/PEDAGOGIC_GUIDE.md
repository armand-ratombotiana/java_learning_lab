# Pedagogic Guide: NLP Pipeline

## Learning Path

### Phase 1: Text Processing (Week 1)
- Text normalization techniques
- Tokenization strategies
- Stemming vs lemmatization

### Phase 2: Feature Engineering (Week 1-2)
- Bag of Words model
- TF-IDF weighting
- Word embeddings

### Phase 3: NLP Tools (Week 2-3)
- Stanford CoreNLP
- Pipeline construction
- Entity extraction

### Phase 4: Advanced Tasks (Week 3-4)
- Sentiment analysis
- Text summarization
- Topic modeling

## Key Concepts

| Concept | Description |
|---------|-------------|
| Tokenization | Splitting text into tokens |
| Stemming | Reducing words to root form |
| Lemmatization | Dictionary-based word reduction |
| TF-IDF | Term frequency-inverse document frequency |
| LDA | Latent Dirichlet Allocation |

## Recommended Resources

- "Speech and Language Processing"
- Stanford CoreNLP Documentation
- NLTK Book

- Build complete text processor
