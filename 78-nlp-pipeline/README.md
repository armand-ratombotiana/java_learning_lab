# 78 - NLP Pipeline

A comprehensive module for building end-to-end Natural Language Processing pipelines.

## Overview

This module covers complete NLP pipelines from raw text processing to advanced tasks like sentiment analysis, summarization, and topic modeling.

## Key Topics

- **Text Preprocessing**: Tokenization, stemming, lemmatization
- **Feature Extraction**: Bag of Words, TF-IDF, Word embeddings
- **Stanford CoreNLP**: Complete NLP toolkit
- **Sentiment Analysis**: VADER, CoreNLP sentiment
- **Text Summarization**: Extractive and abstractive
- **Topic Modeling**: LDA, NMF

## Technology Stack

- Stanford CoreNLP
- Apache OpenNLP
- Apache Spark MLlib
- TF-IDF implementations

## Getting Started

```bash
cd 78-nlp-pipeline
mvn compile exec:java
```

## Requirements

- Java 11+
- Maven 3.6+
