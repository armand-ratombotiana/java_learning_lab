## 🧪 Testing Strategy

### Each Module Includes:

**Unit Tests**
```java
@Test
@DisplayName("Should process message successfully")
void shouldProcessMessage() {
    // Given
    Message input = createTestMessage();
    
    // When
    Result result = processor.process(input);
    
    // Then
    assertThat(result).isNotNull();
    assertThat(result.getStatus()).isEqualTo(Status.SUCCESS);
}
```

**Integration Tests**
```java
@SpringBootTest
@Testcontainers
class IntegrationTest {
    
    @Container
    static PostgreSQLContainer<?> postgres = 
        new PostgreSQLContainer<>("postgres:15");
    
    @Test
    void shouldIntegrateWithDatabase() {
        // Test implementation
    }
}
```

**Performance Tests**
```java
@Test
void shouldHandleHighLoad() {
    // JMeter or Gatling integration
    LoadTestResult result = loadTester.run(
        requests: 10000,
        concurrency: 100
    );
    
    assertThat(result.getAverageResponseTime())
        .isLessThan(Duration.ofMillis(100));
}
```

## 🚀 Quick Start for New Modules

### Template Structure

```bash
# Create new module
mkdir -p 16-apache-camel/01-camel-basics
cd 16-apache-camel/01-camel-basics

# Initialize Maven project
mvn archetype:generate \
  -DgroupId=com.learning \
  -DartifactId=camel-basics \
  -DarchetypeArtifactId=maven-archetype-quickstart

# Add Docker support
cat > docker-compose.yml << 'EOF'
version: '3.8'
services:
  app:
    build: .
    ports:
      - "8080:8080"
EOF

# Add Kubernetes manifests
mkdir -p k8s
cat > k8s/deployment.yaml << 'EOF'
apiVersion: apps/v1
kind: Deployment
metadata:
  name: camel-app
spec:
  replicas: 3
  selector:
    matchLabels:
      app: camel-app
  template:
    metadata:
      labels:
        app: camel-app
    spec:
      containers:
      - name: camel-app
        image: camel-app:latest
        ports:
        - containerPort: 8080
EOF
```
