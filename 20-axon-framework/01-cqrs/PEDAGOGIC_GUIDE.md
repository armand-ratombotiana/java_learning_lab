# Pedagogic Guide: Axon Framework

## Learning Path
1. Understand CQRS and Event Sourcing
2. Create aggregates with @Aggregate
3. Define commands and events
4. Implement query handlers
5. Build saga patterns

## Key Concepts
- **CQRS**: Command Query Responsibility Segregation
- **Event Sourcing**: Store events instead of state
- **Aggregate**: Business entity boundaries
- **Commands**: Intent to change state
- **Events**: State changes representation
- **Saga**: Long-running transactions
