# Axon CQRS

Learn Axon Framework for CQRS and Event Sourcing patterns.

## Overview
Axon Framework provides building blocks for CQRS and Event Sourcing architectures. It helps build scalable applications with clear separation between commands and queries.

## Key Features
- CQRS implementation
- Event Sourcing support
- Distributed saga processing
- Spring Boot integration
- Event store integration

## Version
- Axon Framework: 4.9.0
- Java: 21
