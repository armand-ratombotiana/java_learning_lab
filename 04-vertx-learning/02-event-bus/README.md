# Event Bus

Learn Vert.x Event Bus for inter-verticle communication.

## Overview

The Event Bus is the backbone of Vert.x applications, enabling:
- Point-to-point messaging
- Request-response patterns
- Publish-subscribe patterns

## Key Features

- Distributed messaging
- Async message handling
- Multiple message formats

## Source Code

- `EventBusLab.java` - Event bus demonstrations
- `EventBusTraining.java` - Training material

## Building

```bash
mvn compile exec:java
```
