# Exercises - Vert.x Basics

## Exercise 1: Vertx Instance Creation
Create multiple Vert.x instances and explore their configurations.

## Exercise 2: Event Loop Handler
Implement a custom event handler that processes requests asynchronously.

## Exercise 3: HTTP Server Extension
Extend the basic HTTP server to handle different routes and methods.

## Exercise 4: Verticle Deployment
Create and deploy multiple verticles to understand deployment patterns.

## Exercise 5: Configuration Management
Load and apply configuration to the Vert.x instance.
