# Apache Pulsar Basics

Learn Apache Pulsar distributed messaging.

## Overview
Apache Pulsar is a distributed messaging platform that provides pub-sub and queuing capabilities. It offers strong ordering guarantees and multi-tenancy support.

## Key Features
- Pub-sub messaging
- Persistent storage
- Multi-tenancy
- Schema registry
- Pulsar Functions
- Geo-replication

- Pulsar Client: 3.1.0
