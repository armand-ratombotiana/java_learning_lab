# Pedagogic Guide: Apache Pulsar

## Learning Path
1. Understand messaging concepts
2. Create producer/consumer
3. Manage subscriptions
4. Use schemas
5. Implement functions

## Key Concepts
- **Topics**: Message channels
- **Producers**: Message senders
- **Consumers**: Message receivers
- **Subscriptions**: Consumer groups
- **Schemas**: Message structure
- **Functions**: Serverless processing
