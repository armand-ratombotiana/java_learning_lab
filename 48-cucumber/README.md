# Cucumber - BDD Testing

## Overview
Cucumber is a BDD (Behavior-Driven Development) tool that enables teams to write executable specifications in human-readable Gherkin language.

## Key Features
- Gherkin syntax (Given-When-Then)
- Step definitions in Java
- Integration with JUnit/TestNG
- Scenario outlines for data-driven tests

## Project Structure
```
48-cucumber/
  cucumber-bdd/
    src/main/java/com/learning/cucumber/CucumberLab.java
```

## Running
```bash
cd 48-cucumber/cucumber-bdd
mvn compile exec:java
```

## Concepts Covered
- Feature file syntax

- Background and hooks
