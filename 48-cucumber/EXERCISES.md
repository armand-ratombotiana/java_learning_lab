# Exercises - Cucumber

## Exercise 1: Write Feature File
Create a feature file for user registration with valid and invalid scenarios.

## Exercise 2: Implement Step Definitions

## Exercise 3: Scenario Outline

## Exercise 4: Hooks
Add @Before and @After hooks to set up and clean test data.

## Exercise 5: Background
Use Background to define common steps for all scenarios in a feature.
