# Pedagogic Guide: Micrometer

## Learning Path
1. Understand metrics fundamentals
2. Create basic metrics
3. Add timers and gauges
4. Configure exporters
5. Integrate with Spring Boot

## Key Concepts
- **Meter Registry**: Metrics storage
- **Counter**: Cumulative count
- **Gauge**: Point-in-time value
- **Timer**: Duration measurement
- **Tags**: Metric metadata
- **Publishers**: Export destinations
