# Pedagogic Guide: OpenTelemetry

## Learning Path
1. Understand observability: traces, metrics, logs
2. Create manual spans
3. Add auto instrumentation
4. Configure exporters
5. Implement context propagation

## Key Concepts
- **Traces**: Request journey through system
- **Spans**: Individual operations
- **Metrics**: Quantitative measurements
- **Attributes**: Span metadata
- **Exporters**: Data export destinations
- **Context**: Propagated state
