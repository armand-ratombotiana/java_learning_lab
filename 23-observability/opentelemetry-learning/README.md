# OpenTelemetry Learning

Learn OpenTelemetry for distributed tracing and metrics.

## Overview
OpenTelemetry provides vendor-neutral APIs for collecting telemetry data (traces, metrics, logs). It enables observability across distributed systems.

## Key Features
- Distributed tracing
- Custom metrics
- Auto and manual instrumentation
- Multiple exporters
- Context propagation
- Spring Boot integration

## Version
- Spring Boot: Latest
- Java: 21
