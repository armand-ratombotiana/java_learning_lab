# 72 - LangChain4j

A module for building LLM-powered applications using LangChain4j in Java.

## Overview

LangChain4j provides a Java API for working with Large Language Models, enabling chat applications, RAG systems, and AI agents.

## Key Topics

- **Chat Models**: Integration with OpenAI, Anthropic, Ollama
- **Prompt Templates**: Dynamic prompt construction
- **Chat Memory**: Conversation context management
- **Document Processing**: PDF, text parsing and splitting
- **RAG**: Retrieval-Augmented Generation implementation
- **AI Services**: Declarative AI service creation

## Technology Stack

- LangChain4j Core
- OpenAI Client
- Ollama Client
- Document Parsers (PDF, HTML)
- Embedding Models

## Getting Started

```bash
cd 72-langchain4j
mvn compile exec:java
```

## Requirements

- Java 17+
- Maven 3.8+
- API keys for LLM providers (OpenAI, etc.)
