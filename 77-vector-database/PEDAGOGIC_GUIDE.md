# Pedagogic Guide: Vector Databases

## Learning Path

### Phase 1: Embeddings (Week 1)
- Understand word embeddings
- Learn transformer-based embeddings
- Explore embedding models

### Phase 2: Vector Search (Week 1-2)
- Understand similarity metrics
- Learn approximate nearest neighbors
- Implement basic vector search

### Phase 3: Vector Databases (Week 2-3)
- Compare database options
- Learn indexing strategies
- Implement CRUD operations

### Phase 4: RAG Systems (Week 3-4)
- Build retrieval pipeline
- Integrate with LLMs
- Optimize for production

## Key Concepts

| Concept | Description |
|---------|-------------|
| Embedding | Numerical vector representation of data |
| Vector Store | Database optimized for vector storage |
| Cosine Similarity | Measures angle between vectors |
| ANN | Approximate Nearest Neighbor search |
| HNSW | Hierarchical Navigable Small World |
