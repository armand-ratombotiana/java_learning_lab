# Flashcards: Java Basics

## Q1: What are the 8 primitive types in Java?
**A:** byte, short, int, long, float, double, boolean, char

## Q2: What is the default value of an int?
**A:** 0

## Q3: What is the default value of a reference type?
**A:** null

## Q4: What is the difference between `==` and `.equals()`?
**A:** `==` compares references, `.equals()` compares content (for Strings)

## Q5: What is the size of int in Java?
**A:** 32 bits (4 bytes)

## Q6: What is the range of values for a byte?
**A:** -128 to 127

## Q7: What does JVM stand for?
**A:** Java Virtual Machine

## Q8: What is the entry point of a Java program?
**A:** public static void main(String[] args)

## Q9: What is the difference between while and do-while?
**A:** while checks condition first; do-while executes at least once

## Q10: What is an array?
**A:** A fixed-size collection of elements of the same type

## Q11: What is the index of the first element in an array?
**A:** 0

## Q12: What is method overloading?
**A:** Multiple methods with same name but different parameters

## Q13: What is pass-by-value in Java?
**A:** Arguments are copied to parameters; reference values copy the reference

## Q14: What is the default value of a boolean?
**A:** false

## Q15: What is String in Java - primitive or object?
**A:** Object (reference type), not primitive

## Q16: What is the result of 5 / 2 in Java?
**A:** 2 (integer division)

## Q17: What is the result of 5.0 / 2?
**A:** 2.5 (floating point division)

## Q18: What is the increment operator?
**A:** ++ (pre-increment ++i or post-increment i++)

## Q19: What is the modulus operator?
**A:** % (returns remainder of division)

## Q20: What is autoboxing?
**A:** Automatic conversion from primitive to wrapper class
