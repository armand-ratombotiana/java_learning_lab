## Operators Precedence

```
1. () [] .                           (postfix)
2. ++ -- ! ~ unary                   (unary)
3. * / %                             (multiplicative)
4. + -                               (additive)
5. << >> >>>                         (shift)
6. < > <= >= instanceof             (relational)
7. == !=                             (equality)
8. &                                 (bitwise AND)
9. ^                                 (bitwise XOR)
10. |                                 (bitwise OR)
11. &&                                (logical AND)
12. ||                                (logical OR)
13. ? :                               (ternary)
14. = += -= *= /= %=                 (assignment)
```
