## Key Concepts

### Data Types
- **Primitive**: `byte`, `short`, `int`, `long`, `float`, `double`, `boolean`, `char`
- **Reference**: Objects, arrays, strings

### Control Flow
- Conditionals: `if`, `else if`, `else`, `switch`
- Loops: `for`, `while`, `do-while`, enhanced for-each

### Methods
- Declaration, parameters, return types
- Method overloading (same name, different signatures)

### Exception Handling
- `try-catch-finally`, `throw`, `throws`
- Checked vs unchecked exceptions
- Custom exceptions

### Object-Oriented Basics
- Classes and objects
- Constructors
- Access modifiers: `public`, `private`, `protected`, default
- `static` members

## Quick Tips

1. **Use enhanced for-loop** for iterating collections
2. **Prefer StringBuilder** over string concatenation in loops
3. **Always close resources** - use try-with-resources
4. **Use BigDecimal** for precise decimal calculations
