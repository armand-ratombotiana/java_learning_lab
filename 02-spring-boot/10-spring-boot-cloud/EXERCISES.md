# Spring Boot Cloud - Exercises

## Lab Exercises

### Exercise 1: Run Application
```bash
mvn spring-boot:run
```
- Cloud Native features enabled
- Service configuration loaded

### Exercise 2: Cloud Native Patterns
- Identify Spring Cloud features
- Understand cloud integration
- Review configuration sources

### Exercise 3: Service Discovery
- Register with service registry
- Discover other services
- Load balancing

### Exercise 4: Configuration Management
- Externalized configuration
- Environment-specific profiles
- Configuration refresh

### Exercise 5: Resiliency Patterns
- Circuit breaker implementation
- Retry mechanisms
- Fallback strategies

### Exercise 6: Cloud Deployments
- Deploy to cloud platform
- Kubernetes integration (optional)
- Cloud Foundry (optional)

## Build & Run
```bash
mvn clean install
mvn spring-boot:run
```
