# Pedagonic Guide: Javalin

## Learning Path
1. Create simple HTTP server
2. Implement REST endpoints
3. Add request/response handling
4. Integrate templates and static files
5. Add authentication

## Key Concepts
- **Routing**: URL pattern matching
- **Context**: Request/response handling
- **Plugins**: Middleware extension
- **WebSockets**: Real-time communication
- **Templates**: View rendering
