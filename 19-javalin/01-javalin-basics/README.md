# Javalin Basics

Learn Javalin simple and lightweight web framework.

## Overview
Javalin is a lightweight web framework for Java and Kotlin. It provides a simple API for creating REST APIs and web applications without the complexity of larger frameworks.

## Key Features
- Simple and intuitive API
- Kotlin and Java support
- RESTful routing
- WebSocket support
- Template rendering
- Middleware/plugin system
