# Exercises: Records & Sealed Classes

## Basic Exercises

1. **Records**: Create immutable data carriers
   - Define record with fields
   - Use built-in methods (equals, hashCode, toString)

2. **Record Constructors**: Define constructors
   - Use canonical constructor
   - Use compact syntax

3. **Sealed Classes**: Restrict inheritance
   - Declare sealed interface
   - Implement permitted classes

## Intermediate Exercises

4. **Pattern Matching**: Use pattern matching
   - Use instanceof with pattern variable
   - Use switch patterns

5. **Record Methods**: Override methods
   - Add custom methods to record
   - Implement validation in constructor

## Advanced Exercises

6. **Nested Records**: Use nested records
   - Define nested record types
   - Use records in collections
