# Pedagogic Guide: Records & Sealed Classes

## Learning Path

### Phase 1: Foundation (Day 1)
- Understand Java 17+ features
- Learn records for immutable data
- Study sealed classes

### Phase 2: Core Skills (Day 2-3)
- Create records with constructors
- Implement sealed hierarchies
- Use pattern matching

### Phase 3: Advanced (Day 4-5)
- Combine records with sealed classes
- Use pattern matching in switch
- Apply to domain modeling

## Key Concepts

| Concept | Description |
|---------|-------------|
| Record | Immutable data carrier |
| Sealed Class | Restricted inheritance |
| Pattern Matching | Type check + extraction |

## Prerequisites
- Java 17+
- OOP fundamentals

## Common Pitfalls
- Mutable records (don't do it!)
- Forgetting permits clause
- Using records where class needed
