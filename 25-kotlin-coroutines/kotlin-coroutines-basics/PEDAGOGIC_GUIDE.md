# Pedagogic Guide: Kotlin Coroutines

## Learning Path
1. Understand async vs sync programming
2. Create suspend functions
3. Use coroutine builders
4. Implement Flows
5. Handle structured concurrency

## Key Concepts
- **Suspend Functions**: Pausable functions
- **Coroutine Builders**: launch, async, runBlocking
- **Dispatchers**: Thread pools
- **Flow**: Reactive streams
- **Structured Concurrency**: Scope management
- **Coroutines**: Lightweight threads
