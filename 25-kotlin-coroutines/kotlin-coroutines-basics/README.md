# Kotlin Coroutines Basics

Learn Kotlin Coroutines for asynchronous programming.

## Overview
Kotlin Coroutines provide a way to write asynchronous code in a sequential manner. They simplify concurrent operations and enable efficient multitasking.

## Key Features
- Suspend functions
- Coroutine builders (launch, async, Flow)
- Dispatchers for threading
- Structured concurrency
- Flow for reactive streams
- Channel for communication

## Building
```bash
mvn clean compile
mvn exec:java -Dexec.mainClass="kotlin.MainKt"
```

- Kotlin: 1.9.22
