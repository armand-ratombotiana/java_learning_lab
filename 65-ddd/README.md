# Domain-Driven Design

## Overview
DDD is a software development approach that focuses on modeling software around business domains with bounded contexts.

## Key Concepts
- Entities and Value Objects
- Aggregates and Repositories
- Domain Services
- Bounded Contexts
- Ubiquitous Language
