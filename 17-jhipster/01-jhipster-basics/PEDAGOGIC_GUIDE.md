# Pedagogic Guide: JHipster

## Learning Path
1. Install JHipster and generate first app
2. Understand project structure
3. Generate entities with JDL
4. Customize frontend and backend
5. Add security and deployment

## Key Concepts
- **JDL**: JHipster Domain Language for entity definition
- **Yeoman**: Generator system used by JHipster
- **Spring Boot**: Backend framework
- **Angular/React**: Frontend options
- **Microservices**: Registry, gateway, and microservice apps
