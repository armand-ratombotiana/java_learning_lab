# JHipster Basics

Learn JHipster generator for Spring Boot + Angular/React applications.

## Overview
JHipster is a development platform that generates production-ready Spring Boot applications with a modern frontend (Angular or React).

## Key Features
- Rapid application generation
- Spring Boot + Angular/React integration
- Entity generation with JDL
- Microservices support
- Docker/Kubernetes deployment
- JWT authentication

## Version
- Spring Boot: Latest
- Java: 21
