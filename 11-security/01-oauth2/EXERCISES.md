# Exercises: OAuth 2.0

## Basic Exercises

1. **Authorization Code Flow**: Implement complete flow
   - Create authorization endpoint
   - Exchange code for tokens

2. **Client Credentials**: Use client credentials grant
   - Configure client registration
   - Request access token

3. **Password Grant**: Implement resource owner password grant
   - Validate credentials
   - Issue tokens

4. **Token Refresh**: Implement token refresh
   - Use refresh token
   - Obtain new access token

## Intermediate Exercises

5. **OpenID Connect**: Add OIDC to OAuth2
   - Configure OIDC provider
   - Get ID token with access token

6. **JWT Tokens**: Use JWT access tokens
   - Sign tokens with JWT
   - Validate token signature

7. **Token Validation**: Validate tokens
   - Verify token signature
   - Check token claims

## Advanced Exercises

8. ** scopes**: Implement scope-based access
   - Define scopes
   - Check scope in resource server
