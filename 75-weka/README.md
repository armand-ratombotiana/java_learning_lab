# 75 - Weka

A comprehensive module for machine learning using Weka (Waikato Environment for Knowledge Analysis).

## Overview

Weka provides a collection of ML algorithms for data mining tasks including classification, regression, clustering, and association rule mining.

## Key Topics

- **Data Loading**: ARFF, CSV file handling
- **Classifiers**: Decision trees, SVM, Random Forest, Neural Networks
- **Preprocessing**: Filters for normalization, discretization, attribute selection
- **Cross-Validation**: Model evaluation and metrics
- **Clustering**: K-Means, hierarchical clustering
- **Association Rules**: Apriori algorithm

## Technology Stack

- Weka library
- ARFF data format
- Maven dependencies

## Getting Started

```bash
cd 75-weka
mvn compile exec:java
```

## Requirements

- Java 11+
- Maven 3.6+
- Weka data files (.arff, .csv)
