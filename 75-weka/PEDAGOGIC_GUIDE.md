# Pedagogic Guide: Weka

## Learning Path

### Phase 1: ML Fundamentals (Week 1)
- Understand supervised vs unsupervised learning
- Learn about training and testing data
- Explore model evaluation basics

### Phase 2: Weka Basics (Week 1-2)
- Load and explore datasets
- Apply basic classifiers
- Understand attribute types

### Phase 3: Classification (Week 2-3)
- Decision trees (J48)
- Ensemble methods (Random Forest)
- Support Vector Machines
- Neural Networks

### Phase 4: Preprocessing (Week 3-4)
- Data normalization
- Feature discretization
- Attribute selection

### Phase 5: Advanced Mining (Week 4)
- Clustering algorithms
- Association rules
- Feature engineering

## Key Concepts

| Concept | Description |
|---------|-------------|
| ARFF | Weka's attribute-relation file format |
| Cross-Validation | Technique for model evaluation |
| Confusion Matrix | Table showing prediction vs actual |
| Overfitting | Model memorizing training data |
| Ensemble | Combining multiple models |
