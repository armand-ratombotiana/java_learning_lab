# Micronaut Basics

Introduction to Micronaut framework fundamentals.

## Overview

Learn Micronaut fundamentals:
- Application creation
- Dependency injection
- Configuration management

## Source Code

- `MicronautLab.java` - Basic application
- `MicronautBasicsTraining.java` - Training material

## Key Features

- Compile-time DI
- Reactive runtime
- Cloud-native ready

## Building

```bash
mvn compile
mvn mn:run
```
