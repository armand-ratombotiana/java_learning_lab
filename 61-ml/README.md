# Machine Learning with Java

## Overview
Java offers multiple ML libraries including Deeplearning4j, Tribuo, Smile, and Weka for building and deploying ML models.

## Key Features
- Deeplearning4j for deep learning
- Tribuo for structured data
- Smile for statistical ML
- Weka for classic algorithms
- Integration with TensorFlow

## Project Structure
```
61-ml/
  machine-learning/
    src/main/java/com/learning/ml/MachineLearningLab.java
```

## Running
```bash
cd 61-ml/machine-learning
mvn compile exec:java
```

## Concepts Covered
- Linear regression implementation
- Deeplearning4j overview
- Tribuo and Smile
- Model training and prediction
