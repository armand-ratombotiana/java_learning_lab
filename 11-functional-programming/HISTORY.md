# History of Functional Programming

## Origins

### 1930s: Lambda Calculus
Alonzo Church developed lambda calculus—a formal system for computation using functions.

### 1950s-60s: Lisp
John McCarthy created Lisp, first functional programming language, introducing:
- First-class functions
- Recursion
- Garbage collection

### 1970s: Haskell
Pure functional language with lazy evaluation, type system innovations.

### 1980s: ML, OCaml
Functional languages with type inference.

## Java FP Journey

| Version | Feature |
|---------|---------|
| Java 1.0 (1996) | No FP support |
| Java 5 (2004) | Generics |
| Java 8 (2014) | Lambdas, Streams, Default methods |
| Java 9 (2017) | Enhanced streams |
| Java 14+ (2020+) | Records (immutable data) |

## Modern FP Languages

- Scala: JVM functional + OOP
- Kotlin: FP features in pragmatic language
- Clojure: Lisp on JVM
- Elixir: Actor model + FP

## Influences on Java 8

- LINQ (C#)
- Scala collections
- Haskell concepts
