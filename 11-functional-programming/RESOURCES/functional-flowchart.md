## Common Patterns

### Transformation Pipeline
```java
list.stream()
    .filter(x -> x > 0)           // Predicate
    .map(x -> x * 2)              // Function
    .forEach(System.out::println); // Consumer
```

### Reducing/Folding
```java
list.stream()
    .reduce(0, Integer::sum);    // BinaryOperator
```

### Conditional Logic
```java
Predicate<String> isLong = s -> s.length() > 10;
Predicate<String> hasDigit = s -> s.matches(".*\\d.*");

isLong.and(hasDigit);            // Combined predicate
isLong.or(hasDigit.negate());    // Negated
```

### Factory Pattern
```java
Supplier<List<String>> listFactory = ArrayList::new;
List<String> newList = listFactory.get();
```
