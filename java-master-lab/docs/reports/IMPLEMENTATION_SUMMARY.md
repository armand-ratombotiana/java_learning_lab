## 🚀 Getting Started

### Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/java-master-lab.git
cd java-master-lab

# Run verification
./scripts/verify-all.sh --quick

# Start with Lab 00
cd labs/00-environment-setup
cat README.md
```

### For Contributors

```bash
# Create a new lab
cp -r templates/maven-starter labs/02-operators-control-flow
cd labs/02-operators-control-flow

# Edit README.md using LAB_TEMPLATE.md as reference
# Implement the mini-project
# Add tests
# Submit pull request
```
