# Web3 and Blockchain

## Overview
Web3 refers to decentralized applications running on blockchain networks, with Java libraries for Ethereum and smart contract interaction.

## Key Features
- Web3j library for Ethereum
- Smart contract deployment
- Wallet and transaction management
- ERC-20 token standards
- Besu enterprise Ethereum

## Project Structure
```
60-blockchain/
  web3-java/
    src/main/java/com/learning/blockchain/web3/Web3Lab.java
```

## Running
```bash
cd 60-blockchain/web3-java
mvn compile exec:java
```

## Concepts Covered
- Basic blockchain implementation
- Hash and nonce
- Web3j library overview
- Smart contract basics

## Dependencies
- Web3j (Ethereum)
