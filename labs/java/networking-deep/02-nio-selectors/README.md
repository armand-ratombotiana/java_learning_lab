# NIO Selectors
## Overview
Selector, SelectionKey, OP_ACCEPT/OP_READ/OP_WRITE, non-blocking IO, Reactor pattern, single-threaded event loop
## Learning Objectives
- Understand the internal mechanics of NIO Selectors
- Implement working code that demonstrates core concepts
- Analyze performance characteristics and trade-offs
- Apply this knowledge in real-world Java applications
## Prerequisites
- Java 21+
- Basic understanding of Java I/O and networking
- Familiarity with TCP/IP protocol stack
## Lab Structure
This micro-lab follows the standard 24-file pedagogical structure with 7 subdirectories.
## Quick Start
1. Navigate to the source directory: src/main/java/com/javalab/02/
2. Review the main implementation class
3. Run tests using: mvn test or gradle test
4. Complete the exercises in EXERCISES.md
5. Build the MINI_PROJECT to cement understanding
