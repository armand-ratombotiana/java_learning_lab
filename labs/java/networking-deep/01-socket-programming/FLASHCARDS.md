# Socket Programming -- Flashcards
## Spaced-Repetition Cards
Card 1: Q: What port range is reserved for well-known services? A: 0-1023
Card 2: Q: What is the purpose of TCP sequence numbers? A: Ordered delivery and retransmission
Card 3: Q: What does Selector.select() do? A: Blocks until at least one channel is ready
Card 4: Q: What is NIO? A: New I/O - non-blocking I/O API introduced in JDK 1.4
Card 5: Q: What is the Reactor pattern? A: Event demultiplexing and handler dispatch
Card 6: Q: How does HTTP/2 improve over HTTP/1.1? A: Multiplexing, header compression, server push
Card 7: Q: What is Netty's ChannelHandler? A: Component that processes I/O events in a pipeline
Card 8: Q: What is gRPC's transport layer? A: HTTP/2
Card 9: Q: What is a ByteBuf? A: Netty's buffer type supporting reference counting and pooling
Card 10: Q: What is the C10K problem? A: Handling 10,000 concurrent network connections
