# Lab 26: Date and Time API in Java

## Overview

The `java.time` package, introduced in Java 8, provides a comprehensive, immutable, and thread-safe date and time API. Designed by the lead developer of Joda-Time (Stephen Colebourne), it addresses the deficiencies of the legacy `java.util.Date` and `java.util.Calendar` classes.

### Key Concepts Covered

- LocalDate, LocalTime, LocalDateTime for date/time without timezone
- ZonedDateTime and OffsetDateTime for timezone-aware operations
- Instant for machine timestamps
- Duration and Period for date/time amounts
- DateTimeFormatter for formatting and parsing
- TemporalAdjusters for complex date calculations
- Interoperability with legacy code

### Prerequisites

- Java 8+ JDK
- Basic understanding of dates, times, and timezones

### What You'll Learn

By the end of this lab, you'll be able to work confidently with dates and times in Java, perform calculations, handle timezones, format and parse dates, and avoid common pitfalls.
