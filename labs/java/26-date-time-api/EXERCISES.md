# Exercises: Date-Time API

## Exercise 1: Age Calculator
Write a method that calculates age from birth date to today, returning years, months, and days.

## Exercise 2: Meeting Scheduler
Create a meeting scheduler that finds available time slots within business hours (9 AM - 5 PM) on weekdays, excluding holidays.

## Exercise 3: Timezone Converter
Write a utility that converts a given date-time from one timezone to another and displays both in user-friendly format.

## Exercise 4: Recurring Event Generator
Generate dates for recurring events (weekly on Monday and Wednesday, monthly on 15th, yearly on March 15).

## Exercise 5: Duration Calculator
Calculate the exact duration between two ZonedDateTimes, accounting for DST changes.

## Exercise 6: Legacy Code Migration
Take a piece of code using `java.util.Date`/`Calendar` and migrate it to `java.time`.

## Exercise 7: Custom TemporalAdjuster
Create a custom `TemporalAdjuster` for "next business day that is not a holiday."

## Exercise 8: Date Range Utility
Write a utility that generates all dates between two dates, supports inclusive/exclusive boundaries.

## Exercise 9: Format Detection
Write a method that detects the format of a date-time string and parses it correctly.

## Exercise 10: Complete Calendar Application
Build a calendar application with event management, recurrence, timezone support, and conflict detection.
