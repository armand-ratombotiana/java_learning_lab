# Arrays & Strings — Visual Guide

## Array Memory Layout

```
Single-dimensional int[5]:
 ┌──────┬──────┬──────┬──────┬──────┬──────┬──────┐
 │header│length│  [0]  │  [1]  │  [2]  │  [3]  │  [4]  │
 │12-16B│  5   │   0   │   0   │   0   │   0   │   0   │
 └──────┴──────┴──────┴──────┴──────┴──────┴──────┘
```

## 2D Array (int[3][4])

```
         ┌──────────┐
matrix ─→│int[3] ref│     ┌────────────────────────────────┐
         ├──────────┤     │  int[4]                        │
         │    [0]   ├────→│ [0] [1] [2] [3]               │
         ├──────────┤     └────────────────────────────────┘
         │    [1]   ├──┐  ┌────────────────────────────────┐
         ├──────────┤  └─→│  int[4]                        │
         │    [2]   │     │ [0] [1] [2] [3]               │
         └──────────┘     └────────────────────────────────┘
                          ┌────────────────────────────────┐
                          │  int[4]                        │
                          │ [0] [1] [2] [3]               │
                          └────────────────────────────────┘
Total: 4 array objects (1 outer + 3 inner)
```

## String Internals (Java 9+ Compact Strings)

```
String s = "Hello";
┌──────┬──────┬────────────────────────────────────┐
│header│ coder│   byte[] value                      │
│12-16B│  0   │ ['H','e','l','l','o'] (Latin-1)    │
└──────┴──────┴────────────────────────────────────┘

String jp = "こんにちは";
┌──────┬──────┬────────────────────────────────────────────┐
│header│ coder│   byte[] value                              │
│12-16B│  1   │ [0x30,053,0x93,0x93,0x30,08B,...] (UTF-16)│
└──────┴──────┴────────────────────────────────────────────┘
```

## String Pool

```
   ┌───────────────────────────────────────┐
   │          String Pool (Heap)            │
   ├───────────────────────────────────────┤
   │ "hello" ────── String("hello")        │
   │ "world" ────── String("world")        │
   │ "hello" ────── (shared, same ref!)    │
   └───────────────────────────────────────┘
         ↑                    ↑
     String s1 = "hello"    String s2 = "hello"
     s1 == s2 → true
```

## StringBuilder Growth

```
StringBuilder sb = new StringBuilder();  // capacity = 16
┌────────────────────────────────────────────────┐
│ [ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ][ ] │ 16 chars
└────────────────────────────────────────────────┘

sb.append("Hello, World! More text...");
┌────────────────────────────────────────────────────────────────┐
│ [H][e][l][l][o][,][ ][W][o][r][l][d][!][ ][M][o][r][e][t][e] │
└────────────────────────────────────────────────────────────────┘
// capacity exceeded → grow to (16*2)+2 = 34
```
