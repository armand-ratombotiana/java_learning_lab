# Arrays & Strings — Quiz

1. What is the index of the first element in an array? **0**
2. How do you get the length of an array? **arr.length (field, not method)**
3. `String` is mutable or immutable? **Immutable**
4. What is the default value of array elements? **0, null, or false depending on type**
5. What is StringBuilder used for? **Efficient mutable string operations**
6. What does `"hello".length()` return? **5**
7. How do you compare two strings for equality? **s1.equals(s2)**
8. What happens if you access array[-1]? **ArrayIndexOutOfBoundsException**
9. What is the String pool? **Cache of interned String literals**
10. What does `Arrays.toString(arr)` do? **Returns string representation of array**
11. Is StringBuffer synchronized? **Yes**
12. What does `split("\\s+")` do? **Splits string by whitespace**
13. What is a jagged array? **Array of arrays with different lengths**
14. `"abc".charAt(3)` — what happens? **StringIndexOutOfBoundsException**
15. What does `new int[5]` create? **Array of 5 ints (all 0)**
16. Can arrays be resized after creation? **No**
17. What is `String.join(", ", "a", "b")`? **"a, b"**
18. What is a text block? **Multi-line string with """ (Java 15+)**
19. What does `"hello" == new String("hello")` return? **false**
20. What is StringBuilder's default capacity? **16**
