# Arrays & Strings — Overview

Arrays store collections of elements of the same type. Strings represent sequences of characters. Together they form the foundation of data manipulation in Java.

## Learning Objectives

- Declare, initialize, and access single and multi-dimensional arrays
- Understand the String pool and immutability
- Use StringBuilder and StringBuffer for mutable strings
- Apply common array and string algorithms
- Choose between String, StringBuilder, and StringBuffer

## Prerequisites

- Data Types lab completed
- Methods lab completed

- **Theory**: 50 minutes
- **Practice**: 60 minutes
- **Exercises**: 40 minutes
