# Arrays & Strings — Flashcards

1. **Q: Array first index?** A: 0
2. **Q: Array length property?** A: arr.length (field)
3. **Q: String immutability?** A: Strings cannot be changed after creation
4. **Q: StringBuilder vs StringBuffer?** A: StringBuilder = unsynchronized (faster)
5. **Q: String pool location?** A: Heap (Java 7+)
6. **Q: Default array values?** A: 0, 0.0, false, '\u0000', null
7. **Q: Arrays.copyOf()?** A: Copies array to new length
8. **Q: Multi-dimensional array?** A: Array of arrays
9. **Q: Jagged array?** A: Array of arrays with different lengths
10. **Q: ArrayIndexOutOfBoundsException?** A: Thrown for invalid index
11. **Q: `==` vs `.equals()` for String?** A: == checks reference, equals checks content
12. **Q: Compact Strings (Java 9+)?** A: byte[] instead of char[]
13. **Q: Text blocks?** A: Multi-line string with """ (Java 15+)
14. **Q: String::isBlank (Java 11+)?** A: Checks if empty/whitespace
15. **Q: String::repeat (Java 11+)?** A: Repeats string n times
16. **Q: StringBuilder default capacity?** A: 16
17. **Q: StringBuilder growth?** A: (old * 2) + 2
18. **Q: Arrays.binarySearch requires?** A: Sorted array
19. **Q: String.intern()?** A: Returns canonical String pool reference
20. **Q: Split returns?** A: String[] (regex delimiter)
