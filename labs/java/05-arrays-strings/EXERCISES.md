# Arrays & Strings — Exercises

## Exercise 1: Array Stats
Write a program that creates an array of 10 random integers and prints min, max, average, sum.

## Exercise 2: Reverse Array
Reverse an array in-place without using a second array.

## Exercise 3: Matrix Multiplication
Multiply two 3×3 matrices and print the result.

## Exercise 4: Anagram Checker
Check if two strings are anagrams (same characters, different order).

## Exercise 5: Character Frequency
Count frequency of each character in a string.

## Exercise 6: Palindrome Checker
Check if a string is a palindrome (same forwards and backwards).

## Exercise 7: Remove Duplicates
Remove duplicate characters from a string.

## Exercise 8: Caesar Cipher
Implement a Caesar cipher: shift each letter by a fixed amount.

## Exercise 9: String Compression
Compress "aaabbbcc" → "a3b3c2". If compressed isn't shorter, return original.

## Exercise 10: Spiral Matrix
Print a 2D array in spiral order (clockwise from top-left).

## Exercise 11: StringBuilder vs String
Compare the time to concatenate 100,000 strings using `+` vs StringBuilder.

## Exercise 12: Text Block Formatting
Create a formatted invoice using a text block. Include variable substitution with `.formatted()`.
