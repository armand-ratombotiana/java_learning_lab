# History — Reflection in Java

- **1996:** Java 1.0 — Core reflection (Class, Method, Field) in `java.lang.reflect`
- **1999:** Java 1.2 — Reflection enhancements, `Proxy` class
- **2004:** Java 5 — Annotations, `Type` interface for generics, `ParameterizedType` etc.
- **2006:** Java 6 — `MethodHandle` preview (JSR 292)
- **2011:** Java 7 — `MethodHandle` GA, `invokedynamic` instruction
- **2014:** Java 8 — `Parameter` class, `Executable.getParameters()`
- **2017:** Java 9 — Module system restricts reflection (`--add-opens`)
- **2021:** Java 17 — Strong encapsulation via `--illegal-access=deny` (default)

Key figure: **John Rose** — `MethodHandle`, `invokedynamic`.
