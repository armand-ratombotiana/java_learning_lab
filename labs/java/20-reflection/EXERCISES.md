# Exercises — Reflection

## Beginner
1. Print all methods (name + return type) of `java.util.List`.
2. Create an instance of `ArrayList` using reflection.
3. Get and print the value of a private field in a class.

## Intermediate
4. Write a generic `toString()` method using reflection (prints all field names/values).
5. Implement a simple JUnit-like test runner: discover `@Test` methods, invoke them, report pass/fail.
6. Use `Proxy` to create a timing wrapper for an interface.

## Advanced
7. Build a minimal DI container:
   - Scan for `@Inject` on constructors/fields
   - Resolve dependencies recursively
   - Return fully wired instances
8. Implement a `@Retry` annotation + proxy that retries a method on exception.
9. Write a reflective ORM that maps `@Column`-annotated fields to/from JDBC ResultSet.

## Debugging
10. Enable proxy class file dumping and examine the generated bytecode.
11. Compare performance: direct call vs `Method.invoke()` vs `MethodHandle` for 1M invocations.

## Reflection
12. Use `MethodHandles.Lookup` to access a private method and compare with `setAccessible`.
