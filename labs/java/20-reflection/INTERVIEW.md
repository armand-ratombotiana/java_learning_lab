# Interview Questions — Reflection

## Beginner
1. What is reflection? When is it used?
2. How do you obtain a `Class` object in Java?
3. What is the difference between `getMethod()` and `getDeclaredMethod()`?

## Intermediate
4. How do you invoke a private method using reflection?
5. What is `InvocationTargetException` and how do you handle it?
6. How do dynamic proxies work? What is `InvocationHandler`?
7. What is the difference between reflection and `MethodHandle`?

## Advanced
8. Explain the performance characteristics of reflection. How can you optimise it?
9. How does the Java module system (Java 9+) affect reflection?
10. How would you design a simple DI container using reflection?
11. What is the `ProxyGenerator` and how are proxy classes created?

## Problem-Solving
12. "Implement a `@Cached` annotation with a reflective caching proxy."
13. "Our legacy system uses reflection for plugin loading. How would you modernise it?"
14. "A method invoked via reflection throws `InvocationTargetException` — how do you extract the original exception?"
15. "We get `IllegalAccessException` on Java 17. What changed and how do we fix it?"
