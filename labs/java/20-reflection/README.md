# Lab 20: Reflection

**Objective:** Master Java Reflection API — inspecting classes, methods, fields, invoking methods dynamically, and creating proxies.

**Key Concepts:**
- `Class` objects and class loading
- `Method`, `Field`, `Constructor` reflection
- Dynamic method invocation
- Accessible object handling (`setAccessible`)
- `java.lang.reflect.Proxy` and dynamic proxies

**Prerequisites:** Java 8+, annotations (lab 19)

**Estimated time:** 120 minutes
