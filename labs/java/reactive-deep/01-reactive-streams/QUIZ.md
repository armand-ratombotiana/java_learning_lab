# Reactive Streams -- Quiz
## Question 1: What is a Publisher?
A: Source of data in reactive streams
## Question 2: What is backpressure?
A: Flow control mechanism
## Question 3: Difference between Flux and Mono?
A: Flux is 0-N items, Mono is 0-1 items
## Question 4: What is flatMap?
A: Maps each item to a Publisher and flattens
## Question 5: What is StepVerifier used for?
A: Testing reactive streams
