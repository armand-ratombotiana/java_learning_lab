# Reactive Streams
## Overview
Publisher/Subscriber/Subscription/Processor, backpressure, request(n), onSubscribe/onNext/onError/onComplete contracts
## Learning Objectives
- Understand Reactive Streams
- Implement reactive solutions
- Apply reactive patterns in real projects
## Prerequisites
- Java 21+
- Basic understanding of reactive programming concepts
## Quick Start
1. Navigate to src/main/java/com/javalab/01/
2. Review the main implementation
3. Run tests: mvn test or gradle test
