# Project Reactor -- Step-by-Step
## Step 1: Add Reactor dependency
Add reactor-core and reactor-test to build.
## Step 2: Create Publisher
Use Flux.just(), Flux.range(), or Mono.just().
## Step 3: Apply operators
Chain map, filter, flatMap as needed.
## Step 4: Subscribe
Add subscriber with error and completion handlers.
## Step 5: Test
Use StepVerifier to verify the reactive chain.
