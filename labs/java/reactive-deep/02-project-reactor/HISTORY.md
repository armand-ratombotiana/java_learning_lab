# Project Reactor -- History
## Reactive Evolution
### Reactive Extensions (.NET 2009)
Rx.NET introduced Observable pattern for .NET.
### RxJava (2013)
Netflix ported Rx to Java as RxJava 1.
### Reactive Streams (2015)
Standard specification for reactive libraries.
### Project Reactor (2016)
Pivotal's reactive library for Spring.
### RSocket (2017)
Binary protocol for reactive communication.
