# Project Reactor
## Overview
Flux/Mono, map/flatMap/concatMap, publishOn/subscribeOn, Schedulers, backpressure strategies, hot vs cold publishers
## Learning Objectives
- Understand Project Reactor
- Implement reactive solutions
- Apply reactive patterns in real projects
## Prerequisites
- Java 21+
- Basic understanding of reactive programming concepts
## Quick Start
1. Navigate to src/main/java/com/javalab/02/
2. Review the main implementation
3. Run tests: mvn test or gradle test
## Time Estimate
- Total: 2-3 hours
