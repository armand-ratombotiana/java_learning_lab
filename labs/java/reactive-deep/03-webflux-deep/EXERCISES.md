# WebFlux Deep Dive -- Exercises
## Beginner Exercises
### Exercise 1: Create a simple Flux
Create a Flux that emits 1 to 10 and subscribe to print each item.
### Exercise 2: Transform data
Use map() to transform strings to uppercase.
## Intermediate Exercises
### Exercise 3: Backpressure
Implement a custom Subscriber that requests items one at a time.
### Exercise 4: Error handling
Use onErrorResume, onErrorReturn, retry operators.
## Advanced Exercises
### Exercise 5: WebClient
Use WebClient to make reactive HTTP calls and process responses.
### Exercise 6: StepVerifier
Write StepVerifier tests for complex reactive chains.
