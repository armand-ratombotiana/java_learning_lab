# RSocket
## Overview
RSocket interaction models (request/response, fire-and-forget, streaming, channel), setup payload, routing, resumability
## Learning Objectives
- Understand RSocket
- Implement reactive solutions
- Apply reactive patterns in real projects
## Prerequisites
- Java 21+
- Basic understanding of reactive programming concepts
## Quick Start
1. Navigate to src/main/java/com/javalab/04/
2. Review the main implementation
3. Run tests: mvn test or gradle test
## Time Estimate
- Total: 2-3 hours
