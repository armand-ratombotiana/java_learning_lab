# Control Flow — Overview

Control flow statements determine the order in which individual statements, instructions, or function calls are executed. Java provides selection statements (if, switch) and iteration statements (for, while, do-while).

## Learning Objectives

- Write branching logic with if-else and switch
- Choose appropriate loop constructs for different scenarios
- Understand break, continue, and labeled statements
- Use switch expressions for concise branching
- Avoid common control flow pitfalls

## Prerequisites

- Java Syntax lab completed
- Basic understanding of boolean expressions

## Estimated Time

- **Theory**: 45 minutes
- **Practice**: 60 minutes
- **Exercises**: 45 minutes
- **Total**: 2–3 hours
