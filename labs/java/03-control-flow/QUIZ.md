# Control Flow — Quiz

1. What is the output of `if (false) { ... }`? **The block never executes**
2. Can you switch on a `String`? **Yes (Java 7+)**
3. What does `break` do in a switch? **Exits the switch block**
4. How many times does a `do-while` loop execute minimum? **Once**
5. What does `continue` do? **Skips to next iteration**
6. `for (;;) { }` — what is this? **Infinite loop**
7. What does `||` do? **Logical OR (short-circuit)**
8. Can you switch on a `long`? **No**
9. What does `break outer;` do? **Exits labeled loop**
10. `5 + 3 * 2` evaluates to? **11 (not 16)**
11. What is the output of `switch(1) { case 1: case 2: print("hi"); }`? **hi (fall-through)**
12. A switch expression must be? **Exhaustive**
13. What does `-Xlint:fallthrough` do? **Warns about fall-through**
14. `while (true) { break; }` — how many iterations? **One**
15. What is an enhanced for-loop? **For-each loop**
16. Can you modify a collection while iterating with for-each? **No (ConcurrentModificationException)**
17. What is a labeled statement? **A statement with an identifier for break/continue**
18. Does `if (x = 5)` compile? **No (Java)**
19. What operator has highest precedence? **Postfix (variable++, variable--)**
20. What is the result of `true && false || true`? **true**
