## break and continue

```
for (i = 0; i < 10; i++) {
    в”Ӯ
    в”ңв”Җв”Җ if (i == 3) continue; в”Җв”Җв”ҖвҶ’ skip to i++ (next iteration)
    в”Ӯ
    в”ңв”Җв”Җ if (i == 7) break; в”Җв”Җв”Җв”Җв”Җв”ҖвҶ’ exit loop entirely
    в”Ӯ
    в””в”Җв”Җ System.out.println(i);
}
// Output: 0, 1, 2, 4, 5, 6
```
