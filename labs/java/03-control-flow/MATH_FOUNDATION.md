# Math Foundation for Control Flow

No specific math foundation required. Control flow concepts rely on boolean logic:

- `true` and `false` values
- AND (`&&`), OR (`||`), NOT (`!`) operations
- Comparison operators (`<`, `>`, `<=`, `>=`, `==`, `!=`)

For loop conditions, understanding arithmetic comparisons is sufficient. Basic understanding of iteration and counting (increment by 1, check if less than N) is all that's necessary.

For `switch` expressions, no math beyond basic value matching is required.
