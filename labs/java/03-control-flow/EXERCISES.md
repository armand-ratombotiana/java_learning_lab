# Control Flow — Exercises

## Exercise 1: FizzBuzz
Print numbers 1-100. For multiples of 3 print "Fizz", multiples of 5 print "Buzz", both print "FizzBuzz".

## Exercise 2: Prime Checker
Check if a number is prime. Use a for loop up to sqrt(n). Optimize by checking 2 separately, then odd numbers only.

## Exercise 3: Multiplication Table
Print a 12×12 multiplication table with proper alignment.

## Exercise 4: Days in Month
Use a switch expression to return days in a month (handle leap years).

## Exercise 5: Number Pattern
Print this pattern using nested loops:
```
1
22
333
4444
55555
```

## Exercise 6: Collatz Conjecture
Start with any positive integer n. If even, n = n/2. If odd, n = 3n+1. Print the sequence until reaching 1.

## Exercise 7: Menu System
Create a text-based menu with do-while loop. Options: 1. Add, 2. Subtract, 3. Show result, 4. Exit.

## Exercise 8: Find All Factors
Print all factors of a given number.

## Exercise 9: GCD with Euclid
Implement Euclid's GCD algorithm using a while loop.

## Exercise 10: Word Counter
Read a sentence and count words, characters (without spaces), and vowels using control flow.

## Exercise 11: Guess the Number
Random number 1-100. User guesses with hints (higher/lower). Count attempts.

## Exercise 12: Labeled Break Practice
Find the first 5x5 submatrix with all positive values in a larger matrix. Use labeled break.
