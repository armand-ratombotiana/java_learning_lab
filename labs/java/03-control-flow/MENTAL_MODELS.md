# Control Flow — Mental Models

## Model 1: The Road Intersection

Execution is a car driving down a road. `if-else` is a fork in the road. `switch` is a roundabout with multiple exits. `for` is a circular track with a lap counter. `while` is a guard gate that checks before letting you pass. `do-while` is a guard gate at the exit — you pass through once before being checked.

## Model 2: The Kitchen Timer

Loops are kitchen timers. A `for` loop pre-sets the time: "bake for 30 minutes." A `while` loop checks a condition: "while (not golden brown) keep baking." A `do-while` loop bakes first, then checks: "taste, then decide if more needed."

## Model 3: The Choose-Your-Own-Adventure Book

`if-else` is a book where "if you go left, turn to page 42; if you go right, turn to page 87." The `switch` statement is a numbered menu — jump directly to chapter 3. `break` prevents reading into the next chapter accidentally.

## Model 4: The Conveyor Belt

A for-each loop is a conveyor belt. Each item passes by, gets processed, and moves on. `break` stops the entire belt. `continue` skips an item but keeps the belt running. Labels are like telling the belt to stop at a specific station, not the nearest one.
