# Java Syntax — References

## Official Documentation

- [Java Language Specification (JLS)](https://docs.oracle.com/javase/specs/jls/se21/html/) — The definitive reference for Java syntax. Chapter 3 (Lexical Structure) and Chapter 4 (Types, Values, and Variables) are particularly relevant.
- [Oracle Java Tutorials — Language Basics](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/)
- [OpenJDK Source Code — javac Parser](https://github.com/openjdk/jdk/tree/master/src/jdk.compiler/share/classes/com/sun/tools/javac/parser)

## Books

- *Java: The Complete Reference* — Herbert Schildt (Comprehensive syntax coverage)
- *Core Java, Volume I* — Cay S. Horstmann (Excellent explanations with examples)
- *Effective Java* — Joshua Bloch (Item-level best practices for syntax usage)
- *The Java Programming Language* — Ken Arnold, James Gosling, David Holmes (Written by Java's creator)
- *Java Precisely* — Peter Sestoft (Concise syntax reference)

## Java Enhancement Proposals (JEPs) for Syntax

- [JEP 286: Local-Variable Type Inference (var)](https://openjdk.org/jeps/286) — Java 10
- [JEP 354: Switch Expressions](https://openjdk.org/jeps/354) — Java 14 (standard in Java 17)
- [JEP 355: Text Blocks](https://openjdk.org/jeps/355) — Java 15
- [JEP 359: Records](https://openjdk.org/jeps/359) — Java 16
- [JEP 394: Pattern Matching for instanceof](https://openjdk.org/jeps/394) — Java 16
- [JEP 440: Record Patterns](https://openjdk.org/jeps/440) — Java 21

- [Google Java Style Guide](https://google.github.io/styleguide/javaguide.html) — Industry standard for Java code formatting
- [Checkstyle](https://checkstyle.org/) — Static analysis tool for syntax conventions
- [PMD](https://pmd.github.io/) — Source code analyzer
- [IntelliJ IDEA](https://www.jetbrains.com/idea/) — IDE with real-time syntax checking
- [javap](https://docs.oracle.com/en/java/javase/21/docs/specs/man/javap.html) — Class file disassembler (included in JDK)
