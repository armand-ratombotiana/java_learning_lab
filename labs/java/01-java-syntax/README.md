# Java Syntax — Overview

Java syntax encompasses the set of rules that define how Java programs are written and interpreted. It governs everything from how classes are declared to how statements are terminated.

## Learning Objectives

- Write syntactically correct Java source files
- Understand Java's lexical structure: whitespace, tokens, comments, and keywords
- Recognize and use Java keywords, operators, and separators
- Apply standard Java naming conventions
- Use primitive literals and reference type literals
- Structure a Java program with packages, imports, classes, and methods

## Prerequisites

- Basic understanding of programming concepts (variables, loops, functions)
- A Java Development Kit (JDK) 21+ installed
- A text editor or IDE (VS Code, IntelliJ IDEA, or Eclipse)

## Estimated Time

- **Theory**: 45 minutes
- **Practice**: 60 minutes
- **Exercises**: 45 minutes
- **Total**: 2–3 hours
