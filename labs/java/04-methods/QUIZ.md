# Methods — Quiz

1. What is a method signature? **Name + parameter types**
2. Is return type part of method signature? **No**
3. What does varargs `int...` compile to? **int[]**
4. Can varargs be the first parameter? **No (must be last)**
5. What does `static` mean? **Belongs to the class, not instance**
6. Java uses pass-by-value or pass-by-reference? **Pass-by-value**
7. Can a method have multiple varargs parameters? **No**
8. What is method overloading? **Same name, different parameters (compile-time)**
9. What is recursion? **A method calling itself**
10. What causes StackOverflowError? **Too many recursive calls or infinite recursion**
11. Can a constructor be private? **Yes (singleton pattern)**
12. What is the default constructor? **No-arg constructor provided by compiler**
13. Can an abstract method have a body? **No**
14. What is a void method? **Method that returns nothing**
15. Can you call a method before it's defined? **Yes (Java doesn't require forward declaration)**
16. What does `return` do? **Exits method and optionally returns a value**
17. Can you have multiple methods with same name and params? **No**
18. What is a static method? **Method callable without instance**
19. Can a static method access instance variables? **No**
20. What is the `main` method signature? **public static void main(String[])**
