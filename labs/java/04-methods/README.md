# Methods — Overview

Methods define executable behavior in Java. They encapsulate a sequence of statements under a name, accept parameters, return values, and can be reused across an application.

## Learning Objectives

- Declare methods with correct syntax and access modifiers
- Understand pass-by-value semantics
- Implement method overloading
- Use varargs for flexible parameter lists
- Apply recursion to solve problems

## Prerequisites

- Control Flow lab completed
- Basic understanding of scope and variables

## Estimated Time

- **Theory**: 40 minutes
- **Practice**: 60 minutes
- **Exercises**: 50 minutes
- **Total**: 2–3 hours
