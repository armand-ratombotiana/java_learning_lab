# Methods — Exercises

## Exercise 1: Method Writing
Write methods for: `max(int, int)`, `min(int, int)`, `abs(int)`, `isEven(int)`, `isPositive(int)`.

## Exercise 2: Temperature Converter
Write methods `celsiusToFahrenheit(double)` and `fahrenheitToCelsius(double)`.

## Exercise 3: Method Overloading
Overload a `mean` method that works with int[], double[], and int.

## Exercise 4: Varargs Average
Write a method `average(double... numbers)` that returns the average.

## Exercise 5: Recursive Power
Implement `power(base, exp)` recursively.

## Exercise 6: Palindrome Checker
Write a recursive method to check if a string is a palindrome.

## Exercise 7: Array Utilities
Write a utility class `ArrayUtils` with static methods: `reverse(int[])`, `findMax(int[])`, `isSorted(int[])`.

## Exercise 8: Method as Parameter
Write a method `repeat(int n, Runnable action)` that runs the action n times.

## Exercise 9: Password Validator
Write a method that validates: min 8 chars, at least 1 digit, 1 uppercase, 1 lowercase, 1 special char.

## Exercise 10: Percentage Calculator
Write overloaded methods for percentage calculations: `percentOf(double, double)`, `percentChange(double, double)`.

## Exercise 11: GCD (Recursive)
Implement GCD using Euclid's algorithm recursively.

## Exercise 12: String Reverser
Write `reverse(String)` recursively (without StringBuilder).
