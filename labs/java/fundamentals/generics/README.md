# Generics and Type Erasure

Welcome to the atomic mastery lab for **Generics**. This lab is part of the Java Academy's Fundamentals module.

## 🧠 What You Will Master
- The need for compile-time type safety.
- Type Erasure and why it was chosen for backward compatibility.
- Bounded Wildcards (Upper vs Lower).
- The PECS (Producer Extends, Consumer Super) principle.
- Bridge Methods and Heap Pollution.
