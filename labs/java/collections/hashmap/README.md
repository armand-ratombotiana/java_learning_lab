# HashMap Deep Dive

Welcome to the atomic mastery lab for `HashMap` in Java. This lab is part of the Universal Engineering Master Lab and deconstructs `HashMap` into its absolute primitives.

## 🧠 What You Will Master
- Hash functions and the `hashCode()` contract.
- Bucket design and memory layout.
- Collision resolution (Chaining vs. Probing).
- Load factor and the resizing/rehashing process.
- Treeification (how bins convert to Red-Black trees).
- Concurrency issues and why `HashMap` is not thread-safe.
