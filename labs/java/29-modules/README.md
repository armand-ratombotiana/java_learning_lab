# Lab 29: Java Module System (JPMS)

## Overview
The Java Platform Module System (JPMS), introduced in Java 9 via Project Jigsaw, provides a module system for Java. Modules define explicit dependencies, control package visibility, and enable reliable configuration and strong encapsulation.

## Key Concepts
- module-info.java declaration
- exports, requires, opens directives
- ServiceLoader and provides/uses
- Module accessibility and reflection
- Module migration strategies
- Modular JARs and module path
- Transitive dependencies and qualified exports
- Open modules and open packages
- Module resolution and configuration
