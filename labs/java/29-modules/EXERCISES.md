# Exercises: Modules

## Exercise 1: Create a Simple Module
Create a module with a single exported package. Write another module that uses it.

## Exercise 2: Service Provider
Create a service interface in one module, an implementation in another, and a consumer in a third.

## Exercise 3: Transitive Dependencies
Create a module hierarchy demonstrating requires transitive.

## Exercise 4: Open Packages for Reflection
Create a module with a package opened for reflection. Use a simple framework-like class to access private fields via reflection.

## Exercise 5: jlink Custom Runtime
Create a custom runtime image with jlink for your application module.

## Exercise 6: Migration
Take a non-modular library and add module-info.java to it.

## Exercise 7: Qualified Exports
Create a module that exports a package only to specific modules.

## Exercise 8: Module Resolution Debugging
Use --show-module-resolution to understand how modules are resolved.
