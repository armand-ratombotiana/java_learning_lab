# Abstraction & Interfaces — Overview

Abstraction hides implementation details and exposes only essential features. Java supports abstraction through abstract classes and interfaces, enabling loose coupling and contract-based design.

## Learning Objectives

- Create and extend abstract classes
- Define and implement interfaces
- Use default and static methods in interfaces
- Choose between abstract classes and interfaces
- Apply functional interfaces with lambdas

## Prerequisites

- Polymorphism lab completed
- Solid understanding of inheritance

## Estimated Time

- **Theory**: 60 minutes
- **Practice**: 75 minutes
- **Exercises**: 50 minutes
- **Total**: 3–4 hours
