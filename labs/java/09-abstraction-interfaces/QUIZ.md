# Abstraction & Interfaces — Quiz

1. Can you instantiate an abstract class? **No**
2. What keyword declares an abstract class? **abstract**
3. Can abstract class have constructors? **Yes**
4. What is an interface? **Contract defining abstract methods**
5. What keyword implements an interface? **implements**
6. Can an interface have fields? **Only public static final**
7. When were default methods added? **Java 8**
8. What is a functional interface? **Interface with exactly one abstract method**
9. What annotation marks a functional interface? **@FunctionalInterface**
10. Can an interface extend another interface? **Yes**
11. What is the diamond problem with default methods? **Two interfaces define same default method**
12. When to use abstract class vs interface? **Abstract: shared state; Interface: contract**
13. What is a sealed interface? **Restricts which classes implement it (Java 17+)**
14. Can an interface have private methods? **Yes (Java 9+)**
15. What is the Adapter pattern? **Wrap one interface with another**
16. What is the Template Method pattern? **Abstract class defines algorithm skeleton**
17. Can a class implement multiple interfaces? **Yes**
18. What is a marker interface? **Interface with no methods (e.g., Serializable)**
19. What is the Dependency Inversion Principle? **Depend on abstractions, not concretions**
20. What is lambda expression? **Concise syntax for functional interface implementation**
