# Inheritance — Overview

Inheritance establishes an IS-A relationship between classes. A subclass inherits fields and methods from a superclass, enabling code reuse and establishing type hierarchies.

## Learning Objectives

- Create subclasses using the `extends` keyword
- Use `super` to access superclass members
- Override methods correctly with `@Override`
- Prevent inheritance with `final` classes and methods
- Understand the Object class and its methods

## Prerequisites

- OOP Basics lab completed
- Understanding of access modifiers
