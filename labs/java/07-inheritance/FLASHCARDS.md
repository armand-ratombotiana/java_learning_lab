# Inheritance — Flashcards

1. **Q: extends keyword?** A: Indicates class inherits from another
2. **Q: Single inheritance?** A: A class can extend only one superclass
3. **Q: Object class?** A: Root of class hierarchy
4. **Q: super keyword?** A: Reference to superclass
5. **Q: super() call?** A: Must be first statement in constructor
6. **Q: Method overriding?** A: Subclass replaces superclass method
7. **Q: @Override annotation?** A: Ensures method actually overrides
8. **Q: final class?** A: Cannot be extended
9. **Q: final method?** A: Cannot be overridden
10. **Q: Covariant return type?** A: Override returns subtype (Java 5+)
11. **Q: equals/hashCode contract?** A: Equal objects → equal hash codes
12. **Q: toString()?** A: String representation of object
13. **Q: clone()?** A: Creates and returns object copy
14. **Q: instanceof operator?** A: Checks object's type
15. **Q: Bridge method?** A: Compiler-generated for generics type safety
16. **Q: Constructor chaining?** A: Super→sub constructor called in order
17. **Q: Composition vs inheritance?** A: HAS-A vs IS-A; prefer composition
18. **Q: Vtable?** A: Virtual method table for dispatch
19. **Q: Fragile base class problem?** A: Changes to base break subclasses
20. **Q: Liskov Substitution?** A: Subtypes must be replaceable for supertypes
