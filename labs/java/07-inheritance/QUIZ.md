# Inheritance — Quiz

1. What keyword is used for inheritance? **extends**
2. Does Java support multiple class inheritance? **No**
3. What is the root class of all Java classes? **Object**
4. What is method overriding? **Subclass provides specific implementation**
5. What annotation should be used for overriding? **@Override**
6. What does `super` do? **Refers to superclass**
7. Can a constructor be inherited? **No**
8. What is a final class? **Cannot be subclassed**
9. What is a final method? **Cannot be overridden**
10. What happens if you omit `super()` in constructor? **Compiler adds it implicitly**
11. Can you override a static method? **No (it's hidden, not overridden)**
12. What is the Liskov Substitution Principle? **Subtype should be substitutable for supertype**
13. What is the fragile base class problem? **Changes to base class break subclasses**
14. What is an IS-A relationship? **Inheritance relationship**
15. What is a HAS-A relationship? **Composition relationship**
16. What does `getClass()` return? **Runtime Class object**
17. What is the contract between equals and hashCode? **Equal objects must have equal hash codes**
18. Can you extend a final class? **No**
19. What is the diamond problem? **Ambiguity from multiple inheritance (solved in interfaces)**
20. What is composition? **Using object references instead of inheritance**
