# Streams — Overview

The Stream API provides a functional approach to processing sequences of elements. Streams support declarative operations (filter, map, reduce) that can run sequentially or in parallel, enabling concise and efficient data processing pipelines.

## Learning Objectives

- Create streams from collections, arrays, and generators
- Use intermediate operations: filter, map, flatMap, distinct, sorted, peek
- Use terminal operations: collect, reduce, forEach, count, anyMatch, findFirst
- Build custom collectors
- Understand parallel stream execution and pitfalls

- Collections lab completed
- Lambdas lab completed
