# Mini Project

This directory contains the Mini Project for Lab 48: Structured Concurrency & Scoped Values.

## Purpose

The Mini Project is designed to provide hands-on experience with the concepts covered in this lab.

## How to Use

1. Read the lab materials (THEORY.md, HOW_IT_WORKS.md, etc.)
2. Complete the exercises in the main lab directory
3. Use this directory for extended practice and exploration

## Prerequisites

- Completion of all exercises in the main lab
- Understanding of the core concepts covered in THEORY.md
- Java 21+ runtime environment

## Expected Outcome

By completing this Mini Project, you will reinforce your understanding of Structured Concurrency & Scoped Values concepts and gain practical experience applying them in realistic scenarios.
