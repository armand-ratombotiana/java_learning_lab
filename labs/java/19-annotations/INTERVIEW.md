# Interview Questions — Annotations

## Beginner
1. What are annotations in Java? How are they different from comments?
2. Name four built-in annotations.
3. What is the purpose of `@Override`?

## Intermediate
4. How do you create a custom annotation? Explain `@Retention` and `@Target`.
5. How do you read annotations at runtime?
6. What is the difference between SOURCE, CLASS, and RUNTIME retention policies?
7. How do repeatable annotations work?

## Advanced
8. Explain how annotation processing works in javac.
9. How does Spring use annotations for dependency injection?
10. How does Lombok use annotations to generate code?
11. What is the `@Inherited` meta-annotation? What are its limitations?

## Problem-Solving
12. "Design a validation framework using annotations (`@NotNull`, `@Min`, `@Max`)."
13. "Our team wants to move from XML-based configuration to annotation-based. Outline the steps."
14. "How would you implement a compile-time check that prevents certain annotations from being used together?"
