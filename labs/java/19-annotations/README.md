# Lab 19: Annotations

**Objective:** Understand built-in annotations, custom annotations, retention policies, and annotation processing.

**Key Concepts:**
- `@Override`, `@Deprecated`, `@SuppressWarnings`, `@FunctionalInterface`
- Custom annotations with `@interface`
- Retention policies (`SOURCE`, `CLASS`, `RUNTIME`)
- Target types and repeated annotations
- Annotation processors (APT)

**Prerequisites:** Java 8+

**Estimated time:** 90 minutes
