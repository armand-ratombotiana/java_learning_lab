# History — Annotations in Java

- **2002:** JSR 175 — A Metadata Facility for Java
- **2004:** Java 5 — Annotations introduced
  - Built-in: `@Override`, `@Deprecated`, `@SuppressWarnings`
  - `@interface`, `@Retention`, `@Target`
- **2006:** Java 6 — Annotation Processing API (javax.annotation.processing)
- **2009:** Java 7 — No major annotation changes
- **2011:** Java 7 — `@SafeVarargs`
- **2014:** Java 8 — Repeatable annotations (`@Repeatable`), type annotations (`ElementType.TYPE_USE`, `ElementType.TYPE_PARAMETER`)
- **2017:** Java 9 — No major changes
- **2021:** Java 17 — `@Serial`, `@SuppressWarnings("preview")`

Annotations were influenced by **C# attributes** and **JavaDoc tags**.
