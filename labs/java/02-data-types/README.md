# Data Types — Overview

Java's type system distinguishes between primitive types (stored directly in memory) and reference types (pointers to objects). Understanding this distinction is fundamental to writing correct and efficient Java code.

## Learning Objectives

- Identify all 8 primitive types and their memory sizes
- Understand automatic type promotion and explicit casting
- Use wrapper classes and autoboxing/unboxing
- Recognize the difference between value and reference semantics
- Apply type conversion correctly in expressions

## Prerequisites

- Java Syntax lab completed or equivalent knowledge
- Basic understanding of binary number representation

## Estimated Time

- **Theory**: 45 minutes
- **Practice**: 60 minutes
- **Exercises**: 45 minutes
- **Total**: 2–3 hours
