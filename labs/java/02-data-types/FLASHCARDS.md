# Data Types — Flashcards

1. **Q: How many bits in a byte?** A: 8 bits
2. **Q: Range of int?** A: -2^31 to 2^31-1
3. **Q: Integer cache range?** A: -128 to 127
4. **Q: Compare wrappers with?** A: .equals(), not ==
5. **Q: Default value of int field?** A: 0
6. **Q: `0x1A` in decimal?** A: 26
7. **Q: `0b1010` in decimal?** A: 10
8. **Q: Wrapper for int?** A: Integer
9. **Q: Float size?** A: 32 bits
10. **Q: Double size?** A: 64 bits
11. **Q: Char size?** A: 16 bits
12. **Q: Suffix for long?** A: L
13. **Q: Suffix for float?** A: f
14. **Q: Which type cannot be null?** A: Primitives
15. **Q: What is autoboxing?** A: Automatic primitive→wrapper conversion
16. **Q: NaN == NaN?** A: false
17. **Q: BigDecimal for what?** A: Precise decimal arithmetic (money)
18. **Q: Use char[] instead of String for?** A: Passwords
19. **Q: Underscore in literals?** A: For readability: 1_000_000
20. **Q: Narrowing conversion requires?** A: Explicit cast
