# Data Types — Quiz

1. How many bits does a `byte` have? **A) 8**
2. What is the default value of an `int` field? **0**
3. What happens if you unbox a null Integer? **NullPointerException**
4. Which wrapper class caches values -128 to 127? **Integer**
5. What is the size of a `double`? **64 bits**
6. `3 + 4 + "5"` evaluates to? **"75"**
7. What is the range of a `char`? **0 to 65535**
8. Can you assign a `double` to an `int` without a cast? **No**
9. `10 / 3` evaluates to? **3 (integer division)**
10. What does `0b1010` equal in decimal? **10**
11. Which is NOT a primitive type? **String**
12. What is the size of a `long`? **64 bits**
13. `Float.NaN == Float.NaN` is? **false**
14. What does `Integer.valueOf(200) == Integer.valueOf(200)` return? **false**
15. Which suffix is required for `float` literals? **f**
16. `int x = 1_000_000;` — valid? **Yes**
17. What is the wrapper class for `boolean`? **Boolean**
18. What is the max value of an `int`? **2,147,483,647**
19. What does `(int) 3.99` return? **3**
20. Can a `boolean` be assigned to an `int`? **No**
