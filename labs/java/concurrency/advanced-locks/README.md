# Advanced Locks & Synchronizers

Welcome to the atomic mastery lab for **Advanced Locks**. This lab is part of the Java Academy's Concurrency module.

## 🧠 What You Will Master
- The limitations of intrinsic `synchronized` blocks.
- `ReentrantLock` and fairness policies.
- `ReentrantReadWriteLock` for read-heavy workloads.
- `StampedLock` for optimistic reading.
- The AbstractQueuedSynchronizer (AQS) framework.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Why we need explicit locks.
2. [INTERNALS.md](./INTERNALS.md) - Deep dive into AQS and StampedLock mechanics.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementations of advanced locking strategies.
