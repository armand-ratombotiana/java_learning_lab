# Thread Pools and ThreadPoolExecutor

Welcome to the atomic mastery lab for **Thread Pools**. This lab is part of the Java Academy's Concurrency module.

## 🧠 What You Will Master
- The cost of creating and destroying threads.
- The `ExecutorService` and `ThreadPoolExecutor` architecture.
- Core vs. Maximum pool size and how they interact.
- Work Queues: `LinkedBlockingQueue` vs `SynchronousQueue`.
- Rejection Policies (Abort, CallerRuns, Discard).

1. [THEORY.md](./THEORY.md) - The need for resource reuse.
2. [INTERNALS.md](./INTERNALS.md) - Deep dive into the ThreadPoolExecutor state machine.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Practical tuning and custom pool implementations.
