# Phaser Synchronizer

Welcome to the atomic mastery lab for **Phaser**. This lab is part of the Java Academy's Concurrency module.

## 🧠 What You Will Master
- The limitations of `CyclicBarrier` and `CountDownLatch`.
- Dynamic registration and deregistration of parties.
- Cyclic vs. One-shot phases.
- Tiered Phasers for massive scalability.
- Implementing multi-phase parallel tasks in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The reusable, flexible synchronizer.
2. [INTERNALS.md](./INTERNALS.md) - State management and phase transitions.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Simulating a multi-stage data processing pipeline.
