# Virtual Threads (Project Loom)

Welcome to the atomic mastery lab for **Virtual Threads**. This lab is part of the Java Academy's Concurrency module.

## 🧠 What You Will Master
- The limitations of OS (Platform) Threads.
- The concept of Virtual Threads (Project Loom) introduced in Java 21.
- Carrier threads and continuations.
- Thread-per-request model revival.
- Implementing high-throughput concurrent applications using Virtual Threads.

1. [THEORY.md](./THEORY.md) - The problem with OS threads and the Loom solution.
2. [INTERNALS.md](./INTERNALS.md) - How continuations and carrier threads work under the hood.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Creating and benchmarking Virtual Threads.
