# Fork/Join Framework

Welcome to the atomic mastery lab for the **Fork/Join Framework**. This lab is part of the Java Academy's Concurrency module.

## 🧠 What You Will Master
- The Divide and Conquer strategy for parallel processing.
- The `RecursiveTask` and `RecursiveAction` classes.
- The **Work-Stealing Algorithm** and why it maximizes CPU utilization.
- Comparing Fork/Join to standard Thread Pools and Streams.
- Implementing a parallel Merge Sort using Fork/Join.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Divide and Conquer and Work-Stealing intuition.
2. [INTERNALS.md](./INTERNALS.md) - How the ForkJoinPool manages double-ended queues (deques).
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a parallel recursive task.
