# JUnit 5 Advanced -- Flashcards
Card 1: Q: @Test annotation? A: Marks a method as a test
Card 2: Q: @Mock? A: Creates a mock object
Card 3: Q: @InjectMocks? A: Injects mocks into the tested object
Card 4: Q: verify()? A: Verifies mock interactions
Card 5: Q: @Testcontainers? A: Manages Docker containers
Card 6: Q: assertSoftly? A: Collects all assertion failures
Card 7: Q: @ParameterizedTest? A: Runs test with multiple inputs
Card 8: Q: @ExtendWith? A: Registers JUnit 5 extension
