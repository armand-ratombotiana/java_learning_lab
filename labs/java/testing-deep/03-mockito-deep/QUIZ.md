# Mockito Deep Dive -- Quiz
## Question 1: What annotation marks a test method?
A: @Test
## Question 2: What is the purpose of @BeforeEach?
A: Setup method before each test
## Question 3: How do you verify a mock was called?
A: verify(mock).method()
## Question 4: What does assertThrows do?
A: Asserts that a specific exception is thrown
## Question 5: What is a Testcontainer?
A: A JUnit 5 extension for docker containers
