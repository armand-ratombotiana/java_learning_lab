# Mockito Deep Dive -- Mathematical Foundation
## 1. Code Coverage Mathematics
### Line Coverage

### Branch Coverage
branches_executed / total_branches * 100%
### Path Coverage
paths_tested / total_paths * 100%
## 2. Test Effectiveness Metrics
### Mutation Score
killed_mutants / total_mutants * 100%
### Fidelity Score
passed_tests / total_tests * 100%
