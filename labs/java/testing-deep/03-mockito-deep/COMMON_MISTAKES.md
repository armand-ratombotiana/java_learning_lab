# Mockito Deep Dive -- Common Mistakes
## Top 10 Common Mistakes
1. Tests that depend on execution order
2. Testing implementation details instead of behavior
3. Not testing edge cases
4. Over-mocking (testing mock behavior, not real behavior)
5. Flaky tests due to shared mutable state
6. Tests with no assertions
7. Ignoring test failures
8. Slow tests that aren't optimized
9. Not cleaning up resources
10. Testing too much in one test
