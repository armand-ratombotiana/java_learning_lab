# Testing Deep - Micro-Labs

## Overview
This module contains 5 deep-dive micro-labs covering JUnit 5 fundamentals and advanced features, Mockito, Testcontainers, and AssertJ.

## Lab Index
| # | Lab | Description |
|---|-----|-------------|
| 01 | [JUnit 5 Fundamentals](./01-junit5-fundamentals/) | @Test, assertions, assumptions, @BeforeEach/@AfterEach, @BeforeAll/@AfterAll, @DisplayName, test lifecycle, nested tests |
| 02 | [JUnit 5 Advanced](./02-junit5-advanced/) | @ParameterizedTest, @ValueSource/@CsvSource/@MethodSource/@EnumSource, @RepeatedTest, @TestFactory, dynamic tests, extensions |
| 03 | [Mockito Deep Dive](./03-mockito-deep/) | @Mock/@InjectMocks, when/thenReturn, verify, ArgumentCaptor, @Spy, @Answer, BDDMockito, static mocking (MockedStatic) |
| 04 | [Testcontainers](./04-testcontainers/) | @Testcontainers, GenericContainer, module containers (PostgreSQL, Kafka, Redis), startup timeout, container reuse, network |
| 05 | [AssertJ Deep Dive](./05-assertj-deep/) | AssertJ soft assertions, extracting, flatExtracting, tuple, condition, exception assertion, recursive comparison, custom assertions |

## How to Use
Each micro-lab is self-contained with 24 markdown files, Java source code, JUnit 5 tests, and 7 subdirectories for hands-on work. Start from lab 01 and progress sequentially.
