# JUnit 5 Fundamentals -- Exercises
## Beginner Exercises
### Exercise 1: Write your first test
Write a test that verifies basic arithmetic operations.
### Exercise 2: Test exceptions
Write tests that verify exceptions are thrown correctly.
## Intermediate Exercises
### Exercise 3: Parameterized tests
Write parameterized tests for multiple input combinations.
### Exercise 4: Mock external dependencies
Use mocks to isolate the class under test from its dependencies.
## Advanced Exercises
### Exercise 5: Integration test with database
Use Testcontainers to test database interactions.
