# JUnit 5 Fundamentals
## Overview
@Test, assertions, assumptions, @BeforeEach/@AfterEach, @BeforeAll/@AfterAll, @DisplayName, test lifecycle, nested tests
## Learning Objectives
- Understand JUnit 5 Fundamentals
- Write comprehensive tests
- Apply testing patterns in real projects
## Prerequisites
- Java 21+
- Basic understanding of testing concepts
## Lab Structure
This micro-lab follows the standard 24-file pedagogical structure with 7 subdirectories.
## Quick Start
1. Navigate to src/main/java/com/javalab/01/
2. Review the main implementation class
3. Run tests using: mvn test or gradle test
4. Complete the exercises in EXERCISES.md
## Time Estimate
- Total: 2-3 hours
