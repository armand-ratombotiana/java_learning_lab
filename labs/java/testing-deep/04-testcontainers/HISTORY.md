# Testcontainers -- History
## Testing Framework Evolution
### JUnit 4 (2006)
Annotations introduced (@Test, @Before, @After).
### JUnit 5 (2017)
Modular architecture, Java 8 features, extensions.
### Mockito (2008)
Simple mocking API for unit tests.
### Testcontainers (2015)
Docker containers for integration testing.
### AssertJ (2013)
Fluent assertions for better readability.
