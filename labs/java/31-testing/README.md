# Lab 31: Testing — JUnit 5, Mockito & TDD

## Overview
Master the art of Java testing: from JUnit 5 fundamentals and Mockito mocking to full TDD workflows. Learn to write fast, reliable, and maintainable tests that give you confidence in your code.

## Learning Objectives
- Write and run JUnit 5 tests with assertions and assumptions
- Use Mockito mocks, stubs, spies, and verify interactions
- Apply the TDD cycle (Red-Green-Refactor)
- Create parameterized tests with @CsvSource, @ValueSource, @MethodSource
- Measure and improve test coverage
- Distinguish unit tests from integration tests

## Prerequisites
- Java 21+
- Familiarity with Maven or Gradle
- Completion of Labs 01–10 (OOP, interfaces, exceptions)

## Estimated Time
- **Theory**: 60 minutes
- **Practice**: 90 minutes
- **Exercises**: 60 minutes
- **Total**: 3–4 hours
