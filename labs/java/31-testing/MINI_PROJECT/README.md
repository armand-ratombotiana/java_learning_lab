# MINI_PROJECT — TDD String Calculator Kata

## Goal
Implement a String Calculator using strict TDD. Follow the Red-Green-Refactor cycle for each requirement.

## Requirements
1. An empty string returns 0
2. A single number returns that number
3. Two numbers separated by comma return their sum
4. Numbers separated by newline also work
5. Support custom delimiter: "//[delimiter]\n[numbers]"
6. Negative numbers throw an exception listing all negatives
7. Numbers > 1000 are ignored
