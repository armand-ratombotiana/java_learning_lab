# Exercises: String Handling

## Exercise 1: CSV Parser
Parse a CSV line into fields, handling quoted fields and escaped quotes.

## Exercise 2: Template Engine
Create a simple template engine that replaces `{{variable}}` placeholders with values.

## Exercise 3: StringBuilder Report Generator
Generate an HTML report using StringBuilder with proper formatting and indentation.

## Exercise 4: Regex Validator
Write validators for email, phone, URL, and IP address using compiled regex patterns.

## Exercise 5: String Compression
Implement run-length encoding and decoding of strings.

## Exercise 6: Text Block SQL Builder
Build SQL queries using text blocks with dynamic parameters (placeholder replacement).

## Exercise 7: Palindrome Detector
Check if a string is a palindrome ignoring case and punctuation.

## Exercise 8: Word Frequency Counter
Count word frequencies in a text using Stream API and Collectors.joining.

## Exercise 9: Logger Formatting
Create a log formatter that produces consistent log entries with timestamp, level, message, and stack traces.

## Exercise 10: JSON Formatter
Write a pretty-printer for JSON strings that adds proper indentation.
