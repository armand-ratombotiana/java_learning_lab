## Compact Strings (Java 9+)
```
Java 8: "hello" вҶ’ char[5] = 10 bytes

Java 9: "hello" вҶ’ byte[5], coder=LATIN1 = 5 bytes (+1 byte coder)
         "hГ©llo" вҶ’ byte[5], coder=UTF16 = 10 bytes (+1 byte coder)
```
