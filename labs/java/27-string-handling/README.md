# Lab 27: String Handling in Java

## Overview
String handling is fundamental to Java programming. This lab covers Java's String class (immutability, String pool, common operations), StringBuilder/StringBuffer for mutable strings, text blocks (Java 13+), string formatting, pattern matching with regex, and best practices for efficient string manipulation.

## Key Concepts
- String immutability and the String pool
- StringBuilder vs StringBuffer vs String concatenation
- Text blocks for multi-line strings
- String formatting (String.format, Formatter, MessageFormat)
- Regular expressions with Pattern/Matcher
- StringJoiner and Collectors.joining()
- Indent, transform, and other modern String methods
- Performance considerations and best practices
