# Why Concurrency Exists

## Hardware Reality
CPUs have multiple cores. Utilising them requires concurrent execution.

## Responsiveness
UI applications must remain responsive while performing background work.

## Throughput
Servers handle thousands of requests simultaneously — concurrency enables high throughput.

## Latency Hiding
While waiting for I/O (disk, network), the CPU can work on other tasks.

## Moore's Law Shift
Clock speeds plateaued around 2005; performance gains now come from parallelism.
