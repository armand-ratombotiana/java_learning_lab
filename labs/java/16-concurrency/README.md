# Lab 16: Concurrency

**Objective:** Understand Java concurrency — threads, synchronization, locks, executor service, and CompletableFuture.

**Key Concepts:**
- Thread creation and lifecycle
- `synchronized` keyword and monitors
- `java.util.concurrent.locks` (ReentrantLock, ReadWriteLock)
- ExecutorService and thread pools
- CompletableFuture for async programming

**Prerequisites:** Java 8+, lambdas

**Estimated time:** 150 minutes
