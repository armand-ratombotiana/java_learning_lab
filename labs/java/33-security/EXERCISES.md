# Exercises

## Exercise 1: Basic Implementation
**Difficulty:** Easy

Implement the core concept from scratch. Create a simple example that demonstrates the fundamental pattern.

## Exercise 2: Error Handling
**Difficulty:** Medium

Extend your implementation with comprehensive error handling. Define custom exceptions and handle at least three error conditions.

## Exercise 3: Integration
**Difficulty:** Medium

Create an integration scenario with at least three interacting components. Use dependency injection to wire them together.

## Exercise 4: Performance Optimization
**Difficulty:** Hard

Profile an existing implementation. Identify the top three bottlenecks. Apply optimizations and document before/after measurements.

## Exercise 5: Refactoring
**Difficulty:** Medium

Take poorly structured code and refactor it. Apply at least three refactoring patterns. Ensure all tests still pass.

## Exercise 6: Design Review
**Difficulty:** Medium

Review a code sample and critique its design. Identify violations of best practices and suggest specific improvements.

## Exercise 7: Advanced Pattern
**Difficulty:** Hard

Research and implement an advanced pattern. Compare with established library implementations. Write comprehensive tests.

## Exercise 8: Code Review
**Difficulty:** Medium

Review code from the lab source files. Identify strengths and areas for improvement. Write a detailed review comment for each file.
