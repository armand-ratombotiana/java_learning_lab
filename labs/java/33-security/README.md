# Lab 33: Security — JCA, JCE, Crypto

## Overview
Deep dive into Security in Java. This lab provides comprehensive coverage of JCA, JCE, Crypto with practical examples, theory, and exercises.

## Learning Objectives
- Understand core concepts of Security
- Apply JCA, JCE, Crypto in real-world scenarios
- Recognize common patterns and anti-patterns
- Master debugging, optimization, and best practices

## Prerequisites
- Java 21+
- Completion of Labs 01-20

## Estimated Time
- **Theory**: 60 minutes
- **Practice**: 90 minutes
- **Exercises**: 60 minutes
- **Total**: 3-4 hours
