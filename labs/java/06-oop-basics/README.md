# OOP Basics — Overview

Object-oriented programming is the core paradigm of Java. Classes define blueprints for objects; objects encapsulate state and behavior. This lab covers classes, objects, constructors, and the static keyword.

## Learning Objectives

- Define classes with fields, constructors, and methods
- Create and use objects
- Understand constructors and constructor overloading
- Distinguish between instance and static members
- Apply the `this` keyword correctly

## Prerequisites

- Methods lab completed
- Arrays and Strings lab completed

## Estimated Time

- **Theory**: 60 minutes
- **Practice**: 90 minutes
- **Exercises**: 60 minutes
- **Total**: 3–4 hours
