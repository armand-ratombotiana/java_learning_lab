# OOP Basics — Quiz

1. What is a class? **Blueprint for creating objects**
2. What is an object? **Instance of a class**
3. What is encapsulation? **Bundling data and methods, controlling access**
4. What is a constructor? **Special method for initializing objects**
5. What is the `this` keyword? **Reference to the current object**
6. What does `static` mean? **Belongs to class, not instance**
7. Can a static method access `this`? **No**
8. What is the default constructor? **Compiler-generated no-arg constructor**
9. What is a field? **Variable declared in a class**
10. What is an access modifier? **public, private, protected, default**
11. What is garbage collection? **Automatic memory reclamation**
12. Can a class have multiple constructors? **Yes (overloading)**
13. What is an instance variable? **Field unique to each object**
14. What is a class variable? **Static field shared by all instances**
15. Does Java have destructors? **No (GC handles cleanup)**
16. What is a fluent API? **Method chaining via returning `this`**
17. Can `this` be used in a static context? **No**
18. What is a package? **Namespace for organizing classes**
19. What is `var`? **Local variable type inference (Java 10+)**
20. What is a record? **Immutable data class (Java 16+)**
