# OOP Basics — Exercises

## Exercise 1: Simple Class
Create a `Book` class with title, author, ISBN. Add getters, setters, and a display method.

## Exercise 2: Car Class
Create a `Car` class with make, model, year, speed. Add `accelerate()`, `brake()`, `displaySpeed()`.

## Exercise 3: Bank Account
Create a `BankAccount` class with account number, balance, owner. Implement `deposit()`, `withdraw()`, `transfer()`.

## Exercise 4: Student with Grading
Create a `Student` class with name, grades array. Compute average, letter grade, highest/lowest.

## Exercise 5: Rectangle Class
Create a `Rectangle` class with width, height. Add `getArea()`, `getPerimeter()`, `isSquare()`.

## Exercise 6: Fraction Class
Create a `Fraction` class with numerator, denominator. Implement `add()`, `multiply()`, `simplify()`.

## Exercise 7: Counter with Static
Create a `Counter` class that tracks both instance count and total count across all instances.

## Exercise 8: Fluent API
Create a `Pizza` class with a fluent builder: `new Pizza().size("large").cheese(true).pepperoni(true).bake()`.

## Exercise 9: Immutable Point
Create an immutable `Point` class with x, y. Methods that "modify" return new instances.

## Exercise 10: Circle with Static Factory
Create a `Circle` class with static factory methods: `fromRadius()`, `fromDiameter()`, `fromCircumference()`.

## Exercise 11: Shopping Cart
ShoppingCart class with add/remove items, calculate total. Item class with name, price, quantity.

## Exercise 12: Employee Hierarchy
Employee class with name, salary. FullTimeEmployee and PartTimeEmployee subclasses (extend).
