# Quiz

## Question 1
What is the primary benefit of good software design?
A) Faster execution speed
B) Better code organization and maintainability
C) Less memory usage
D) Automatic error handling
**Answer: B**

## Question 2
Which principle states that a class should have only one reason to change?
A) Open-Closed
B) Liskov Substitution
C) Single Responsibility
D) Dependency Inversion
**Answer: C**

## Question 3
What is the recommended way to handle dependencies?
A) Hard-code them in constructors
B) Use dependency injection
C) Access them as static fields
D) Create new instances each time
**Answer: B**

## Question 4
Which is NOT a benefit of good design?
A) Improved testability
B) Reduced coupling
C) Guaranteed performance improvements
D) Better code readability
**Answer: C**

## Question 5
When should you optimize code?
A) During initial implementation
B) After profiling identifies bottlenecks
C) Never, let the JIT handle it
D) Only when users complain
**Answer: B**

## Question 6
What is the purpose of interfaces?
A) To define contracts between components
B) To improve performance
C) To reduce memory usage
D) To enable multiple inheritance
**Answer: A**

## Question 7
Which testing approach is most appropriate?
A) Only manual testing
B) Unit tests with mocked dependencies
C) Only integration tests
D) No testing for well-designed code
**Answer: B**

## Question 8
What is the Open-Closed principle?
A) Classes should be open for extension, closed for modification
B) Classes should be open for modification, closed for extension
C) Classes should be both open and closed
D) Classes should neither be open nor closed
**Answer: A**

## Question 9
What indicates tight coupling?
A) Using interfaces
B) Creating dependencies with new inside constructors
C) Passing dependencies via constructor
D) Using dependency injection
**Answer: B**

## Question 10
What is defensive copying?
A) Copying mutable objects to protect internal state
B) Making copies of all objects
C) Preventing any object copying
D) Copying only immutable objects
**Answer: A**
