# Lab 34: Logging — SLF4J, Logback, Log4j2

## Overview
Deep dive into Logging in Java. This lab provides comprehensive coverage of SLF4J, Logback, Log4j2 with practical examples, theory, and exercises.

## Learning Objectives
- Understand core concepts of Logging
- Apply SLF4J, Logback, Log4j2 in real-world scenarios
- Recognize common patterns and anti-patterns
- Master debugging, optimization, and best practices

## Prerequisites
- Java 21+
- Completion of Labs 01-20

## Estimated Time
- **Theory**: 60 minutes
- **Practice**: 90 minutes
- **Exercises**: 60 minutes
- **Total**: 3-4 hours
