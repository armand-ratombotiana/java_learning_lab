# History — Virtual Threads

- **2018:** Project Loom announced (Ron Pressler, Alan Bateman)
- **2019:** Early prototypes in OpenJDK
- **2021:** Loom preview in Java 19 (JEP 425)
- **2022:** Second preview in Java 20
- **2023:** Virtual threads GA in Java 21 (JEP 444)
- **2023 (Sept):** Structured concurrency preview (JEP 428)
- **2024+:** Structured concurrency incubation, scoped values

Project Loom was inspired by:
- Erlang processes (lightweight, isolated)
- Go goroutines (M:N scheduling)
- Fibers in other JVM languages (Quasar, Kilim)
