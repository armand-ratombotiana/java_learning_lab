# JVM Memory Model

Welcome to the atomic mastery lab for the **JVM Memory Model**. This lab is part of the Java Academy's JVM Internals module.

## 🧠 What You Will Master
- The architecture of JVM memory (Heap, Stack, Metaspace, PC Register).
- How objects are allocated and referenced.
- Garbage Collection Roots (GC Roots) and reachability.
- The difference between Pass-by-Value and Pass-by-Reference in Java.
- Diagnosing OutOfMemoryError and StackOverflowError.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The different memory areas of the JVM.
2. [INTERNALS.md](./INTERNALS.md) - Object headers, references, and GC roots.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Simulating memory leaks and stack overflows in Java.
