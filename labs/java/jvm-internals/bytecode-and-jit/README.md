# Bytecode and JIT Compilation

Welcome to the atomic mastery lab for **Bytecode and JIT**. This lab is part of the Java Academy's JVM Internals module.

## 🧠 What You Will Master
- The structure of a `.class` file and Java Bytecode.
- The JVM Execution Engine (Interpreter vs JIT).
- C1 (Client) and C2 (Server) Compilers.
- Tiered Compilation and Deoptimization.
- Analyzing JIT compilation logs.

1. [THEORY.md](./THEORY.md) - From source code to machine code.

3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Simulating method warming and JIT logs.
