# JVM Classloading

Welcome to the atomic mastery lab for **Classloading**. This lab is part of the Java Academy's JVM Internals module.

## 🧠 What You Will Master
- The JVM Classloading lifecycle (Loading, Linking, Initialization).
- The Delegation Hierarchy (Bootstrap, Platform, Application classloaders).
- Why `ClassNotFoundException` is different from `NoClassDefFoundError`.
- How Application Servers (like Tomcat) isolate web applications.
- Writing a Custom Classloader in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The lifecycle of a Java Class.
2. [INTERNALS.md](./INTERNALS.md) - The Delegation Model and common exceptions.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a Custom Classloader.
