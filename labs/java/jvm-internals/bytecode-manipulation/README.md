# Bytecode Manipulation

Welcome to the atomic mastery lab for **Bytecode Manipulation**. This lab is part of the Java Academy's JVM Internals module.

## 🧠 What You Will Master
- The structure of the Java Class File.
- ASM vs. ByteBuddy vs. Javassist.
- Runtime class generation and modification.
- Java Agents and the `premain` method.
- Real-world use cases: AOP, Profiling, and Mocking frameworks.
