# JVM Profiling and Diagnostics

Welcome to the atomic mastery lab for **JVM Profiling**. This lab is part of the Java Academy's JVM Internals module.

## 🧠 What You Will Master
- The importance of "Measure, don't guess".
- Profiling types: CPU, Memory, and Threading.
- Modern tools: Java Flight Recorder (JFR) and JVisualVM.
- Flame Graphs and identifying bottlenecks.
- Understanding Safepoints and Bias.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The methodology of performance analysis.
2. [INTERNALS.md](./INTERNALS.md) - How profilers work (Sampling vs. Instrumentation).
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Writing code that triggers JFR recordings.
