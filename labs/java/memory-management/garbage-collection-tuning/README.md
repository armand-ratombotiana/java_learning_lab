# Garbage Collection Tuning

Welcome to the atomic mastery lab for **GC Tuning**. This lab is part of the Java Academy's Memory Management module.

## 🧠 What You Will Master
- The trade-off between Throughput, Latency, and Footprint.
- Choosing the right GC for your workload (Parallel vs G1 vs ZGC).
- Key JVM flags for tuning heap size and pause times.
- Analyzing GC logs using tools like GCEasy or JClarity.
- Identifying and fixing "Premature Promotion" and "Humongous Allocations".

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The methodology of performance tuning.
2. [INTERNALS.md](./INTERNALS.md) - How different GCs structure the heap (Regions vs Generations).
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Hands-on tuning exercise to reduce pause times.
