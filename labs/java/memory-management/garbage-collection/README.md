# Garbage Collection

Welcome to the atomic mastery lab for **Garbage Collection**. This lab is part of the Java Academy's Memory Management module.

## 🧠 What You Will Master
- The Generational Hypothesis (Young vs. Old Generation).
- Stop-The-World (STW) pauses and their impact.
- GC Algorithms: Serial, Parallel, G1GC, and ZGC.
- Tuning GC parameters for throughput vs. latency.
- Analyzing GC logs.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The Generational Hypothesis and GC phases.
2. [INTERNALS.md](./INTERNALS.md) - Deep dive into G1GC and ZGC mechanics.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Simulating GC behavior and tuning flags.
