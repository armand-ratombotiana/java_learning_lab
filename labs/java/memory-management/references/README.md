# Reference Types

Welcome to the atomic mastery lab for **Reference Types**. This lab is part of the Java Academy's Memory Management module.

## 🧠 What You Will Master
- The four types of references in Java: Strong, Soft, Weak, and Phantom.
- How the Garbage Collector treats each reference type differently.
- Implementing memory-sensitive caches using `SoftReference`.
- Avoiding memory leaks with `WeakHashMap`.
- The `ReferenceQueue` and post-mortem cleanup.
