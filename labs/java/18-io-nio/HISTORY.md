# History — I/O & NIO in Java

- **1996:** Java 1.0 — `InputStream`, `OutputStream`, `FileInputStream`, `FileOutputStream`
- **1997:** Java 1.1 — `Reader`, `Writer`, `BufferedReader`, `ObjectOutputStream`
- **2002:** Java 1.4 — NIO (channels, buffers, selectors, `ByteBuffer`)
- **2004:** Java 5 — `Scanner`, `Formatter`, `printf`
- **2011:** Java 7 — NIO.2 (`Path`, `Files`, `FileVisitor`, `WatchService`, `SeekableByteChannel`)
- **2014:** Java 8 — `Files.lines()`, `Files.list()`, `Files.walk()` (Stream API integration)
- **2017:** Java 9 — `InputStream.readAllBytes()`, `InputStream.transferTo()`
- **2021:** Java 17 — `HexFormat`, `java.io.Serial` (deprecated serialization)

Key figures: Doug Lea, Alan Bateman, Mark Reinhold.
