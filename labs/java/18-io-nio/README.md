# Lab 18: I/O and NIO

**Objective:** Master Java I/O — streams, readers/writers, NIO.2 channels, and the Path/Files API.

**Key Concepts:**
- InputStream / OutputStream (byte-oriented)
- Reader / Writer (character-oriented)
- Buffered streams and efficient I/O
- NIO.2 Path, Files, FileVisitor
- Channels, ByteBuffers, Selectors
- Memory-mapped files

**Prerequisites:** Java 8+

**Estimated time:** 120 minutes
