# Lab 14: Lambdas

**Objective:** Understand lambda expressions, functional interfaces, method references, and closures in Java.

**Key Concepts:**
- Lambda syntax and scope
- Functional interfaces (`@FunctionalInterface`)
- Method and constructor references
- Variable capture (closures)
- `java.util.function` package

**Prerequisites:** Java 8+, basics of interfaces and anonymous classes.

**Estimated time:** 90 minutes
