# History — Lambdas

- **1997:** Anonymous inner classes introduced in Java 1.1 (JDK 1.1)
- **2004:** Java 5 with generics — paved way for type-safe functional interfaces
- **2009:** Project Lambda proposed as JSR 335
- **2014:** Java 8 released with lambdas, method references, and Streams API
- **Design led by:** Brian Goetz, with input from the broader Java community
- **Influenced by:** ML, Haskell, Scala, C# LINQ

The key challenge was retrofitting closures into an existing type system with backward compatibility — solved by mapping lambdas to functional interfaces.
