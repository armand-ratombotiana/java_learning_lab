# Polymorphism — Overview

Polymorphism allows objects of different types to respond to the same method call in their own way. Java supports compile-time polymorphism (overloading) and runtime polymorphism (overriding).

## Learning Objectives

- Understand dynamic method dispatch
- Distinguish method overloading from overriding
- Apply covariant return types
- Use polymorphic collections and parameters
- Recognize when polymorphism simplifies design

## Prerequisites

- Inheritance lab completed
- Understanding of method overriding

- **Theory**: 50 minutes
- **Practice**: 75 minutes
