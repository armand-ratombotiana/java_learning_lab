# History — Functional Programming in Java

- **1996:** Java 1.0 — purely OOP, no functional features
- **2004:** Java 5 — generics, enhanced for-loop
- **2014:** Java 8 — lambdas, streams, Optional, functional interfaces
- **2017:** Java 9 — factory methods for collections (immutable by default)
- **2018:** Java 11 — `Optional::isEmpty`, `String::repeat`
- **2021:** Java 16 — `record` types (concise immutable data carriers)
- **2022:** Java 17 — sealed classes, pattern matching (preview)
- **2023+:** Java 21 — pattern matching, record patterns, virtual threads

Java functional programming was heavily influenced by:
- **Haskell** (pure functions, monads)
- **Scala** (collections, Option type)
- **Clojure** (persistent data structures)
- **C# LINQ** (fluent query patterns)
