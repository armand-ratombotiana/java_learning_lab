# Lab 15: Functional Programming in Java

**Objective:** Master functional programming principles — immutability, pure functions, monads (Optional), and combinators.

**Key Concepts:**
- Immutability and pure functions
- Side-effect-free programming
- Optional as a monad
- Function composition and combinators
- Recursion over iteration

**Prerequisites:** Java 8+, lambdas, streams

**Estimated time:** 120 minutes
