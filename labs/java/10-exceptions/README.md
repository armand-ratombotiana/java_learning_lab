# Exceptions — Overview

Exceptions are Java's mechanism for handling error conditions without cluttering normal control flow. The try-catch-finally structure separates error handling from business logic.

## Learning Objectives

- Use try-catch-finally blocks effectively
- Distinguish checked from unchecked exceptions
- Create and throw custom exceptions
- Use try-with-resources for automatic resource management
- Understand exception propagation and stack traces

## Prerequisites

- OOP Basics lab completed
- Understanding of method call stacks

- **Theory**: 45 minutes
- **Practice**: 45 minutes
- **Exercises**: 40 minutes
