# Exercises: Enums

## Exercise 1: Days Until
Add a method to `DayOfWeek` that calculates days until a given target day.

## Exercise 2: Calculator Enum
Create an enum `CalculatorOperation` (ADD, SUBTRACT, MULTIPLY, DIVIDE) with an `apply` method.

## Exercise 3: EnumSet Permissions
Model file permissions (READ, WRITE, EXECUTE) using EnumSet. Write methods to check permissions.

## Exercise 4: EnumMap Configuration
Use EnumMap to store configuration values for different environments (DEV, TEST, STAGING, PRODUCTION).

## Exercise 5: State Machine
Implement a traffic light state machine using enums (RED, GREEN, YELLOW → next state transitions).

## Exercise 6: Enum Singleton
Create a thread-safe connection pool using enum singleton pattern.

## Exercise 7: Enum with JSON Serialization
Create an enum that serializes to custom JSON values (not just names).

## Exercise 8: Enum as Strategy
Implement a payment processing system where each payment method (CREDIT_CARD, PAYPAL, BANK_TRANSFER) has its own processing logic via an abstract method.
