# History of Enums

- **Java 1.0-1.4**: `public static final int` constants and typesafe enum pattern
- **Java 5 (2004)**: Enums introduced (JSR 201), EnumMap/EnumSet added
- **Java 7**: Enums in switch with String (String in switch)
- **Java 17+**: Pattern matching for switch extends to enums with type patterns
- **Java 21**: Sealed classes and records provide alternatives for more complex hierarchical constants

The enum design was heavily influenced by Joshua Bloch's "Typesafe Enum" pattern from Effective Java. Bloch consulted on the Java 5 enum implementation.
