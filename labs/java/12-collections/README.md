# Collections — Overview

The Java Collections Framework provides a unified architecture for storing and manipulating groups of objects. It includes interfaces (List, Set, Map, Queue, Deque), implementations (ArrayList, HashSet, HashMap, etc.), and algorithms (sorting, searching, shuffling).

## Learning Objectives

- Choose the correct collection type for different use cases
- Understand performance tradeoffs of each implementation
- Use List, Set, Map, Queue, and Deque effectively
- Apply collection utility methods (sort, binarySearch, shuffle)
- Create unmodifiable and synchronized collections

## Prerequisites

- Generics lab completed
- Basic understanding of hash tables and balanced trees

- **Theory**: 90 minutes
- **Practice**: 60 minutes
- **Exercises**: 60 minutes
