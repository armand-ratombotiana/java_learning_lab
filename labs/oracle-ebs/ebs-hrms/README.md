# EBS HRMS

## Overview

Oracle HRMS covering people, assignments, positions, organizations, payroll, time and labor, talent management, Oracle Learning Management (OLM), Advanced Benefits, legislative data groups, absence management, and government reporting.

## Prerequisites
- Lab 01 (EBS Architecture)
- Java 21+
- Oracle Database concepts

## What You Will Learn
- Core EBS ebs-hrms concepts
- Java simulation of EBS components
- Testing with JUnit 5
