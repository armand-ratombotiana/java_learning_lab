# EBS Manufacturing

## Overview

Oracle Manufacturing covering WIP discrete and flow manufacturing, repetitive manufacturing, lot-based manufacturing, engineering (ECO/ECN), BOM/routing, work centers, scheduling (MPS/MRP/DRP), capacity (CRP), shop floor control, and Oracle Quality.

## Prerequisites
- Lab 01 (EBS Architecture)
- Java 21+
- Oracle Database concepts

## What You Will Learn
- Core EBS ebs-manufacturing concepts
- Java simulation of EBS components
- Testing with JUnit 5
