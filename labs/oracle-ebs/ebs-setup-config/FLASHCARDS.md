# Flashcards: EBS Setup and Configuration

## Card 1
Q: What is the EBS database schema?
A: APPS

## Card 2
Q: MOAC stands for?
A: Multi-Org Access Control

## Card 3
Q: Which tier runs Oracle Forms?
A: Application Tier (Forms Server)

## Card 4
Q: What tool manages online patching?
A: ADOP

## Card 5
Q: Name the five CEMLI categories.
A: Configuration, Extension, Modification, Localization, Integration

A: FND_GLOBAL

A: APPL_TOP

A: BC4J (Business Components for Java)

A: MO_DEFAULT_ORG_ID
