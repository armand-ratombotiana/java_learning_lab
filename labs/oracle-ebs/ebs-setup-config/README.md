# EBS Setup and Configuration

## Overview

Oracle EBS R12.2 installation types, Rapid Install, pre-install checks, system administrator responsibility, profile levels, MOAC setup, flexfields, document sequences, auditing, and concurrent programs setup.

## Prerequisites
- Lab 01 (EBS Architecture)
- Java 21+
- Oracle Database concepts

## What You Will Learn
- Core EBS ebs-setup-config concepts
- Java simulation of EBS components
- Testing with JUnit 5
