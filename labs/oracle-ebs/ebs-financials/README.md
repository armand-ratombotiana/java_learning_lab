# EBS Financials

## Overview

Oracle Financials covering General Ledger (GL), Accounts Payables (AP), Accounts Receivables (AR), Cash Management (CE), Fixed Assets (FA), chart of accounts, journal entries, intercompany accounting, multi-currency, subledger accounting (SLA), and payment processing.

## Prerequisites
- Lab 01 (EBS Architecture)
- Java 21+
- Oracle Database concepts

## What You Will Learn
- Core EBS ebs-financials concepts
- Java simulation of EBS components
- Testing with JUnit 5
