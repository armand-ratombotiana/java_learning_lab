# History: EBS Technical Architecture

## 1. Origins

Oracle EBS traces roots to Oracle Financials (1990). Expanded through acquisitions and development.

## 2. Major Releases

- 10.7 (1995) - First web-enabled
- 11i (2000) - Internet architecture
- R12 (2007) - SOA
- R12.1 (2008) - UX enhancements
- R12.2 (2013) - Online patching via EBR

## 3. Technology Stack Evolution

Database: 9i -> 10g -> 11g -> 12c -> 19c
Forms: 6i -> 10g -> 12c
JVM: 1.4 -> 5 -> 7 -> 8 -> 11

## 4. Community

MOS, OAUG, UKOUG, independent consultants worldwide.
