# Quiz: EBS Technical Architecture

## Question 1
What is the primary database schema used by all Oracle EBS modules?
A. SYSTEM
B. APPS
C. DBSNMP
D. OUTLN

Answer: B

## Question 2
Which component handles online patching in R12.2?
A. AD Worker
B. ADOP
C. FNDLOAD
D. Forms Personalization

Answer: B

## Question 3
MOAC stands for?
A. Multi-Org Access Control
B. Multiple Oracle Access Control
C. Multi-Object Access Control
D. Managed Oracle Application Control

Answer: A

## Question 4
Which directory contains EBS application files?
A. ORACLE_HOME
B. APPL_TOP
C. DB_HOME
D. INST_TOP

Answer: B

## Question 5
What does CEMLI stand for?
A. Configuration Extension Modification Localization Integration
B. Core Enterprise Management Library Interface
C. Central EBS Module Language Integration
D. Custom Enterprise Middleware Localization Interface

Answer: A

## Question 6
Which view provides VPD support in EBS?
A. FND_MOBS
B. FND_GLOBAL
C. FND_PROFILE
D. FND_USER

Answer: A

## Question 7
The concurrent manager phase Running indicates?
A. Request is queued
B. Request is executing
