# Quiz — Distributed Queues

1. SQS is which AWS service? Answer: Simple Queue Service
2. Pulsar uses what for storage? Answer: Apache BookKeeper
3. Exactly-once requires? Answer: Dedup + idempotent consumers
4. At-least-once can cause? Answer: Duplicates
5. Pulsar exclusive subscription allows? Answer: One consumer
6. DLQ stores? Answer: Failed messages after max retries
7. Partition key determines? Answer: Partition assignment
8. Exponential backoff uses? Answer: Multiplicative delay increase
9. Pulsar shared subscription does? Answer: Round-robin consumer distribution
10. Message ordering within partition? Answer: Preserved (FIFO)
