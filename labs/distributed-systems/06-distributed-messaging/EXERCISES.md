# Distributed Messaging: Exercises

## Exercise 1: Kafka Producer/Consumer
Write a Java program that produces and consumes messages from Kafka.

## Exercise 2: Message Ordering
Implement order-preserving processing with Kafka partitions.

## Exercise 3: Exactly-Once Semantics
Configure Kafka for exactly-once delivery and verify.

## Exercise 4: Custom Message Router
Build a simple RabbitMQ-style exchange with routing keys.

## Exercise 5: Stream Processing
Implement a Kafka Streams application that aggregates events into a time window.
