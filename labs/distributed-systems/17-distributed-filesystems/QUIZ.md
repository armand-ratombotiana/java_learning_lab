# Quiz — Distributed Filesystems

1. HDFS default block size? Answer: 128MB
2. HDFS single point of failure? Answer: NameNode
3. Ceph uses which algorithm for placement? Answer: CRUSH
4. Erasure coding (10+4) tolerates how many failures? Answer: 4
5. 3x replication storage overhead? Answer: 300%
6. MinIO uses what API? Answer: S3-compatible REST API
7. Ceph MON provides? Answer: Consensus via Paxos
8. HDFS Namenode stores? Answer: Metadata (file-to-block mapping)
9. Erasure coding reduces? Answer: Storage overhead vs replication
10. Reed-Solomon is which type of code? Answer: Erasure code
