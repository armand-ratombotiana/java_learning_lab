# Exercises — Distributed Filesystems

1. **File Replicator (Easy)**: Implement block re-replication on failure
2. **Erasure Codec (Medium)**: Implement Reed-Solomon (k=4, m=2)
3. **MinIO Client (Easy)**: Implement S3-compatible object operations
4. **HDFS Client (Medium)**: Implement HDFS read/write with block awareness
5. **Metadata Store (Medium)**: Implement in-memory metadata cache
6. **Rack Awareness (Hard)**: Implement rack-aware replica placement
7. **File Splitter (Easy)**: Split large files into blocks
8. **Block Recovery (Hard)**: Implement degraded read from erasure-coded fragments
9. **Benchmark (Medium)**: Compare replication vs erasure coding performance
10. **Consistent Hash Ring (Hard)**: Implement CRUSH-like data placement
