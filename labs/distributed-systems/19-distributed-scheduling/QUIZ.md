# Quiz — Distributed Scheduling

1. Quartz uses what for cluster coordination? Answer: Database locks
2. Leader election ensures? Answer: Single executor
3. Cron * * * * * means? Answer: Every minute
4. Misfire means? Answer: Job didn't run on time
5. Partitioned model distributes by? Answer: Hash or range
6. Quartz JDBC JobStore provides? Answer: Persistent job storage
7. At-most-once may? Answer: Skip job execution
8. Fencing token prevents? Answer: Double execution
9. Raft election time depends on? Answer: Heartbeat timeout
10. Consistent hash minimizes? Answer: Job movement on rebalance
