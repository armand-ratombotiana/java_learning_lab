# Failure Detection: Exercises

## Exercise 1: Heartbeat Detector
Implement a simple heartbeat-based failure detector with configurable timeout.

## Exercise 2: Phi-Accrual Detector
Implement the phi-accrual failure detector as used in Cassandra.

## Exercise 3: Gossip Protocol
Build a gossip-based membership protocol that propagates failure information.

## Exercise 4: SWIM Implementation
Implement the SWIM failure detection protocol with suspicion.

## Exercise 5: Integration
Integrate your failure detector with a Raft-like leader election system.
