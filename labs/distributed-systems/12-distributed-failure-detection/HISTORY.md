# History of Failure Detection

## Timeline
- **1996**: Chandra and Toueg define failure detector abstractions
- **2002**: SWIM protocol for scalable failure detection
- **2004**: Phi-Accrual failure detector by Hayashibara et al.
- **2007**: Cassandra adopts Phi-Accrual
- **2013**: HashiCorp Serf implements SWIM
- **2015**: Kubernetes node controller uses lease-based detection

## Key Figures
- **Tushar Chandra**: Failure detector formalization
- **Sam Toueg**: Failure detector abstraction
- **Naohiro Hayashibara**: Phi-Accrual failure detector
- **Amitabha Das**: SWIM protocol
- **Mark Shapiro**: Gossip-based detection

## Evolution
From simple fixed-timeout heartbeats to adaptive, probabilistic failure detectors that adjust to network conditions.
