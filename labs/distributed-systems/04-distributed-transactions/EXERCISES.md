# Distributed Transactions: Exercises

## Exercise 1: Implement 2PC
Build a two-phase commit coordinator in Java.

## Exercise 2: Implement SAGA
Create a SAGA orchestrator for an e-commerce order flow (inventory, payment, shipping).

## Exercise 3: Compensating Transactions
Design compensatory actions for: payment, inventory reservation, shipment scheduling.

## Exercise 4: Test Failure Scenarios
Test: coordinator crash, participant crash, network timeout during prepare.

## Exercise 5: Compare Protocols
Benchmark 2PC vs SAGA for an order placement flow.
