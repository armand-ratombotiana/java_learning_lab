# History of Distributed Transactions

## Timeline
- **1976**: X/Open XA specification begins
- **1987**: SAGA pattern introduced by Garcia-Molina and Salem
- **1990s**: CORBA OTS, JTA/JTS specifications
- **2001**: WS-AtomicTransaction for web services
- **2014**: SAGA resurrected for microservices
- **2018**: Seata (Alibaba) open-source distributed transaction framework

## Key Figures
- **Hector Garcia-Molina**: SAGA pattern, distributed database pioneer
- **Kenneth Salem**: SAGA co-author
- **Jim Gray**: Transaction processing pioneer

## Evolution
From heavyweight XA transactions to lightweight SAGA patterns, distributed transactions evolved with architecture trends.
