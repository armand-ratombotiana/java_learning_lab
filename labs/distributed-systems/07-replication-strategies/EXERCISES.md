# Replication: Exercises

## Exercise 1: Leader/Follower Replication
Implement a single-leader replication system with asynchronous followers.

## Exercise 2: Quorum-Based Replication
Build a quorum-based replicated store with configurable R and W.

## Exercise 3: Multi-Leader Conflict Resolution
Implement last-write-wins and custom merge conflict resolution.

## Exercise 4: Replication Lag Monitoring
Write a tool to measure replication lag across nodes.

## Exercise 5: Failover Simulation
Simulate leader failure and implement automatic follower promotion.
