# Mental Models for Replication

## The Newspaper Model
- **Leader**: Central printing press
- **Followers**: Distribution trucks
- Synchronous: Everyone waits for all trucks to load
- Asynchronous: Trucks leave when ready, may have different editions

## The Group Chat Model
- **Leader**: Only one person can type (single-leader)
- **Multi-leader**: Everyone can type, messages merge
- **Leaderless**: WhatsApp-style (everyone has copy, sync when connected)

## The Library Model
- **Single-leader**: One master copy, all changes go through librarian
- **Multi-leader**: Regional libraries, each has local master copy
- **Leaderless**: Everyone has their own copy, periodic sync
