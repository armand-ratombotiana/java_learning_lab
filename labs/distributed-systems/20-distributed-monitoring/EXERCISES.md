# Exercises — Distributed Monitoring

1. **Metric Registry (Easy)**: Implement thread-safe counter and gauge
2. **Prometheus Exporter (Medium)**: Export JVM metrics in Prometheus format
3. **Histogram (Medium)**: Implement latency histogram with buckets
4. **Tracing Span (Easy)**: Implement OpenTelemetry-style span
5. **Alert Rule (Medium)**: Implement threshold-based alert evaluation
6. **Downsampling (Hard)**: Implement time-based metric downsampling
7. **Aggregation Pipeline (Medium)**: Implement streaming aggregation
8. **PromQL Client (Hard)**: Implement basic PromQL parsing and evaluation
9. **Health Check (Easy)**: Implement distributed health check endpoint
10. **SLO Calculator (Medium)**: Implement SLO compliance calculation
