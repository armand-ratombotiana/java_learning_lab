# Time and Ordering: Exercises

## Exercise 1: Lamport Clock
Implement a Lamport clock and use it to timestamp events in a multi-process simulation.

## Exercise 2: Vector Clock
Implement a vector clock that correctly determines causality between events.

## Exercise 3: Causal Broadcast
Build a causal broadcast system using vector clocks.

## Exercise 4: HLC Implementation
Implement a Hybrid Logical Clock (HLC) and compare with physical clocks.

## Exercise 5: Clock Drift Simulation
Simulate NTP-synchronized clocks with drift and measure maximum skew.
