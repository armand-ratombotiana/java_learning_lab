# 02 - Consistency Models

## Overview
Consistency models define the guarantees a distributed data store provides about read and write operations. Different models offer different tradeoffs between performance, correctness, and developer convenience.

## Lab Structure
- **MINI_PROJECT/** - Small focused exercises
- **REAL_WORLD_PROJECT/** - Production-scale implementations
- **CHALLENGE/** - Advanced problem-solving tasks
- **TESTS/** - Unit and integration test suites
- **BENCHMARK/** - Performance measurement tools
- **DIAGRAMS/** - Visual representations of consistency concepts
- **SOLUTION/** - Reference implementations
