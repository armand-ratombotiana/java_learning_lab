# Distributed Consensus: Exercises

## Exercise 1: Implement Leader Election

## Exercise 2: Implement Log Replication
Extend your leader to replicate entries to followers.

## Exercise 3: Paxos Implementation
Implement single-decree Paxos in Java.

## Exercise 4: Test Failure Scenarios
Write tests for: leader crash, network partition, log inconsistency.

## Exercise 5: Performance Benchmark
Measure throughput of your consensus implementation with varying cluster sizes.
