# History of Distributed Consensus

## Timeline
- **1988**: Paxos published by Leslie Lamport
- **1992**: FLP impossibility result
- **2006**: Zab protocol (ZooKeeper) developed
- **2013**: Raft consensus published by Ongaro and Ousterhout
- **2014**: etcd (Raft-based) released
- **2017**: Multi-Raft and parallel consensus variants

## Key Figures
- **Leslie Lamport**: Paxos, LaTeX, distributed systems pioneer
- **Barbara Liskov**: Practical Byzantine Fault Tolerance
- **Miguel Castro**: PBFT co-author
- **Diego Ongaro**: Raft consensus algorithm
- **John Ousterhout**: Raft co-author

## Evolution
From theoretical Paxos to practical Raft, consensus algorithms have evolved from academic curiosities to production-critical infrastructure components.
