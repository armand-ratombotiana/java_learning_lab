# Partitioning: Flashcards

## Front: What is sharding?
**Back**: Dividing data across multiple nodes (horizontal partitioning).

## Front: What is range sharding?
**Back**: Data divided by key value ranges.

## Front: What is hash sharding?
**Back**: Data divided by hash of key.

## Front: What is consistent hashing?
**Back**: Hash ring minimizing data movement on node changes.

## Front: What is a shard key?
**Back**: Attribute(s) determining data placement.

## Front: What is rebalancing?
**Back**: Moving data when nodes added/removed.

## Front: What are virtual nodes?
**Back**: Multiple tokens per real node for better distribution.

## Front: What is scatter/gather?
**Back**: Query all shards and merge results.
