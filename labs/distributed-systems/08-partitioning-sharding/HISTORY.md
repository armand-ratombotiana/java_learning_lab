# History of Partitioning

## Timeline
- **1990s**: Manual sharding in large web apps (LiveJournal, eBay)
- **1997**: Consistent hashing proposed by Karger et al.
- **2004**: Dynamo uses consistent hashing
- **2006**: BigTable uses range partitioning
- **2010**: MongoDB auto-sharding
- **2012**: Spanner uses directory-based partitioning
- **2016**: Vitess (YouTube sharding) open-sourced

## Key Figures
- **David Karger**: Consistent hashing inventor
- **Google team**: BigTable, Spanner partitioning
- **YouTube**: Vitess sharding layer

## Evolution
From manual application-level sharding to automated distributed database partitioning with dynamic rebalancing.
