# Partitioning: Exercises

## Exercise 1: Consistent Hashing
Implement a consistent hash ring with virtual nodes.

## Exercise 2: Range Sharding
Build a range-sharded key-value store with automatic splitting.

## Exercise 3: Shard Rebalancing
Simulate adding and removing nodes, measuring keys moved.

## Exercise 4: Cross-Shard Query
Implement a scatter/gather query across all shards.

## Exercise 5: Shard Key Analysis
Given a dataset, determine the best shard key for balanced distribution.
