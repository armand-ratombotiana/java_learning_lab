# Security — Gossip Protocols

## Threats
- Byzantine nodes spreading false information
- Sybil attacks (forging identities)
- Eavesdropping on gossip messages
- Flooding (excessive gossip traffic)

## Mitigations
- Message signing and verification
- Reputation systems for peer trust
- Rate limiting per peer
- Authenticated encryption for gossip channels
- Anomaly detection for unusual gossip patterns
