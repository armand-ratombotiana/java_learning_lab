# Exercises — Gossip Protocols

1. **Basic Gossip (Easy)**: Implement push gossip with random peer selection
2. **Pull Gossip (Medium)**: Implement pull-based gossip
3. **Push-Pull (Medium)**: Combine push and pull for faster convergence
4. **SWIM Protocol (Hard)**: Implement full SWIM with suspicion
5. **Failure Detection (Medium)**: Add phi-accrual failure detection
6. **Membership Protocol (Hard)**: Implement gossip-based membership
7. **Convergence Test (Easy)**: Verify all nodes converge within expected rounds
8. **Simulation (Medium)**: Build a gossip simulator with network delay
9. **Optimization (Hard)**: Implement adaptive fan-out
10. **Benchmark (Medium)**: Compare push vs pull vs push-pull convergence
