# History of Distributed ID Generation

## Timeline
- **1987**: UUID specification (RFC 4122)
- **1990s**: Database sequences (Oracle, PostgreSQL)
- **2005**: Flickr ticketing service (distributed sequence)
- **2010**: Twitter Snowflake (64-bit time-sorted IDs)
- **2014**: Sonyflake (Snowflake variant)
- **2022**: UUID v7 standardized

## Key Figures
- **Twitter team**: Snowflake (Rob Wultsch, others)
- **Flickr team**: Distributed ticket servers
- **IETF UUID WG**: UUID v6, v7, v8 standards

## Evolution
From simple UUIDs to custom time-based compact IDs optimized for specific infrastructure needs.
