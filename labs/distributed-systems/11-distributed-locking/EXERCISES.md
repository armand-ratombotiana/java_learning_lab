# Distributed Locking: Exercises

## Exercise 1: ZooKeeper Lock
Implement a distributed lock using ZooKeeper sequential ephemeral nodes.

## Exercise 2: etcd Lock
Implement a lock using etcd's lease mechanism.

## Exercise 3: Redlock
Implement the Redis Redlock algorithm with fencing tokens.

## Exercise 4: Fencing Token
Add fencing token protection to a shared resource access.

## Exercise 5: Leader Election
Build a leader election system using distributed locks.
