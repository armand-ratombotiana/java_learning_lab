# Distributed Caching: Exercises

## Exercise 1: Cache-Aside Implementation
Implement a cache-aside pattern with Redis as the distributed cache.

## Exercise 2: Consistent Hashing
Build a distributed cache that uses consistent hashing for key distribution.

## Exercise 3: Cache Invalidation
Implement a cache invalidation system that broadcasts invalidation messages.

## Exercise 4: Benchmarking
Compare throughput and latency of local vs distributed caching.

## Exercise 5: Hazelcast Configuration
Configure a Hazelcast cluster with near caching and map eviction policies.
