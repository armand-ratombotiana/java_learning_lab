# History of Distributed Caching

## Timeline
- **2000s**: Memcached for LiveJournal/Scalable web caching
- **2009**: Redis released by Salvatore Sanfilippo
- **2010**: Hazelcast founded (in-memory data grid)
- **2015**: Redis Cluster (native sharding)
- **2018**: Redis Streams and Modules

## Key Figures
- **Salvatore Sanfilippo**: Redis creator
- **Brad Fitzpatrick**: Memcached creator (LiveJournal)
- **Tal Kelman**: Hazelcast founder

## Evolution
From simple key-value caching (Memcached) to feature-rich data structure servers (Redis) to in-memory data grids (Hazelcast, Apache Ignite).
