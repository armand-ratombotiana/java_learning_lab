# Mental Models — Distributed ID Generation

1. **Factory Assembly Line**: Parts (bits) assembled into IDs
2. **License Plate System**: Unique identifiers per region (worker)
3. **Ticket Booth**: Sequential tickets per lane (sequence per ms)
4. **Shipping Labels**: Timestamp + location + batch number
5. **Social Security Numbers**: Structured to encode information
6. **Library Call Numbers**: Categorization + sequence for ordering
