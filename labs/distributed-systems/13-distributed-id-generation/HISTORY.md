# History — Distributed ID Generation

- **2010**: Twitter Snowflake paper published
- **2012**: Snowflake open-sourced
- **2014**: ULID specification created
- **2016**: UUID v6/v7 draft proposals
- **2022**: UUID v7 standardized in RFC 9562
- **2023**: Widespread adoption of Snowflake-like IDs
- **2024**: UUID v7 support in major databases (MySQL, Postgres)
