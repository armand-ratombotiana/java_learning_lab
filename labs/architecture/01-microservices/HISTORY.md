# History of Microservices

## Origins
- **2011** - First workshop on microservices at Venice, Italy
- **2012** - ThoughtWorks uses "microservice" as architectural style
- **2014** - Martin Fowler and James Lewis publish seminal article "Microservices"

## Key Milestones
- **Netflix** - Migrated from monolithic DVD rental to streaming microservices (2008-2015)
- **Amazon** - Bezos mandate (2002) requiring all teams to communicate via APIs
- **Uber** - Microservices to handle global ride-hailing scale
- **Spotify** - Squad model aligned with microservice boundaries

## Technology Evolution
- **2014** - Spring Cloud / Netflix OSS ecosystem matures
- **2016** - Docker containerization becomes standard
- **2018** - Service Mesh (Istio, Linkerd) adoption
- **2020+** - Kubernetes becomes de facto orchestration platform

## Java Ecosystem Timeline
- Spring Boot 1.0 (2014) - Simplified microservice creation
- Spring Cloud (2015) - Distributed systems patterns
- Spring Boot 2.0 (2018) - Reactive microservices
- Spring Boot 3.0 (2022) - Native image support with GraalVM
