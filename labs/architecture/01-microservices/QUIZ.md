# Microservices Quiz

## Question 1
What is the primary benefit of database-per-service pattern?
- a) Reduced storage costs
- b) Loose coupling and independent scaling
- c) Easier joins across services
- d) Single point of backup

**Answer: b)**

## Question 2
Which pattern prevents cascading failures in microservices?
- a) API Gateway
- b) Circuit Breaker
- c) Service Discovery
- d) Distributed Tracing

**Answer: b)**

## Question 3
What is the Strangler Fig pattern used for?
- a) Database sharding
- b) Gradual migration from monolith to microservices
- c) Circuit breaking logic
- d) Service discovery

**Answer: b)**

## Question 4
Which of the following is NOT a microservices benefit?
- a) Technology diversity
- b) Independent deployment
- c) Simplified distributed transactions
- d) Organizational alignment

**Answer: c)**

## Question 5
What does the CAP theorem state about distributed systems?
- a) Consistency, Availability, Performance
- b) Consistency, Availability, Partition Tolerance
- c) Cost, Availability, Performance
- d) Consistency, Atomicity, Partition Tolerance

**Answer: b)**

## Question 6
When should you NOT use microservices?
- a) Small team with simple domain
- b) Large engineering organization
- c) Complex domain with clear boundaries
- d) Need for independent scaling

**Answer: a)**
