# History of Event Sourcing

## Origins
- **2005** - Martin Fowler describes Event Sourcing pattern
- **2006** - Greg Young and others develop event sourcing concepts
- **2010** - CQRS and Event Sourcing become paired patterns

## Evolution
- **2011** - Event Store database created by Greg Young
- **2012** - Axon Framework for Java with event sourcing support
- **2013** - Kafka used as event store
- **2014** - Event Sourcing gains traction with microservices
- **2016** - Eventuate.io platform for event sourcing
- **2018** - Axon Server for dedicated event store
- **2020+** - Event Sourcing becomes mainstream pattern

## Key Influences
- Domain-Driven Design (Eric Evans, 2003)
- CQRS (Greg Young, 2009)
- Reactive Manifesto (2014)
- Microservices movement

## Java Ecosystem
- 2012: Axon Framework with ES support
- 2015: Spring Data event sourcing
- 2018: Axon 4.0 with improved ES
- 2021: Axon Server for dedicated ES
- 2023: Event Sourcing in Spring Modulith
