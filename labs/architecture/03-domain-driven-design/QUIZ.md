# DDD Quiz

## Question 1
What is a bounded context?
- a) A database schema boundary
- b) An explicit boundary where a particular domain model applies
- c) A microservice boundary
- d) A transaction boundary

**Answer: b)**

## Question 2
What distinguishes an Entity from a Value Object?
- a) Entity has identity, Value Object does not
- b) Value Object has identity, Entity does not
- c) Both have identity
- d) Neither has identity

**Answer: a)**

## Question 3
What is an Aggregate Root?
- a) Any entity in the model
- b) The root entity that controls access to an aggregate
- c) A database table
- d) A service class

**Answer: b)**

## Question 4
What is ubiquitous language?
- a) A programming language used everywhere
- b) A shared language between domain experts and developers
- c) SQL queries
- d) JSON schema

**Answer: b)**

## Question 5
What is an anemic domain model?
- a) A model with rich business logic
- b) A model with only data and no behavior
- c) A database-first approach
- d) A reactive programming model

**Answer: b)**

## Question 6
What is an anticorruption layer used for?
- a) Encrypting domain data
- b) Translating between bounded contexts
- c) Preventing SQL injection
- d) Caching database queries

**Answer: b)**
