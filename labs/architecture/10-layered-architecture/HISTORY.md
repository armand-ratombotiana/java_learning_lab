# History of Layered Architecture

## Origins
- **1970s** - Early layered operating systems (Dijkstra's THE system)
- **1980s** - OSI model (7-layer network model)
- **1990s** - Three-tier architecture for enterprise applications

## Evolution
- **1998** - Martin Fowler "Patterns of Enterprise Application Architecture"
- **2002** - J2EE BluePrints promote layered architecture
- **2003** - Spring Framework simplifies layering
- **2006** - Spring MVC formalizes web layer
- **2010** - Spring Data JPA for persistence layer
- **2014** - Spring Boot makes layered structure easy
- **2018+** - Microservices challenge traditional layering

## Java Evolution
- EJB 2.0 (heavy layers) -> Spring (lightweight layers)
- JSP/Servlet -> Spring MVC -> Spring Boot REST
- JDBC -> JPA/Hibernate -> Spring Data JPA
- Transaction demarcation: EJB CMT -> Spring @Transactional
