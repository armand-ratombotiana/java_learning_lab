# Layered Architecture Quiz

## Question 1
What are the three primary layers in layered architecture?
a) UI, Network, Database
b) Presentation, Business, Persistence
c) Client, Server, Data
d) Frontend, Backend, DevOps

**Answer: b)**

## Question 2
What is a layer violation?
a) Using too many layers
b) A dependency that skips a layer
c) Having too few layers
d) Using design patterns

**Answer: b)**

## Question 3
What belongs in the service layer?
a) HTTP request handling
b) Business logic and validation
c) Database queries
d) HTML rendering

**Answer: b)**

## Question 4
Why should controllers not directly access repositories?
a) Performance reasons
b) It violates the single responsibility principle
c) It requires more code
d) It's not possible in Spring

**Answer: b)**

## Question 5
What is a DTO used for?
a) Direct database access
b) Transferring data between layers without exposing entities
c) HTML rendering
d) Business validation

**Answer: b)**

## Question 6
What annotation marks a business layer component in Spring?
a) @Repository
b) @Service
c) @Controller
d) @Component

**Answer: b)**
