# History of CQRS

## Origins
- **2009** - Greg Young introduces the term CQRS
- **2010** - Udi Dahan popularizes CQRS in his talks
- **2010** - CQRS often paired with Event Sourcing

## Evolution
- **2010-2013** - Early adoption in .NET community
- **2012** - Axon Framework for Java launched
- **2014** - CQRS gains traction with microservices wave
- **2016** - Eventuate.io platform for CQRS/ES
- **2018** - Spring Modulith adds CQRS support
- **2020+** - CQRS becomes standard pattern for complex domains

## Key Influences
- Command-Query Separation (CQS) by Bertrand Meyer (1988)
- DDD by Eric Evans (2003)
- Event Sourcing by Martin Fowler
- Microservices architecture (2014+)

## Java Timeline
- 2012: Axon Framework
- 2016: Spring Cloud Stream
- 2019: Axon 4.0 released
- 2021: Axon Framework 4.5 with improved Spring Boot support
- 2023: CQRS support in Spring Modulith
