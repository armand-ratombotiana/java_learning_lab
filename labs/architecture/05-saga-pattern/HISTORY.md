# History of Saga Pattern

## Origins
- **1987** - Hector Garcia-Molina and Kenneth Salem introduce the Saga pattern in a paper
- **1989** - Paper "Sagas" published in ACM SIGMOD Record

## Evolution in Distributed Systems
- **2000s** - Sagas used in database research, not widely adopted
- **2014** - Chris Richardson revives Saga pattern for microservices
- **2015** - Eventuate framework for sagas in microservices
- **2016** - Axon Framework adds saga support
- **2018** - Camunda BPM for saga orchestration
- **2020** - Serverless saga implementations emerge
- **2022** - Temporal.io workflow engine for sagas

## Key Contributions
- **Chris Richardson** - Microservices patterns book, Eventuate framework
- **Caitie McCaffrey** - Distributed sagas at Twitter
- **Nikolay Ashanin** - Saga pattern implementations

## Java Ecosystem
- 2015: Eventuate Tram for saga coordination
- 2016: Axon Framework saga support
- 2018: Camunda Spring Boot integration
- 2020: Temporal Java SDK
- 2022: Spring Modulith saga support
