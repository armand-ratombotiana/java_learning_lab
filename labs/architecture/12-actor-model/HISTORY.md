# History of Actor Model

## Origins
- **1973** - Carl Hewitt introduces Actor Model at MIT
- **1977** - Hewitt and Baker formalize Actor Model
- **1985** - Gul Agha publishes "Actors" book

## Evolution
- **1986** - Erlang language developed at Ericsson (actor-like)
- **1998** - Erlang open-sourced, actor model gains attention
- **2006** - Scala language with actors in standard library
- **2009** - Akka framework for Scala/Java (Typesafe/Lightbend)
- **2012** - Akka 2.0 with typed actors (prototype)
- **2014** - Akka Cluster and Akka Persistence
- **2018** - Akka Typed becomes stable (Akka 2.6)
- **2022** - Akka 2.7 with improved typed API
- **2023** - Akka changes license, Apache Pekko forks

## Key Influences
- Erlang/OTP - Supervision, fault tolerance
- Scala actors - JVM actor model
- Akka - Industrial-strength actor model for Java/Scala
- Orleans (Microsoft) - Virtual actors
- Proto.Actor - Cross-platform actors

## Java Ecosystem
- 2009: Akka 1.0
- 2012: Akka 2.0 (untyped actors)
- 2018: Akka Typed (stable 2.6)
- 2021: Akka 2.7
- 2023: Apache Pekko (Akka fork)
