# Actor Model Quiz

## Question 1
What is an actor?
a) A database record
b) A fundamental unit of computation with state, behavior, and mailbox
c) A REST controller
d) A design pattern

**Answer: b)**

## Question 2
How do actors communicate?
a) Shared memory
b) Direct method calls
c) Message passing
d) Database writes

**Answer: c)**

## Question 3
What is the purpose of a supervisor?
a) To monitor performance
b) To handle actor failures and decide recovery strategy
c) To route messages
d) To manage configuration

**Answer: b)**

## Question 4
What does location transparency mean?
a) Actors don't have a location
b) Actors can be local or remote with the same programming model
c) Actors can move between machines
d) Actors don't need addresses

**Answer: b)**

## Question 5
What is a mailbox in actor model?
a) An email address
b) A queue where incoming messages are stored
c) A database table
d) A network endpoint

**Answer: b)**

## Question 6
Who introduced the Actor Model?
a) Martin Fowler
b) Carl Hewitt
c) Eric Evans
d) Robert C. Martin

**Answer: b)**
