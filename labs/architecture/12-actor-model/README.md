# Actor Model

## Overview
The Actor Model is a conceptual model for building concurrent and distributed systems. Actors are the fundamental units of computation - they encapsulate state and behavior, communicate exclusively through message passing, and process messages asynchronously.

## Topics Covered
- Actor fundamentals (state, behavior, mailbox)
- Message passing and immutability
- Supervision hierarchies and fault tolerance
- Actor lifecycle and state management
- Akka framework for Java
- Cluster and distributed actors
- Testing actor systems

## Java/Spring Stack
- Akka framework (akka-actor, akka-typed)
- Akka Cluster for distributed actors
- Akka Persistence for event-sourced actors
- Akka HTTP for actor-based web services
- Pekko (Apache incubation, Akka fork)
- Lightbend Lagom for microservices with actors
