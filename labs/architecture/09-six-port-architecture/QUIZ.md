# Six-Port Architecture Quiz

## Question 1
How many port types does Six-Port Architecture define?
a) 4
b) 6
c) 8
d) 3

**Answer: b)**

## Question 2
Which port type handles REST API calls?
a) Outbound Driven
b) Inbound Driving
c) Event Publisher
d) Notification

**Answer: b)**

## Question 3
What is an Outbound Driven Port used for?
a) Receiving HTTP requests
b) Persistence operations
c) Sending emails
d) Consuming events

**Answer: b)**

## Question 4
What suffix should port interfaces use in Six-Port?
a) Interface
b) Port
c) Service
d) Adapter

**Answer: b) Port**

## Question 5
Which port type would a Stripe payment client implement?
a) Notification Port
b) Outbound Driving Port
c) Inbound Driving Port
d) Event Publisher Port

**Answer: b) Outbound Driving Port**

## Question 6
What is the key difference from Hexagonal Architecture?
a) Six-Port has more layers
b) Six-Port explicitly defines 6 port types
c) Six-Port doesn't use dependency injection
d) No difference

**Answer: b)**
