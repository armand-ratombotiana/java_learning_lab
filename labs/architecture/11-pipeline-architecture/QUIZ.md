# Pipeline Architecture Quiz

## Question 1
What is a pipeline stage?
a) A database table
b) A processing unit that transforms input to output
c) A network endpoint
d) A user interface component

**Answer: b)**

## Question 2
What is the Chain of Responsibility pattern?
a) A security pattern
b) A pattern where multiple handlers each decide to process or pass the request
c) A database pattern
d) A UI pattern

**Answer: b)**

## Question 3
What does a filter stage do?
a) Transforms data format
b) Removes items based on criteria
c) Adds external data
d) Combines multiple items

**Answer: b)**

## Question 4
Why should pipeline stages be stateless?
a) Performance reasons
b) To ensure they can be composed and tested independently
c) Security requirements
d) Database constraints

**Answer: b)**

## Question 5
What is the origin of the pipeline pattern?
a) Object-oriented programming
b) Unix pipes
c) Database systems
d) Web services

**Answer: b)**

## Question 6
How does a parallel pipeline improve performance?
a) By running stages on different threads/cores
b) By reducing the number of stages
c) By skipping validation
d) By using faster algorithms

**Answer: a)**
