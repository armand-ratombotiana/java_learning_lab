# History of 09-authorization-rbac

## Timeline

### 1990s
- HTTP Basic Authentication defined in RFC 2617
- Early web applications use simple session cookies
- SSL 3.0 introduces secure transport

### 2000s
- JAAS introduced in Java 1.4
- Acegi Security (predecessor to Spring Security) released in 2004
- OWASP Top 10 established

### 2010s
- Spring Security 3.0 introduces Java configuration
- OAuth 2.0 (RFC 6749) published in 2012
- JWT (RFC 7519) standardized in 2015
- Spring Security 5.0 introduces reactive security support

### 2020s
- Spring Security 6.0 removes deprecated features
- Zero Trust architecture becomes mainstream
- OAuth 2.1 consolidates best practices

## Key Milestones in Java/Spring

| Year | Milestone |
