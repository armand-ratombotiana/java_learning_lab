# 12-GitOps - GitOps Workflow

## Overview
GitOps is a methodology for managing infrastructure and applications using Git as the single source of truth. This lab covers ArgoCD, Flux, declarative deployments, and GitOps workflows.

## Prerequisites
- Kubernetes fundamentals (Lab 02)
- Git fundamentals
- Helm basics (Lab 06)

## Learning Objectives
- Understand GitOps principles and workflow
- Deploy and configure ArgoCD
- Implement GitOps with Flux
- Manage applications declaratively via Git
- Implement progressive delivery and auto-remediation
