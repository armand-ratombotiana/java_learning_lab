# 11-Secrets Management - Vault & Secrets

## Overview
Secrets management covers storing, accessing, and rotating sensitive data (passwords, API keys, certificates). This lab focuses on HashiCorp Vault and secrets management best practices.

## Prerequisites
- Basic Linux command-line
- Docker fundamentals (Lab 01)

## Learning Objectives
- Understand secrets management principles
- Deploy and configure HashiCorp Vault
- Implement dynamic secrets and database credentials
- Configure secret rotation policies
- Integrate Vault with applications and Kubernetes
