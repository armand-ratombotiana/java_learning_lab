# Cloud Deployment Exercises

## Exercise 1: Cloud CLI
Install and configure your cloud provider CLI. Verify authentication.

## Exercise 2: Create K8s Cluster
Create an EKS/GKE/AKS cluster using CLI commands. Connect kubectl.

## Exercise 3: Deploy a Sample App
Deploy a stateless application (e.g., nginx) to your cloud K8s cluster. Expose via LoadBalancer.

## Exercise 4: Managed Database
Create a managed database (RDS/Cloud SQL/Azure SQL). Configure the app to use it.

## Exercise 5: IAM Configuration
Create a service account with workload identity. Limit permissions to S3 read.

## Exercise 6: CI/CD Pipeline
Set up GitHub Actions to build, push to container registry, and deploy to cloud K8s.

## Exercise 7: Auto-scaling
Configure HPA + cluster autoscaler on your cloud cluster. Test with load generation.

## Exercise 8: Cost Analysis
Use cloud cost explorer/ billing tools to analyze current spending. Set up budget alerts.

## Exercise 9: Multi-Region
Deploy the same application in two regions. Set up global load balancing.
