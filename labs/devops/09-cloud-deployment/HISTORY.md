# History of Cloud Deployment

- **2006 (Mar)**: AWS launches S3 (Simple Storage Service).
- **2006 (Aug)**: AWS launches EC2 (Elastic Compute Cloud) — beginning of IaaS.
- **2008**: Google App Engine launches (first PaaS).
- **2009**: AWS Elastic Beanstalk; Heroku launches.
- **2010**: Microsoft Azure IaaS launches; OpenStack (open-source cloud) announced.
- **2011**: AWS launches IAM, VPC; Netflix open-sources cloud tools.
- **2013**: Docker containers; Google Cloud Platform full launch.
- **2014**: AWS Lambda (serverless); Kubernetes open-sourced.
- **2015**: AWS ECS (container service); Azure Container Service.
- **2016**: AWS EKS announced; GCP Cloud Functions; Azure Functions.
- **2018-2020**: Serverless adoption explodes; multi-cloud strategies emerge.
- **2021-2024**: Cloud-native becomes default; edge computing; AI/ML cloud services.

Cloud evolved from simple VMs to a complete ecosystem of managed services, containers, serverless, and AI.
