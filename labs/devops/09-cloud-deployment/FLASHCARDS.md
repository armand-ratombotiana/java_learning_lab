# Cloud Deployment Flashcards

**Q: What is EKS?**
A: Amazon Elastic Kubernetes Service — managed K8s on AWS.

**Q: What is GKE?**
A: Google Kubernetes Engine — managed K8s on GCP.

**Q: What is AKS?**
A: Azure Kubernetes Service — managed K8s on Azure.

**Q: What is IRSA?**
A: IAM Roles for Service Accounts (AWS) — maps K8s SA to IAM role.

**Q: What is Workload Identity?**
A: GCP feature mapping K8s SA to GCP IAM service accounts.

**Q: What is a VPC?**
A: Virtual Private Cloud — isolated network within a cloud provider.

**Q: What is a NAT Gateway?**
A: Allows private subnet resources to access internet (outbound only).

**Q: What is the shared responsibility model?**
A: Cloud provider secures infrastructure; customer secures data and configurations.

**Q: What is CloudFront?**
A: AWS CDN service for low-latency content delivery globally.

**Q: What is an ALB vs NLB?**
A: ALB (L7 HTTP/gRPC), NLB (L4 TCP/UDP) — AWS load balancer types.
