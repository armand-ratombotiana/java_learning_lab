# 09-Cloud Deployment - Cloud-Native Deployments

## Overview
Cloud deployment covers deploying applications to AWS, GCP, and Azure using managed services and cloud-native patterns. This lab focuses on Kubernetes in the cloud, managed services, and deployment best practices.

## Prerequisites
- Kubernetes basics (Lab 02)
- Terraform basics (Lab 04)
- A cloud provider account (AWS/GCP/Azure)

## Learning Objectives
- Deploy Kubernetes clusters on major cloud providers
- Use managed cloud services (RDS, Cloud SQL, Cosmos DB)
- Implement cloud-native storage, networking, and security patterns
- Configure CI/CD pipelines for cloud deployments
- Understand multi-cloud and hybrid cloud strategies
