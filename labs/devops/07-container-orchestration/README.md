# 07-Container Orchestration - Advanced Kubernetes

## Overview
Advanced Kubernetes orchestration concepts including auto-scaling, rolling updates, probes, resource management, and cluster operations.

## Prerequisites
- Kubernetes fundamentals (Lab 02)
- Helm basics (Lab 06)

## Learning Objectives
- Configure Horizontal and Vertical Pod Autoscaling
- Implement advanced deployment strategies (blue/green, canary)
- Configure liveness, readiness, and startup probes
- Manage cluster resources with quotas and limits
- Use Pod Disruption Budgets for high availability
