# History of Monitoring & Logging

- **1990s**: SNMP-based monitoring (MRTG, RRDtool); syslog for logs.
- **2000s**: Nagios (alerting), Cacti (graphs), Splunk (log management, proprietary).
- **2006**: Graphite — first scalable time-series database and graphing tool.
- **2010**: Elasticsearch 0.4 released; Kibana, Logstash follow (ELK stack born).
- **2012**: Prometheus started at SoundCloud; InfluxDB released.
- **2014**: Grafana v1.0 released (fork of Kibana 3 for Graphite). Prometheus open-sourced.
- **2015**: Prometheus graduated in CNCF; Elasticsearch becomes most popular search engine.
- **2016**: Prometheus 1.0; Grafana 3.0 with alerting; Jaeger (tracing) open-sourced by Uber.
- **2018**: Grafana 5.0 with native Prometheus support; Loki (log aggregation) announced.
- **2020-2024**: OpenTelemetry becomes standard for observability data collection; eBPF enables kernel-level observability; AIOps emerges.

The field evolved from simple SNMP checks to comprehensive, open-source observability platforms.
