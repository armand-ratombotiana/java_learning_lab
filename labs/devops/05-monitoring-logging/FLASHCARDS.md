# Monitoring & Logging Flashcards

**Q: What is Prometheus?**
A: Open-source time-series database and monitoring system (CNCF graduated).

**Q: What is Grafana?**
A: Open-source observability and data visualization platform.

**Q: What is the ELK stack?**
A: Elasticsearch (store), Logstash (process), Kibana (visualize).

**Q: What is a Counter in Prometheus?**
A: Cumulative metric that only increases (requests total, errors total).

**Q: What is a Gauge in Prometheus?**
A: Point-in-time metric that can go up or down (CPU, memory, temperature).

**Q: What is PromQL?**
A: Prometheus Query Language for selecting and aggregating time-series data.

**Q: What does Alertmanager do?**
A: Handles alert deduplication, grouping, routing, silencing, and notifications.

**Q: What is Filebeat?**
A: Lightweight log shipper that tails log files and forwards them.

**Q: What is structured logging?**
A: Logging in JSON format with key-value pairs for machine parsing.

**Q: What is an SLO?**
A: Service Level Objective — target for service reliability (e.g., 99.9% uptime).
