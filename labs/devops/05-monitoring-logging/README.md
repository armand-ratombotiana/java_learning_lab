# 05-Monitoring-Logging - Monitoring & Logging

## Overview
Monitoring and logging are essential for observability in distributed systems. This lab covers Prometheus, Grafana, ELK Stack, metrics collection, log management, and alerting.

## Prerequisites
- Docker and Kubernetes basics (Labs 01-02)
- Understanding of application metrics

## Learning Objectives
- Deploy and configure Prometheus for metric collection
- Create Grafana dashboards for visualization
- Set up ELK stack (Elasticsearch, Logstash, Kibana) for log aggregation
- Define alerting rules and notification channels
- Understand metrics, logs, and traces (three pillars of observability)
