# Kubernetes Interview Questions

## Junior
1. What is a Pod and how is it different from a container?
2. How do you scale a Deployment?
3. What is kubectl and how do you use it?

## Mid-Level
4. Explain the Kubernetes control plane components.
5. How does the scheduler work?
6. What's the difference between a Service and an Ingress?
7. How do ConfigMaps and Secrets work?

## Senior
8. Explain etcd's role and quorum requirements.
9. How do network policies work and why are they important?
10. Design a multi-tenant Kubernetes cluster.
11. How does the API server handle authentication and authorization?
12. Explain the container runtime interface (CRI).

## Principal/Architect
13. Design a hybrid cloud Kubernetes platform across AWS and on-premises.
14. How do you implement disaster recovery and backup for K8s?
15. Compare service mesh approaches (Istio vs Linkerd vs Cilium).
