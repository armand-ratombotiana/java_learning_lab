# History of Kubernetes

- **2003-2006**: Google develops Borg (internal cluster management system) and Omega.
- **2014 (Jun)**: Kubernetes v1.0 announced at Google I/O, open-sourced.
- **2015 (Jul)**: Kubernetes v1.0 released; donated to CNCF (Cloud Native Computing Foundation).
- **2016**: Ingress, Deployments, DaemonSets, StatefulSets features added.
- **2017**: Container Runtime Interface (CRI); API aggregation layer; Custom Resource Definitions (CRDs).
- **2018**: Kubernetes graduates from CNCF incubation (v1.10); Helm becomes CNCF project.
- **2019-2020**: Stability improvements; IPv6 support; Windows containers GA; ephemeral containers.
- **2021**: CronJobs GA; topology-aware routing; PodSecurityPolicy replaced by Pod Security Standards.
- **2022-2024**: Sidecar containers; Ingress v1 GA; KMS v2 for encryption; Kubestronaut program.

Kubernetes went from Google's internal tool to the de facto standard for container orchestration.
