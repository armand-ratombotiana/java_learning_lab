# 02-Kubernetes - Container Orchestration

## Overview
Kubernetes (K8s) is an open-source platform for automating container deployment, scaling, and management. This lab covers architecture, pods, deployments, services, ConfigMaps, Secrets, and Ingress.

## Prerequisites
- Docker fundamentals (Lab 01)
- Basic YAML knowledge

## Learning Objectives
- Understand K8s architecture and control plane components
- Deploy and manage pods, deployments, and services
- Configure applications with ConfigMaps and Secrets
- Expose services via Ingress controllers
