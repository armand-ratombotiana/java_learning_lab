# 06-Helm - Kubernetes Package Manager

## Overview
Helm is the package manager for Kubernetes, simplifying deployment and management of applications. This lab covers Helm charts, templates, values, releases, and package management.

## Prerequisites
- Kubernetes basics (Lab 02)
- kubectl configured with a cluster

## Learning Objectives
- Understand Helm architecture and chart structure
- Create and customize Helm charts
- Manage releases with Helm CLI
- Package and distribute charts via repositories
- Use advanced template features
