# Helm Interview Questions

## Junior
1. What is Helm and why is it useful?
2. How do you install and upgrade a chart?
3. What are values in Helm?

## Mid-Level
4. Explain the Helm 3 architecture and how it differs from Helm 2.
5. How do you handle chart dependencies?
6. What is the three-way merge strategy?
7. How do you create reusable template functions?

## Senior
8. Design a Helm chart for a microservice with ConfigMaps, Secrets, and Ingress.
9. How do you handle secrets in Helm charts securely?
10. Explain the trade-offs of using Helm vs Kustomize.
11. How do you implement CI/CD for Helm charts?

## Principal/Architect
12. Design a Helm chart governance strategy for an enterprise.
13. How do you manage 100+ Helm releases across multiple clusters?
14. Compare OCI vs HTTP chart repositories for enterprise use.
15. How do you implement policy enforcement for Helm deployments?
