# Docker Interview Questions

## Junior
1. What is Docker and why use it?
2. Explain the difference between docker run and docker start.
3. How do you view logs from a container?

## Mid-Level
4. Explain the Docker build cache mechanism.
5. How do you handle persistent storage in Docker?
6. What's the difference between bridge and host networking?
7. Describe multi-stage builds and their benefits.

## Senior
8. How does Docker achieve isolation without a hypervisor?
9. Explain the architecture of dockerd, containerd, and runc.
10. How would you secure a production Docker host?
11. Design a CI/CD pipeline using Docker for a microservice architecture.
12. How do you handle secrets in Docker without exposing them in images?

## Principal/Architect
13. Compare Docker Swarm vs Kubernetes for container orchestration.
14. Design a multi-tenant container platform with security isolation.
15. How do you implement container image governance at scale?
