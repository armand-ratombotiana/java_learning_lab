# 01-Docker - Docker Fundamentals

## Overview
Docker is a containerization platform that enables developers to package applications and dependencies into lightweight, portable containers. This lab covers images, containers, Dockerfile, Docker Compose, and multi-stage builds.

## Prerequisites
- Basic Linux command-line knowledge
- Docker installed (v20.10+ recommended)

## Learning Objectives
- Understand container vs VM differences
- Build and run Docker containers
- Write Dockerfiles using best practices
- Orchestrate multi-container apps with Docker Compose
- Optimize images with multi-stage builds

## Lab Structure
- `THEORY.md` - Core containerization concepts
- `CODE_DEEP_DIVE.md` - Hands-on Dockerfile examples
- `EXERCISES.md` - Practice exercises
- All other standard DevOps Academy files
