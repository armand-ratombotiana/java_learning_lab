# History of Docker

- **2008**: LXC (Linux Containers) project launched, providing kernel-level isolation via cgroups and namespaces.
- **2013 (Mar)**: Docker Inc. (then dotCloud) releases Docker as open-source at PyCon.
- **2013 (Aug)**: Docker 0.6 with `docker run`, `docker pull`, Docker Hub.
- **2014 (Jun)**: Docker 1.0 released at DockerCon.
- **2014 (Dec)**: Docker Compose (formerly Fig) acquired and open-sourced.
- **2015**: OCI (Open Container Initiative) formed to standardize container runtimes and image formats.
- **2016**: Docker Swarm mode built into Docker Engine; containerd spun out to CNCF.
- **2017**: Moby project created; Docker Enterprise edition launched.
- **2019**: Docker Inc. sells Enterprise business to Mirantis.
- **2020-2024**: Docker Desktop growth; WSL2 integration; Docker Scout supply chain features.

Docker's creation sparked the containerization revolution that reshaped DevOps.
