# 03-CI-CD-Pipelines - CI/CD Pipelines

## Overview
Continuous Integration and Continuous Delivery automate building, testing, and deploying code changes. This lab covers GitHub Actions, Jenkins pipelines, and CI/CD best practices.

## Prerequisites
- Git fundamentals
- Docker basics (Lab 01)

## Learning Objectives
- Understand CI/CD principles and pipeline stages
- Build GitHub Actions workflows
- Create Jenkins declarative pipelines
- Implement build, test, security scan, and deploy stages
