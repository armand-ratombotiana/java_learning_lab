# CI/CD Interview Questions

## Junior
1. What is CI/CD and why is it important?
2. What's the difference between build, test, and deploy stages?
3. How do you view pipeline logs?

## Mid-Level
4. Explain how GitHub Actions workflows are structured.
5. How do you handle secrets in CI/CD pipelines?
6. What strategies can you use to speed up a slow pipeline?
7. How do you implement deployment approval gates?

## Senior
8. Design a multi-environment CI/CD pipeline with security scanning.
9. How do you ensure supply chain security in your pipeline?
10. Compare GitHub Actions vs Jenkins vs GitLab CI.
11. How do you handle database migrations in CI/CD?
12. What metrics do you track for pipeline health?

## Principal/Architect
13. Design a GitOps-based CI/CD platform for 200 microservices.
14. How do you implement canary deployments in CI/CD?
15. How do you build a self-service CI/CD platform for multiple teams?
