# Service Mesh Interview Questions

## Junior
1. What problem does a service mesh solve?
2. What is a sidecar proxy?
3. How does Istio improve security?

## Mid-Level
4. Explain the difference between Istio control plane and data plane.
5. How do VirtualService and DestinationRule work together?
6. How does mTLS work in Istio?
7. What metrics does a service mesh provide?

## Senior
8. Design a multi-cluster service mesh with global load balancing.
9. How do you migrate a legacy application to Istio without downtime?
10. Compare Istio, Linkerd, and Cilium service mesh implementations.
11. How do you troubleshoot mTLS failures in production?

## Principal/Architect
12. Design a zero-trust network architecture using service mesh.
13. How do you manage service mesh across multiple Kubernetes clusters in different regions?
14. Compare sidecar vs sidecar-less (Ambient Mesh) approaches.
15. How does service mesh integrate with existing security compliance frameworks?
