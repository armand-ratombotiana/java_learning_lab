# Terraform Interview Questions

## Junior
1. What is Infrastructure as Code and why use it?
2. Explain `terraform init`, `plan`, and `apply`.
3. What is the Terraform state file?

## Mid-Level
4. How does Terraform manage dependencies between resources?
5. What's the difference between `count` and `for_each`?
6. How do you handle secrets and sensitive data in Terraform?
7. Explain remote state backends and state locking.

## Senior
8. How would you structure a Terraform repository for a large organization?
9. Explain the Terraform provider plugin architecture.
10. How do you handle Terraform upgrades across a team?
11. What strategies do you use for Terraform CI/CD?
12. How do you handle state migration during refactoring?

## Principal/Architect
13. Design a multi-cloud Terraform strategy with compliance enforcement.
14. Compare Terraform Cloud, Terragrunt, and OpenTofu.
15. How do you implement Terraform policy as code at enterprise scale?
