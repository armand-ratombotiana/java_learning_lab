# 04-Terraform - Infrastructure as Code

## Overview
Terraform by HashiCorp is an Infrastructure as Code (IaC) tool for building, changing, and versioning infrastructure safely and efficiently. This lab covers Terraform syntax, providers, state management, and modules.

## Prerequisites
- Cloud fundamentals (AWS/GCP/Azure)
- Basic HCL syntax knowledge

## Learning Objectives
- Write Terraform configurations using HCL
- Understand provider architecture and authentication
- Manage Terraform state (local and remote)
- Create and use reusable modules
- Apply infrastructure changes safely with plans
