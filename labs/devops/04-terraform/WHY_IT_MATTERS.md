# Why Terraform Matters

Terraform has become the standard IaC tool:

- **Multi-Cloud**: Manage AWS, GCP, Azure, and on-premises with one tool and language.
- **Version Control**: Infrastructure changes go through code review like application code.
- **Self-Service**: Teams provision infrastructure via approved modules without cloud console access.
- **Compliance**: Policy as Code (Sentinel/OPA) enforces guardrails on infrastructure changes.
- **Disaster Recovery**: Reproduce entire environments from code in minutes.
- **Collaboration**: Remote state locking prevents concurrent modifications.

Terraform Cloud/Enterprise provides collaboration, policy enforcement, and cost estimation. Over 80% of organizations use Terraform for IaC (2024 HashiCorp user survey).
