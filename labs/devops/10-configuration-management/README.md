# 10-Configuration Management - Ansible, Puppet, Chef

## Overview
Configuration management automates server provisioning, configuration, and maintenance. This lab covers Ansible, Puppet, Chef, and idempotent infrastructure principles.

## Prerequisites
- Linux system administration basics
- Infrastructure as Code concepts (Lab 04)

## Learning Objectives
- Understand configuration management principles and idempotency
- Write Ansible playbooks and roles
- Understand Puppet manifests and Chef cookbooks
- Manage infrastructure at scale with automation
- Implement configuration drift detection and remediation
