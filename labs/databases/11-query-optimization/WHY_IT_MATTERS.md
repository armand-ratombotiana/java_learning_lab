# Why Query Optimization Matters

Query optimization directly impacts:

- **User experience**: Slow pages drive users away (53% abandon >3s load)
- **Infrastructure costs**: Efficient queries need fewer/smaller database instances
- **Scalability**: Optimized queries handle 10x more traffic on same hardware
- **Bottleneck prevention**: Database is often the first system bottleneck
- **Release confidence**: No surprise production degradation from new queries
- **Sleep quality**: No 3am pages about database CPU at 100%
