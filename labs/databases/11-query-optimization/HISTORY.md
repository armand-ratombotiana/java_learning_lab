# History: Query Optimization

- **1970s** – System R introduces cost-based query optimization
- **1980s** – B-tree indexes become standard, R*-trees for spatial
- **1990s** – Database optimizers mature (Oracle CBO, PostgreSQL)
- **2000s** – EXPLAIN ANALYZE tools, graphical plan viewers
- **2005** – pgFouine (PostgreSQL log analyzer) released
- **2010** – Automatic query tuning in commercial databases
- **2015** – pg_stat_statements for query performance monitoring
- **2018** – eBPF-based database monitoring, JIT compilation for queries
- **2020s** – ML-based query optimization, auto-indexing features
