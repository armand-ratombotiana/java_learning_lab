# History: R2DBC

- **2017** – R2DBC initiative announced by Spring team (Mark Paluch, Oliver Drotbohm)
- **2018** – R2DBC 0.8 preview: PostgreSQL, H2 support
- **2019** – R2DBC 0.9: Connection pooling, transaction management
- **2020** – Spring Data R2DBC 1.0 GA, R2DBC 1.0.0 RELEASE (Aranea)
- **2021** – R2DBC 1.1: MSSQL support, MySQL driver maturity
- **2022** – R2DBC 1.2: SPI improvements, connection lifecycle hooks
- **2023** – R2DBC 1.3: Observability (Micrometer), better PostGIS support
- **2024** – R2DBC 1.4+: Reactive transaction demarcation improvements
