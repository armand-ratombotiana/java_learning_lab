# 02 - SQL Fundamentals

## Focus
SQL syntax, SELECT, JOIN, GROUP BY, subqueries, window functions with Java/JDBC/JPA examples.

## Topics Covered
- SELECT queries and filtering
- JOIN types (INNER, LEFT, RIGHT, FULL, CROSS)
- GROUP BY and aggregate functions
- Subqueries (correlated vs non-correlated)
- Window functions (ROW_NUMBER, RANK, LAG, LEAD)
- Common Table Expressions (CTEs)
- JDBC query execution
- JPA @Query and native queries
