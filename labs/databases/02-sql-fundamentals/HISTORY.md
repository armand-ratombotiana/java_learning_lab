# History of SQL

## Timeline

| Year | Event |
|---|---|
| 1970 | Codd proposes relational model |
| 1974 | Chamberlin & Boyce develop SEQUEL (later SQL) |
| 1979 | Oracle ships first commercial SQL database |
| 1986 | SQL-86 (ANSI) first standard |
| 1989 | SQL-89 (minor revision) |
| 1992 | SQL-92 (major revision) – the standard most DBs target |
| 1999 | SQL:1999 – recursive queries, triggers, OO features |
| 2003 | SQL:2003 – XML, window functions, sequences |
| 2006 | SQL:2006 – XML-related features |
| 2008 | SQL:2008 – INSTEAD OF triggers, TRUNCATE |
| 2011 | SQL:2011 – temporal tables (periods) |
| 2016 | SQL:2016 – JSON support, polymorphic tables |
| 2019 | SQL:2019 – multidimensional arrays |
| 2023 | SQL:2023 – property graphs, SQL/PGQ |

## Key Contributors
- **Donald Chamberlin**: Co-creator of SQL (SEQUEL)
- **Raymond Boyce**: Co-creator of SQL
- **ISO/IEC JTC 1/SC 32**: Maintains the SQL standard
