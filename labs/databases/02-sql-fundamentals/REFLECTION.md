# Reflection: SQL Fundamentals

## Key Takeaways
- SQL is declarative – you describe WHAT, not HOW
- Understanding execution order is critical for debugging
- NULLs require special handling (3-valued logic)
- Window functions are powerful alternatives to complex subqueries
- PreparedStatements are mandatory for security

## Questions to Consider
