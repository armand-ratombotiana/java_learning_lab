# LEETCODE_SOLUTIONS — SQL Fundamentals

## Basic SQL Queries

| Problem | Topic | Oracle Solution |
|---------|-------|-----------------|
| LeetCode 175 | Combining tables | LEFT JOIN |
| LeetCode 176 | Second highest salary | OFFSET / subquery with MAX |
| LeetCode 182 | Duplicate emails | GROUP BY + HAVING |
| LeetCode 183 | Customers no orders | LEFT JOIN + IS NULL |
| LeetCode 595 | Big countries | WHERE OR condition |
| LeetCode 596 | Classes 5+ students | GROUP BY + HAVING |
| LeetCode 620 | Not boring movies | MOD + ORDER BY |
| LeetCode 627 | Swap salary | CASE expression |

### Key Takeaway
Master basic JOINs, GROUP BY with HAVING, and CASE expressions before moving to window functions.
