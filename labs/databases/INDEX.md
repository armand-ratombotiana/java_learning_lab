## Overview
This expanded academy covers three pillars of Oracle technology. Master analytical SQL, procedural database logic with PL/SQL, and low-code application development with Oracle APEX.

## Curriculum Map

### Databases Academy — Level 5: SQL + PL/SQL Deep Dive

| # | Lab | Focus | Duration |
|---|-----|-------|----------|

| 23 | [PL/SQL Advanced](./23-pl-sql-advanced/) | Pipelined table functions, result cache, VPD, scheduling, AQ, ORDS, JSON, SODA, security, profiling, unit testing, code coverage | 8-10 hrs |
