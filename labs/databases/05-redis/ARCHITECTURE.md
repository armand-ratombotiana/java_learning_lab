# Architecture: Redis

## Single Node Application
```
┌─────────────────────────────────────────────────────┐
│ Application Server                                  │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐  │
│  │ Controller│  │ Service  │  │ RedisTemplate    │  │
│  └──────────┘  └────┬─────┘  └────────┬─────────┘  │
│                      │                 │            │
└──────────────────────┼─────────────────┼────────────┘
                       │                 │
              ┌────────┴─────────┐       │
              │   PostgreSQL     │       │
              │   (Primary DB)   │       │
              └──────────────────┘       │
                                  ┌──────┴──────┐
                                  │   Redis      │
                                  │   (Cache)    │
                                  └─────────────┘
```

```
                         ┌─────────────┐
                         │  Sentinel   │

                         └──────┬──────┘
                                │ monitoring
        ┌───────────────────────┼───────────────────────┐
        │                       │                       │
   ┌────┴────┐           ┌─────┴─────┐           ┌─────┴─────┐

```
