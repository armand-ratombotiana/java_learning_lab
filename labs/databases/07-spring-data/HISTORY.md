# History: Spring Data JPA

- **2008** – Spring Source launches Spring Data project to simplify data access
- **2009** – Spring Data JPA 1.0 released alongside Spring 3.0
- **2011** – Query DSL integration, Specifications support added
- **2013** – Spring Data JPA 1.4 introduces `@Query` with SpEL, auditing
- **2015** – Spring Data Commons 1.13, JPA 2.1 support
- **2017** – Spring Data Kay (2.0): `Optional` return types, immutable entities
- **2019** – Spring Data Neumann (2.3): Kotlin support, `QuerydslPredicateExecutor`
- **2021** – Spring Data 2021.0: Hibernate 5.5, Jakarta EE 9 migration path
- **2023** – Spring Data 3.x: Jakarta EE 10, Hibernate 6.x, native queries with records
