# Flashcards: Multi-Model & Polyglot

**Q:** What is polyglot persistence?
**A:** Using multiple different database technologies in the same application.

**Q:** What is a multi-model database?
**A:** A single database engine that supports multiple data models (e.g., ArangoDB, Cosmos DB).

**Q:** What is the best data model for ACID transactions?
**A:** Relational (RDBMS).

**Q:** What is the best data model for flexible schemas?
**A:** Document (MongoDB, Couchbase).

**Q:** What is the best data model for relationship traversal?
**A:** Graph (Neo4j, JanusGraph).

**Q:** What is the best data model for high-speed caching?
**A:** Key-Value (Redis, Memcached).

**Q:** What is the CAP theorem?
**A:** Distributed databases can only guarantee two of: Consistency, Availability, Partition Tolerance.

**Q:** What is the Saga pattern?
**A:** A sequence of local transactions with compensating actions for rollback across microservices.
