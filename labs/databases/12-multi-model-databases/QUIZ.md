# Quiz: Multi-Model & Polyglot

## Question 1
What is polyglot persistence?
- A) Using multiple databases of different types for different data needs
- B) Using multiple schemas in the same database
- C) Writing database queries in multiple languages
- D) Using a single database that supports multiple data models

<details><summary>Answer</summary>A</details>

## Question 2
Which data model is best for recommendation engines?
- A) Relational
- B) Document
- C) Graph
- D) Key-Value

<details><summary>Answer</summary>C</details>

## Question 3
What is a multi-model database?
- A) A database with multiple tables
- B) A single database engine supporting multiple data models
- C) A database cluster with multiple nodes
- D) A database with multiple indexes

<details><summary>Answer</summary>B</details>

## Question 4
True or False: Polyglot persistence guarantees strong consistency across all databases.

<details><summary>Answer</summary>False – heterogeneous databases cannot participate in a global distributed transaction.</details>

## Question 5
Which pattern is recommended for synchronizing data across heterogeneous databases?
- A) Distributed transactions (2PC)
- B) XA transactions
- C) Outbox pattern with CDC
- D) Dual writes without coordination

<details><summary>Answer</summary>C</details>
