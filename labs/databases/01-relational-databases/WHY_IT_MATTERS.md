# Why Relational Databases Matter

Relational databases remain the **dominant data storage technology** in enterprise software.

## Business Impact
- **Data integrity** through constraints, transactions, and ACID
- **Ad-hoc querying** enables BI and reporting without code changes
- **Standardization** – SQL is a universal language across vendors
- **Maturity** – decades of optimization, tooling, and expertise

## Industry Adoption
- ~80% of enterprise applications use a relational database
- Oracle, PostgreSQL, MySQL, SQL Server dominate
- Cloud RDS offerings (AWS RDS, Azure SQL, Cloud SQL) are core infrastructure

## Java Ecosystem
- JDBC is the lowest-level database API in Java
- JPA/Hibernate is the #1 ORM choice
- Spring Data JPA builds on JPA repositories
- Flyway/Liquibase manage schema migrations

## When to Choose Relational
- Structured data with clear schema
- Strong consistency requirements
- Complex relationships and joins
- ACID transactions needed
- Reporting and analytics
