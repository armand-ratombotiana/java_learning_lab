# History of Relational Databases

## Timeline

| Year | Event |
|---|---|
| 1970 | E.F. Codd publishes "A Relational Model of Data for Large Shared Data Banks" |
| 1974 | IBM starts System R project |
| 1979 | Oracle (Relational Software, Inc.) ships first commercial RDBMS |
| 1983 | IBM releases DB2 |
| 1986 | SQL standard (ANSI) published |
| 1989 | First JDB specification (JDBC prototype) |
| 1995 | MySQL released |
| 1996 | PostgreSQL 6.0 released |
| 1997 | Java JDBC 1.0 in JDK 1.1 |
| 2001 | Microsoft SQL Server 2000 |
| 2006 | Hibernate 3.0 (JPA 1.0 standard) |
| 2010 | NoSQL movement gains traction |
| 2014 | NewSQL (CockroachDB, Spanner) |
| 2020s | Cloud-native relational databases, serverless RDS |

## Key Contributions
- **Codd**: Relational model, normalization
- **Date & Codd**: Relational algebra and calculus
- **Gray**: Transaction processing, ACID
- **Stonebraker**: Ingres, PostgreSQL, column stores
- **Gavin King**: Hibernate ORM
