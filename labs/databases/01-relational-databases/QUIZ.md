# Quiz: Relational Databases

## Question 1
What is the primary difference between 2NF and 3NF?

a) 3NF removes transitive dependencies
b) 2NF requires atomic columns
c) 3NF removes partial dependencies
d) There is no difference

**Answer: a) 3NF removes transitive dependencies**

## Question 2
Which isolation level prevents dirty reads but not non-repeatable reads?

a) READ_UNCOMMITTED
b) READ_COMMITTED
c) REPEATABLE_READ
d) SERIALIZABLE

**Answer: b) READ_COMMITTED**

## Question 3
In JPA, what happens when you call `entityManager.persist(entity)`?

a) SQL INSERT is executed immediately
b) Entity is scheduled for INSERT on flush
c) Entity is deleted
d) Nothing

**Answer: b) Entity is scheduled for INSERT on flush**

## Question 4
What is the purpose of a foreign key?

a) Speed up queries
b) Enforce referential integrity
c) Store large objects
d) Create indexes

**Answer: b) Enforce referential integrity**

## Question 5
Which MVCC-related mechanism clean up dead tuples in PostgreSQL?

a) CHECKPOINT
b) VACUUM
c) REINDEX
d) CLUSTER

**Answer: b) VACUUM**
