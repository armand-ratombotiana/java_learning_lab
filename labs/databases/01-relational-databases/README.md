# 01 - Relational Databases

## Focus
RDBMS concepts, tables, relationships, normalization, ACID properties with Java/JDBC/JPA examples.

## Topics Covered
- Relational model fundamentals
- Table design and relationships (1:1, 1:N, N:M)
- Normalization (1NF, 2NF, 3NF, BCNF)
- ACID properties
- Constraints (PK, FK, UNIQUE, CHECK)
- JDBC metadata API
- JPA entity relationships

## Prerequisites
- Core Java
- Basic SQL knowledge
- JDK 17+, Maven/Gradle
