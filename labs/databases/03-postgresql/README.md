# 03 - PostgreSQL

## Focus
PostgreSQL setup, data types, indexing, JSONB, full-text search, extensions with Java/JDBC/JPA examples.

## Topics Covered
- PostgreSQL architecture
- Data types (native, custom, JSONB, arrays, ranges)
- Index strategies (B-tree, Hash, GiST, GIN, BRIN)
- JSONB operations and indexing
- Full-text search (tsvector, tsquery)
- Extensions (PostGIS, pg_stat_statements, pgcrypto)
- JDBC for PostgreSQL
- JPA with PostgreSQL-specific features
