# History: Database Migrations

- **2007** – Ruby on Rails popularizes migrations with ActiveRecord
- **2009** – Liquibase 1.0 released as first major Java migration tool
- **2010** – Flyway 1.0 released, focused on simplicity and SQL-first
- **2011** – Liquibase adds Maven plugin support
- **2013** – Flyway adopts gradle plugin, Java-based migrations
- **2016** – Flyway 4.0: callback mechanism, placeholder support
- **2018** – Flyway 5.0: baseline on migrate, repeatable migrations
- **2019** – Flyway acquired by Redgate, Liquibase 3.7
- **2021** – Flyway 7.x: Java 11+, Dry-run output
- **2023** – Flyway 9.x, Liquibase 4.20+: SQL-first YAML changelogs, Pro features
