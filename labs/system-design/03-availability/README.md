# Availability

## Overview
This lab covers high availability design patterns, fault tolerance, redundancy, disaster recovery, and the tools for building systems that survive failures.

## Prerequisites
- Java 17+
- Understanding of distributed systems
- Familiarity with Spring Boot

## Lab Structure
| File | Description |
|------|-------------|
| THEORY.md | Comprehensive theory of availability |
| CODE_DEEP_DIVE.md | Full Java implementation examples |
| EXERCISES.md | Hands-on exercises |
| QUIZ.md | Knowledge assessment |
| FLASHCARDS.md | Key concept review |

## Learning Objectives
- Understand the 9s of availability (99.9% to 99.999%)
- Design for redundancy and failover
- Implement circuit breakers and retries
- Plan disaster recovery strategies

```bash
cd 03-availability
# Review theory first
# Then explore code deep dive
```
