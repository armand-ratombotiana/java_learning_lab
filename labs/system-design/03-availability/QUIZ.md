# Availability - QUIZ

**Q1**: What does 99.99% availability equal in annual downtime?
- A) 52.6 minutes
- B) 8.76 hours
- C) 5.26 minutes
- D) 87.6 hours

**Answer: A**

**Q2**: What does RPO stand for?
- A) Recovery Point Objective
- B) Real-time Processing Option
- C) Redundancy Protection Output
- D) Request Processing Order

**Answer: A**

**Q3**: Which pattern has one active and one passive instance?
- A) Active-Active
- B) N+1
- C) Active-Passive
- D) Distributed

**Answer: C**

**Q4**: What is the primary purpose of a circuit breaker?
- A) Increase throughput
- B) Prevent cascading failures
- C) Cache data
- D) Balance load

**Answer: B**

**Q5**: What are the four golden signals?
- A) CPU, Memory, Disk, Network
- B) Latency, Traffic, Errors, Saturation
- C) Uptime, Downtime, Response, Throughput
- D) Read, Write, Update, Delete

**Answer: B**
