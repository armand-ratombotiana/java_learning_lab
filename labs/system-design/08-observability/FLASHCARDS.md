# Observability - FLASHCARDS

### Card 1
**Q:** Three pillars?
**A:** Metrics (measurements), Logs (events), Traces (flows).

### Card 2
**Q:** Counter metric?
**A:** Cumulative count. Use for: requests, errors.

### Card 3
**Q:** Gauge metric?
**A:** Current value. Use for: connections, queue size.

### Card 4
**Q:** Histogram metric?
**A:** Distribution. Use for: latency, response size.

### Card 5
**Q:** Trace ID purpose?
**A:** Correlate all spans of a single request.

### Card 6
**Q:** Health check types?
**A:** Liveness (is running), Readiness (can serve traffic).

### Card 7
**Q:** OpenTelemetry?
**A:** Vendor-neutral observability standard for metrics, logs, traces.

**Total: 7 flashcards**
