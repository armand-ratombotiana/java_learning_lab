# Observability

## Overview
This lab covers observability in distributed systems: logging, metrics, distributed tracing, alerting, and the tools for understanding system behavior in production.

## Prerequisites
- Java 17+
- Understanding of microservices
- Familiarity with Spring Boot

## Lab Structure
| File | Description |
|------|-------------|
| THEORY.md | Comprehensive theory of observability |
| CODE_DEEP_DIVE.md | Full Java implementation examples |
| EXERCISES.md | Hands-on exercises |
| QUIZ.md | Knowledge assessment |
| FLASHCARDS.md | Key concept review |

## Learning Objectives
- Implement structured logging with correlation IDs
- Collect and visualize metrics with Micrometer + Prometheus
- Set up distributed tracing with OpenTelemetry
- Design meaningful alerts and dashboards

## Quick Start
```bash
cd 08-observability
# Review theory first
# Then explore code deep dive
# Complete exercises
```
