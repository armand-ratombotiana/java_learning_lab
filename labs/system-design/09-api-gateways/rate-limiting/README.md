# Rate Limiting

Welcome to the atomic mastery lab for **Rate Limiting**. This lab is part of the System Design Academy's API Gateways module.

## 🧠 What You Will Master
- Why rate limiting is critical for API reliability and security.
- The Token Bucket algorithm.
- The Leaky Bucket algorithm.
- Fixed Window vs. Sliding Window Log algorithms.
- Distributed rate limiting using Redis.
- Implementing a Token Bucket rate limiter in Java.
