# Consistency Models

## Overview
This lab covers consistency models in distributed systems: strong consistency, eventual consistency, causal consistency, CAP theorem, and the trade-offs between consistency, availability, and latency.

## Prerequisites
- Java 17+
- Understanding of distributed databases
- Familiarity with ACID and BASE

## Lab Structure
| File | Description |
|------|-------------|
| THEORY.md | Comprehensive theory of consistency |
| CODE_DEEP_DIVE.md | Full Java implementation examples |
| EXERCISES.md | Hands-on exercises |
| QUIZ.md | Knowledge assessment |
| FLASHCARDS.md | Key concept review |

## Learning Objectives
- Understand the consistency spectrum (strong → eventual)
- Apply CAP theorem to real-world trade-offs
- Implement distributed consensus with Raft/Zab
- Handle read-after-write and monotonic read guarantees
