# Architecture Patterns

## Overview
This lab covers the four fundamental architecture patterns used in modern distributed systems: Layered Architecture, Microservices, Event-Driven Architecture, and CQRS.

## Prerequisites
- Java 17+
- Basic understanding of REST APIs
- Familiarity with Spring Boot

## Lab Structure
| File | Description |
|------|-------------|
| THEORY.md | Comprehensive theory of architecture patterns |
| CODE_DEEP_DIVE.md | Full Java implementation examples |
| EXERCISES.md | Hands-on exercises |
| QUIZ.md | Knowledge assessment |
| FLASHCARDS.md | Key concept review |

## Learning Objectives
- Understand layered architecture and its strict layering principle
- Design microservices with proper service boundaries
- Implement event-driven communication patterns
- Apply CQRS for read/write separation
