# Raft Consensus Algorithm

Welcome to the atomic mastery lab for the **Raft Algorithm**. This lab is part of the System Design Academy's Consensus module.

## 🧠 What You Will Master
- The problem of distributed consensus (Byzantine Generals Problem).
- Why Paxos is notoriously difficult to understand and implement.
- Raft's Leader Election mechanism (Terms, Heartbeats, Randomized Timers).
- Log Replication and Commits.
- Implementing a conceptual Raft node in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The need for consensus in distributed systems.
2. [INTERNALS.md](./INTERNALS.md) - Deep dive into leader election and log replication.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java simulation of Raft leader election.
