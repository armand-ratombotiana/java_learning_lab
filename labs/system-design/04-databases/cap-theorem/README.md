# CAP Theorem

Welcome to the atomic mastery lab for the **CAP Theorem**. This lab is part of the System Design Academy's Databases module.

## 🧠 What You Will Master
- The definitions of Consistency, Availability, and Partition Tolerance.
- Why you can only choose two out of three in a distributed system.
- Real-world database examples (CP vs AP systems).
- Eventual Consistency and PACELC theorem.
- Simulating network partitions in a Java distributed environment.

1. [THEORY.md](./THEORY.md) - The trade-offs of distributed data.
