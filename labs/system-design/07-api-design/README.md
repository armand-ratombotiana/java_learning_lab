# API Design

## Overview
This lab covers RESTful API design, GraphQL, gRPC, API versioning, pagination, error handling, and best practices for designing APIs that scale.

## Prerequisites
- Java 17+
- Understanding of HTTP and REST
- Familiarity with Spring Boot

## Lab Structure
| File | Description |
|------|-------------|
| THEORY.md | Comprehensive theory of API design |
| CODE_DEEP_DIVE.md | Full Java implementation examples |
| EXERCISES.md | Hands-on exercises |
| QUIZ.md | Knowledge assessment |
| FLASHCARDS.md | Key concept review |

## Learning Objectives
- Design RESTful APIs following best practices
- Implement proper error handling and validation
- Version APIs without breaking clients
- Choose between REST, GraphQL, and gRPC

## Quick Start
```bash
cd 07-api-design
# Review theory first
# Then explore code deep dive
# Complete exercises
```
