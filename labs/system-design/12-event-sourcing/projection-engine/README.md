# Projection Engine

Welcome to the atomic mastery lab for the **Projection Engine**. This lab is part of the System Design Academy's Event Sourcing module.

## 🧠 What You Will Master
- The role of Projections in Event-Sourced systems.
- Transforming an Event Stream into a Read Model.
- Idempotency and Checkpointing.
- Replaying events to rebuild state (The Blue-Green Projection).
- Implementing a simple Projection Engine in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Why we separate storage from query state.
2. [INTERNALS.md](./INTERNALS.md) - Sequential processing, batching, and error handling.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java simulation of an Event Projection loop.
