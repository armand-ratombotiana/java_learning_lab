# Messaging

## Overview
This lab covers messaging systems, message brokers, event-driven architecture, pub/sub patterns, and reliable message processing in distributed systems.

## Prerequisites
- Java 17+
- Understanding of distributed systems
- Familiarity with Kafka concepts

## Lab Structure
| File | Description |
|------|-------------|
| THEORY.md | Comprehensive theory of messaging |
| CODE_DEEP_DIVE.md | Full Java implementation examples |
| EXERCISES.md | Hands-on exercises |
| QUIZ.md | Knowledge assessment |
| FLASHCARDS.md | Key concept review |

## Learning Objectives
- Understand pub/sub and point-to-point messaging
- Implement producers and consumers with Kafka
- Design reliable message processing (at-least-once, exactly-once)
- Handle message ordering, idempotency, and dead letters

cd 06-messaging
