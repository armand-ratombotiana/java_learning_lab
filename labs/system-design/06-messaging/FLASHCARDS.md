# Messaging - FLASHCARDS

### Card 1
**Q:** Point-to-point?
**A:** One producer, one consumer per message. Queue-based.

### Card 2
**Q:** Pub/Sub?
**A:** One producer, multiple subscribers. Topic-based.

### Card 3
**Q:** Kafka use case?
**A:** High-throughput event streaming, log aggregation.

### Card 4
**Q:** RabbitMQ use case?
**A:** Task queues, request/reply, complex routing.

### Card 5
**Q:** Dead letter queue?
**A:** Queue for failed messages that need manual processing.

### Card 6
**Q:** Consumer group?
**A:** Set of consumers sharing message processing.

**Total: 6 flashcards**
