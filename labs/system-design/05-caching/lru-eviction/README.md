# LRU Cache Eviction

Welcome to the atomic mastery lab for **LRU (Least Recently Used) Eviction**. This lab is part of the System Design Academy's Caching module.

## 🧠 What You Will Master
- Cache eviction policies and why LRU is the industry standard.
- The architectural trade-offs of LRU vs LFU vs FIFO.
- Designing an O(1) LRU Cache using a HashMap and a Doubly-Linked List.
- Thread-safety considerations for concurrent caches.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Cache coherence and eviction strategies.
2. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a thread-safe LRU Cache.
