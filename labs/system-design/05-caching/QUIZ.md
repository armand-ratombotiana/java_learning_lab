# Caching Strategies - QUIZ

**Q1: Cache-aside pattern means:**
- A) Always read from cache
- B) Check cache, load from DB on miss, populate cache
- C) Write to cache only
- D) No cache

**Q2: Write-through ensures:**
- A) Fast writes
- B) Cache always has latest data
- C) No DB writes
- D) Async operation

**Q3: TTL stands for:**
- A) Time To Load
- B) Time To Live
- C) Total Time Limit
- D) Throughput Time Limit

**Q4: Cache warming is:**
- A) Clearing cache
- B) Preloading hot data
- C) Testing cache speed
- D) Deleting old data

**Q5: For session data, use:**
- A) Long TTL
- B) Short TTL with write-through
- C) No caching
- D) Write-behind

**Answers**: 1-B, 2-B, 3-B, 4-B, 5-B
