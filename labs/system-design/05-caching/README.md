# Caching

## Overview
This lab covers caching strategies, cache invalidation, distributed caching, CDNs, and patterns for improving application performance through intelligent data caching.

## Prerequisites
- Java 17+
- Understanding of database performance
- Familiarity with Spring Boot

## Lab Structure
| File | Description |
|------|-------------|
| THEORY.md | Comprehensive theory of caching |
| CODE_DEEP_DIVE.md | Full Java implementation examples |
| EXERCISES.md | Hands-on exercises |
| QUIZ.md | Knowledge assessment |
| FLASHCARDS.md | Key concept review |

## Learning Objectives
- Understand cache-aside, read-through, write-through patterns
- Implement distributed caching with Redis
- Design cache invalidation strategies
- Apply CDN caching for static assets

cd 05-caching
# Review theory first
