## Deployment Topology

| Component | Instances | Scaling Policy |
|-----------|-----------|---------------|
| Web App | 3-20 | CPU > 70% scale out |
| API Gateway | 2-10 | Request count > 5000 |
| Redis Cache | 3-6 cluster | Memory > 80% add node |
| PostgreSQL | 1 master + 3 replicas | Read replica on high query count |
| Kafka | 3-12 brokers | Disk > 70% add partition |
