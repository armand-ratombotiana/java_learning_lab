# Consistent Hashing

Welcome to the atomic mastery lab for **Consistent Hashing**. This lab is part of the System Design Academy's Scalability module.

## 🧠 What You Will Master
- The problem with standard modulo hashing in distributed systems.
- The concept of the Hash Ring.
- Virtual Nodes and load balancing.
- Implementing a Consistent Hashing algorithm in Java.
- Applications in real-world systems (DynamoDB, Cassandra, Memcached).

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The architectural problem and the hash ring solution.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - The mathematics of uniform distribution and virtual nodes.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a consistent hash ring.
