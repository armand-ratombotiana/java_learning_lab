# The Saga Pattern

Welcome to the atomic mastery lab for the **Saga Pattern**. This lab is part of the System Design Academy's Transactions module.

## 🧠 What You Will Master
- The problem with distributed transactions (Two-Phase Commit).
- The Saga Pattern: Local transactions and compensating actions.
- Choreography vs. Orchestration.
- Idempotency and eventual consistency.
- Implementing an Orchestration-based Saga in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Why 2PC fails in microservices and how Sagas solve it.
2. [INTERNALS.md](./INTERNALS.md) - Deep dive into Choreography vs Orchestration.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a Saga Orchestrator.
