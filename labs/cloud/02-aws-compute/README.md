# 02 - AWS Compute

## Overview
Explore AWS compute services beyond EC2: Lambda (serverless), ECS (containers), EKS (Kubernetes), Fargate (serverless containers), Batch, and Elastic Beanstalk. Focus on Java-centric deployment patterns.

## Prerequisites
- AWS Fundamentals (Lab 01)
- Java 11+ with Maven/Gradle
- Docker Desktop (for ECS/EKS sections)
