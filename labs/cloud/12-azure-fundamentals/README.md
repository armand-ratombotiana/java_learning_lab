# 12 Ś Azure Fundamentals

## Overview
Azure VMs, Blob Storage, Azure SQL, AKS, managed identities. This lab provides hands-on experience with real Java implementations and cloud SDK patterns.

## Prerequisites
- Java 21+ development environment
- Basic cloud computing knowledge
- Familiarity with REST APIs and JSON
- An Azure subscription (free tier)

## What You Will Learn
- Provision and manage Azure VMs programmatically
- Work with Azure Blob Storage using Java SDK
- Connect to Azure SQL Database from Java
- Deploy containers to AKS
- Configure managed identities for Azure resources
