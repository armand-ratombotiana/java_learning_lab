# 15 — Cloud Cost Optimization

## Overview
Reserved instances, savings plans, rightsizing, spot instances, FinOps. This lab provides hands-on experience with real Java implementations and cloud SDK patterns.

## Prerequisites
- Java 21+ development environment
- Basic cloud computing knowledge
- Familiarity with REST APIs and JSON
- Multi-cloud awareness

## What You Will Learn
- Analyze cloud spending across AWS, Azure, and GCP
- Implement rightsizing recommendations in Java
- Model reserved instance and savings plan purchases
- Simulate spot instance workloads
- Build a FinOps cost allocation dashboard
