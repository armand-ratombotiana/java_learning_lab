# 13 — GCP Fundamentals

## Overview
GCP Compute Engine, Cloud Storage, Cloud SQL, GKE, IAM. This lab provides hands-on experience with real Java implementations and cloud SDK patterns.

## Prerequisites
- Java 21+ development environment
- Basic cloud computing knowledge
- Familiarity with REST APIs and JSON
- A GCP project (free tier)

## What You Will Learn
- Create and manage GCP Compute Engine instances
- Store and retrieve objects with Cloud Storage
- Connect to Cloud SQL from Java applications
- Deploy workloads on GKE
- Configure GCP IAM roles and service accounts

- **Cloud Storage** — GCP object storage
- **Cloud SQL** — Managed MySQL/PostgreSQL
- **GKE** — Google Kubernetes Engine
- **GCP IAM** — Identity and access management
