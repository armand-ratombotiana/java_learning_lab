# 09 — AWS Security

## Overview
IAM policies, KMS encryption, Security Groups, WAF, Shield, GuardDuty. This lab provides hands-on experience with real Java implementations and cloud SDK patterns.

## Prerequisites
- Java 21+ development environment
- Basic cloud computing knowledge
- Familiarity with REST APIs and JSON
- Multi-cloud awareness

## What You Will Learn
- Configure IAM policies and roles programmatically
- Encrypt and decrypt data with KMS
- Define Security Group rules in code
- Implement WAF rules for web application protection
- Analyze threats with GuardDuty simulation

## Key Topics
- **IAM** — Identity and access management policies
- **KMS** — Key Management Service encryption
- **Security Groups** — Instance-level firewall rules
- **WAF** — Web application firewall
- **Shield** — DDoS protection
- **GuardDuty** — Threat detection service
