# AWS Storage - Flashcards

### S3 (Simple Storage Service)
Object storage with unlimited capacity and 99.999999999% durability.

### S3 Storage Class
Categorization of data based on access patterns and retention needs.

### S3 Lifecycle Policy
Rule to automatically transition objects between storage classes.

### S3 Versioning
Feature maintaining multiple versions of objects in bucket.

### S3 Replication
Automatic asynchronous copying of objects to another bucket.

### S3 Pre-signed URL
Time-limited URL granting temporary access to private S3 object.

### S3 Server-Side Encryption
Encryption at rest using AWS-managed or customer-managed keys.

### EBS (Elastic Block Store)
Block storage volumes for EC2 instances.

### EBS Volume Type
Category determining performance characteristics (gp3, io2, st1, etc.).

### EBS Snapshot
Point-in-time backup of EBS volume stored in S3.

### EBS Multi-Attach
Feature allowing io2 volumes to attach to multiple instances.

### EBS Elastic Volumes
Ability to modify volume size and type without downtime.

### EFS (Elastic File System)
Managed NFS file system for cloud workloads.

### EFS Performance Mode
Configuration for throughput characteristics (General Purpose vs Max I/O).

### EFS Throughput Mode
Bursting or Provisioned throughput settings.

### EFS Lifecycle Management
Automatic transition to Infrequent Access storage class.

### EFS Access Point
Entry point with specific POSIX permissions and path.

### NFS (Network File System)
Protocol for file sharing across network.

### Storage Class Analysis
S3 feature to analyze access patterns and recommend storage class.

### S3 Select
Feature to query data directly in S3 using SQL-like syntax.

### S3 Inventory
Reporting tool providing daily or weekly file listing.

### S3 Glacier
Archive storage with retrieval times from minutes to hours.

### S3 Object Lock
Feature preventing object deletion or overwrite for specified period.

### Storage Gateway
Hybrid storage solution connecting on-premises to AWS cloud storage.

### Amazon S3
Object storage service fundamental to cloud architectures.

### EBS Encrypted Volume
Volume with encryption enabled using KMS keys.

### EFS Mount Target
Network endpoint for accessing EFS file system.

### S3 Bucket Policy
IAM-based access control at bucket level.

### Cold Storage
Storage for rarely accessed data with lower cost but higher retrieval time.
