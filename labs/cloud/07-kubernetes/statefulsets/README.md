# StatefulSets

Welcome to the atomic mastery lab for **StatefulSets**. This lab is part of the Cloud Academy's Kubernetes module.

## 🧠 What You Will Master
- The difference between Stateless (Deployments) and Stateful (StatefulSets) applications.
- Persistent Volumes (PV) and Persistent Volume Claims (PVC).
- Headless Services and stable network identities.
- Ordered deployment, scaling, and deletion.
- Deploying a stateful database cluster (e.g., PostgreSQL or Kafka) on Kubernetes.

1. [THEORY.md](./THEORY.md) - Why Deployments fail for databases.
