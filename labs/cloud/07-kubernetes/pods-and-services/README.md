# Kubernetes Pods & Services

Welcome to the atomic mastery lab for **Pods and Services**. This lab is part of the Cloud Academy's Kubernetes module.

## 🧠 What You Will Master
- The architecture of a Kubernetes Cluster (Control Plane vs Worker Nodes).
- Why Pods are the smallest deployable unit, not containers.
- Ephemeral IP addresses and the need for Services.
- Service Types: ClusterIP, NodePort, LoadBalancer.
- Implementing and deploying a Java application to a local K8s cluster.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The problem with raw containers and the K8s abstraction.
2. [INTERNALS.md](./INTERNALS.md) - Kube-proxy, iptables, and service discovery.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Writing YAML manifests for a Java Spring Boot app.
