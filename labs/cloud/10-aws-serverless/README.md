# 10 — AWS Serverless

## Overview
Lambda advanced, Step Functions, EventBridge, SAM framework. This lab provides hands-on experience with real Java implementations and cloud SDK patterns.

## Prerequisites
- Java 21+ development environment
- Basic cloud computing knowledge
- Familiarity with REST APIs and JSON
- Multi-cloud awareness

## What You Will Learn
- Build and deploy advanced Lambda functions with Java
- Orchestrate workflows with Step Functions
- Implement event-driven architectures with EventBridge
- Package serverless apps using the SAM framework
- Handle async invocations and dead-letter queues
