# 04 - AWS Database

## Overview
Master AWS database services: RDS (relational), DynamoDB (NoSQL), ElastiCache (in-memory), Aurora (MySQL/PostgreSQL-compatible), and Redshift (data warehousing). Learn Java integration patterns.

## Prerequisites
- AWS Fundamentals (Lab 01)
- Java 11+ with JDBC or DynamoDB SDK
- Understanding of SQL and NoSQL concepts
