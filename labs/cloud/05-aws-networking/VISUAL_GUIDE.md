## 3. Route 53 Routing Policies

```
┌──────┐  Simple:              ┌──────┐  Weighted:
│www   │───► A record (1 IP)   │api   │───► v1 (80%)
└──────┘                       │      │───► v2 (20%)
                                └──────┘

┌──────┐  Latency-based:       ┌──────┐  Failover:
│app   │───► us-east-1 (10ms)  │db    │───► Primary
│      │───► ap-southeast (150ms)    │───► Secondary (standby)
└──────┘                       └──────┘

┌──────┐  Geolocation:
│shop  │───► NA → us-east-1
│      │───► EU → eu-west-1
│      │───► APAC → ap-southeast-1
└──────┘
```
