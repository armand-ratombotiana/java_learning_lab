# 11 — AWS Observability

## Overview
CloudWatch, X-Ray, Prometheus on AWS, AMP, AMG. This lab provides hands-on experience with real Java implementations and cloud SDK patterns.

## Prerequisites
- Java 21+ development environment
- Basic cloud computing knowledge
- Familiarity with REST APIs and JSON
- Multi-cloud awareness

## What You Will Learn
- Instrument Java applications with CloudWatch metrics
- Trace requests end-to-end with X-Ray
- Expose application metrics in Prometheus format
- Query metrics with Amazon Managed Service for Prometheus
- Build dashboards with Amazon Managed Grafana
