# 01 - AWS Fundamentals

## Overview
Master the foundational AWS services: EC2, S3, IAM, VPC, and the global infrastructure. This lab builds the core knowledge required for all subsequent cloud labs.

## Prerequisites
- Basic AWS account (free tier)
- AWS CLI installed and configured
- Java 11+ development environment

## What You Will Learn
- Launch and manage EC2 instances
- Store and retrieve objects from S3
- Configure IAM users, groups, roles, and policies
- Design a VPC with public/private subnets
- Navigate AWS regions, AZs, and edge locations

## Key Services
- **EC2** — Virtual machines in the cloud
- **S3** — Object storage at global scale
- **IAM** — Identity and access management
- **VPC** — Isolated network environments
- **Route 53** — DNS and domain management
