# Exploratory Data Analysis

Welcome to the atomic mastery lab for **Exploratory Data Analysis**. This lab is part of the Data Science Academy.

## What You Will Master
- Univariate and multivariate exploration techniques
- Distribution analysis and summary statistics
- Correlation analysis and association metrics
- Visual patterns discovery with Java tools
- Hypothesis generation from data
