# References: Exploratory Data Analysis

## Papers & Books
- Tukey, J.W. (1977). *Exploratory Data Analysis*. Addison-Wesley.
- Anscombe, F.J. (1973). *Graphs in Statistical Analysis*. The American Statistician, 27(1), 17–21.
- Cleveland, W.S. (1993). *Visualizing Data*. Hobart Press.
- Wickham, H. & Grolemund, G. (2017). *R for Data Science* (Ch. 7: Exploratory Data Analysis).
- NIST/SEMATECH. *e-Handbook of Statistical Methods*. https://www.itl.nist.gov/div898/handbook/

## Java Libraries
- [Tablesaw](https://github.com/jtablesaw/tablesaw) — DataFrame with built-in EDA summaries
- [Smile](https://github.com/haifengl/smile) — Statistical Machine Intelligence (correlation, PCA, MDS)
- [Apache Commons Math](https://commons.apache.org/proper/commons-math/) — Descriptive statistics, distributions
- [Hipparchus](https://hipparchus.org/) — Mathematics and statistics library (successor to Commons Math)

## Tools
- Pandas Profiling (Python) — automated EDA reports
- Sweetviz (Python) — visual EDA comparison
- D-Tale (Python) — interactive EDA GUI
