# Mini Project: Customer Data EDA

## Objective
Perform comprehensive exploratory data analysis on customer purchase data.

## Dataset
Customer purchase data with:
- Customer demographics
- Purchase history
- Product categories
- Transaction dates
- Payment methods

## Tasks

### 1. Data Loading and Overview
- Load customer data
- Check dimensions
- Identify data types
- Calculate missing value percentages

### 2. Univariate Analysis
- Analyze each numeric variable (mean, std, min, max, percentiles)
- Analyze each categorical variable (counts, modes)
- Create distribution plots

### 3. Bivariate Analysis
- Correlation between numeric variables
- Compare categories across groups
- Identify relationships

### 4. Outlier Detection
- IQR method
- Z-score method
- Visual inspection

### 5. Statistical Tests
- Normality tests
- Hypothesis tests
- Confidence intervals

### 6. Visualization
- All key charts
- Summary dashboard

## Output
Complete EDA report with statistics and visualizations.
