# References: Data Visualization

## Papers & Books
- Tufte, E. (1983). *The Visual Display of Quantitative Information*. Graphics Press.
- Cleveland, W.S. (1993). *Visualizing Data*. Hobart Press.
- Bertin, J. (1967). *Sémiologie Graphique*. (English: *Semiology of Graphics*, 1983)
- Wickham, H. (2010). *A Layered Grammar of Graphics*. Journal of Computational and Graphical Statistics.
- Few, S. (2004). *Show Me the Numbers: Designing Tables and Graphs to Enlighten*.
- Cairo, A. (2012). *The Functional Art: An Introduction to Information Graphics and Visualization*.

## Java Libraries
- [JFreeChart](https://www.jfree.org/jfreechart/) — Full-featured charting library
- [XChart](https://github.com/knowm/XChart) — Lightweight plotting library
- [Charts4j](https://github.com/charts4j/charts4j) — Declarative charting via annotations
- [Tablesaw Plot](https://github.com/jtablesaw/tablesaw) — DataFrame-integrated plotting
- [Apache Batik](https://xmlgraphics.apache.org/batik/) — SVG rendering for Java
