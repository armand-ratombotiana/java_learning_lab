# Data Visualization

Welcome to the atomic mastery lab for **Data Visualization**. This lab is part of the Data Science Academy.

## What You Will Master
- Principles of effective visual communication
- Chart types and when to use each
- Building visualizations with Java charting libraries
- Interactive and static plotting techniques
- Visual diagnostics for model evaluation

## Lab Structure
1. [THEORY.md](./THEORY.md) — Visual perception and graphic design principles
2. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) — Java plotting with Charts4j, XChart, and JFreeChart
3. [EXERCISES.md](./EXERCISES.md) — Hands-on visualization tasks
4. [QUIZ.md](./QUIZ.md) — Test your knowledge
5. [FLASHCARDS.md](./FLASHCARDS.md) — Key terms and concepts
