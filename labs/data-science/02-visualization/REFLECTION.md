# Reflection: Data Visualization

## Key Takeaways

- Every chart encodes data through visual channels — choose channels that match the perceptual strengths of human vision
- Simpler charts are harder to misinterpret; maximize data-ink ratio
- Color, scale, and geometry choices are not neutral — they shape the viewer's interpretation
- Interactive visualization adds exploration depth; static charts add communication clarity

## Questions to Internalize

1. If I remove the legend and color, does the chart still communicate the main finding?
2. Am I using color to encode a dimension, or just to decorate?
3. Would a non-expert viewer immediately understand what this chart says?
4. Could this chart be used to mislead someone? How?

## Growth Areas

| Skill | Beginner | Proficient | Master |
|---|---|---|---|
| Chart choice | Default scatter | Match chart to question | Novel visualizations |
| Color theory | Default palettes | Sequential/diverging by purpose | Perceptually uniform + accessible |
| Interactivity | Static only | Tooltips, zoom | Linked brushing, animation |
| Visual perception | Follow rules | Understanding why rules work | Breaking rules intentionally |
