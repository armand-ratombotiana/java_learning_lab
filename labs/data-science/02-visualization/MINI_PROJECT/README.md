# Mini Project: Sales Dashboard

## Objective
Create a complete interactive sales dashboard with multiple chart types visualizing sales data.

## Features

### 1. Data Visualization Components
- Line chart: Monthly sales trends
- Bar chart: Sales by region
- Pie chart: Product category breakdown
- Scatter plot: Revenue vs units sold
- Histogram: Order value distribution

### 2. Dashboard Layout
- Header with title and date range
- KPI cards (total sales, orders, average order value)
- Multiple charts arranged in grid
- Legend and interactivity

### 3. Data Processing
- Load sales data from CSV
- Aggregate by various dimensions
- Calculate KPIs
- Filter by date range

## Implementation

Create a complete Java Swing application with:
1. Main frame with menu
2. Chart panels for each visualization
3. KPI display cards
4. Interactive filtering

## Sample Data
- sales_data.csv with columns: date, product, category, region, quantity, unit_price, total

## Output
- Working dashboard application
- All chart types demonstrated
- Interactive filtering capability
