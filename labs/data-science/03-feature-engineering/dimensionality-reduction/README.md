# Dimensionality Reduction

Welcome to the atomic mastery lab for **Dimensionality Reduction**. This lab is part of the Data Science Academy's Feature Engineering module.

## 🧠 What You Will Master
- The Curse of Dimensionality.
- Principal Component Analysis (PCA) intuition and math.
- t-Distributed Stochastic Neighbor Embedding (t-SNE) concepts.
- Eigenvectors, Eigenvalues, and Covariance Matrices.
- Implementing PCA from scratch in Java.

1. [THEORY.md](./THEORY.md) - Intuition behind compressing feature spaces.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - Linear algebra of PCA.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of PCA.
