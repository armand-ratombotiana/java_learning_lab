# Handling Categorical Data

Welcome to the atomic mastery lab for **Categorical Data**. This lab is part of the Data Science Academy's Feature Engineering module.

## 🧠 What You Will Master
- Nominal vs. Ordinal data.
- Label Encoding and its pitfalls.
- One-Hot Encoding and the Dummy Variable Trap.
- Target Encoding for high-cardinality features.
- Binary and Hashing encoders.
- Implementing encoders from scratch in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Intuition behind mapping text to numbers.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - The math of high-cardinality encoding.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of One-Hot and Target Encoders.
