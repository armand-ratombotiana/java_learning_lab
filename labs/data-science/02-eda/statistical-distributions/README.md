# Statistical Distributions

Welcome to the atomic mastery lab for **Statistical Distributions**. This lab is part of the Data Science Academy's EDA (Exploratory Data Analysis) module.

## 🧠 What You Will Master
- Discrete vs. Continuous distributions.
- Normal (Gaussian) Distribution and the Central Limit Theorem.
- Bernoulli and Binomial Distributions.
- Poisson Distribution (modeling rare events).
- Visualizing distributions and calculating PDF/CDF in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Intuition behind probability density.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - Formulas for Mean, Variance, and Skewness.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of distribution samplers.
