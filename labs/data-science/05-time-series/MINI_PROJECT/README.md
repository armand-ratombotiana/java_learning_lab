# Mini Project: Sales Forecasting

## Objective
Build time series forecasting model for monthly sales data.

## Dataset
Monthly sales data for 5 years (60 months).

## Tasks
1. Load and visualize data
2. Identify components (trend, seasonality)
3. Test stationarity
4. Apply forecasting methods (MA, Exponential Smoothing, ARIMA)
5. Evaluate and compare methods
6. Generate forecasts

## Output
Complete forecasting pipeline with results.
