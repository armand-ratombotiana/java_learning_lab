# Mini Project: A/B Test Causal Analysis

## Objective
Analyze A/B test results using causal inference methods.

## Dataset
Website test data with:
- User ID, group (control/treatment)
- Features (age, session count, prior purchases)
- Conversion outcome

## Tasks
1. Check randomization balance
2. Estimate propensity scores
3. Apply IPW estimation
4. Use matching for subgroup analysis
5. Calculate confidence intervals

## Output
Causal effect estimate with sensitivity analysis.
