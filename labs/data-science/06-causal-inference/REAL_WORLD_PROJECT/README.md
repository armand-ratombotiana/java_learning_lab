# Real World Project: Marketing Campaign Analysis

## Objective
Evaluate causal impact of marketing campaign on customer behavior.

## Data
- Customer exposure to campaign (treated/control)
- Pre/post customer metrics
- Customer characteristics

## Tasks
1. Propensity score matching
2. Difference-in-differences
3. Heterogeneous treatment effects
4. Cost-benefit analysis

## Output
Causal impact estimate with recommendations.
