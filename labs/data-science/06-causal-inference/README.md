# Causal Inference

Welcome to the atomic mastery lab for **Causal Inference**. This lab is part of the Data Science Academy.

## What You Will Master
- Distinguishing correlation from causation
- Directed Acyclic Graphs (DAGs) for causal reasoning
- Confounding, collider bias, and mediation
- Randomized experiments and natural experiments
- Causal effect estimation methods
