# Causal Inference Exercises

### Exercise 4: Identify Assumptions

### Exercise 5: Propensity Score
Estimate propensity score using logistic regression.

### Exercise 6: IPW Estimator
Implement inverse probability weighting.

### Exercise 7: Matching

### Exercise 9: Balance Check
Check balance of covariates after matching.

### Exercise 10: Interpret Results
Interpret causal effect estimate with confidence intervals.
