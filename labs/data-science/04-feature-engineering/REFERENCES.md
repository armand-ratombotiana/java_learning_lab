# References: Feature Engineering

## Papers & Books
- Kuhn, M. & Johnson, K. (2019). *Feature Engineering and Selection*. CRC Press.
- Zheng, A. & Casari, A. (2018). *Feature Engineering for Machine Learning*. O'Reilly.
- Ng, A. (2017). *Machine Learning Yearning* (Chapters on feature engineering).
- Kanter, J.M. & Veeramachaneni, K. (2015). *Deep Feature Synthesis: Towards Automating Data Science Endeavors*. IEEE DSAA.
- Pedregosa, F. et al. (2011). *Scikit-learn: Machine Learning in Python* (feature_extraction module).

## Java Libraries
- [Smile](https://github.com/haifengl/smile) — Feature extraction, PCA, feature selection
- [Tablesaw](https://github.com/jtablesaw/tablesaw) — Add/replace columns for feature creation
- [Apache Commons Math](https://commons.apache.org/proper/commons-math/) — Polynomial expansion, PCA
- [Feast](https://feast.dev/) — Feature store (Java SDK available)

## Tools
- Featuretools (Python) — Automated feature engineering on relational data
- TSFresh (Python) — Automated time series feature extraction
- Alteryx / DataRobot — Visual feature engineering
