# Data Wrangling

Welcome to the atomic mastery lab for **Data Wrangling**. This lab is part of the Data Science Academy.

## 🧠 What You Will Master
- Identifying and handling missing or corrupt data (Imputation vs. Deletion).
- Data normalization and standardization (Min-Max, Z-score).
- Encoding categorical variables (One-Hot, Label Encoding).
- Outlier detection and treatment.
- Implementing data cleaning pipelines in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The importance of data quality and preprocessing strategies.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - The statistics of normalization and outlier detection.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a data preprocessing pipeline.
