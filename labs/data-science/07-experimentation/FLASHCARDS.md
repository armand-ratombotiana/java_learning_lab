# Experimentation Flashcards

**Card 1:** H0 (Null Hypothesis) - No effect or difference; status quo

**Card 2:** H1 (Alternative Hypothesis) - There is an effect; what we want to prove

**Card 3:** Type I Error - Rejecting H0 when it's true (false positive); probability = α

**Card 4:** Type II Error - Failing to reject H0 when it's false (false negative); probability = β

**Card 5:** P-value - Probability of observing data if H0 is true; smaller = stronger evidence

**Card 6:** Statistical vs Practical Significance - Statistical means unlikely by chance; practical means meaningful in real world

**Card 7:** One-tailed test - Tests direction of effect (greater or less than)

**Card 8:** Two-tailed test - Tests for any difference (not specific direction)

**Card 9:** T-test - Compares means when population std unknown or sample small

**Card 10:** Paired t-test - Compares before/after for same subjects

**Card 11:** ANOVA - Analysis of variance; compares 3+ group means using F-distribution

**Card 12:** F-statistic - Ratio of between-group variance to within-group variance

**Card 13:** Post-hoc tests - After significant ANOVA, determine which groups differ (Tukey, Scheffe)

**Card 14:** Chi-square - Tests categorical data; goodness of fit or independence

**Card 15:** Effect size - Standardized measure of magnitude; d = (mean1 - mean2) / pooled_std

**Card 16:** Cohen's d: < 0.2 negligible, 0.2-0.5 small, 0.5-0.8 medium, ≥ 0.8 large

**Card 17:** Power - Probability of detecting true effect; 1 - β

**Card 18:** Sample size formula - n = 2((z_α + z_β)/d)² for two-group comparison

**Card 19:** Bonferroni correction - Divide α by number of tests to control FWER

**Card 20:** Benjamini-Hochberg - Controls FDR (expected proportion of false discoveries)
