# Mini-Project: 3D Graphics Engine

## Objective
Build a simple 3D graphics engine using transformation matrices.

## Requirements
1. Define 3D points and wireframe objects (cube, pyramid, sphere approximation)
2. Implement transformation matrices: translation, rotation (x/y/z), scaling
3. Apply perspective projection to map 3D → 2D
4. Render wireframe using simple ASCII or Java graphics (Swing/JavaFX)

## Extensions
- Add a camera system (view matrix)
- Implement object hierarchy (scene graph)
- Add polygon fill and basic lighting (flat shading)
- Support loading simple 3D model files (OBJ format)

## Evaluation Criteria
- Correct matrix transformations
- Proper projection
- Interactive controls (rotation, zoom)
- Code organization
