# Why Discrete Mathematics Matters

Discrete math is the mathematical backbone of computer science.

## Applications

| Domain | Use |
|--------|-----|
| Algorithms | Correctness proofs, complexity analysis |
| Databases | Relational algebra, query optimization |
| Cryptography | RSA, Diffie-Hellman, elliptic curve crypto |
| Compilers | Parse trees, type checking, optimization |
| Networking | Routing, error correction, protocol verification |
| AI/ML | Boolean circuits, decision trees, logic programming |

## In Java

```java
// Every boolean expression is propositional logic
boolean isValid = (age >= 18) && hasLicense;

// Every Set/Map uses hash functions (modular arithmetic)
Set<String> users = new HashSet<>();

// Every thread synchronization uses discrete logic
synchronized (lock) { ... }
```
