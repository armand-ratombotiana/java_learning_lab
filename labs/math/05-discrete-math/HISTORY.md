# History of Discrete Mathematics

## Timeline

| Period | Development |
|--------|------------|
| ~300 BCE | Euclid's *Elements* — number theory, GCD algorithm |
| ~250 BCE | Eratosthenes' sieve for primes |
| 1654 CE | Pascal and Fermat: probability theory foundations |
| 1847 CE | Boole: *The Mathematical Analysis of Logic* |
| 1879 CE | Frege: predicate logic, quantifiers |
| 1936 CE | Turing: computability, halting problem |
| 1937 CE | Shannon: Boolean algebra for digital circuits |
| 1948 CE | Shannon: information theory |
| 1960s | Knuth: analysis of algorithms |
| 1976 CE | Diffie-Hellman: public-key cryptography |
| 1977 CE | RSA: modular arithmetic for encryption |

## Key Figures

- **George Boole** (1815–1864) — Boolean algebra
- **Gottlob Frege** (1848–1925) — modern logic
- **Alan Turing** (1912–1954) — computability theory
- **Claude Shannon** (1916–2001) — information theory
