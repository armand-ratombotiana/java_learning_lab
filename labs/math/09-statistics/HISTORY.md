# History of Statistics

## Timeline

| Year | Development |
|------|------------|
| 1662 | John Graunt: first mortality tables (demographics) |
| 1710 | Arbuthnott: first statistical test (sex ratio) |
| 1805 | Legendre: method of least squares |
| 1888 | Galton: correlation and regression |
| 1900 | Pearson: chi-squared test |
| 1908 | Student's t-distribution (Gosset) |
| 1925 | Fisher: *Statistical Methods for Research Workers* |
| 1935 | Neyman-Pearson: hypothesis testing framework |
| 1950s | Bayesian statistics revival |
| 2000s | Big data: statistical learning, data science |

## Key Figures

- **Francis Galton** (1822–1911) — correlation, regression to the mean
- **Karl Pearson** (1857–1936) — chi-squared, correlation coefficient
- **R.A. Fisher** (1890–1962) — ANOVA, maximum likelihood, experimental design
- **William Gosset** (1876–1937) — Student's t-distribution
- **George Box** (1919–2013) — Box-Jenkins, "All models are wrong, but some are useful"
