# Number Theory

The study of integers and their properties, forming the foundation of modern cryptography.

## Scope

Prime numbers, modular arithmetic, GCD, RSA, Chinese Remainder Theorem, primality testing

## Java Implementation

The number theory package (com.math10) provides production-quality implementations.

## Prerequisites

- Strong foundation in high-school mathematics
- Java 21+ programming experience
- Familiarity with algorithmic thinking

## How to Use This Lab

1. Read THEORY.md for comprehensive mathematical treatment
2. Review MATH_FOUNDATION.md for prerequisite review
3. Study CODE_DEEP_DIVE.md for implementation patterns
4. Complete EXERCISES.md problem sets
5. Build the MINI_PROJECT for hands-on application
