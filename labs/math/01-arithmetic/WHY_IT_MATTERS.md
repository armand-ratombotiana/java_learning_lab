# Why Arithmetic Matters

Arithmetic is the most universally applied branch of mathematics.

## Real-World Impact

| Domain | Application |
|--------|------------|
| Finance | Interest calculation, budgeting, taxes |
| Engineering | Stress analysis, dimensions, tolerances |
| Computer Science | Hash codes, memory addresses, bit manipulation |
| Medicine | Dosage calculations, vital statistics |
| Daily Life | Shopping, cooking, time management, fuel economy |

## Java Context

Every Java program performs arithmetic — from simple counters to complex financial calculations. The `BigDecimal` class exists solely to handle the precision arithmetic demands of monetary computation.
