# History of Arithmetic

## Timeline

| Period | Development |
|--------|------------|
| ~30000 BCE | Tally marks on bone (Ishango bone) |
| ~3400 BCE | Sumerian numeral system |
| ~300 BCE | Euclid's *Elements* — number theory foundations |
| 628 CE | Brahmagupta formalizes zero and negative numbers |
| 1202 CE | Fibonacci's *Liber Abaci* introduces Hindu-Arabic numerals to Europe |
| 1614 CE | Napier invents logarithms |
| 1642 CE | Pascal builds the first mechanical calculator |
| 1945 CE | ENIAC ushers in electronic arithmetic |

## Key Figures

- **Brahmagupta** (598–668 CE) — rules for zero, negatives
- **Al-Khwarizmi** (780–850 CE) — algorithms for arithmetic operations
- **John von Neumann** (1903–1957) — binary arithmetic in stored-program computers
