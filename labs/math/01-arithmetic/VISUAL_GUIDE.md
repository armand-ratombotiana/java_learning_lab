# Visual Guide to Arithmetic

## The Four Operations

```
  Addition          Subtraction       Multiplication      Division
  a + b = c         a - b = c         a × b = c           a ÷ b = c
    a                    a                 a                 a
   +|                   -|                ×|                ÷|
    b                    b                 b                 b
  ---                  ---               ---               ---
   =c                   =c                =c                =c
```

## Number Line Operations

```
Add 2 to 3:   0---1---2---3---4---5
                          |---→   = 5

Subtract 2 from 5:  0---1---2---3---4---5
                              ←---|---   = 3
```

## The Carry Concept

   47
