# Mental Models for Arithmetic

## The Number Line

Visualize numbers as points on an infinite line. Addition moves right, subtraction moves left.

```
<--|----|----|----|----|----|----|-->
  -3   -2   -1    0    1    2    3
```

## The Balance Scale

An equation like $a + b = c$ is a balanced scale. Whatever you do to one side, you must do to the other.

## The Area Model

Multiplication $a \times b$ is the area of a rectangle with sides $a$ and $b$.

```
+-------+
|       | b
+-------+ a
```

## Stacking (Column Arithmetic)

Align numbers by place value, compute from rightmost column, carry overflow.

## Java HashMap Analogy

The `hashCode()` method uses arithmetic (typically prime multiplication) to distribute keys — a direct application of modular arithmetic.
