# Visual Guide to Geometry

## Coordinate System

```
(-x, +y)  │  (+x, +y)
          │
──────────┼──────────
          │
(-x, -y)  │  (+x, -y)
```

## Triangles

```
     /\
    /  \
 a /    \ b
  /      \
 /________\
     c

Area = ½ × base × height
```

## Circle

```
   .---.
  /     \
 |   r   |
  \     /
   '---'

Area = πr²
Circumference = 2πr
```
