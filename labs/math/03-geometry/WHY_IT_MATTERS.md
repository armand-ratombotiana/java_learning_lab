# Why Geometry Matters

Geometry is essential for interacting with and constructing the physical world.

## Applications

| Domain | Use |
|--------|-----|
| Computer Graphics | Rendering, ray tracing, collision detection |
| GIS & Maps | Projections, spatial queries, routing |
| Robotics | Path planning, inverse kinematics, sensor fusion |
| CAD/CAM | 3D modeling, manufacturing tolerances |
| Physics Engine | Hitboxes, bounding volumes, force vectors |

## In Java

```java
// Java 2D uses geometric transforms extensively
AffineTransform transform = new AffineTransform();
transform.translate(100, 50);
transform.rotate(Math.PI / 4);
transform.scale(2.0, 2.0);
```
