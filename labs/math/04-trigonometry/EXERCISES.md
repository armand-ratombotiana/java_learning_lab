# Exercises: Trigonometry

**1.** Find sin(30°), cos(60°), tan(45°)
- Solution: sin(30°) = ½, cos(60°) = ½, tan(45°) = 1

**2.** Simplify: sin²(x) + cos²(x)
- Solution: sin²(x) + cos²(x) = 1 (Pythagorean identity)

**3.** Solve: sin(x) = 0.5 for 0 ≤ x < 2π
- Solution: x = π/6, 5π/6

**4.** Prove: tan(x) = sin(x)/cos(x)
- Solution: tan(x) = sin(x)/cos(x) by definition, since sin/cos = tan. Dividing both sides of sin²+cos²=1 by cos² gives tan²+1=sec², confirming identity.

**5.** Find: sin(2θ) if sin(θ) = 3/5
- Solution: If sin(θ)=3/5, then cos(θ)=4/5. sin(2θ) = 2 sin(θ)cos(θ) = 2×3/5×4/5 = 24/25

**6.** Verify: sin(π/2 - x) = cos(x)
- Solution: Using identity sin(a-b) = sin(a)cos(b)-cos(a)sin(b), sin(π/2 - x) = sin(π/2)cos(x) - cos(π/2)sin(x) = 1·cos(x) - 0 = cos(x). Verified.

**7.** Solve: cos²(x) - sin²(x) = 0
- Solution: cos²(x) - sin²(x) = cos(2x) = 0 → 2x = π/2 + kπ → x = π/4 + kπ/2

**8.** Find all solutions: 2sin²(x) + sin(x) - 1 = 0
- Solution: Let u = sin(x). 2u² + u - 1 = 0 → (2u-1)(u+1) = 0 → u = ½ or u = -1. So sin(x) = ½ → x = π/6 + 2kπ or 5π/6 + 2kπ. sin(x) = -1 → x = 3π/2 + 2kπ.

**9.** Prove: sin(a+b) = sin(a)cos(b) + cos(a)sin(b)
- Solution: This is the sum formula for sine. Derived from Euler's formula: sin(a+b) = (e^(ia+b) - e^(-ia-b))/2i = sin(a)cos(b) + cos(a)sin(b). QED.

**10.** Find area of triangle with sides 5, 6, angle 60° between them
- Solution: Area = ½ × a × b × sin(C) = ½ × 5 × 6 × sin(60°) = 15 × (√3/2) = (15√3)/2 ≈ 12.99
