# Trigonometry: Functions, Identities, and Applications

## 1. Trigonometric Functions

### 1.1 Right Triangle Definitions
- sin(θ) = opposite/hypotenuse
- cos(θ) = adjacent/hypotenuse
- tan(θ) = opposite/adjacent = sin(θ)/cos(θ)

### 1.2 Reciprocal Functions
- csc(θ) = 1/sin(θ) = hypotenuse/opposite
- sec(θ) = 1/cos(θ) = hypotenuse/adjacent
- cot(θ) = 1/tan(θ) = adjacent/opposite

### 1.3 Unit Circle Definitions
- sin(θ) = y-coordinate
- cos(θ) = x-coordinate
- tan(θ) = y/x

## 2. Trigonometric Identities

### 2.1 Pythagorean Identities
- sin²(θ) + cos²(θ) = 1
- 1 + tan²(θ) = sec²(θ)
- 1 + cot²(θ) = csc²(θ)

### 2.2 Co-Function Identities
- sin(π/2 - θ) = cos(θ)
- cos(π/2 - θ) = sin(θ)
- tan(π/2 - θ) = cot(θ)

### 2.3 Double Angle Formulas
- sin(2θ) = 2sin(θ)cos(θ)
- cos(2θ) = cos²(θ) - sin²(θ) = 2cos²(θ) - 1 = 1 - 2sin²(θ)
- tan(2θ) = 2tan(θ)/(1 - tan²(θ))

### 2.4 Half Angle Formulas
- sin(θ/2) = ±√[(1-cos(θ))/2]
- cos(θ/2) = ±√[(1+cos(θ))/2]
- tan(θ/2) = sin(θ)/(1+cos(θ)) = (1-cos(θ))/sin(θ)

### 2.5 Sum/Difference Formulas
- sin(α±β) = sin(α)cos(β) ± cos(α)sin(β)
- cos(α±β) = cos(α)cos(β) ∓ sin(α)sin(β)
- tan(α±β) = (tan(α)±tan(β))/(1∓tan(α)tan(β))

## 3. Inverse Trigonometric Functions

### 3.1 Definitions
- arcsin(x): sin⁻¹(x) where -π/2 ≤ θ ≤ π/2
- arccos(x): cos⁻¹(x) where 0 ≤ θ ≤ π
- arctan(x): tan⁻¹(x) where -π/2 < θ < π/2

### 3.2 Properties
- sin(sin⁻¹(x)) = x for x ∈ [-1,1]
- sin⁻¹(sin(θ)) = θ only for θ ∈ [-π/2,π/2]
- sin⁻¹(x) + arccos(x) = π/2

## 4. Solving Trigonometric Equations

### 4.1 Basic Equations
- sin(x) = a → x = arcsin(a) + 2πk or x = π - arcsin(a) + 2πk
- cos(x) = a → x = arccos(a) + 2πk or x = -arccos(a) + 2πk
- tan(x) = a → x = arctan(a) + πk

### 4.2 Strategies
1. Factor expressions
2. Use identities to simplify
3. Convert to single function
4. Find principal values, then add period

## 5. Law of Sines and Cosines

### 5.1 Law of Sines
sin(A)/a = sin(B)/b = sin(C)/c

### 5.2 Law of Cosines
c² = a² + b² - 2ab·cos(C)
Used when: SAS, SSS cases

### 5.3 Area Formulas
- A = (1/2)ab·sin(C)
- A = √[s(s-a)(s-b)(s-c)] (Heron)

## 6. Graphs of Trigonometric Functions

### 6.1 Sine and Cosine
- Period: 2π
- Amplitude: |A|
- Phase shift: -B/C
- Vertical shift: D

### 6.2 Tangent
- Period: π
- Vertical asymptotes at odd multiples of π/2

### 6.3 Secant and Cosecant
- Reciprocals of cosine and sine
- Same period as their reciprocals
