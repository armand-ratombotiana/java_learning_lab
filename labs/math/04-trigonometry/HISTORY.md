# History of Trigonometry

## Timeline

| Period | Development |
|--------|------------|
| ~1800 BCE | Plimpton 322 tablet — Pythagorean triples |
| ~150 BCE | Hipparchus creates first trigonometric table (chords) |
| ~100 CE | Menelaus: spherical trigonometry |
| ~500 CE | Aryabhata: sine (jya) and cosine (kojya) tables |
| ~830 CE | Al-Khwarizmi: accurate sine/cosine tables |
| 1543 CE | Copernicus uses trigonometry for heliocentric model |
| 1614 CE | Napier invents logarithms — simplifies trig calculations |
| 1748 CE | Euler: $e^{i\theta} = \cos\theta + i\sin\theta$ |
| 1822 CE | Fourier: any periodic function is sum of sines/cosines |

## Key Figures

- **Hipparchus** (c. 150 BCE) — father of trigonometry
- **Ptolemy** (c. 150 CE) — *Almagest* chord tables
- **Leonhard Euler** (1707–1783) — modern notation, Euler's formula
- **Jean-Baptiste Joseph Fourier** (1768–1830) — Fourier series
