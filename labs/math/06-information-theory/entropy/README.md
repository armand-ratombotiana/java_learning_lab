# Entropy & Information Theory

Welcome to the atomic mastery lab for **Entropy**. This lab is part of the Math Academy's Information Theory module.

## 🧠 What You Will Master
- The concept of Uncertainty and Information Surprise.
- Shannon Entropy formula and derivation.
- Cross-Entropy and its role in Machine Learning loss functions.
- KL Divergence (measuring difference between distributions).
- Implementing Entropy calculators in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Intuition behind bits of surprise.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - Formal Shannon and Cross-Entropy math.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of Entropy and KL Divergence.
