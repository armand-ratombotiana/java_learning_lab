# Why Algebra Matters

Algebra is the language of quantitative relationships.

## Applications

| Domain | Use |
|--------|-----|
| Engineering | Ohm's law $V=IR$, stress-strain $\sigma = E\varepsilon$ |
| Computer Science | Big-O analysis, type systems, database queries |
| Economics | Supply-demand equilibrium, compound interest |
| Physics | $F=ma$, $E=mc^2$, kinematics equations |
| Graphics | Linear transformations, rotations, projections |

## In Java

Every generic type uses algebraic thinking (`List<T>` is like $f(x)$). Every SQL query is relational algebra. Every sorting algorithm analyzes complexity using algebraic growth rates.
