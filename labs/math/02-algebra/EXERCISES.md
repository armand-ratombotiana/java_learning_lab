# Exercises: Algebra

## Level 1: Linear Equations

**1.** Solve: 3x + 7 = 22
- Solution: 3x = 15 → x = 5

**2.** Solve: 2x - 5 = 3x + 1
- Solution: -5 - 1 = 3x - 2x → x = -6

**3.** Solve: 4(x - 2) = 3x + 6
- Solution: 4x - 8 = 3x + 6 → x = 14

**4.** Solve: (x + 1)/2 = (x - 1)/3
- Solution: 3(x+1) = 2(x-1) → 3x+3 = 2x-2 → x = -5

**5.** Solve: 0.5x + 1.5 = 3.5
- Solution: 0.5x = 2 → x = 4

## Level 2: Systems of Equations

**6.** Solve: 2x + 3y = 7, x - y = 3
- Solution: From (2): x = y + 3. Substitute: 2(y+3) + 3y = 7 → 5y + 6 = 7 → y = 0.2 → x = 3.2

**7.** Solve: 3x + 4y = 10, 2x - y = 3
- Solution: y = 2x - 3. Substitute: 3x + 4(2x-3)=10 → 11x = 22 → x=2, y=1

**8.** Solve: x + y = 5, 2x + 2y = 10 (what does this tell you?)
- Solution: Second equation is 2(x+y)=10, same as first. Infinite solutions along line x+y=5.

**9.** Solve: x + 2y + 3z = 6, 2x + y + z = 6, x + y + z = 4
- Solution: Subtract third from first: y+2z=2. Subtract third from second: y=2. Then z=0, x=2

**10.** Find k such that system has infinite solutions: 2x + ky = 6, 4x + 2ky = 12
- Solution: Second eq is 2×first. For infinite solutions, both equations must be identical, so k=2.

## Level 3: Quadratic Equations

**11.** Solve: x² - 5x + 6 = 0
- Solution: (x-2)(x-3)=0 → x = 2 or 3

**12.** Solve: 2x² + 5x - 3 = 0
- Solution: (2x-1)(x+3)=0 → x = 1/2 or x = -3

**13.** Solve: x² + x + 1 = 0
- Solution: Discriminant = 1-4=-3, roots = (-1 ± i√3)/2 (complex)

**14.** Solve by completing square: x² + 6x + 5 = 0
- Solution: (x+3)² = 4 → x+3 = ±2 → x = -1 or x = -5

**15.** Find discriminant and nature of roots for: 3x² - 4x + 1 = 0
- Solution: Δ = 16 - 12 = 4 > 0, two distinct real roots

## Level 4: Polynomials

**16.** Factor: x³ - 6x² + 11x - 6
- Solution: (x-1)(x-2)(x-3)

**17.** Factor: 2x³ - 5x² - 3x + 6
- Solution: (x-2)(2x+3)(x-1)

**18.** Divide: (x⁴ - 2x³ - 5x² + 7x - 3) ÷ (x - 2)
- Solution: Quotient: x³ - 5x - 3, remainder: -3

**19.** Find all roots of: x⁴ - 5x² + 4 = 0
- Solution: Let u=x², u²-5u+4=0 → u=1,4 → x=±1, ±2

**20.** Use synthetic division to evaluate P(3) where P(x) = x³ - 2x² + 4x - 1
- Solution: Synthetic: 3 | 1 -2 4 -1 → 1 1 7 20. P(3) = 20

## Level 5: Factoring

**21.** Factor: x² - 9
- Solution: (x+3)(x-3)

**22.** Factor: 4x² - 12x + 9
- Solution: (2x-3)²

**23.** Factor: x³ + 8
- Solution: (x+2)(x² - 2x + 4)

**24.** Factor: x⁴ - 16
- Solution: (x²+4)(x²-4) = (x²+4)(x+2)(x-2)

**25.** Factor by grouping: x³ + 3x² + 2x + 6
- Solution: x²(x+3) + 2(x+3) = (x²+2)(x+3)

## Level 6: Rational Expressions

**26.** Simplify: (x² - 9)/(x² - 6x + 9)
- Solution: (x+3)(x-3)/(x-3)² = (x+3)/(x-3), x≠3

**27.** Simplify: (x² - 4)/(x + 2) · (x + 1)/(x² - 1)
- Solution: (x-2)·(x+1)/(x+1)(x-1) = (x-2)/(x-1), x≠±1

**28.** Add: 1/(x - 1) + 1/(x + 1)
- Solution: (x+1 + x-1)/(x²-1) = 2x/(x²-1)

**29.** Simplify: (x/(x-1) - 1/x) ÷ (1/x + 1/(x-1))
- Solution: = (x²-x-(x-1))/(x(x-1)) ÷ ((x-1)+x)/(x(x-1)) = (x²-2x+1)/(2x-1) = (x-1)²/(2x-1)

**30.** Find partial fractions: 1/(x² - 1)
- Solution: 1/(x-1)(x+1) = ½/(x-1) - ½/(x+1)

## Level 7: Sequences and Series

**31.** Find 15th term of arithmetic sequence with a₁ = 3, d = 5
- Solution: a₁₅ = 3 + (15-1)×5 = 3 + 70 = 73

**32.** Find sum of first 20 terms of arithmetic sequence: 2, 5, 8, ...
- Solution: a₂₀ = 2 + 19×3 = 59, S₂₀ = 20(2+59)/2 = 610

**33.** Find 10th term of geometric sequence with a₁ = 3, r = 2
- Solution: a₁₀ = 3×2⁹ = 3×512 = 1536

**34.** Find sum of infinite geometric series: 1/2 + 1/4 + 1/8 + ...
- Solution: a₁=½, r=½, S = a₁/(1-r) = ½/(½) = 1

**35.** Evaluate: Σₖ₌₁¹⁰ k²
- Solution: n(n+1)(2n+1)/6 = 10×11×21/6 = 385

## Level 8: Binomial Theorem

**36.** Find coefficient of x⁵ in (2x - 3)⁸
- Solution: C(8,5)(2x)⁵(-3)³ = C(8,5)·32·(-27) = 56·32·(-27) = -48384

**37.** Find the middle term of (x + y)¹⁰
- Solution: 6th term (k=5): C(10,5)x⁵y⁵ = 252x⁵y⁵

**38.** Expand: (1 + x)⁶
- Solution: 1 + 6x + 15x² + 20x³ + 15x⁴ + 6x⁵ + x⁶

**39.** Find sum: C(5,0) + C(5,1) + ... + C(5,5)
- Solution: 2⁵ = 32

**40.** Prove: Σₖ₌₀ⁿ C(n,k) = 2ⁿ
- Solution: (1+1)ⁿ = ΣC(n,k)1ⁿ⁻ᵏ1ᵏ = ΣC(n,k) = 2ⁿ. QED.

## Level 9: Inequalities

**41.** Solve: |2x - 3| < 5
- Solution: -5 < 2x-3 < 5 → -2 < 2x < 8 → -1 < x < 4

**42.** Solve: x² - 3x - 4 > 0
- Solution: (x-4)(x+1) > 0 → x > 4 or x < -1

**43.** Solve: (x - 1)/(x + 2) ≤ 3
- Solution: (x-1)/(x+2) - 3 ≤ 0 → (x-1-3x-6)/(x+2) ≤ 0 → (-2x-7)/(x+2) ≤ 0. Solution: x < -2 or x ≥ -3.5

**44.** Find all solutions: |x + 1| + |x - 2| = 5
- Solution: Test intervals: x<-1: -x-1 -x+2 = 5 → -2x+1=5 → x=-2. -1≤x≤2: x+1-x+2=3≠5. x>2: x+1+x-2=5 → 2x-1=5 → x=3. Solutions: x=-2, x=3

**45.** Solve system: x + y ≥ 4, x - y ≤ 2, x ≥ 0, y ≥ 0
- Solution: Feasible region bounded by lines in first quadrant

## Level 10: Complex Numbers

**46.** Simplify: (3 + 4i) + (2 - 5i)
- Solution: (5 - i)

**47.** Find (1 + i)²
- Solution: 1 + 2i + i² = 1 + 2i - 1 = 2i

**48.** Find modulus of (3 + 4i)
- Solution: √(3² + 4²) = 5

**49.** Convert to polar form: 1 + i
- Solution: r = √2, θ = π/4. Polar form: √2(cos π/4 + i sin π/4)

**50.** Find all cube roots of 8
- Solution: 8 = 8∠0. Cube roots: 2∠0°, 2∠120°, 2∠240° = 2, -1+√3i, -1-√3i

## Challenge Problems

**51.** Find all real x such that: x⁴ - 5x² + 6 = 0
- Solution: Let u=x², u²-5u+6=0 → u=2,3 → x=±√2, ±√3

**52.** Solve: 2ˣ = x² + 1
- Solution: x=0, x=1 work (1=1, 2=2). By inspection, solutions are x=0 and x=1

**53.** If α, β are roots of x² - 3x + 1 = 0, find α⁴ + β⁴
- Solution: α+β=3, αβ=1. α²+β² = (α+β)²-2αβ = 9-2=7. α⁴+β⁴ = (α²+β²)²-2(α²β²) = 49-2=47

**54.** Find polynomial of degree 4 with roots: 1, 2, 3, 4
- Solution: (x-1)(x-2)(x-3)(x-4) = x⁴ - 10x³ + 35x² - 50x + 24

**55.** Prove: √2 + √3 is irrational
- Solution: Suppose rational = a/b. Then √2 = a/b - √3 → square both sides, get 5 - 2√6 = a²/b². Left has √6, right doesn't. Contradiction.
