# Visual Guide to Algebra

## The Cartesian Plane

```
      y
      ↑
      |   / y = 2x + 1
      |  /
    3-│ /
    2-│/
    1-│----→ x
      |  1  2
```

## Solving $x + 3 = 7$

```
  x + 3 = 7
  ──┬── ─┬─
    │    └── result
    └──────── unknown

  Step 1: x + 3 - 3 = 7 - 3
  Step 2: x = 4
```

## Quadratic Parabola $y = x^2 - 4$

```
      y
  4---│
  3---│
  2---│
  1---│
  0---│---│---→ x
     -2   2
      ↑   ↑
    roots: x = -2, x = 2
```

## System of Equations

Intersection of two lines:

$$
\begin{cases}
y = 2x + 1 \\
y = -x + 4
\end{cases}
$$

Solution is $(1, 3)$ — the point where both lines cross.
