# Mini-Project: Logistic Regression Classifier

## Objective
Implement binary logistic regression using gradient descent optimization.

## Requirements
1. Generate or load a binary classification dataset
2. Implement the sigmoid function and logistic loss (cross-entropy)
3. Train using gradient descent (any variant)
4. Visualize the decision boundary
5. Evaluate accuracy, precision, recall, F1-score

## Extensions
- Add L1/L2 regularization
- Implement multiclass logistic regression (softmax)
- Compare optimizer performance (SGD, Momentum, Adam)
- Add feature engineering (polynomial features)

## Evaluation Criteria
- Correct gradient computation
- Convergence of training loss
- Good classification performance
- Visualization of learning curve and decision boundary
