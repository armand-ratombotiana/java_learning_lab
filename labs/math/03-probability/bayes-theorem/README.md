# Bayes' Theorem

Welcome to the atomic mastery lab for **Bayes' Theorem**. This lab is part of the Math Academy's Probability module.

## 🧠 What You Will Master
- Conditional probability and independence.
- The intuition behind updating beliefs with new evidence.
- Priors, Likelihoods, and Posteriors.
- Naive Bayes classification.
- Implementing a Naive Bayes classifier in Java.

1. [THEORY.md](./THEORY.md) - Intuition behind Bayesian thinking.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - The mathematical formula and derivations.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a Naive Bayes classifier.
