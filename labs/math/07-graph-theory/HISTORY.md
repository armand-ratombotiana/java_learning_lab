# History of Graph Theory

## Timeline

| Year | Development |
|------|------------|
| 1736 | Euler solves Königsberg bridge problem — first graph theorem |
| 1852 | Four color conjecture (proved 1976: first computer-assisted proof) |
| 1847 | Kirchhoff: graph theory for electrical circuits |
| 1857 | Cayley: counting chemical isomers using trees |
| 1956 | Ford-Fulkerson: max flow / min cut |
| 1959 | Dijkstra's shortest path algorithm |
| 1969 | Stanley Milgram: "six degrees of separation" (small-world) |
| 1998 | PageRank algorithm (Google) |
| 2000s | Network science: scale-free networks (Barabási–Albert) |

## Key Figures

- **Leonhard Euler** (1707–1783) — father of graph theory
- **Gustav Kirchhoff** (1824–1887) — circuit theory, spanning trees
- **Paul Erdős** (1913–1996) — random graphs, extremal graph theory
- **Edsger Dijkstra** (1930–2002) — shortest paths, graph algorithms
