# References

- *Introduction to Algorithms* by CLRS (chapters on graph algorithms)
- *Network Flows* by Ahuja, Magnanti, Orlin
- *Graph Theory* by Reinhard Diestel
- *The Algorithm Design Manual* by Steven Skiena

- [JGraphT](https://jgrapht.org/) — comprehensive graph library for Java
- [Guava Graph](https://github.com/google/guava) — Google's graph API
- [Neo4j](https://neo4j.com/) — graph database (Java-based)
- [Apache TinkerPop](https://tinkerpop.apache.org/) — graph computing framework
