# References

## Books

- *Calculus* by Michael Spivak (rigorous)
- *Calculus: Early Transcendentals* by James Stewart (comprehensive)
- *Introduction to Calculus and Analysis* by Courant & John
- *Numerical Methods* by Richard Burden (numerical calculus)
- *Deep Learning* by Goodfellow, Bengio, Courville (calculus in ML)

- [Apache Commons Math](https://commons.apache.org/proper/commons-math/) — integration, ODEs, optimization
- [ND4J](https://nd4j.org/) — scientific computing and auto-diff for Java
- [DeepLearning4J](https://deeplearning4j.konduit.ai/) — deep learning with automatic differentiation
- [DJL](https://djl.ai/) — Deep Java Library
