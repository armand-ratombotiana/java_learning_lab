# Why Combinatorics Matters

Combinatorics is essential for analyzing possibilities and complexity.

## Applications

| Domain | Use |
|--------|-----|
| Cryptography | Key space analysis, password cracking difficulty |
| Algorithm Analysis | Worst-case path counting, combinatorial explosion |
| Machine Learning | Feature selection, decision tree complexity |
| Network Design | Topology optimization, routing tables |
| Biology | DNA sequence analysis, protein folding |
| Game Theory | Strategy enumeration, optimal play |

## In Java

```java
// Number of ways to choose k items from n
int combinations = binomial(52, 5); // 2,598,960 poker hands

// Number of permutations of n items
long perms = factorial(10); // 3,628,800 arrangements
```
