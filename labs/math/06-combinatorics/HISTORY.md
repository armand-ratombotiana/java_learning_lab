# History of Combinatorics

## Timeline

| Period | Development |
|--------|------------|
| ~3000 BCE | Chinese hexagrams (I Ching) — 64 arrangements |
| ~300 BCE | Euclid studies permutation counts |
| 1654 CE | Pascal and Fermat correspondence on gambling problems |
| 1666 CE | Leibniz: *De Arte Combinatoria* |
| 1713 CE | Bernoulli: *Ars Conjectandi* — binomial theorem |
| 1751 CE | Euler: partitions, generating functions |
| 1850s | Cayley: counting chemical compounds (trees) |
| 1930s | Ramsey theory: order in large structures |
| 1960s | Pólya enumeration theory |

## Key Figures

- **Blaise Pascal** (1623–1662) — Pascal's triangle, binomial coefficients
- **Leonhard Euler** (1707–1783) — generating functions, graph theory
- **Paul Erdős** (1913–1996) — probabilistic method, Ramsey theory
- **Gian-Carlo Rota** (1932–1999) — modern combinatorics
