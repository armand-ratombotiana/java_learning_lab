# Gradient Descent

Welcome to the atomic mastery lab for **Gradient Descent**. This lab is part of the Math Academy's Optimization module.

## 🧠 What You Will Master
- The intuition behind minimizing a cost function.
- The mathematics of the gradient vector.
- The role of the Learning Rate (Step Size).
- Variants: Batch, Stochastic (SGD), and Mini-Batch Gradient Descent.
- Implementing Gradient Descent from scratch in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The mountain climber intuition and cost functions.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - Derivatives, partial derivatives, and the update rule.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of Gradient Descent for Linear Regression.
