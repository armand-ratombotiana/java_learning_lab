# HTTP Protocol - Quiz

1. Which HTTP method is idempotent and safe? A) POST B) PUT C) GET D) DELETE
2. What port does HTTPS use by default? A) 80 B) 443 C) 8080 D) 8443
3. Which HTTP version uses QUIC? A) HTTP/1.1 B) HTTP/2 C) HTTP/3 D) HTTP/1.0
4. What header is used for content negotiation? A) Content-Type B) Accept C) Allow D) Vary
5. What status code indicates resource created? A) 200 B) 201 C) 202 D) 204
6. Which HTTP/2 feature solves head-of-line blocking? A) Header compression B) Multiplexing C) Server push D) Binary
7. What is the purpose of ETag? A) Authentication B) Cache validation C) Content type D) Encoding

Answers: 1-C, 2-B, 3-C, 4-B, 5-B, 6-B, 7-B
