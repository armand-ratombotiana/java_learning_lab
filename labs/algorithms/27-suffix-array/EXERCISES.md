# Exercises — Suffix Array

## Beginner
1. Compute all suffixes of "algorithm" and sort them manually
2. Implement a function to check if a suffix array is valid (suffixes are sorted)
3. Compute the LCP array manually for a given string
4. Find the number of distinct substrings in "ababa"

## Intermediate
5. Implement the prefix-doubling suffix array construction
6. Implement Kasai's LCP algorithm
7. Find the longest repeated substring in a string
8. Count the number of occurrences of a pattern using binary search on SA

## Advanced
9. Implement the SA-IS linear-time suffix array construction
10. Implement pattern matching with LCP-accelerated binary search
11. Compute the longest common substring of two strings using suffix arrays
12. Build the Burrows-Wheeler transform from the suffix array
