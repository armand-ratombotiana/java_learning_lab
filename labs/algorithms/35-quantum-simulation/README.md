# Quantum Algorithm Simulation — Overview

This lab simulates quantum computing concepts classically using Java. Topics include qubit state representation, quantum gates (Hadamard, Pauli, CNOT, Toffoli), Grover's search algorithm simulation, and the quantum Fourier transform (QFT). While classical simulation cannot provide quantum speedup, it serves as an educational tool for understanding quantum algorithms.

## Learning Objectives

- Simulate single and multi-qubit states as complex vectors
- Implement common quantum gates as unitary matrices
- Simulate Grover's search algorithm for unstructured search
- Implement the quantum Fourier transform
- Understand measurement and probability amplitudes

## Prerequisites

- Complex number arithmetic
- Matrix multiplication
- Basic understanding of quantum mechanics concepts (superposition, entanglement)

## Estimated Time

- **Theory**: 100 minutes
- **Practice**: 110 minutes
- **Exercises**: 100 minutes
