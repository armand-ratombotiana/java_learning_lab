# Exercises — Quantum Simulation

## Beginner
1. Compute H|0> and H|1> manually
2. Apply CNOT to |01> and |11>
3. Verify that X^2 = I, H^2 = I
4. Compute the probability of measuring |0> in state |+> = (|0>+|1>)/sqrt(2)

## Intermediate
5. Implement the single qubit simulator with all Pauli gates
6. Implement CNOT for two-qubit systems
7. Simulate Grover's algorithm for 2 qubits
8. Implement the Hadamard gate on any qubit in multi-qubit system

## Advanced
9. Implement QFT for up to 4 qubits
10. Simulate Deutsch-Jozsa algorithm for n=3
11. Implement the Toffoli gate and verify its truth table
12. Verify Grover's algorithm works for a 3-qubit (8-element) search
