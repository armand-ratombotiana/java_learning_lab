# Theory
Pollard's Rho: uses Floyd's cycle detection with f(x) = x^2 + c mod n to find nontrivial factors in O(sqrt(p)) time for prime factor p. Elliptic curves: y^2 = x^3 + ax + b over finite fields, with group law for point addition. Continued fractions: periodic expansion of sqrt(n) for solving Pell's equation.
