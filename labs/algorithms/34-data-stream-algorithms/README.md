# Data Stream Algorithms — Overview

This lab covers algorithms for processing massive data streams using sublinear memory. Topics include reservoir sampling for uniform random samples, the Alon-Matias-Szegedy (AMS) algorithm for frequency moment estimation, frequent items (Misra-Gries, Lossy Counting, Space-Saving), Count-Min Sketch, and sliding window distinct counting. These algorithms are essential for network monitoring, database query optimization, and big data analytics.

## Learning Objectives

- Implement reservoir sampling for streaming data
- Estimate the second frequency moment using the AMS algorithm
- Find frequent items using Misra-Gries and Space-Saving
- Count distinct elements in a sliding window
- Compute streaming mean and variance in O(1) space

## Prerequisites

- Basic probability and statistics
- Hash functions
- Big O notation for space complexity
