# Quick Sort Algorithm

Welcome to the atomic mastery lab for **Quick Sort**. This lab is part of the Algorithms Academy's Sorting module.

## 🧠 What You Will Master
- The Divide and Conquer paradigm.
- Pivot selection strategies and their impact on complexity.
- Partitioning schemes: Lomuto vs. Hoare.
- Worst-case $O(n^2)$ scenarios and how to avoid them (Randomized Quick Sort).
- In-place sorting and space efficiency.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Intuition and partitioning logic.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - Average and worst-case time complexity proofs.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation using the Hoare partition scheme.
