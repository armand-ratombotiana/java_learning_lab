# Exercises — Bit Manipulation

## Beginner
1. Implement isPowerOfTwo without conditionals
2. Write a function to count set bits using Kernighan's algorithm
3. Convert binary to Gray code and back
4. Find the lowest set bit of a number

## Intermediate
5. Build an XOR basis from an array of integers
6. Implement the maximum XOR subset query
7. Generate all subsets of a set using bitmasks
8. Solve TSP with 10 cities using bit DP

## Advanced
9. Implement bit reversal using divide-and-conquer
10. Solve the Hamiltonian path problem with bit DP
11. Implement Gray code sequence generation for n bits
12. Find the next higher number with the same number of set bits
