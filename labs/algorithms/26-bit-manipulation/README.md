# Bit Manipulation Algorithms — Overview

This lab covers essential bit manipulation techniques used in competitive programming and systems programming: bitwise tricks, Gray code generation, XOR basis (linear basis over GF(2)), subset enumeration via bitmasks, and bit DP for NP-hard problems like the Traveling Salesman Problem. Bit manipulation exploits the parallel nature of bitwise operations to achieve constant-factor speedups and sometimes reduce exponential complexity classes.

## Learning Objectives

- Implement common bit tricks (isPowerOf2, population count, trailing zeros, next power of two)
- Generate Gray code sequences and convert between binary and Gray representations
- Build an XOR basis for linear algebra over GF(2)
- Enumerate subsets and supersets using bitmask iteration
- Solve TSP and Hamiltonian path problems using bit DP with O(n^2 * 2^n) complexity

## Prerequisites

- Java integer types and bitwise operators (&amp;, |, ^, ~, &lt;&lt;, &gt;&gt;, &gt;&gt;&gt;)
- Binary number representation
- Basic understanding of dynamic programming
