# Interview Questions — Bit Manipulation

1. Check if a number is a power of two (Amazon, Google)
2. Count set bits of a 32-bit integer (Microsoft)
3. Find the only non-repeating element in an array where all others repeat twice (XOR solution)
4. Find the two non-repeating elements (XOR + bit isolation)
5. Reverse bits of a 32-bit integer (Apple)
6. Compute the next power of two (Uber)
7. Maximum XOR of two numbers in an array (building XOR trie) (Google)
8. Subset sum via meet-in-the-middle (hard, for competitive programming roles)
9. Solve traveling salesman with DP (for algorithm-heavy roles)
10. Generate Gray code sequence (Amazon)
