# Exercises — Online Algorithms

## Beginner
1. Trace LRU on page sequence [1,2,3,4,1,2,5,1,2,3] with k=3
2. Compute competitive ratio of a given paging algorithm
3. Solve ski rental with B=5 for total days 3, 5, and 10
4. Simulate the secretary problem for n=5 candidates

## Intermediate
5. Implement LRU and FIFO and compare fault counts
6. Implement Marker algorithm with randomized eviction
7. Implement the deterministic ski rental strategy
8. Run secretary problem simulation 10000 times to verify 1/e

## Advanced
9. Implement epsilon-greedy with epsilon annealing
10. Implement UCB algorithm and compare with epsilon-greedy
11. Prove the k-competitive lower bound for deterministic paging
12. Implement the optimal randomized algorithm for ski rental
