# Online Algorithms — Overview

This lab covers online algorithms that process input incrementally without knowledge of future requests. Topics include competitive analysis, paging algorithms (LRU, FIFO, Marker), the ski rental problem, the secretary problem, and multi-armed bandits with epsilon-greedy strategies. Online algorithm design is crucial for real-time systems, caching, and reinforcement learning.

## Learning Objectives

- Implement LRU, FIFO, and Marker paging algorithms
- Analyze competitive ratios for paging algorithms
- Solve the ski rental problem with optimal deterministic strategy
- Simulate the secretary problem (optimal stopping)
- Implement epsilon-greedy multi-armed bandit algorithm

## Prerequisites

- Basic data structures (queues, linked lists)
- Probability fundamentals
- Algorithmic thinking for adversarial settings
