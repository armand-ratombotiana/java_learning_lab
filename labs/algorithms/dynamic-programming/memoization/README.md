# Memoization & Dynamic Programming

Welcome to the atomic mastery lab for **Memoization**. This lab is part of the Algorithms Academy's Dynamic Programming module.

## 🧠 What You Will Master
- The core philosophy of Dynamic Programming (DP).
- Overlapping Subproblems and Optimal Substructure.
- Top-Down (Memoization) vs Bottom-Up (Tabulation) approaches.
- Time complexity reduction from Exponential $O(2^N)$ to Linear $O(N)$.
- Implementing a Memoized recursive algorithm in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The intuition behind caching subproblem results.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - The recurrence relation and time complexity analysis.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of the Fibonacci sequence using Memoization.
