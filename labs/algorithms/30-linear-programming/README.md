# Linear Programming (Simplex Method) — Overview

This lab covers linear programming with the simplex method, a fundamental optimization technique. The simplex algorithm navigates vertices of the feasible polytope to find optimal solutions. Topics include the tableau representation, the two-phase method for problems without an obvious basic feasible solution, duality theory, and sensitivity analysis.

## Learning Objectives

- Formulate linear programs in standard form
- Implement the simplex tableau solver
- Apply the two-phase method for general LPs
- Understand duality and complementary slackness
- Perform sensitivity analysis on constraint coefficients

## Prerequisites

- Matrix operations and linear algebra
- Understanding of systems of linear equations
- Basic optimization concepts
