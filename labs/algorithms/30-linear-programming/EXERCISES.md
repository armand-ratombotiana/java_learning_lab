# Exercises — Linear Programming

## Beginner
1. Convert an LP with <= constraints to standard form
2. Solve a 2-variable LP graphically
3. Identify basic and non-basic variables in a given tableau
4. Perform one pivot of the simplex method manually

## Intermediate
5. Implement the simplex tableau solver
6. Add the two-phase method for problems without BFS
7. Extract optimal solution and dual values from the final tableau
8. Implement sensitivity analysis for RHS changes

## Advanced
9. Implement the dual simplex method
10. Solve a transportation problem using LP
11. Formulate max flow as an LP and solve with simplex
12. Implement steepest edge pivot selection for faster convergence
