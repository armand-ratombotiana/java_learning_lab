# Binary Search Variants

Welcome to the atomic mastery lab for **Binary Search Variants**. This lab is part of the Algorithms Academy's Searching module.

## 🧠 What You Will Master
- Standard Binary Search and its invariants.
- Finding First and Last occurrences in sorted arrays.
- Searching in Rotated Sorted Arrays.
- Binary Search on Answer (Optimization problems).
- Avoiding Integer Overflow in midpoint calculation.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The power of logarithmic reduction.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - Proof of $O(\log N)$ and decision space analysis.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of 5 different search variants.
