# Exercises — FFT

## Beginner
1. Compute the DFT of [1,0,0,0] manually
2. Draw the butterfly diagram for N=4 FFT
3. Compute the convolution of [1,2] and [3,4] manually
4. Find the primitive root for modulus 998244353

## Intermediate
5. Implement the Cooley-Tukey FFT with complex numbers
6. Multiply two polynomials using FFT
7. Implement the inverse FFT and verify round-trip
8. Implement NTT with modular arithmetic

## Advanced
9. Implement FFT for non-power-of-two sizes using mixed-radix
10. Solve the "matching with wildcards" problem using FFT
11. Implement a high-precision large integer multiplier using FFT
12. Compare FFT vs naive convolution performance for various sizes
