# Fast Fourier Transform — Overview

This lab covers the Cooley-Tukey Fast Fourier Transform (FFT), its application to polynomial multiplication, and the Number Theoretic Transform (NTT) for modular arithmetic. The FFT reduces the complexity of the Discrete Fourier Transform from O(n^2) to O(n log n), enabling efficient convolution in signal processing, image filtering, and computational mathematics.

## Learning Objectives

- Implement the Cooley-Tukey FFT using complex numbers
- Multiply polynomials using FFT-based convolution
- Implement NTT for modular arithmetic (inverse under prime modulus)
- Understand convolution and its applications
- Apply FFT to basic signal processing tasks

## Prerequisites

- Complex number arithmetic
- Recursive algorithm design
- Modular arithmetic (for NTT)
