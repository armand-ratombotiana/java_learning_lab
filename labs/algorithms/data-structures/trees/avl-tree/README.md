# AVL Tree

Welcome to the atomic mastery lab for **AVL Trees**. This lab is part of the Algorithms Academy's Data Structures module.

## 🧠 What You Will Master
- The concept of strict balancing in Binary Search Trees.
- The Balance Factor ($height(left) - height(right)$).
- The 4 types of rotations: LL, RR, LR, RL.
- Time complexity proofs for search, insert, and delete.
- Implementing an AVL Tree from scratch in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Strict balance vs. the Red-Black Tree.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - Calculating balance and rotation logic.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of self-balancing insertions.
