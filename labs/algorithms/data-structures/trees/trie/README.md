# Trie (Prefix Tree)

Welcome to the atomic mastery lab for the **Trie**. This lab is part of the Algorithms Academy's Data Structures module.

## 🧠 What You Will Master
- The concept of Prefix-based storage.
- Why Tries beat HashMaps for string prefix searches.
- Space vs. Time trade-offs in Trie nodes.
- Compressed Tries (Radix Trees).
- Implementing Autocomplete and Spell-check logic.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Intuition behind character-level branching.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - $O(L)$ time complexity analysis.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a Trie with search and prefix methods.
