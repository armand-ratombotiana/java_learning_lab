# Red-Black Trees

Welcome to the atomic mastery lab for **Red-Black Trees**. This lab is part of the Algorithms Academy's Data Structures module.

## 🧠 What You Will Master
- The problem with unbalanced Binary Search Trees (BSTs).
- The 5 mathematical properties of a Red-Black Tree.
- Time complexity of balanced vs. unbalanced trees.
- Rotations (Left and Right) and Color Flipping.
- Why Java's `TreeMap` and `HashMap` (in Java 8+) use Red-Black trees.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - The need for self-balancing trees.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - The 5 rules and height proofs.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of Red-Black insertions and rotations.
