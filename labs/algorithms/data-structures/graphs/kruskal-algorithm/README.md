# Kruskal's Algorithm

Welcome to the atomic mastery lab for **Kruskal's Algorithm**. This lab is part of the Algorithms Academy's Graph module.

## 🧠 What You Will Master
- Minimum Spanning Trees (MST) and their properties.
- The Greedy strategy for MST construction.
- Disjoint Set Union (DSU) / Union-Find data structure.
- Path Compression and Union by Rank optimizations.
- Implementing Kruskal's from scratch in Java.

1. [THEORY.md](./THEORY.md) - MST intuition and Kruskal's vs Prim's.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - Proof of correctness and complexity analysis.
