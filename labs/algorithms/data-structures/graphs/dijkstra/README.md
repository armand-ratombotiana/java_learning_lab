# Dijkstra's Shortest Path Algorithm

Welcome to the atomic mastery lab for **Dijkstra's Algorithm**. This lab is part of the Algorithms Academy's Graph module.

## 🧠 What You Will Master
- The concept of weighted graphs and shortest paths.
- The Greedy strategy behind Dijkstra's.
- Using Priority Queues to optimize search time.
- Time and Space complexity analysis.
- Implementing Dijkstra from scratch in Java.

1. [THEORY.md](./THEORY.md) - Intuition and greedy logic.
