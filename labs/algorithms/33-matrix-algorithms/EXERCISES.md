# Exercises — Matrix Algorithms

## Beginner
1. Multiply two 3x3 matrices manually
2. Solve a 3x3 linear system with Gaussian elimination by hand
3. Compute the determinant of a 3x3 matrix
4. Find the eigenvalues of a 2x2 matrix

## Intermediate
5. Implement Strassen's matrix multiplication
6. Implement Gaussian elimination with partial pivoting
7. Implement LU decomposition
8. Compute the determinant via LU decomposition

## Advanced
9. Implement the matrix inverse via LU
10. Implement power iteration for eigenvalues
11. Implement the SVD using the one-sided Jacobi method
12. Compare Strassen vs naive multiplication performance for various sizes
