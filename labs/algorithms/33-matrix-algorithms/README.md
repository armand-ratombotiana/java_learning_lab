# Matrix Algorithms — Overview

This lab covers advanced matrix algorithms essential in scientific computing and machine learning: Strassen's sub-cubic matrix multiplication, Gaussian elimination with partial pivoting, LU decomposition, determinant and inverse computation, and singular value decomposition (SVD). These algorithms form the computational backbone of linear algebra applications.

## Learning Objectives

- Implement Strassen's matrix multiplication (O(n^2.807))
- Perform Gaussian elimination with partial pivoting
- Compute LU decomposition, determinant, and matrix inverse
- Understand the SVD algorithm for dimensionality reduction
- Apply power iteration for dominant eigenvalue computation

## Prerequisites

- Matrix multiplication and basic linear algebra
- Recursive algorithms
- Floating-point arithmetic awareness
