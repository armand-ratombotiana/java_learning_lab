# Exercises — Push-Relabel

## Beginner
1. Compute max flow manually on a simple 4-node network
2. Identify the min cut in a flow network
3. Trace push-relabel on a triangle network
4. Verify the max-flow min-cut theorem on a small example

## Intermediate
5. Implement the generic push-relabel algorithm
6. Add the gap heuristic optimization
7. Implement the FIFO selection rule
8. Find the min cut from the residual graph after push-relabel

## Advanced
9. Implement min-cost max flow with potentials
10. Implement capacity scaling for max flow
11. Solve a bipartite matching problem using max flow
12. Implement the relabel-to-front variant
