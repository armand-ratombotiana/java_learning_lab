# Exercises — Exact Exponential Algorithms

## Beginner
1. Enumerate all subsets of {1,2,3,4,5} using bitmasks
2. Count subsets summing to 10 in {3,4,5,6,7}
3. Apply inclusion-exclusion to count multiples of 2 or 3 up to 30
4. Compute the zeta transform of a simple function

## Intermediate
5. Implement meet-in-the-middle for subset sum
6. Implement inclusion-exclusion for counting
7. Implement the zeta and Moebius transforms
8. Count independent sets in small graphs using inclusion-exclusion

## Advanced
9. Implement fast subset convolution
10. Implement a branching algorithm for vertex cover
11. Solve the maximum clique problem using meet-in-the-middle
12. Implement the split-and-list technique for knapsack
