# Exact Exponential Algorithms — Overview

This lab covers exact exponential algorithms for solving NP-hard problems faster than brute force. Topics include meet-in-the-middle (subset sum), inclusion–exclusion principle, split-and-list, fast subset convolution, and branching algorithms. These techniques reduce the base of the exponential, often achieving O(2^{n/2}) or O(1.618^n) instead of O(2^n).

## Learning Objectives

- Implement meet-in-the-middle for subset sum (O(2^{n/2}))
- Apply inclusion–exclusion for counting problems
- Implement fast subset convolution for DP over subsets
- Design branching algorithms for vertex cover and SAT
- Understand split-and-list technique

## Prerequisites

- Bitmask enumeration
- Recursive backtracking
- Basic probability and combinatorics
