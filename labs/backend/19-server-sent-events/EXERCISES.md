# Exercises: Server-Sent Events

## Exercise 1: Basic SSE Stream
Create an SSE endpoint that streams timestamp events every second.

## Exercise 2: Named Events
Create multiple named event types (update, heartbeat, notification).

## Exercise 3: Reactive SSE
Implement reactive SSE with WebFlux flux publishing.

## Exercise 4: Event Reconnection
Implement last-event-id tracking for reconnection.

## Exercise 5: Stock Ticker
Simulate a stock price feed using SSE with random price updates.
