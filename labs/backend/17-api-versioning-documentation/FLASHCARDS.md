# Flashcards: API Versioning & Documentation

**Card 1**: Q: What annotation adds operation documentation? A: @Operation

**Card 2**: Q: Default SpringDoc API docs path? A: /v3/api-docs

**Card 3**: Q: How to group APIs by version in SpringDoc? A: GroupedOpenApi bean

**Card 4**: Q: Which header indicates API removal date? A: Sunset

**Card 5**: Q: What is contract-first development? A: Writing OpenAPI spec before implementation

**Card 6**: Q: URL versioning vs header versioning trade-off? A: URL is explicit but clutters, header is clean but less visible
