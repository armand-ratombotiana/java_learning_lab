# History

- **2000**: Roy Fielding defines REST in his PhD dissertation
- **2002**: eBay API launches (one of the first REST APIs)
- **2006**: Amazon Web Services S3 REST API
- **2007**: Twitter REST API v1
- **2008**: GitHub API launched
- **2010**: Swagger specification created
- **2012**: RESTful API design becomes mainstream
- **2014**: OpenAPI Specification (fork of Swagger)
- **2016**: OpenAPI 3.0 released
- **2020**: OpenAPI 3.1 (semantic version, JSON Schema alignment)
- **2023**: Spring Framework 6 with declarative HTTP clients

REST evolved from a theoretical concept to the dominant API paradigm through simplicity and web alignment.
