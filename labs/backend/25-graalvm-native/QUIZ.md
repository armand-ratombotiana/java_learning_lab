# Quiz: GraalVM Native

Q1: What does AOT stand for? A: Ahead-of-Time
Q2: What is SubstrateVM? A: GraalVM's minimal runtime for native images
Q3: Default GC in GraalVM native images? A: Serial GC
Q4: What must be declared for reflection in native? A: Reachability metadata
Q5: Spring Boot AOT processing version? A: 3.x
