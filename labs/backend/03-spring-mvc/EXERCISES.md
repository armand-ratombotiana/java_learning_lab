# Exercises for 03 Spring Mvc

## Exercise 1: Basic Setup
Configure and run a basic 03 Spring Mvc setup with Spring Boot.

## Exercise 2: Configuration
Create custom configuration properties and use them in your service.

## Exercise 3: Error Handling
Implement proper error handling with custom exceptions and @ControllerAdvice.

## Exercise 4: Testing
Write unit and integration tests for your implementation.

## Exercise 5: Production Readiness
Add monitoring, health checks, and metrics for production deployment.

## Exercise 6: Performance
Profile and optimize your implementation for better performance.

## Bonus: Extension
Add a new feature or integration that extends the basic functionality.
