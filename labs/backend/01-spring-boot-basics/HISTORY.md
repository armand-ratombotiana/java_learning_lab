# History

- **2002**: Spring Framework 1.0 released by Rod Johnson
- **2013**: Spring Boot 1.0.0.RC1 announced at SpringOne 2GX
- **2014**: Spring Boot 1.0 GA released with Java 6/7 support
- **2015**: Spring Boot 1.3 adds caching, scheduling
- **2017**: Spring Boot 2.0 with Spring Framework 5, Reactor, WebFlux
- **2019**: Spring Boot 2.2 introduces RSocket support
- **2020**: Spring Boot 2.4 with config import, Docker buildpacks
- **2022**: Spring Boot 3.0 with Java 17 baseline, GraalVM native images
- **2023**: Spring Boot 3.2 with Virtual Threads (Project Loom) support

The evolution reflects the industry shift from monolithic J2EE applications to cloud-native microservices.
