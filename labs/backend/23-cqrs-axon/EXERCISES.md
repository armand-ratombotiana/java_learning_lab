# Exercises: CQRS Axon

## Exercise 1: Basic Aggregate
Create an Order aggregate with CreateOrderCommand and OrderCreatedEvent.

## Exercise 2: Command Handler
Implement command handling for placing and canceling orders.

## Exercise 3: Query Side
Create a projection that listens to events and builds a read model.

## Exercise 4: Event Sourcing
Implement event sourcing for the order aggregate.

## Exercise 5: Saga
Create a saga that coordinates order payment and shipping.
