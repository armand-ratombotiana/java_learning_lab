# Quiz: Backend Security Deep Dive

Q1: What is a CSRF token? A: A unique, secret value included in forms to verify request origin.

Q2: What does CORS stand for? A: Cross-Origin Resource Sharing.

Q3: Which header prevents MIME sniffing? A: X-Content-Type-Options: nosniff.

Q4: What does parameterized query prevent? A: SQL injection.

Q5: What is the OWASP? A: Open Web Application Security Project.
