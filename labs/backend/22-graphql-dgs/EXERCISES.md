# Exercises: GraphQL DGS

## Exercise 1: Basic Query
Create a Query resolver that returns a list of shows.

## Exercise 2: Mutation
Implement a mutation to add a review for a show.

## Exercise 3: Data Loader
Implement a DataLoader for reviews to solve N+1.

## Exercise 4: Custom Scalar
Define and implement a custom scalar type (Date).

## Exercise 5: Subscription
Create a subscription for live review updates.
