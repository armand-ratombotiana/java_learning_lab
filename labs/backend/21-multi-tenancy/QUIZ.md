# Quiz: Multi-Tenancy

Q1: Which pattern has highest isolation? A: Database-per-tenant.
Q2: Which pattern shares the same tables? A: Discriminator column.
Q3: What stores tenant ID for current request? A: ThreadLocal.
Q4: Hibernate interface for multi-tenancy? A: CurrentTenantIdentifierResolver.
Q5: Which pattern is most scalable? A: Database-per-tenant.
