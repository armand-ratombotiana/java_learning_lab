# History: Spring Batch

Spring Batch originated from the co-operation between SpringSource and Accenture in 2007. It was designed to replace complex mainframe batch processing with a modern Java framework. Version 1.0 was released in 2008. Major versions include 2.0 (2009), 3.0 (2013), 4.0 (2017), and 5.0 (2023) with Jakarta namespace support.
