# Interview Questions: Federated Learning

## Conceptual Questions
1. What is Federated Learning and how does it differ from traditional programming?
2. Explain overfitting and how to prevent it.
3. What is the bias-variance tradeoff?
4. How does gradient descent work?
5. What is backpropagation and why is it important?

## Implementation Questions
1. Implement the forward pass of a basic model.
2. Write a function to compute gradient of a loss function.
3. How would you debug a model that isn't converging?
4. What techniques improve training performance?

## System Design Questions
1. Design a system for large-scale training.
2. How would you serve a model in production?
3. Design a monitoring system for deployed models.

## Mathematical Questions
1. Derive the gradient update rule.
2. Explain the relationship between MLE and cross-entropy.
3. What is the condition number and why does it matter?

## Behavioral Questions
1. Describe a challenging ML project you've worked on.
2. How do you stay current with ML research?
3. How do you explain technical concepts to non-technical stakeholders?
