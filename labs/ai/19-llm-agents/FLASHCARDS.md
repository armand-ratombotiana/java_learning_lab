# LLM Agents - FLASHCARDS

### Card 1
**Q:** What is ReAct agent?
**A:** Reasoning + Acting - interleaves thought generation with tool execution

### Card 2
**Q:** What is tool use in agents?
**A:** Ability to call external functions/services (search, calculator, APIs)

### Card 3
**Q:** Why use planning in agents?
**A:** Breaks complex tasks into manageable steps before execution

### Card 4
**Q:** What is agent memory?
**A:** Storage for past interactions - short-term (current session) and long-term

### Card 5
**Q:** What is reflection in agents?
**A:** Agent evaluates its own outputs and adjusts behavior

### Card 6
**Q:** What is the agent loop?
**A:** Continuous cycle:感知 → Think → Act → Observe → Learn

### Card 7
**Q:** Why use multi-step reasoning?
**A:** Complex tasks require multiple thought-action-observation cycles

### Card 8
**Q:** What is tool registry?
**A:** Catalog of available tools with descriptions for agent to choose from

### Card 9
**Q:** What is function calling?
**A:** Model generates structured output calling specific functions with parameters

### Card 10
**Q:** How do agents handle errors?
**A:** Try alternate approaches, use reflection to adjust strategy

### Card 11
**Q:** What is autonomous vs guided agent?
**A:** Autonomous: acts independently. Guided: follows explicit instructions

### Card 12
**Q:** Why build agents over plain LLMs?
**A:** Can take actions, use tools, handle multi-step tasks

**Total: 12 flashcards**
