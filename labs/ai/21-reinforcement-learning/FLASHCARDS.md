# Flashcards: Reinforcement Learning

**Q**: What is 
**A**: Learning patterns from data without explicit programming.

**Q**: What is overfitting?
**A**: Model memorizes training data but fails on new data.

**Q**: What is underfitting?
**A**: Model is too simple to capture underlying patterns.

**Q**: How does gradient descent work?
**A**: Iteratively update parameters opposite to gradient direction.

**Q**: What is backpropagation?
**A**: Computing gradients using chain rule backward through network.

**Q**: What is regularization?
**A**: Penalty on complexity to prevent overfitting.

**Q**: What is bias-variance tradeoff?
**A**: Balance between simplifying assumptions and sensitivity to data.

**Q**: What is cross-validation?
**A**: Split data into folds, train on some, validate on others.

**Q**: What is transfer learning?
**A**: Using pre-trained model as starting point for new task.

**Q**: What is a loss function?
**A**: Measures prediction error between predicted and true values.

**Q**: What is a learning rate?
**A**: Step size for parameter updates in gradient descent.

**Q**: What is an epoch?
**A**: One complete pass through the entire training dataset.

**Q**: What is a batch?
**A**: Subset of training data processed in one iteration.

**Q**: What is feature engineering?
**A**: Creating input features from raw data to improve performance.

**Q**: What is an activation function?
**A**: Non-linear transformation applied to layer outputs.

**Q**: What is the curse of dimensionality?
**A**: Data becomes sparse in high-dimensional space, making learning harder.

**Q**: What is an ensemble?
**A**: Combining multiple models for better predictions.

**Q**: What is data augmentation?
**A**: Creating modified versions of data to expand training set.

**Q**: What is gradient clipping?
**A**: Limiting gradient magnitude to prevent explosion.

**Q**: What is early stopping?
**A**: Halting training when validation performance stops improving.
