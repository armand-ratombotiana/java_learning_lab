# The Perceptron

Welcome to the atomic mastery lab for the **Perceptron**. This lab is part of the AI Academy's Neural Networks Foundations.

- The mathematical model of a Perceptron (Weights, Bias, Activation).

- Implementing a Perceptron from scratch in Java.
