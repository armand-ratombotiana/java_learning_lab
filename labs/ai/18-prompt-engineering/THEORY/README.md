# Prompt Engineering - Theory

## 1. Basic Techniques

### Zero-shot
- No examples, direct task description

### Few-shot
- 1-5 examples
- Demonstrates format

### Chain-of-Thought
- Include reasoning steps
- "Let me think step by step"

## 2. Prompt Structure

### Components
1. **Context**: Background info
2. **Task**: Clear instructions
3. **Examples**: Few-shot demonstrations
4. **Constraints**: Format/size limits
5. **Style**: Tone, persona

## 3. Advanced Patterns

### Role Prompting
- "You are a Python expert..."
- "Act as a data scientist..."

### Template Prompts
- Variable substitution
- Reusable structures

### Chain of Prompts
- Multi-step reasoning
- Sequential processing

### Self-Correction
- Verify and refine
- "Let me double-check..."

## 4. Optimization

### A/B Testing
- Compare different prompts
- Measure performance

### Prompt Tuning
- Learn embeddings for prompts
- Soft prompts

## 5. Common Patterns

### Classification
- "Classify the following..."

### Extraction
- "Extract the key information..."

### Summarization
- "Summarize this article..."

### Translation
- "Translate to Spanish..."

### Code Generation
- "Write a Python function..."
