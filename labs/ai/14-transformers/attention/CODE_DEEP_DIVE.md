# Self-Attention Code Deep Dive

This lab provides a pure Java implementation of Scaled Dot-Product Attention, demonstrating the matrix multiplications and the softmax application.

## 💻 Pure Java Implementation

```java file="labs/ai/14-transformers/attention/SOLUTION/ScaledDotProductAttention.java"
package ai.transformers.attention;

import java.util.Arrays;

/**
 * A fundamental implementation of Scaled Dot-Product Attention.
 */
public class ScaledDotProductAttention {

    /**
     * Computes the attention weights and the final output context vectors.
     * 
     * @param q Queries matrix [seq_length x d_k]
     * @param k Keys matrix [seq_length x d_k]
     * @param v Values matrix [seq_length x d_v]
     * @return The context matrix [seq_length x d_v]
     */
    public static double[][] computeAttention(double[][] q, double[][] k, double[][] v) {
        int seqLength = q.length;
        int dK = q[0].length;
        int dV = v[0].length;

        // 1. Q * K^T (Calculate raw scores)
        double[][] scores = new double[seqLength][seqLength];
        for (int i = 0; i < seqLength; i++) {
            for (int j = 0; j < seqLength; j++) {
                double dotProduct = 0;
                for (int d = 0; d < dK; d++) {
                    scores[i][j] += q[i][d] * k[j][d];
                }
            }
        }

        // 2. Scale by sqrt(d_k)
        double scale = Math.sqrt(dK);
        for (int i = 0; i < seqLength; i++) {
            for (int j = 0; j < seqLength; j++) {
                scores[i][j] /= scale;
            }
        }

        // 3. Apply Softmax to get attention weights (probabilities)
        double[][] weights = new double[seqLength][seqLength];
        for (int i = 0; i < seqLength; i++) {
            double max = Double.NEGATIVE_INFINITY;
            for (int j = 0; j < seqLength; j++) {
                if (scores[i][j] > max) max = scores[i][j];
            }
            
            double sumExp = 0;
            for (int j = 0; j < seqLength; j++) {
                weights[i][j] = Math.exp(scores[i][j] - max); // Subtract max for numerical stability
                sumExp += weights[i][j];
            }
            
            for (int j = 0; j < seqLength; j++) {
                weights[i][j] /= sumExp;
            }
        }
        
        printMatrix(weights, "Attention Weights (Softmax Output)");

        // 4. Multiply weights by V to get the final context
        double[][] context = new double[seqLength][dV];
        for (int i = 0; i < seqLength; i++) {
            for (int j = 0; j < dV; j++) {
                double sum = 0;
                for (int n = 0; n < seqLength; n++) { // n iterates over the sequence length
                    sum += weights[i][n] * v[n][j];
                }
                context[i][j] = sum;
            }
        }

        return context;
    }

    public static void printMatrix(double[][] matrix, String name) {
        System.out.println("--- " + name + " ---");
        for (double[] row : matrix) {
            for (double val : row) {
                System.out.printf("%6.3f ", val);
            }
            System.out.println();
        }
        System.out.println();
    }

    public static void main(String[] args) {
        // Assume a sequence of 3 words, embedded into 4 dimensions
        // In a real network, Q, K, and V are generated by multiplying the input embeddings by learned weight matrices (W_q, W_k, W_v).
        
        // Let's pretend word 1 and word 3 are highly related.
        double[][] q = {
            {1, 0, 1, 0}, // Word 1 Query
            {0, 1, 0, 1}, // Word 2 Query
            {1, 0, 1, 0}  // Word 3 Query
        };
        
        double[][] k = {
            {1, 0, 1, 0}, // Word 1 Key
            {0, 1, 0, 1}, // Word 2 Key
            {1, 0, 1, 0}  // Word 3 Key
        };
        
        // Values can be anything, they carry the actual information
        double[][] v = {
            {10, 20}, // Word 1 Value
            {30, 40}, // Word 2 Value
            {50, 60}  // Word 3 Value
        };

        double[][] output = computeAttention(q, k, v);
        printMatrix(output, "Final Context Output");
    }
}
```

## 🔍 Key Takeaways
1. **The $N^2$ Bottleneck**: Look at the nested loops calculating `Q * K^T`. The outer loops are `i < seqLength` and `j < seqLength`. This means the time and space complexity of Self-Attention is $O(N^2)$, where $N$ is the sequence length. This is why LLMs (like GPT-4) struggle with massive context windows (e.g., 128k tokens) and require optimizations like FlashAttention.
2. **Numerical Stability**: In the Softmax step, we subtract the maximum score from all scores before exponentiating (`Math.exp(scores[i][j] - max)`). Mathematically, this does not change the softmax output, but it prevents `Math.exp()` from returning `Infinity` (overflow) if the scores are very large.
