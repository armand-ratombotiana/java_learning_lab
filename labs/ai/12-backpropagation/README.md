# Backpropagation

Welcome to the atomic mastery lab for **Backpropagation**. This lab is part of the AI Academy's Deep Learning Foundations.

## 🧠 What You Will Master
- The Chain Rule of calculus and its application to neural networks.
- Forward pass vs. Backward pass.
- Computational graphs and automatic differentiation.
- The vanishing and exploding gradient problems.
- Implementing Backpropagation from scratch in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Intuition and computational graphs.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - The Chain Rule and partial derivatives.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a Multi-Layer Perceptron (MLP) with Backpropagation.
