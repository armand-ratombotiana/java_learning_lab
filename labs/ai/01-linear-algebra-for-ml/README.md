# Linear Algebra for Machine Learning

## Overview
Lab 01 introduces the mathematical foundations of linear algebra essential for machine learning. Vectors, matrices, eigenvalues, and SVD form the backbone of virtually all ML algorithms.

## Learning Objectives

By completing this lab, you will:
1. Understand vectors, matrices, and their operations
2. Master matrix decompositions (LU, QR, Cholesky, SVD)
3. Compute eigenvalues and eigenvectors
4. Apply linear algebra to ML problems (PCA, regression)
5. Implement linear algebra operations in Java
