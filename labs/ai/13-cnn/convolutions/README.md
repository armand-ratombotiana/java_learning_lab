# Convolutions

Welcome to the atomic mastery lab for **Convolutions**. This lab is part of the AI Academy's Convolutional Neural Networks (CNN) module.

## 🧠 What You Will Master
- The intuition behind image filtering and feature extraction.
- Kernel/Filter mechanics, strides, and padding.
- The mathematical operation of discrete 2D convolution.
- Pooling operations (Max Pooling, Average Pooling).
- Implementing a 2D Convolution operation from scratch in Java.

## 📂 Lab Structure
1. [THEORY.md](./THEORY.md) - Intuition and visual feature extraction.
2. [MATH_FOUNDATION.md](./MATH_FOUNDATION.md) - The math of 2D convolutions and output shape calculation.
3. [CODE_DEEP_DIVE.md](./CODE_DEEP_DIVE.md) - Pure Java implementation of a Convolutional layer.
