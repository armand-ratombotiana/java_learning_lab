# Low-Rank Adaptation (LoRA)

Welcome to the atomic mastery lab for **LoRA**. This lab is part of the AI Academy's Fine-Tuning module.

## 🧠 What You Will Master
- The cost problem of full fine-tuning for Large Language Models.
- Parameter-Efficient Fine-Tuning (PEFT).
- The mathematics of Low-Rank matrix decomposition.
- How LoRA freezes base weights and trains adapter matrices.
- Implementing a conceptual LoRA layer in Java.
