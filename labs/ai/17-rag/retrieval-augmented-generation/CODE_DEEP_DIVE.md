# RAG Code Deep Dive

This lab provides a pure Java simulation of a RAG pipeline. It demonstrates how a user query is intercepted, augmented with retrieved context, and packaged into a final prompt for the LLM.

## 💻 Pure Java Implementation

```java file="labs/ai/17-rag/retrieval-augmented-generation/SOLUTION/RagPipelineSim.java"
package ai.genai.rag;

import java.util.ArrayList;
import java.util.Comparator;
import java.util.List;

/**
 * A simulation of a Retrieval-Augmented Generation (RAG) pipeline.
 */
public class RagPipelineSim {

    // Simulates a Document Chunk stored in a Vector Database
    static class DocumentChunk {
        String content;
        double[] embedding; // The semantic vector

        DocumentChunk(String content, double[] embedding) {
            this.content = content;
            this.embedding = embedding;
        }
    }

    // Simulates the Vector Database
    private final List<DocumentChunk> vectorDatabase = new ArrayList<>();

    /**
     * Ingestion Phase: Stores chunks with their embeddings.
     */
    public void ingestData(String content, double[] embedding) {
        vectorDatabase.add(new DocumentChunk(content, embedding));
    }

    /**
     * Calculates Cosine Similarity between two vectors.
     */
    private double cosineSimilarity(double[] vectorA, double[] vectorB) {
        double dotProduct = 0.0;
        double normA = 0.0;
        double normB = 0.0;
        for (int i = 0; i < vectorA.length; i++) {
            dotProduct += vectorA[i] * vectorB[i];
            normA += Math.pow(vectorA[i], 2);
            normB += Math.pow(vectorB[i], 2);
        }
        if (normA == 0 || normB == 0) return 0.0;
        return dotProduct / (Math.sqrt(normA) * Math.sqrt(normB));
    }

    /**
     * Retrieval Phase: Finds the Top K most relevant chunks based on vector similarity.
     */
    public List<String> retrieveTopK(double[] queryEmbedding, int k) {
        // Calculate similarity for all chunks
        record ScoredChunk(String content, double score) {}
        
        List<ScoredChunk> scoredChunks = new ArrayList<>();
        for (DocumentChunk chunk : vectorDatabase) {
            double score = cosineSimilarity(queryEmbedding, chunk.embedding);
            scoredChunks.add(new ScoredChunk(chunk.content, score));
        }

        // Sort by score descending and take Top K
        return scoredChunks.stream()
                .sorted(Comparator.comparingDouble(ScoredChunk::score).reversed())
                .limit(k)
                .map(ScoredChunk::content)
                .toList();
    }

    /**
     * Augmentation Phase: Constructs the final prompt for the LLM.
     */
    public String buildPrompt(String userQuery, List<String> retrievedContexts) {
        StringBuilder prompt = new StringBuilder();
        prompt.append("System: You are a helpful corporate assistant. ");
        prompt.append("Answer the user's question using ONLY the context provided below. ");
        prompt.append("If the context does not contain the answer, reply with 'I do not know based on the provided context.'\n\n");
        
        prompt.append("--- CONTEXT ---\n");
        for (int i = 0; i < retrievedContexts.size(); i++) {
            prompt.append("[").append(i + 1).append("] ").append(retrievedContexts.get(i)).append("\n");
        }
        
        prompt.append("\n--- QUESTION ---\n");
        prompt.append(userQuery).append("\n");
        
        return prompt.toString();
    }

    public static void main(String[] args) {
        RagPipelineSim rag = new RagPipelineSim();

        // 1. Ingestion (Pre-computed embeddings for simulation)
        // Imagine these vectors were generated by OpenAI's text-embedding-3-small
        rag.ingestData("The company remote work policy allows 2 days WFH per week.", new double[]{0.9, 0.1, 0.0});
        rag.ingestData("The cafeteria serves pizza on Fridays.", new double[]{0.0, 0.9, 0.1});
        rag.ingestData("Employees get 20 days of PTO per year.", new double[]{0.8, 0.2, 0.0});

        // 2. The User Query
        String query = "How many days can I work from home?";
        // The query is embedded (simulated here as highly similar to the remote work chunk)
        double[] queryEmbedding = new double[]{0.85, 0.15, 0.0}; 

        // 3. Retrieval
        System.out.println("Searching Vector Database...");
        List<String> topContexts = rag.retrieveTopK(queryEmbedding, 2);

        // 4. Augmentation
        System.out.println("\nConstructing Prompt for LLM...\n");
        String finalPrompt = rag.buildPrompt(query, topContexts);
        
        System.out.println(finalPrompt);
        
        // 5. Generation (Simulated)
        System.out.println("\n[LLM Output]: Based on context [1], you can work from home 2 days per week.");
    }
}
```

## 🔍 Key Takeaways
1. **The System Prompt**: Look closely at `buildPrompt`. The magic of RAG is not in the retrieval, but in the strict instructions given to the LLM: *"Answer using ONLY the context provided... If it does not contain the answer, reply 'I do not know'"*. This is what eliminates hallucinations.
2. **The Vector Database**: In this simulation, `retrieveTopK` does a brute-force $O(N)$ scan of all vectors. In a real system, the vector database uses an HNSW index (as built in Capstone 04) to perform this search in $O(\log N)$ time, making it viable for millions of documents.
