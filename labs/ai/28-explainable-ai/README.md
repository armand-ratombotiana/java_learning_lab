## Overview
This lab provides a comprehensive exploration of Explainable AI, covering theoretical foundations, mathematical derivations, and practical Java implementations.

## Prerequisites
- Java 21+ development environment
- Linear algebra and probability fundamentals
- Basic machine learning concepts

## Topics
- Core algorithms and mathematical foundations
- Complete Java implementations with tests
- Performance optimization and debugging
- Mini-project and real-world project

## Structure
24 documentation files, 7 content subdirectories, Java source code with JUnit tests.

## Getting Started
1. Read THEORY.md and MATH_FOUNDATION.md for fundamentals
2. Review CODE_DEEP_DIVE.md for implementation patterns
3. Run the Java source files to see algorithms in action
4. Complete EXERCISES.md to test comprehension
5. Build the MINI_PROJECT and REAL_WORLD_PROJECT
