# Custom Spring Boot Starters

Welcome to the atomic mastery lab for **Custom Starters**. This lab is part of the Spring Boot Internals module.

## 🧠 What You Will Master
- The architecture of a Spring Boot Starter.
- `autoconfigure` module vs. `starter` module.
- Managing transitive dependencies.
- Using `@ConfigurationProperties` for type-safe config.
- Testing starters with `ApplicationContextRunner`.

1. [THEORY.md](./THEORY.md) - The "Starter" design pattern.
2. [INTERNALS.md](./INTERNALS.md) - Deep dive into auto-configuration discovery.
