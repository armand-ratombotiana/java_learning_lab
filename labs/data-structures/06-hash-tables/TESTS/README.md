# Tests: Hash Tables

## Test Files

Unit tests are located in `src/test/java/`.

## Test Coverage

- Core functionality and correctness
- Edge cases and invalid inputs
- Boundary conditions
- Performance assertions

## Run with

```bash
# Using Maven
mvn test

# Using Gradle
gradle test

# Using javac directly
javac -d out src/test/java/**/*.java src/main/java/**/*.java
java -cp out org.junit.platform.console.ConsoleLauncher --scan-classpath
```
