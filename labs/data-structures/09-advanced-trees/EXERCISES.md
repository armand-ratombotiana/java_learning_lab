# Exercises: Advanced Trees

## Basic

1. **BST implementation** — Implement a BST with insert, search, delete, min, max, inorder traversal.

2. **BST validation** — Write a function to check if a binary tree is a valid BST.

3. **Kth smallest** — Find the kth smallest element in a BST (inorder iterative).

4. **BST iterator** — Implement an in-order BST iterator with O(h) memory (stack-based).

## Intermediate

5. **AVL tree** — Implement an AVL tree with insert, delete, and all four rotations.

6. **Red-Black tree** — Implement a Red-Black tree with insert and fix-up (focus on insert; delete is complex).

7. **TreeMap operations** — Use Java's TreeMap to implement a leaderboard (insert score, get rank, get top K).

8. **B-tree basics** — Implement a B-tree with insert (split) and search.

9. **Fenwick tree** — Implement a Fenwick tree for prefix sums and range queries.

10. **Segment tree** — Implement a segment tree for range minimum queries.

## Advanced

11. **Range sum with updates** — Implement a data structure supporting range sum queries and point updates in O(log n).

12. **Lazy segment tree** — Implement a segment tree with lazy propagation for range updates.

13. **Ordered statistics tree** — Augment a BST with subtree sizes to support find-by-rank and rank queries in O(log n).

14. **Interval tree** — Implement an interval tree for overlapping interval queries.

15. **B+ tree** — Implement a B+ tree where all data is in leaves and leaves are linked for sequential access.
