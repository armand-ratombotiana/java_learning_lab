# Mini Project: Rope

Build a hands-on mini project that applies the Rope data structure to a meaningful problem.

## Requirements
1. Implement the core data structure from scratch
2. Create a command-line interactive interface for demonstrating operations
3. Write comprehensive unit tests for all operations (edge cases, stress tests)
4. Document your design decisions with trade-off analysis

## Deliverables
- Source code with full implementation and Javadoc
- Unit tests with 90%+ branch coverage
- Brief report on time/space complexity analysis with empirical validation
