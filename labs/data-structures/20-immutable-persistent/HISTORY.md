# History of Persistent Data Structures

- **1982**: Driscoll et al. formalize persistence
- **1986**: Path copying technique described
- **1989**: Fully persistent arrays
- **1990s**: ML and Haskell popularize immutable data
- **2000**: Okasaki's Purely Functional Data Structures
- **2007**: Clojure introduces persistent collections to the JVM
- **2010s**: Scala, Kotlin adopt persistent collections
- **2020s**: Persistent data structures in mainstream Java (Vavr, Eclipse Collections)

## Key Contributors

- **Chris Okasaki**: Purely Functional Data Structures (1998)
- **Rich Hickey**: Clojure's persistent collections (2007)
- **Phil Bagwell**: HAMT (Hash Array Mapped Trie) for persistent maps
