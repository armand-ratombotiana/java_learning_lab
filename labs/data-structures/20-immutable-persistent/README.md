## Learning Objectives
- Understand immutability and its benefits
- Implement persistent linked lists and stacks
- Build persistent binary search trees
- Master structural sharing for efficiency
- Understand Clojure-style persistent collections

## Topics
- Immutable vs mutable data structures
- Persistent vs ephemeral data structures
- Structural sharing for efficiency
- Persistent singly linked list
- Persistent stack
- Persistent binary search tree
- Immutable pair/tuple
- Path copying in persistent trees
