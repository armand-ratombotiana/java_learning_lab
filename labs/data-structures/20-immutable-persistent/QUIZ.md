# Quiz

1. What is an immutable data structure? Cannot be modified after creation
2. What is persistence? Previous versions are preserved
3. What is structural sharing? New versions share nodes with old versions
4. How many new nodes does a persistent BST insert create? O(log n)
5. What is path copying? Copy nodes along the path from root to modification point
6. Who wrote Purely Functional Data Structures? Chris Okasaki
7. What are transients? Mutable variants for batch modifications
