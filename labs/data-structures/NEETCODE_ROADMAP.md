# NeetCode Roadmap — Data Structures Academy

Maps the 35 academy labs to NeetCode 150, Blind 75, and Grind 75 problem sets. Each lab's `LEETCODE_SOLUTIONS/` directory contains Java implementations for the key problems listed below.
