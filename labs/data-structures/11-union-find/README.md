## Learning Objectives

- Understand the Union-Find data structure and its applications
- Implement path compression and union by rank/size optimizations
- Analyze the nearly O(1) amortized time complexity (inverse Ackermann function)
- Apply DSU to Kruskal's algorithm for Minimum Spanning Tree
- Solve dynamic connectivity and connected components problems
- Detect cycles in undirected graphs using DSU

## Prerequisites

- Basic graph theory (vertices, edges, connected components)
- Tree data structures and path concepts
- Recursion and iterative algorithms

## Topics Covered

- Disjoint set representation with parent arrays
- Find operation with path compression
- Union by rank and union by size heuristics
- Inverse Ackermann function complexity
- Kruskal's Minimum Spanning Tree algorithm
- Dynamic connectivity and cycle detection
- Connected components in graphs

## Exercises

1. Implement DSU with both path compression and union by rank
2. Use DSU to find connected components in an undirected graph
3. Implement Kruskal's algorithm and find MST weight
4. Detect cycles in a graph using DSU
5. Solve the number of islands problem using DSU
6. Implement union by size and compare performance
