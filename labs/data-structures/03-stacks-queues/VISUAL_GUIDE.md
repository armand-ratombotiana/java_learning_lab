## Queue Using Two Stacks

```
enqueue(1), enqueue(2), enqueue(3):
  in: [1, 2, 3]  (top = 3)
  out: []

dequeue():
  transfer: in вҶ’ out
  in: []
  out: [3, 2, 1]  (top = 1)
  pop from out вҶ’ returns 1

enqueue(4):
  in: [4]
  out: [3, 2]

dequeue():
  out not empty вҶ’ pop вҶ’ returns 2
  in: [4]
  out: [3]
```

## Monotonic Stack (Sliding Window Maximum)

```
nums = [1, 3, -1, -3, 5, 3, 6, 7], k = 3

Deque stores indices (decreasing values):

i=0: [1]
i=1: remove 1 вҶ’ [3]
i=2: [-1] (3, -1) вҶ’ window max = 3
i=3: [-3] (3, -1, -3) вҶ’ window max = 3
i=4: clear all вҶ’ [5] вҶ’ window max = 5
i=5: [3] (5, 3) вҶ’ window max = 5
i=6: clear all вҶ’ [6] вҶ’ window max = 6
i=7: clear all вҶ’ [7] вҶ’ window max = 7
```
