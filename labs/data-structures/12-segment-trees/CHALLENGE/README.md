# Challenge: Range Updates with Lazy Propagation

Implement a segment tree supporting:
1. Range assignment (set all values in [l,r] to x)
2. Range addition (add x to all values in [l,r])
3. Range sum query

All operations must be O(log n). Handle both assignment and addition lazily.

## Constraints
- N up to 10^6
- Q up to 10^5
- Values up to 10^9
