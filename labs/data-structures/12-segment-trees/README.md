## Learning Objectives

- Understand segment tree structure and recursive construction
- Implement range queries (sum, min, max) in O(log n)
- Implement point updates in O(log n)
- Master lazy propagation for range updates
- Build iterative (bottom-up) segment trees
- Compare segment trees with Fenwick trees and other structures

## Prerequisites

- Binary tree concepts
- Recursion and divide-and-conquer
- Basic understanding of range query problems

## Topics Covered

- Segment tree construction and memory layout
- Range sum, minimum, and maximum queries
- Point updates and range updates
- Lazy propagation for efficient range updates
- Iterative segment tree implementation
- 2D segment trees
- Dynamic (sparse) segment trees

## Exercises

1. Implement a recursive segment tree for range sum
2. Add lazy propagation for range addition
3. Implement range minimum query segment tree
4. Build an iterative segment tree
5. Solve range update + range query problems
6. Compare performance of recursive vs iterative
