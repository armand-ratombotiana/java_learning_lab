# Quiz: Segment Trees

## Multiple Choice

1. What is the minimum array size needed for a recursive segment tree with n elements?
   a) 2*n
   b) 4*n
   c) n
   d) n^2

2. What is the time complexity of a range sum query on a segment tree?
   a) O(1)
   b) O(log n)
   c) O(n)
   d) O(n log n)

3. Lazy propagation is used for:
   a) Faster point updates
   b) Efficient range updates
   c) Reducing memory usage
   d) Building the tree faster

4. Which of the following is NOT an associative operation suitable for segment trees?
   a) Addition
   b) Minimum
   c) Subtraction
   d) GCD

5. What is the height of a segment tree for n = 1000 elements?
   a) 10
   b) 100
   c) 1000
   d) Depends on the implementation

## True or False

6. A segment tree can answer range sum queries in O(1) time.
7. The iterative segment tree typically uses less memory than the recursive version.
8. Lazy propagation makes range updates O(log n) instead of O(n).
9. Segment trees only work with integer data types.
10. Any binary operation can be used with a segment tree.

## Short Answer

11. Explain the difference between recursive and iterative segment tree implementations.
12. Describe how lazy propagation works and why it's important.
13. Compare segment trees with Fenwick trees. When would you use each?
14. How does a persistent segment tree work? What are its applications?
15. What modifications are needed to support 2D range queries?
