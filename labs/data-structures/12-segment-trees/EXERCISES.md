# Exercises: Segment Trees

## Beginner

1. **Basic Segment Tree**: Implement a recursive segment tree for range sum queries and point updates. Test with n = 10.

2. **Range Minimum Query**: Modify the segment tree to support range minimum queries. What changes are needed?

3. **Point Update + Range Max**: Implement a segment tree that supports range maximum queries and point updates.

## Intermediate

4. **Lazy Propagation**: Extend your segment tree with lazy propagation for range addition updates. Support range sum queries.

5. **Range Assignment + Sum**: Implement lazy propagation for range assignment (set all values in [l, r] to x). Support range sum queries.

6. **Iterative Segment Tree**: Implement an iterative (bottom-up) segment tree for range sum queries. Compare performance.

7. **Segment Tree with GCD**: Implement a segment tree that supports range GCD queries and point updates.

8. **Multiple Aggregates**: Build a segment tree that stores (sum, min, max) at each node and supports all three query types.

## Advanced

9. **Segment Tree with Coordinate Compression**: Implement a dynamic segment tree that handles values up to 10^9 without pre-allocating 4*n space.

10. **2D Segment Tree**: Implement a 2D segment tree for range sum queries on a matrix.

11. **Persistent Segment Tree**: Implement a persistent segment tree that supports queries on previous versions.

12. **Segment Tree Beats**: Implement the "segment tree beats" data structure for range min/max updates with range sum queries.

13. **Lazy Propagation with Multiple Operations**: Support both range addition and range assignment simultaneously. Define which operation takes priority.

14. **Segment Tree for Range XOR**: Implement a segment tree for range XOR queries and point updates.

15. **Range Kth Order Statistic**: Use a segment tree of Fenwick trees to find the kth smallest element in a range.
