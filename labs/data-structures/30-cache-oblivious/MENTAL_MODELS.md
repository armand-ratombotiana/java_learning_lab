# Mental Models for Cache-Oblivious Data Structures

## Core Mental Models

### The Recursive Decomposition Model

A hierarchy of independent but coordinated components delegating work to lower levels.

### The Space-Time Trade-off Model

Every memory layout decision directly impacts performance.

### The Probabilistic Guarantee Model

Guarantees hold with high probability for randomized variants.

## Useful Analogies

Filing cabinets with coordinated indices, telephone networks routing calls, and library classification systems.

## Common Misconceptions

Assuming worst-case equals average-case, ignoring constant factors, and overlooking memory hierarchy effects.
