# Quiz: Skip Lists

1. What is the expected time complexity of search in a skip list? O(log n)
2. What probability is typically used for level generation? 0.5
3. How many levels does the average node have? 2
4. What is the worst-case time complexity? O(n)
5. Who invented skip lists? William Pugh (1989)
6. Which production system famously uses skip lists? Redis
7. What is the space complexity? O(n)
8. How does the update array get used? Stores predecessors for each level
9. What distribution does level generation follow? Geometric
10. True/False: Skip lists are deterministic. False
