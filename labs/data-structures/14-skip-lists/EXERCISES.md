# Exercises: Skip Lists

## Beginner
1. Implement SkipListNode
2. Implement basic skip list with insert and search
3. Implement delete operation
4. Add print method to visualize levels

## Intermediate
5. Implement SkipListSet with add/remove/contains
6. Implement SkipListMap with put/get/remove
7. Add ordered iterator
8. Add reverse iterator

## Advanced
9. Implement concurrent skip list
10. Implement lock-free skip list
11. Analyze level distribution empirically
12. Compare with Treap (another randomized tree)
13. Implement persistent skip list
14. Implement skip list with batch insertions
