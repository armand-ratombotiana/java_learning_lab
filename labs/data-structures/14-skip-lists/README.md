## Learning Objectives

- Understand the skip list structure and probabilistic balancing
- Implement search, insert, and delete in a skip list
- Understand how randomization achieves O(log n) expected time
- Build Set and Map implementations using skip lists
- Compare skip lists with balanced BSTs and hash tables

## Prerequisites

- Linked list concepts
- Basic probability and expected value
- BST and hash table fundamentals

## Topics Covered

- Skip list levels and towers
- Random level generation
- Search/Insert/Delete algorithms
- Expected time complexity analysis
- Space complexity and level distribution
- SkipListSet and SkipListMap implementations
- Comparison with red-black trees and hash tables

## Exercises

1. Implement a basic generic skip list
2. Implement SkipListSet with standard set operations
3. Implement SkipListMap with put/get/remove
4. Compare performance with TreeMap and HashMap
5. Analyze level distribution empirically
