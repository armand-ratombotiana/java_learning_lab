# Exercises: Van Emde Boas Tree

## Basic Exercises

### Exercise 1: Basic Implementation
Implement the core structure with insert, lookup, and delete.

### Exercise 2: Iteration
Add iteration support and compare order with insertion order.

### Exercise 3: Size Tracking
Maintain correct size across all operations.

## Intermediate Exercises

### Exercise 4: Load Factor Tuning
Experiment with load factors and measure performance impact.

### Exercise 5: Custom Hash Function
Implement and compare custom hash functions.

### Exercise 6: Thread Safety
Add thread-safe operations.

## Advanced Exercises

### Exercise 7: Persistent Variant
Implement an immutable version with efficient snapshots.

### Exercise 8: Benchmark Suite
Create comprehensive benchmarks for latency, throughput, and memory.

### Exercise 9: Integration
Integrate into a real application and measure improvement.

### Exercise 10: Research
Implement an advanced variant from recent literature.
