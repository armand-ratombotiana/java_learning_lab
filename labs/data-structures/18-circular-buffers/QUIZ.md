# Quiz

1. What is the time complexity of all circular buffer operations? O(1)
2. How is wrapping typically implemented? Modulo arithmetic
3. What happens when an overwrite buffer is full? Oldest element is overwritten
4. What is an SPSC buffer? Single Producer, Single Consumer
5. True/False: Circular buffers need dynamic allocation at runtime. False
6. What is the Disruptor pattern? High-performance inter-thread messaging
