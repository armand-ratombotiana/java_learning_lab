# References: Arrays

## Books

- **Cormen et al., Introduction to Algorithms (4th ed.)** — Chapter 10: Data Structures (dynamic arrays)
- **Robert Sedgewick, Algorithms (4th ed.)** — Arrays throughout; excellent Java examples
- **Brian Goetz, Java Concurrency in Practice** — Chapter 11: Performance and Scalability (false sharing on arrays)
- **Scott Oaks, Java Performance (2nd ed.)** — Chapter 5: Garbage Collection (array allocation)
