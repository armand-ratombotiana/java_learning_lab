## Learning Objectives

- Understand contiguous memory allocation and indexing
- Implement static and dynamic arrays (ArrayList-like)
- Master insertion, deletion, traversal, search, and sorting operations
- Analyze time and space complexity of array operations
- Work with multi-dimensional and jagged arrays
- Learn resizing strategies (geometric growth) for dynamic arrays

## Prerequisites

- Basic Java syntax (variables, loops, methods)
- Understanding of index-based access
- Java `int[]` and `ArrayList` familiarity

## Topics Covered

- Static vs dynamic arrays
- Contiguous memory layout and cache locality
- Big O analysis: access O(1), search O(n), insertion O(n), deletion O(n)
- Dynamic array resizing and amortized analysis
- Multi-dimensional arrays (row-major vs column-major)
- Utility methods: `java.util.Arrays` and `System.arraycopy`
- Common pitfalls: off-by-one, null elements, capacity vs size

## Exercises

1. Implement a generic dynamic array class from scratch
2. Write a method to rotate an array by k positions
3. Find the maximum subarray sum (Kadane's algorithm)
4. Merge two sorted arrays
5. Implement a sparse array representation

## Estimated Time: 2-3 hours
