# Exercises: Fenwick Tree

## Beginner

1. **Basic BIT**: Implement a Fenwick tree with add and prefixSum operations.

2. **Range Sum**: Add a rangeSum(l, r) method to your BIT.

3. **Building from Array**: Implement fromArray() that builds a BIT in O(n log n) and optimize to O(n).

## Intermediate

4. **Inversion Count**: Count inversions in an array using BIT.

5. **Range Update + Point Query**: Implement BIT with range updates and point queries using difference array.

6. **Range Update + Range Query**: Implement using two BITs for simultaneous range updates and range queries.

7. **Kth Smallest**: Find the kth smallest element in a dynamic set using BIT.

8. **Frequency Array**: Maintain a frequency array with BIT, supporting add and cumulative frequency queries.

## Advanced

9. **2D BIT**: Implement a 2D BIT for submatrix sum queries.

10. **BIT with Coordinate Compression**: Handle values up to 10^9 by compressing coordinates.

11. **Order Statistics Tree**: Build an order statistics tree using BIT that supports insert, delete, and find-by-rank.

12. **BIT for Range XOR**: Implement BIT for prefix XOR queries (point updates).

13. **Persistent BIT**: Implement a BIT that supports queries on previous versions.

14. **BIT on Trees**: Use BIT for path queries on a tree (Euler tour + BIT).

15. **Counting Inversions in 2D**: Count inversions in a 2D array using nested BIT.

16. **Dynamic BIT**: Support resizing the BIT dynamically.

17. **Multi-Dimensional BIT**: Extend to 3D or higher dimensions.

18. **BIT with Lazy Propagation**: Implement BIT with lazy range updates.
