## Learning Objectives

- Understand the binary indexed tree structure
- Implement point updates and prefix sum queries
- Master the LSB (least significant bit) technique
- Extend BIT for range updates and range queries
- Build 2D BIT for matrix queries
- Count inversions using BIT

## Prerequisites

- Binary number representation
- Basic array operations
- Understanding of prefix sums

## Topics Covered

- BIT structure and the LSB trick (i & -i)
- Point update + prefix sum (standard BIT)
- Range update + point query (difference array BIT)
- Range update + range query (two BITs)
- 2D BIT for matrix operations
- Inversion counting using BIT
- BIT vs Segment Tree comparison

## Exercises

1. Implement standard BIT with point update and prefix sum
2. Implement range update + range query using two BITs
3. Build a 2D BIT
4. Count inversions using BIT
5. Find kth smallest element using BIT (order statistic)
