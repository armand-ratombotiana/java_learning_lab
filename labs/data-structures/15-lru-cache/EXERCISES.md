# Exercises: LRU Cache

## Beginner
1. Implement LRU cache with HashMap + DLL
2. Implement LRU using LinkedHashMap
3. Add eviction listener callback

## Intermediate
4. Implement LFU cache
5. Add TTL expiration to LRU cache
6. Make LRU cache thread-safe
7. Add cache statistics (hit/miss count)

## Advanced
8. Implement concurrent LRU cache
9. Implement distributed LRU cache with consistent hashing
10. Implement multi-level cache (L1 LRU + L2 LRU)
11. Implement adaptive cache (switches between LRU and LFU)
