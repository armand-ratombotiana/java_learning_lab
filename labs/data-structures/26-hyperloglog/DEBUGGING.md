# Debugging: HyperLogLog

## Common Debugging Techniques

Print debugging, visual inspection of structure, and unit test isolation.

## Debugging Checklist

1. Validate invariants after each operation
2. Check boundary conditions
3. Verify index calculations
4. Monitor performance degradation

## Tools and Techniques

Assertions, memory profiling, debugger breakpoints, and toString methods.

## Common Bug Patterns

Hash collisions, metadata not updated, wrong rebalancing thresholds, thread safety violations, off-by-one errors.
