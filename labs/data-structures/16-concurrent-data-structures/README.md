## Learning Objectives
- Understand lock-free algorithms and CAS operations
- Implement Treiber stack and Michael-Scott queue
- Build concurrent hash sets using ConcurrentHashMap
- Master atomic classes (AtomicInteger, AtomicLong, AtomicReference)
- Understand memory ordering and visibility guarantees
- Compare lock-free vs lock-based approaches

## Topics Covered
- CAS (Compare-And-Swap) operations
- Treiber stack (lock-free stack)
- Michael-Scott queue (lock-free queue)
- ConcurrentHashMap and ConcurrentHashSet
- CopyOnWriteArrayList
- AtomicInteger, AtomicLong, AtomicReference
- Lock striping and fine-grained locking
- Memory barriers and happens-before relationships
- ABA problem and solutions
