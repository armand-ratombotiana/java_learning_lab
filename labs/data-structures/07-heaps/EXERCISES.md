# Exercises: Heaps

## Basic

1. **Min-heap implementation** — Implement a min-heap from scratch using an array with insert, extractMin, peek, isEmpty.

2. **Max-heap** — Convert your min-heap to a max-heap by changing comparison logic.

3. **Heap sort** — Implement heap sort using your max-heap. Sort an array in ascending order in-place.

4. **Is heap?** — Write a function that checks if an array satisfies the min-heap property.

## Intermediate

5. **K largest elements** — Find the k largest elements in an array using a min-heap of size k.

6. **K smallest elements** — Find the k smallest elements using a max-heap of size k.

7. **Merge k sorted lists** — Merge k sorted linked lists into one sorted list using a min-heap.

8. **Median of stream** — Implement a data structure that supports adding numbers and finding the median in O(log n) per operation (two heaps).

9. **Heapify performance** — Compare the time of building a heap with repeated inserts vs bottom-up heapify.

## Advanced

10. **Nearly sorted array** — Sort an array where each element is at most k positions away from its sorted position (use min-heap of size k+1).

11. **Sliding window median** — Find the median of every sliding window of size k (two heaps + lazy deletion).

12. **Top K frequent words** — Given a list of words, find the k most frequent words (HashMap + min-heap).

13. **Minimum cost to connect ropes** — Given rope lengths, connect all ropes with minimum cost (always connect two shortest).

14. **Maximum CPU profit** — Given tasks with start time, end time, and profit, schedule maximum profit (heap + sorting).
