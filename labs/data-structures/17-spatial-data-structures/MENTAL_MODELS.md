# Mental Models

## 1. The Map Subdivision Model
A quadtree is like dividing a map into quarters, then subdividing each quarter. Continue until each region has few points of interest.

## 2. The Phone Book Model (K-D Tree)
A k-d tree is like a phone book: first sort by last name (x), then first name (y) within each section, then middle initial (z), etc.

## 3. The GPS Navigation Model
To find the nearest gas station, first look in your current quadrant. If none found, expand to neighboring quadrants.
