# Mathematical Foundation of MinHash and SimHash

## Asymptotic Analysis

### Time Complexity

Operations achieve logarithmic expected and amortized time bounds.

### Space Complexity

O(n) for n elements with constant overhead.

## Key Concepts

Recursive decomposition yields sublinear bounds. Randomized variants allow arbitrarily small error probabilities through repetition. Amortized analysis uses potential functions.

## Proof Sketches

Invariants guarantee correctness. Complexity follows from bounded depth, O(1) work per level, and amortized rebalancing.

## Edge Cases

Empty structures, single-element operations, full capacity, and repeated elements.
