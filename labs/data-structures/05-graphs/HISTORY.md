# History: Graphs

## Early Foundations (1700s–1800s)

- **1736**: **Leonhard Euler** solved the **Seven Bridges of Königsberg** problem — the first graph theory result. He proved that a walk crossing every bridge exactly once is impossible. This is considered the birth of graph theory.
- **1847**: **Gustav Kirchhoff** used graph theory for electrical circuits (Kirchhoff's laws)
- **1854**: **Arthur Cayley** developed trees for enumerating chemical isomers

## Algorithmic Era (1900s–1950s)

- **1930**: **Kazimierz Kuratowski** characterized planar graphs
- **1956**: **Edsger Dijkstra** published Dijkstra's algorithm for shortest paths
- **1957**: **Robert Prim** developed Prim's algorithm for minimum spanning trees
- **1959**: **Joseph Kruskal** published Kruskal's algorithm for MSTs

## Modern Graph Algorithms (1960s–1990s)

- **1961**: **Edward F. Moore** published BFS algorithm
- **1962**: **Robert Tarjan** developed strongly connected components algorithm
- **1964**: **Claude Shannon** developed the graph-based mice maze solver (Theseus)
- **1972**: Depth-first search formalized by Tarjan
- **1998**: **PageRank** developed by Larry Page and Sergey Brin at Stanford

## Java Era (1995–present)

- Java Collections Framework did not include graph structures
- Third-party libraries: **JGraphT**, **Guava Graphs**, **Apache TinkerPop**
- **1999**: JGraph (later JGraphX) — Java graph visualization library
- **2009**: Neo4j — first graph database (NoSQL)
- **2015**: Apache Spark GraphX — distributed graph processing
- **2020s**: Graph Neural Networks drive new interest in graph algorithms
