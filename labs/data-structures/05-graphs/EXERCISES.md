# Exercises: Graphs

## Basic

1. **Adjacency list graph** — Implement a graph with `addEdge`, `neighbors`, and `hasEdge` methods.

2. **BFS traversal** — Implement BFS from a start vertex. Print visited vertices in order.

3. **DFS traversal** — Implement DFS (both recursive and iterative). Print visited vertices.

4. **Graph representation comparison** — Implement the same graph as adjacency matrix and adjacency list. Compare memory and edge lookup time.

## Intermediate

5. **Detect cycle (undirected)** — Detect if an undirected graph contains a cycle.

6. **Bipartite graph check** — Determine if a graph can be colored with two colors (BFS with coloring).

7. **Topological sort (Kahn's)** — Implement topological ordering using in-degree counting.

8. **Shortest path (BFS)** — Find the shortest path between two vertices in an unweighted graph.

9. **Dijkstra's algorithm** — Implement shortest path with weighted edges using a PriorityQueue.

10. **Number of connected components** — Count connected components in an undirected graph.

## Advanced

11. **Bellman-Ford** — Implement shortest path with negative weight handling and cycle detection.

12. **Minimum spanning tree (Prim's or Kruskal's)** — Find MST of a weighted undirected graph.

13. **Strongly connected components (Tarjan's)** — Find all SCCs in a directed graph.

14. **Max flow / min cut (Ford-Fulkerson)** — Compute maximum flow in a flow network.

15. **Word ladder** — Given two words and a dictionary, find shortest transformation sequence (BFS on graph of words differing by one letter).
