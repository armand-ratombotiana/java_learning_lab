# Exercises: Linked Lists

## Basic

1. **Singly linked list** — Implement a generic singly linked list with `addFirst`, `addLast`, `removeFirst`, `removeLast`, `get`, `size`, `isEmpty`.

2. **Reverse a linked list** — Reverse a singly linked list in-place (iterative). Then implement recursive reversal.

3. **Find middle node** — Find the middle node of a linked list in one pass (slow/fast pointer).

4. **Detect cycle** — Implement Floyd's cycle detection algorithm. Return the node where the cycle begins.

## Intermediate

5. **Merge two sorted lists** — Given two sorted singly linked lists, merge them into one sorted list.

6. **Remove nth from end** — Remove the nth node from the end of a linked list in one pass.

7. **Intersection of two lists** — Find the node where two singly linked lists intersect.

8. **Palindrome check** — Check if a singly linked list is a palindrome (O(n) time, O(1) space).

9. **Doubly linked list** — Implement a doubly linked list with sentinel nodes.

## Advanced

10. **LRU Cache** — Implement an LRU cache using a doubly linked list + HashMap (get/put O(1)).

11. **Flatten a multilevel doubly linked list** — Given a doubly linked list where nodes may have a `child` linked list, flatten it.

12. **Copy list with random pointer** — Deep copy a linked list where each node has an additional `random` pointer.

13. **Skip list** — Implement a skip list with search, insert, delete in O(log n) expected time.
