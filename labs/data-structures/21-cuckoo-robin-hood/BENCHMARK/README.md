# Benchmark: Performance Analysis

## Benchmark Scenarios
1. Basic operations: Time individual insert/delete/search
2. Batch operations: Process 10K, 100K, 1M items
3. Memory usage: Measure heap consumption
4. Comparison: Against standard library equivalents
5. Worst-case: Deliberately adversarial inputs

## Metrics
- Average, median, and p99 latency
- Throughput (ops/sec)
- Memory overhead per element
- Cache misses (if measurable)
