# History of Sparse Tables

- **1970s**: Range minimum query problem formalized
- **1980s**: Sparse table described in algorithms literature
- **1990s**: Used in competitive programming for O(1) RMQ
- **2000s**: Disjoint sparse table developed for non-idempotent ops
- **2010s**: 2D sparse tables for image processing
- **2020s**: GPU-accelerated sparse table implementations

The sparse table is one of the classic examples of the precomputation trade-off: O(n log n) preprocessing for O(1) queries.
