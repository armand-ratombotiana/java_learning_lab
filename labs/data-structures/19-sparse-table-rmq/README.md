## Learning Objectives
- Understand sparse table structure and precomputation
- Implement O(1) range minimum/maximum queries
- Build sparse tables for sum (idempotent) queries
- Implement disjoint sparse table for non-idempotent operations
- Compare sparse tables with segment trees and BIT

## Topics
- Sparse table construction (O(n log n))
- Range minimum query (RMQ) in O(1)
- Range maximum query
- Range sum with sparse table
- Disjoint sparse table for non-idempotent ops
- Static vs dynamic array scenarios
