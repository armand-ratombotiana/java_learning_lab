# Exercises: Merkle Tree and Hash Trees

## Basic Exercises

### Exercise 1: Basic Implementation
Implement the core Merkle Tree and Hash Trees data structure from scratch. Support insert, lookup, and delete operations.

### Exercise 2: Iteration
Add support for iterating over all elements in the structure. Compare iteration order with insertion order.

### Exercise 3: Size Tracking
Implement size tracking and ensure it is maintained correctly across all operations.

## Intermediate Exercises

### Exercise 4: Load Factor Tuning
Experiment with different load factors and measure the impact on performance. Find the optimal load factor for your use case.

### Exercise 5: Custom Hash Function
Implement a custom hash function and compare its performance with the default implementation.

### Exercise 6: Thread Safety
Add thread-safe operations using appropriate synchronization mechanisms.

## Advanced Exercises

### Exercise 7: Persistent Variant
Implement a persistent (immutable) version of Merkle Tree and Hash Trees that supports efficient snapshots.

### Exercise 8: Benchmark Suite
Create a comprehensive benchmark suite that measures operation latency, throughput, and memory usage.

### Exercise 9: Integration
Integrate Merkle Tree and Hash Trees into a real application and measure the performance improvement over standard alternatives.

### Exercise 10: Research
Research and implement an advanced variant from recent academic literature.
