# Exercises: Trees

## Basic

1. **Binary tree traversals** — Implement recursive preorder, inorder, and postorder traversal.

2. **Level-order traversal** — Implement BFS level-order traversal using a queue.

3. **Iterative inorder** — Implement inorder traversal without recursion (using a stack).

4. **Tree size and height** — Write methods to compute the size (number of nodes) and height of a binary tree.

## Intermediate

5. **Validate BST** — Check if a binary tree satisfies the BST property.

6. **Serialize/Deserialize** — Encode a binary tree to a string and decode it back.

7. **Lowest common ancestor** — Find the LCA of two nodes in a binary tree.

8. **Diameter of a tree** — Find the longest path between any two nodes (may not pass through root).

9. **Kth smallest in BST** — Find the kth smallest element in a BST (inorder traversal).

10. **Symmetrictip tree** — Check if a binary tree is a mirror of itself (symmetric).

## Advanced

11. **Maximum path sum** — Find the path with the maximum sum (can start/end at any node).

12. **Binary tree from traversals** — Construct a binary tree from preorder and inorder traversal arrays.

13. **Tree boundary** — Print the boundary of a binary tree (left boundary, leaves, right boundary in reverse).

14. **Flatten tree to linked list** — Flatten a binary tree to a right-pointing linked list (in-place).

15. **N-ary tree** — Implement an N-ary tree with level-order traversal and maximum depth.
