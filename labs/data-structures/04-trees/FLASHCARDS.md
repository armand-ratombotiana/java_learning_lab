# Flashcards: Trees

---

**Q**: What is a binary search tree property?
**A**: For every node, all values in left subtree < node value < all values in right subtree.

---

**Q**: What is the worst-case height of a BST?
**A**: O(n) when elements are inserted in sorted order (degenerate tree).

---

**Q**: What is inorder traversal order?
**A**: left → root → right

---

**Q**: What data structure does level-order traversal use?
**A**: A queue for BFS.

---

**Q**: What is the lowest common ancestor?
**A**: The deepest node that has both given nodes as descendants.

---

**Q**: How do you check if a tree is balanced?
**A**: Height difference between left and right subtrees ≤ 1 for every node (AVL definition).

---

**Q**: What is serialization of a tree?
**A**: Converting a tree to a string that can be stored/transmitted and reconstructed.

---

**Q**: What is a degenerate tree?
**A**: Each node has at most one child, forming a linked list.

---

**Q**: What is the time for inorder traversal of n nodes?
**A**: O(n) — visits every node exactly once.

---

**Q**: What exception does deep recursion cause in Java?
**A**: StackOverflowError.
