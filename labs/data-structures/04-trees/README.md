## Learning Objectives

- Understand hierarchical data organization
- Implement binary trees and tree traversals (preorder, inorder, postorder, level-order)
- Build tree algorithms recursively and iteratively
- Understand N-ary trees and common variants
- Learn tree serialization and deserialization
- Apply trees in real-world scenarios (file systems, HTML DOM, decision trees)

## Prerequisites

- Recursion fundamentals
- Lab 03: Stacks & Queues (for level-order traversal)

## Topics Covered

- Binary tree structure: node with left/right children
- Depth-first traversals: preorder, inorder, postorder
- Breadth-first (level-order) traversal using a queue
- Recursive vs iterative traversal
- Tree properties: height, depth, size, diameter
- N-ary trees and their representations
- Expression trees, binary search trees (introductory)
- Big O: traversal O(n), search O(n) on standard binary tree
- Common pitfalls: infinite recursion, null root, depth overflow

## Exercises

1. Implement a binary tree with recursive traversals (pre, in, post)
2. Implement iterative in-order traversal using a stack
3. Level-order traversal using a queue
4. Calculate tree height and diameter
5. Serialize and deserialize a binary tree
