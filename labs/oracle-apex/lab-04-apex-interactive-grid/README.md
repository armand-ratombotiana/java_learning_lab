# Lab 04: APEX Interactive Grid

## Focus
Interactive Grid (IG) is APEX's most powerful data presentation and editing component. This lab covers IG configuration, columns, aggregations, control breaks, master-detail, and cell editing.

## Topics Covered
- Interactive Grid vs Interactive Report: key differences
- IG configurations: table attributes, layout, columns
- Column attributes: type, heading, alignment, format mask
- Aggregations: SUM, AVG, COUNT, MIN, MAX per column
- Control breaks: grouping by column values
- Chart view: visualizing data within the grid
- Group by: aggregating and grouping
- Actions menu: filtering, sorting, highlighting, download
- Dynamic actions on IG
- Column groups: organizing columns logically
- Master-detail IG: two grids linked by a foreign key
- Cell editing: inline editing, add row, delete row
- Saving IG state per user

## Prerequisites
- Lab 03: APEX SQL Workshop
- Understanding of SQL SELECT statements
- Familiarity with APEX page creation

## Learning Objectives
- Create and configure Interactive Grids
- Implement aggregations and control breaks
- Set up master-detail relationships
- Enable cell editing with save capabilities
- Customize the actions menu
- Apply dynamic actions on grid events
