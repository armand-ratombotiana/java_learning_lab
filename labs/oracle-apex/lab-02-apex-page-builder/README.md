# Lab 02: APEX Page Builder

## Focus
The APEX Page Designer is the primary development interface. This lab covers every component of the Page Designer: regions, items, buttons, dynamic actions, page processes, branches, computations, and validations.

## Topics Covered
- Page Designer layout and navigation (rendering, processing, page shared components)
- Region types: Static Content, Classic Report, Interactive Report, Chart, PL/SQL Dynamic
- Item types: Text Field, Textarea, Number Field, Date Picker, Select List, Radio Group, Checkbox
- Button types and button templates
- Dynamic Actions: events, selection types, true/false actions
- Page Processes: types (PL/SQL, Execute Code), processing points
- Branches: after processing, after authentication, before header
- Computations: types (SQL, PL/SQL, Static), computation points
- Validations: types (Item, SQL, PL/SQL, Function Body), error messages

## Prerequisites
- Lab 01: APEX Getting Started
- Oracle APEX workspace with developer privileges
- Web browser access to APEX Builder

## Learning Objectives
- Navigate the Page Designer and understand its three-pane layout
- Create and configure all region types
- Add and configure page items with different source types
- Implement dynamic actions with multiple true/false actions
- Set up page processes, branches, computations, and validations
- Troubleshoot common page design issues
