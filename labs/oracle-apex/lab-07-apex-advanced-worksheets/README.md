# Lab 07: APEX Advanced Worksheets

## Focus
Advanced APEX features for building enterprise applications: collections, caching, credentials, mail, printing, PDF generation, Oracle JET charting, and the APEX plugin architecture.

## Topics Covered
- APEX collections (APEX_COLLECTION): create, add, query, delete
- Application-level caching: APEX_CACHE, cache best practices
- APEX_CREDENTIAL: store and manage credentials securely
- APEX_JOURNAL: application change logging
- APEX_MAIL: sending emails from APEX
- APEX_PRINT: PDF and print functionality
- APEX_ZIP: creating ZIP archives
- PDF generation with APEX Office Print (AOP) or BI Publisher
- Charting with Oracle JET: chart types, data series, customization
- APEX plugin architecture: types of plugins
- Dynamic actions plugins: custom client-side behavior
- Item type plugins: custom form controls
- Region type plugins: custom region renderers
- Process type plugins: custom server-side logic

## Prerequisites
- Labs 01-06
- PL/SQL programming knowledge

## Learning Objectives
- Use APEX collections for temporary data storage
- Implement caching strategies for performance
- Send emails and generate PDFs from APEX
- Create interactive charts with Oracle JET
- Develop and use APEX plugins
- Process files with APEX_ZIP
