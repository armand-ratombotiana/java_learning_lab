## Overview

Oracle APEX (Application Express) is a low-code development platform that enables you to build scalable, secure enterprise applications with just a browser. This academy covers the complete APEX application lifecycle — from architecture to deployment and performance tuning.

## Prerequisites
- Basic SQL and PL/SQL knowledge
- Oracle Database access (ADB or on-premise)
- Oracle APEX installed (19.2+, preferably 23.2+)
- Web browser (Chrome/Firefox/Edge)
- No Java knowledge required for core APEX content

---

## How to Use This Academy

### For APEX Beginners
Complete Labs 01–03 to understand APEX fundamentals. Then Lab 04 (Interactive Grid) is essential.

### For PL/SQL Developers
Labs 03 (SQL Workshop) and 06 (RESTful APIs) are most relevant. Lab 05 (Security) is critical.

### For Full-Stack Developers
Combine Lab 06 (RESTful APIs) with Java/Node.js microservices. Lab 04 for data entry screens.

### For APEX Administrators
Labs 05 (Security), 07 (Advanced), and 08 (Performance) are must-haves.
