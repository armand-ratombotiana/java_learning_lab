# Lab 03: APEX SQL Workshop

## Focus
APEX SQL Workshop provides database management capabilities within APEX. This lab covers the Object Browser, SQL Commands, Query Builder, Data Workshop, and RESTful Services integration with ORDS.

## Topics Covered
- SQL Workshop overview and navigation
- Object Browser: browse, create, alter, drop database objects
- SQL Commands: run SQL, bind variables, export results, command history
- Query Builder: visual query construction, joins, conditions
- Data Workshop: CSV/JSON/XML data import and export
- APEX_DATA_PARSER: parsing uploaded files programmatically
- RESTful Services: creating and managing REST endpoints
- ORDS integration with APEX
- Data loading patterns (insert, update, merge)

## Prerequisites
- Lab 02: APEX Page Builder
- Oracle Database schema with sample tables

## Learning Objectives
- Use Object Browser for database object management
- Write and execute SQL via SQL Commands
- Build queries visually with Query Builder
- Import and export data using Data Workshop
- Create RESTful services in APEX
- Use APEX_DATA_PARSER for file uploads
