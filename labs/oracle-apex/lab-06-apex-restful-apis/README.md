# Lab 06: APEX RESTful APIs

## Focus
APEX and ORDS enable creating and consuming RESTful APIs directly from the database. This lab covers RESTful service development, ORDS autorest, APEX_WEB_SERVICE, and APEX_JSON.

## Topics Covered
- RESTful Services in APEX
- ORDS (Oracle REST Data Services) auto-REST for tables and views
- RESTful service templates and resource handlers
- HTTP methods: GET, POST, PUT, DELETE
- Resource parameters: path parameters, query parameters, request body
- Pagination: offset/limit, cursor-based
- ETag for caching and conditional requests
- OAuth2 client configuration and token exchange
- APEX_WEB_SERVICE package: make_rest_request, HTTP methods
- APEX_JSON package: parse, generate, and manipulate JSON
- Consuming external REST APIs from APEX
- Securing REST endpoints with ORDS

## Prerequisites
- Labs 01-04
- Understanding of REST API concepts
- ORDS installed and configured

## Learning Objectives
- Create RESTful service modules with resource handlers
- Implement auto-REST for database objects
- Consume external REST APIs from APEX
- Handle pagination and caching
- Implement OAuth2 client authentication
- Parse and generate JSON with APEX_JSON
