# History of Apache Iceberg

## Timeline
- **2017**: Created by Ryan Blue at Netflix for the data lake use case
- **2018**: Open-sourced by Netflix; joined Apache Incubator
- **2019**: Adopted by Airbnb, Expedia, and LinkedIn
- **2020**: Apache Iceberg top-level project
- **2021**: 1.0 release with stable API; broad adoption
- **2022**: Iceberg 1.1 with Spark 3.3 support, enhanced catalog support
- **2023**: Iceberg 1.3 with REST catalog GA, Flink 1.17 integration
- **2024**: Iceberg 1.5 with multi-engine enhancements, Puffin blob stats
