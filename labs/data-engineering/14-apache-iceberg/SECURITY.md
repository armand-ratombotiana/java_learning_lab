# Security for Apache Iceberg

## Catalog Security
- AWS Glue catalog IAM policies
- Hive Metastore authentication
- REST catalog TLS + API keys

## Data Protection
- Server-side encryption for data files
- Column-level security via Spark SQL views
- Audit logging of metadata operations

## Access Control
- Catalog-level RBAC
- Table-level grant/revoke
- File-level permissions via bucket policies
