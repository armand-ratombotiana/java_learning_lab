# Data Governance Theory

## Pillars
1. **Data Quality**: Accuracy, completeness
2. **Data Catalog**: Discovery, inventory
3. **Data Lineage**: Origin, movement
4. **Data Security**: Access control, encryption
5. **Data Privacy**: PII handling, consent
6. **Compliance**: GDPR, CCPA, HIPAA, SOX

## Metadata Types
- Technical: Schema, types, partitions
- Business: Definitions, owners, tags
- Operational: Run times, error rates
