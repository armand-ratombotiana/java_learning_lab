# Exercises for Apache Flink

1. Implement word count with event-time processing and watermark strategy
2. Create tumbling, sliding, and session window aggregations on sensor data
3. Build a stateful enrichment pipeline using KeyedState (ValueState, MapState)
4. Write Flink SQL queries with Kafka source and JDBC sink
5. Configure checkpointing and savepoints; test failure recovery
6. Implement exactly-once sink semantics with two-phase commit
