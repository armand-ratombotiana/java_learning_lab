# History of Data Catalog

## Timeline
- **2000s**: Enterprise data catalogs emerge (Informatica, Collibra, Alation)
- **2015**: Apache Atlas enters incubation for Hadoop metadata management
- **2016**: LinkedIn open-sources WhereHive (precursor to DataHub)
- **2020**: DataHub open-sourced by LinkedIn; OpenMetadata founded
- **2021**: OpenMetadata enters Apache Incubator; DataHub grows rapidly
- **2022**: OpenMetadata top-level Apache project; wide adoption
- **2024**: AI-powered metadata enrichment; automated classification; real-time lineage
