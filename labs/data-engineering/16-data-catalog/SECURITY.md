# Security for Data Catalog

## Access Control
- Catalog platform authentication (LDAP, OAuth, SAML)
- Role-based access to metadata (Admin, Editor, Viewer)
- Data classification policies (public, internal, restricted, confidential)

## Sensitive Data
- Automatic PII detection and tagging
- Column-level security for sensitive metadata
- Audit logging of all metadata access

## API Security
- API keys for automated ingestion
- TLS for all catalog communications
- Network isolation for catalog deployment
