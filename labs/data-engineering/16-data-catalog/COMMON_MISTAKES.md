# Common Mistakes with Data Catalog

1. No Business Context: Harvesting only technical metadata without business glossary reduces catalog value
2. Stale Metadata: Scheduled ingestion without incremental updates leads to stale information
3. No Ownership: Datasets without owners become orphaned; enforce ownership at ingestion
4. Over-Engineering: Trying to catalog everything at once; start with critical assets, expand iteratively
5. Ignoring Users: Catalog fails without user adoption; invest in training, search, and integration
