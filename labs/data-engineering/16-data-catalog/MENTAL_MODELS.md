# Mental Models for Data Catalog

## 1. The Library Catalog
Data catalog is like a library card catalog. Books = datasets. Cards = metadata. Search = find relevant books. Subject headings = business glossary. Call numbers = storage locations.

## 2. The GPS Navigation
Data lineage is GPS tracking for data. Origin shown as source system. Route = transformations. Destination = consumption (dashboard, ML model, report). Alt routes = alternative data paths.

## 3. The Company Directory
Business glossary is the company directory. Employee name = business term. Department = data domain. Title = technical details. Phone = data owner. Just enough info to find and contact.
