# Exercises for Data Catalog

1. Deploy OpenMetadata using Docker Compose and configure ingestion
2. Harvest metadata from PostgreSQL database and explore the catalog UI
3. Build data lineage for a sample ETL pipeline (source -> transform -> target)
4. Create a business glossary with terms, categories, and relationships
5. Set up automated metadata ingestion via Airflow integration
6. Implement data discovery search with custom facets and filters
