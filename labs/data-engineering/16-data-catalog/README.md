# Data Catalog

## Overview
A data catalog is a metadata management tool that enables data discovery, governance, and collaboration across the organization. This lab covers Apache Atlas, DataHub, OpenMetadata, metadata harvesting, data lineage, and data discovery.

## Key Concepts
- **Metadata Management**: Systematic collection, storage, and access to data about data
- **Data Discovery**: Search and browse capabilities for finding relevant datasets
- **Data Lineage**: Tracking data flow from source through transformations to consumption
- **Apache Atlas**: Open-source data governance and metadata platform (Hadoop ecosystem)
- **DataHub**: Modern open-source metadata platform by LinkedIn
- **OpenMetadata**: Open-source metadata platform with active community and extensive integrations

## Learning Objectives
1. Understand metadata management principles and best practices
2. Deploy and configure a data catalog (Atlas, DataHub, or OpenMetadata)
3. Harvest metadata from databases, data lakes, and ETL pipelines
4. Build and visualize data lineage for critical data assets
5. Implement data discovery with search and business glossary
6. Automate metadata ingestion using REST APIs and crawlers
