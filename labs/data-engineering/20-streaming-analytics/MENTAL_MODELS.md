# Mental Models for Streaming Analytics

## 1. The TV News Feed
Streaming analytics is like a TV news ticker. Data flows continuously (news feed). Queries filter to relevant events (matching topics). Aggregations are headlines (summary statistics). Alerts are breaking news (threshold breaches).

## 2. The Live Sports Scoreboard
Scoreboard updates in real-time as game events happen. Hits, runs, errors = streaming events. Box score = aggregation. Win probability = analytics. No game (no data) means scoreboard is stale.

## 3. The Air Traffic Control
ATC monitors planes in real-time. Position = streaming event. Altitude/speed = metrics. Collision warnings = alerts. Route planning = predictive analytics. Every second counts.
