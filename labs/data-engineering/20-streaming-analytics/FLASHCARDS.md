# Flashcards: Streaming Analytics

## Card 1
**Front**: What is streaming analytics?
**Back**: Real-time analysis of data as it arrives, enabling sub-second insights and immediate action

## Card 2
**Front**: What is Flink SQL?
**Back**: ANSI SQL interface for Apache Flink supporting streaming queries with windowed aggregations

## Card 3
**Front**: What is a materialized view in streaming?
**Back**: Result of streaming query stored in persistent table, updated incrementally as new data arrives

## Card 4
**Front**: What is Kinesis Analytics?
**Back**: AWS managed service running Flink-based streaming SQL without infrastructure management

## Card 5
**Front**: What is Lambda Architecture?
**Back**: Dual pipeline: batch layer (comprehensive, accurate) + speed layer (low-latency, approximate) + serving layer
