# History of Streaming Analytics

## Timeline
- **2000s**: Early streaming: CEP engines (Esper, StreamBase)
- **2011**: Apache Kafka created at LinkedIn
- **2014**: Apache Flink emerges as streaming-first engine
- **2016**: AWS Kinesis Analytics launched
- **2019**: Flink SQL GA; stream processing becomes mainstream
- **2020**: Materialize (streaming SQL database) public launch
- **2022**: ClickHouse gains streaming ingestion capabilities
- **2024**: Streaming analytics for real-time ML; AI-driven dashboards
