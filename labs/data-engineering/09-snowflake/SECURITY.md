# Security for Snowflake Data Cloud

## Authentication
- MFA for user accounts
- Key-pair auth for automated processes
- SCIM provisioning for SSO
- OAuth for applications

## Authorization
- RBAC with custom roles
- Network policies for IP restriction
- Row-level security via secure views
- Column-level masking policies for PII

## Data Protection
- Encryption at rest (AES-256) and in transit (TLS 1.2+)
- Tri-Secret Secure for key management
- Audit logs for all queries and access
