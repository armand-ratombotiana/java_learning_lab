# Reflection: Data Governance

- Data governance requires both technology and organizational commitment
- PII detection at scale is harder than it seems — context matters (is 'zip code' PII? sometimes)
- RBAC must balance security with usability; overly restrictive models cause workarounds
- GDPR compliance is an ongoing process, not a one-time project
- Automation is essential for governance at scale; manual processes don't scale
