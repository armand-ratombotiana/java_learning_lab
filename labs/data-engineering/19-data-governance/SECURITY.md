# Security for Data Governance

## Core Security
- Encryption at rest and in transit for all sensitive data
- Key management for masking and encryption functions
- Network isolation for governance infrastructure

## Access Control
- Strict access to governance tools (PII detection, masking config)
- Separation of duties: data owner != security admin
- Regular access reviews and remediation

## Compliance
- Audit trail of all governance actions
- Breach notification procedures
- Regular compliance assessments
