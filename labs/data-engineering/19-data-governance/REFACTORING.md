# Refactoring Data Governance Pipelines

## Manual to Automated PII Detection
Before: Manual review of schemas for sensitive data
After: Automated PII scanner with confidence scoring

## Static to Dynamic Masking
Before: Masking at ETL stage (permanent)
After: Dynamic masking based on user role (flexible)

## Siloed to Centralized Governance
Before: Each team manages own access policies
After: Centralized RBAC with consistent policies across platform
