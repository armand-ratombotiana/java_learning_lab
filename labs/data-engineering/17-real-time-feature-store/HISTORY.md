# History of Real-Time Feature Store

## Timeline
- **2017**: Uber open-sources Michelangelo (includes feature store concepts)
- **2018**: Tecton founded; Feast project started
- **2019**: Feast v0.1: first open-source feature store
- **2020**: Feast v0.5: stable API, Redis online store
- **2021**: Feast v0.10: improved performance, multi-store support
- **2022**: Feast v0.20: stream feature support, Java SDK
- **2023**: Feast v0.30: improved registry, offline store plugins
- **2024**: Feast v0.35: enhanced streaming, AWS/GCP integrations
