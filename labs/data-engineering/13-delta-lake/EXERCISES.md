# Exercises for Delta Lake

1. Create Delta table from Parquet; inspect _delta_log/ directory structure
2. Implement MERGE INTO with WHEN MATCHED and WHEN NOT MATCHED clauses
3. Query different versions using versionAsOf and timestampAsOf
4. Optimize small files with OPTIMIZE and ZORDER BY
5. Configure Change Data Feed and read row-level changes
6. Simulate concurrent write conflicts and implement retry logic
