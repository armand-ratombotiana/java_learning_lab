# History of Data Quality Engineering

## Timeline
- **1990s**: Data quality recognized as formal discipline; TDWI data quality framework
- **2000s**: ETL tools include basic quality checks (Informatica, DataStage)
- **2015**: Great Expectations founded by Abe Gong at Superconductive
- **2018**: dbt Labs popularizes data testing in transformation layer
- **2020**: Data contracts concept formalized; data mesh influence
- **2022**: Soda and Monte Carlo emerge for data observability
- **2024**: AI-assisted quality monitoring; automated schema evolution handling
