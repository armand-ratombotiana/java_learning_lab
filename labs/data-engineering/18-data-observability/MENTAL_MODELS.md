# Mental Models for Data Observability

## 1. The Car Dashboard
Data observability = car dashboard. Freshness = speedometer. Volume = fuel gauge. Distribution = check engine light. Schema = warning lights. Lineage = GPS tracking route.

## 2. The Health Monitor
Data observability is like a hospital patient monitor. Vital signs (metrics) stream continuously. Alarms (alerting) trigger when readings exceed thresholds. Trend charts show improvement or deterioration over time.

## 3. The Security Camera
Data observability = security cameras for your data platform. Lineage = camera recording showing data flow. Profiling = periodic snapshots. Anomaly detection = motion detection alerting on unusual activity.
