# History of Data Observability

## Timeline
- **2019**: Monte Carlo founded by Barr Moses (former WeWork data lead)
- **2020**: Soda open-sourced by Tom Baeyens and friends
- **2020**: Data observability concept formalized by Monte Carlo (5 pillars)
- **2021**: Widespread adoption of data observability tools
- **2022**: Soda raises Series B; Monte Carlo reaches unicorn status
- **2023**: Open-source alternatives grow; Soda Core gains momentum
- **2024**: AI-powered anomaly detection; observability for streaming data
