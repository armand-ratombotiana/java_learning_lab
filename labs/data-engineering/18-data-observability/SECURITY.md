# Security for Data Observability

## Data Privacy
- Profiling on masked/sampled data for PII protection
- Alert information should not contain actual data values
- Role-based access to observability dashboards

## Access Control
- Read-only access for monitoring tools
- Separate credentials from production systems
- Audit logging of all observability checks

## Compliance
- Retention of observability history for audits
- Data lineage for regulatory traceability
- Quality metrics for data governance reporting
