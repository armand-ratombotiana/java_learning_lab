# Exercises - JUnit 5

## Exercise 1: Parameterized Test
Create a parameterized test that tests a calculator with multiple input pairs.

## Exercise 2: Nested Tests
Use @Nested to group related test cases for a specific feature.

## Exercise 3: Display Names
Add @DisplayName to tests with custom characters and emojis.

## Exercise 4: Exception Testing
Use assertThrows() to verify exception handling in code.

## Exercise 5: Timeout
Use assertTimeout() to verify a method completes within time.
